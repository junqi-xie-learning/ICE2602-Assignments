“2019上海交通大学城市科学秋季论坛系列活动”会议现场 本网实习记者 陈炼/摄　

  　中国社会科学网讯（记者 查建国 夏立 实习记者 陈炼）12月28日，上海交通大学城市科学研究院、上海交通大学出版社主办，上海交通大学出版社人文社科分社、上海交通大学媒体与传播学院协办的“2019上海交通大学城市科学秋季论坛系列活动”在上海交通大学召开。上海交通大学出版社董事长、书记谈毅致辞，上海交通大学城市科学研究院学术委员会副主任蒋宏主持会议。

　　上海交通大学城市科学研究院院长刘士林教授发布《江浙沪毗邻区一体化发展的定位与策略研究报告》，国家发展改革委城市和小城镇改革发展中心学术委员会秘书长冯奎研究员、上海社会科学院经济研究所所长沈开艳研究员分别对报告作点评。

　　《江浙沪毗邻区一体化发展的定位与策略研究报告》介绍了江浙沪毗邻区一体化发展的基本情况并点明了江浙沪毗邻区一体化发展的主要问题。报告提出，毗邻地区是推进一体化的难点和薄弱点，但也是突破点。毗邻地区通常处于城市边缘，较少得到来自核心区域的行政资源支持。其好处是受到较少的行政束缚，如果给予毗邻地区更大的改革自由度开展各类试点，成本投入不高而效果则会非常显著。即便出现一定问题，更正也容易进行。

　　报告有针对性地提出了加快江浙沪毗邻区一体化发展的策略与建议：一是要统一规划，加强区域发展顶层设计，推进实施江浙沪毗邻区一体化发展政策；二是要加快推进上海都市圈基础设施的互联互通；三是需聚焦改革与制度创新，共享长三角创新发展红利；四是要聚焦重点领域突破，打造创新性、融合性、生态化的产业新体系；五是要发挥科技创新产业发展协同效应，打造全球先进制造业发展新高地。

　　在点评环节，冯奎表示，行政壁垒会造成数据分割，数据分割进一步巩固了行政壁垒，要充分发挥数据治理的作用去打破这种循环，其中一个突破口就是从数据采集、共享、使用上入手。数据治理的启动需要上层政府部门、社会组织予以推动，一旦数据治理形成了力量，整个区域的一体化将进入快车道。要明确在毗邻地区的一体化建设进程中不能脱离大区域的一体化进程。此外，亟需注意以下两点：其一，毗邻地区的一体化要选准“切口”；其二，一体化建设进程要成为改革力量的汇集之地，而不是优惠政策的堆积之地。

　　会上，受邀嘉宾共同为“上海交通大学出版社——上海交通大学城市科学研究院共建城市文化传播中心”揭牌。

　　上半场专题研讨以“‘十四五’期间城镇化的理论准备与战略规划”为主题，冯奎，沈开艳，华东师范大学终身教授宁越敏，国家地理空间信息中心战略与规划中心副主任刘敏副研究员，中共中央党校(国家行政学院)经济学教研部李江涛教授，同济大学建筑与城市规划学院景观学系陈蔚镇教授分别作了主旨演讲，由刘士林主持。发言嘉宾表示，当前“十四五”规划研究已经启动，新型城镇化持续推进，城市群相继崛起。都市圈建设的政策出台，大湾区经济快速发展，中心城市的全国布局，快速交通网络的延伸拓展，城市经济的创新驱动等都在重新塑造着中国城市面貌。

　　下半场专题研讨中，上海交通大学城市科学研究院学术委员会副主任高小康、上海社会科学院文学研究所公共文化研究室主任郑崇选研究员、新华社《瞭望》周刊社副总编辑罗海岩、华东理工大学艺术设计与传媒学院副院长于炜、中国传媒大学耿波教授围绕“新时代中国城市文化的研究与传播”分别作了主旨演讲，由同济大学人文学院文化产业管理系王国伟教授主持。发言嘉宾认为，文化是城市发展的不竭动力，一个城市只有具备了有特色的文化，才能增强城市凝聚力和影响力。如何在推进城市化进程中，按照新时代中国特色社会主义的时代要求，保持并强化城市的特色文化，是当前城市文化研究的重要课题。

　　来自国家发展改革委、国家发展研究中心、新华社、中共中央党校、上海社会科学院、复旦大学、同济大学、华东师范大学等单位50余位专家出席此次论坛。
