沪上多所高校发布2018届本科生就业质量报告

越来越多国际学生选择留在中国

青年报记者 刘昕璐

青年报记者 刘昕璐

本报讯 沪上多所高校昨天发布2018届本科毕业生就业质量报告。其中上海纽约大学的报告显示，2018届毕业生中，共18名国际学生留在中国就业或深造，占国际毕业生人数的16%。深厚的中国情结促使他们留在这片土地上继续实现自己的理想。而上海交大发布的报告显示本科毕业生深造率达到了67.29%。

上海纽约大学

在中国就业毕业生平均年薪13.5万元，同比增14％

上海纽约大学昨天正式发布2018届本科毕业生就业质量报告。报告显示，上海纽约大学2018届本科毕业生共254人，其中，中国毕业生143人，国际毕业生111人。他们来自全球43个国家，说着37种语言，组成了一个真正多元化的国际大家庭。根据截至2018年11月30日的统计数据，143名中国学生的就业率为99%，其中，42%毕业后直接工作；57%选择出国（境）深造，人均获得3份录取通知书。在中国就业的毕业生平均年薪为13.5万元，较去年增长14％。同时，首次有毕业生在公益机构和国际组织就职，有10人被世界顶尖高校博士项目直接录取，4人获苏世民等全球知名奖学金等。

在毕业生调查中，国际化视野、通识教育基础、跨文化沟通能力和求职技能成为了毕业生自评的四大优势。与之相呼应，在用人单位对该校毕业生满意度调查中，人际沟通、积极主动性、解决问题、团队协作和跨文化交流被选为了满意度最高的五项能力。

上海纽约大学校长俞立中表示，这些亮眼数据背后的故事更值得关注，真正体现了上海纽约大学人才培养的目标和成效，彰显了立足中国的国际化教育对于学生成长的价值，以及对于构建人类命运共同体的意义。

据统计，2018届上海纽约大学中国毕业生青睐的雇主机构主要集中在五大行业，包括互联网/软件/硬件/信息技术（18%）、金融服务/会计/银行（18%）、广告传媒/市场营销（17%）、管理咨询（17%），以及教育/科研（15%）。和去年相比，选择加入互联网/软件/硬件/信息技术行业的毕业生人数增长了12%。

选择就业的60名中国学生中，就职于三资企业的占比最高，达到62%。值得一提的是，今年首次有毕业生就职于公益组织及国际组织，践行更富于社会与全球意义的职业路径，实现自己的人生价值。

在选择深造的中国毕业生中，有55人被USNEWS世界大学排名前50位的大学录取，占出国（境）留学总人数的68%，就读的名校包括哈佛大学、麻省理工学院、牛津大学、哥伦比亚大学等。其中，6人被世界著名高校的博士项目直接录取。

与此同时，2018届毕业生中，共18名国际学生留在中国就业或深造，占国际毕业生人数的16%。深厚的中国情结促使他们留在这片土地上继续实现自己的理想。

上海交通大学

毕业生就业单位以企业为主本科毕业生深造率达67.29%

上海交通大学昨天发布2018届毕业生就业质量报告。报告显示，本科、硕士和博士毕业生的就业率均达到98%以上,分别为98.01%、99.06%和98.51%。其中，2018届本科毕业生深造率达到了67.29%。

交大2018年毕业生共 8674 人(不含港澳台及留学生)，其中,博士生1209人,占毕业生总人数的13.94%;硕士生3955人，占毕业生总数的45.60%；本科生3510人，占毕业生总数的 40.46%。

按学历层次统计，本科生签约就业比例为24.99%，国内升学比例40.97%，出国(境)深造比例为26.32%，灵活就业比例为5.73%；硕士生签约就业比例为80.69%，国内升学比例为13.22%，出国(境)深造比例为3.94%，灵活就业比例为1.21%；博士生签约就业比例为79.74%，国内升学比例为12.90%，出国(境)深造比例为3.56%，灵活就业比例为2.31%。

另经对签约就业毕业生的就业流向、国内升学和出国(境)深造进行分析（不包含定向委培生和出国就业）后发现，交大签约就业毕业生 4790 人，国内升学毕业生为 2117 人，出国(境)深造(含出国做博士后)毕业生为1123 人。

上海交大积极鼓励学生服务于国家战略需求，引导学生到重点领域、关键行业等 国家国民经济建设主战场建功立业。从就业单位分布结构来看，毕业生就业单位主要以企业为主，其中居首位的是民营企业，比例达到27.04%，其次是医疗卫生单位，比例为19.56%，再次为国有企业，比例为19.47%。

上海电机学院

毕业生学历层次越高择业时针对性越强

新近公布的上海电机学院2018届毕业生就业质量报告显示，该校2018届毕业生总体就业率为98.64%，签约率为91.05%，灵活就业率为4.83%。其中硕士毕业生就业率为100%，本科毕业生就业率为98.65%，专科毕业生就业率为98.48%。分析发现，近五届毕业生均以在长三角区域经济体就业为主，辐射长三角区域社会经济发展成为主旋律。

具体来看，学校2018届毕业生升学比例9.60%，为近几年最高录取率。其中成功考取国内研究生的比例为5.26%，38人考取了985、211工程重点院校；考取国外研究生比例为2.44%。2018届硕士毕业生期初薪酬为6596.56元/月，本科毕业生期初薪酬为4870.54元/月，专科毕业生期初薪酬为4223.94元/月。

从学历来看，毕业生学历层次越高，择业时针对性越强。硕士毕业生就职岗位与所学专业的匹配度达到98.59%，本科毕业生匹配度为78.91%，专科毕业生匹配度为76.96%。总体反映出专业设置比较符合社会需求，毕业生所学专业知识及技能与实际工作的契合度较高，能够学以致用。

学校作为东部地区的应用技术型大学，始终将立足上海，辐射长三角，服务区域社会经济发展作为重要的发展使命。近五届毕业生均以在长三角区域经济体就业为主，占比均在85%以上。
