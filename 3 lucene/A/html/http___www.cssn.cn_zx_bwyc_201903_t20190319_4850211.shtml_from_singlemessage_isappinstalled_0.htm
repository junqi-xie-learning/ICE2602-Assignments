以身作则践行立德树人 ——上海交通大学师生热议学校思想政治理论课教师座谈会
2019年03月19日 18:23 来源：中国社会科学网 作者：查建国 仝薇 字号
打印 推荐

内容摘要：

关键词：

作者简介：

　　中国社会科学网上海讯（记者 查建国 仝薇）3月18日，中共中央总书记、国家主席、中央军委主席习近平在北京主持召开学校思想政治理论课教师座谈会并发表重要讲话。上海交通大学刘西拉教授、陈锡喜教授、施索华教授在座谈会现场聆听了习近平总书记的讲话。他们与校内师生围绕座谈会内容，结合自身学习、工作和生活实际进行了学习交流和表达了感想。

　　把思政教育转化成学生对真善美的追求

　　上海交通大学船舶海洋与建筑工程学院教授刘西拉在“读懂中国”的课堂上，讲解了如今中国大地上世界第二的高楼、高坝，世界第一的高桥、高铁，以及“以柔克刚”抗震建筑的东方智慧；作为思政教育者，他不回避全国接近60%城镇化率带来的资源与环境可持续发展挑战。他表示，出色的土木工程师最重要的就是要有社会责任感，青年学生应当在学习的过程中认识到身为一个建筑人的职业责任和家国责任。

　　上海交通大学讲席教授陈锡喜表示，在课堂上，要以实际行动传递座谈会思政教学理念，把深刻的理论、严密的逻辑与满腔的激情、生动的语言融为一体，并鼓励学生对学术观点提出质疑，大胆提出自己的意见。他认为，马克思主义理论的研究要始终坚持“理直气壮”的理念，才能为马克思主义理论学科建设作出重要贡献。

　　上海交通大学马克思主义学院教授施索华表示，思政课教师的职责，是把理想和信念这杆旗帜，通过教学树立到学生的心里，以自己对国家、对社会、对人民的深情来激励学生。因此，会一如既往地用一份真情把大学生思想政治教育转化成一种跟同学一起进行的对真善美的追求。

　　不断提高学生对思政课教学的获得感

　　上海交通大学马克思主义学院院长王岩教授谈到，在全国两会闭幕之际，习近平总书记来到了思政课教师中间，举行座谈会并发表重要讲话，使广大思政工作者深受鼓舞。作为思政工作者，我们要在这个承载着立德树人精神内涵、担当着教育振兴使命责任的时代中，用自己的真懂、真信、真爱、真做，实现自立、自爱、自强、自信，在实际工作中为培养优秀学子做出卓越贡献。

　　“总书记重要讲话博大精深、内容丰富，需要我们认真学习领会。”上海交通大学马克思主义学院胡涵锦教授表示，办好思政课的关键在教师。实践经验充分表明，青年大学生对思政课学习的态度和热情，在很大程度上是通过对思政课教师的认识来把握的。思政课教师在“课上课下”“网上网下”的一言一行，都体现了思政课的形象。从事思政课教学，培养担当民族复兴大任的时代新人，是思政课教师对党的事业忠诚度的体现。我们要以此次座谈会为重要契机，以习近平总书记重要讲话精神为遵循，加强自身建设，练好内功，不断提高思政课教育教学的针对性和实效性，不断提高学生对思政课教育教学的获得感。

　　上海交通大学马克思主义学院史宏波副教授认为，作为一名年轻教师，要做到用高尚的人格感染学生，用真理的力量感召学生，以深厚的理论功底赢得学生，自觉做为学为人的表率，做让学生喜爱的人。同时，积极投身思政课改革创新，不断增强思政课的思想性、理论性和亲和力、针对性，使思政课能够真正成为使大学生终身受用的课程。

　　上海交通大学人文学院张玉梅教授认为，爱国教育是扎根于厚重的传统文化的，也是基于文化自信的，是最具有说服力和感染力的。因此，我们古代文学教授、中国历史教授、传统哲学教授也承担着传承五千年中华文化之民族魂、家国情怀与优秀传统文化的责任。

　　将思政教育融入到理工科教学科研中

　　上海交通大学机械与动力工程学院常务副院长彭志科教授表示，大学的根本在立德树人，我们教师的天职是教书育人。作为专业教师，我们要主动走进学生，帮助他们规划人生，学到做人、做事的道理。

　　上海交通大学船舶海洋与建筑工程学院刘铸永副研究员表示，作为一名力学专业教师，我深知教师的作用不仅仅是向学生传授专业知识，更重要的是要帮助学生塑造健全的人格。在教学的过程中，我们经常通过讲述自身所从事研究的案例，让学生明白力学在航天工程等领域的重要作用，并以独立自主、艰苦奋斗的航天精神点燃学生的民族自豪感，引导更多的同学投身于航空、航天、船海等国家重点行业，为中华民族的复兴贡献自己的力量。

　　上海交通大学船舶海洋与建筑工程学院刘清风表示，十年树木，百年树人。树人的核心则在于立德。作为一名高校青年教师，通过学习习近平总书记关于学校思想政治理论座谈讲话，更加清楚地认识到品德教育的重要性。我国的经济、科学技术，目前都处在飞速发展的阶段。教育体系尤其是高等教育，担负着培养人才、积蓄力量的重任。而除了知识传输和创造力的培养之外，帮助学生建立完善的价值观和人格，更是我们教师在教学过程中不可回避的重要职责。

　　上海交通大学机械与动力工程学院张滕飞表示，作为一名教师，我能更加清晰地体会到教书育人、立德树人是大学的根本。大学是传播专业知识的地方，更是引导学生立鸿鹄志、做奋斗者、滋养家国情怀的地方；只有培养出政治强、情怀深、思维新、视野广、自律严、人格正的世界一流的学生，才真正称得上是世界一流的大学。

　　上海交通大学电子信息与电气工程学院蔡鸿明研究员表示，教师必先立德树人，才好引导学生践行社会主义核心价值观，培育德才兼备的社会主义建设者。为此，教师应继续坚持做到对人生具有理想，不断提高个体的品德修养和科研业务水平；对社会有热爱，以科技为第一生产力，将社会民生需要作为研究动力，将未来合格社会主义建设者的学生培养作为重要责任；对工作有情怀，既能热情投入开展改善社会民生的应用研究，也能潜心钻研面向国际前沿的基础科学。

　　上海交通大学材料科学与工程学院柳伟副研究员认为，我们作为高校教师，责任十分重大，必须时时重视思想政治原则，遵循教书育人规律，在培养学生专业知识与能力的同时，立德树人，将思政工作贯穿融入在日常教学与科研工作中。

　　“高校思想政治教育工作不仅仅是思政教师的事，更是我们每个高校教师不可推卸的责任。”上海交通大学材料科学与工程学院工程师吴一在学习了习近平总书记在学校思想政治理论课教师座谈会上的重要讲话之后，更清晰地认识到了高校思想政治教育的重要性。作为一名青年教师和科研工作者，平时和学生最主要的接触渠道就是上专业课和指导研究生，这就要求我们把学生的思想政治教育贯彻到这些日常工作中去。

　　上海交通大学化学化工学院党委副书记郑浩谈到，作为一名形势与政策课教师，我深深感受到今天中华民族伟大复兴的事业，为形势与政策课提供了无比丰富的素材。开设这门课是要引导大学生将马克思主义政治经济学基本原理融会贯通到现实中，去读懂、理解和预测国家和世界在宏观、中观乃至微观层面发生的种种变化。我们的教学方法也要与当代大学生认知习惯相适应，善于讲故事，敢于讲理论，真正将“四个自信”根植于当代大学生的头脑之中。

　　上海交通大学生命科学技术学院乔忠东教授结合自身教学经验，深刻认识到作为大学教师，除了传授知识以外，还需要帮助学生树立正确的世界观、价值观和人生观。在教学过程中，要把对学生的价值引领贯彻始终，在介绍科学知识的同时，还注重介绍中国科学家努力工作、奋勇拼搏的实例，以激励学生努力学习，使他们成长为既具有坚实的科学素养，同时又具有正确价值取向的社会主义建设者。

　　上海交通大学农业与生物学院杨志彪副研究员表示，学校能否做好思想政治工作，很重要的一个方面就是要看能否传承和引领中国的优秀传统文化。上海交通大学不仅以理工著称，文脉亦源远流长。随着民族复兴的重任落在我们这一代人的肩上，我们将更加自觉自愿地传承和引领优秀传统文化，做有历史责任感和使命感的新时代学人。

　　上海交通大学环境科学与工程学院思政教师汤淏溟表示，作为高校一线思政教师，我们要坚持党对教育事业的绝对领导，贯彻时代中国特色社会主义思想，坚持社会主义办学方向，牢牢站稳党的立场、国家立场和人民立场。以润物无声的育人方式，塑造掷地有声的话语体系，帮助同学们涵养家国情怀、培育人民情怀、激发行业情怀，引导同学们立足当下，扎根行业，勤于奋斗，科研报国。

　　上海交通大学电子信息与电气工程学院2018级硕士生邵华青，上海交通大学电子信息与电气工程学院网络空间安全专业2018级硕士生沈倩颖，上海交通大学-南加州大学文化创意产业学院2017级硕士生曾玉竹，上海交通大学国际经济与贸易专业2015级本科生戴如吉亮等学生代表也结合自身学习、生活经历分别作专题发言。
