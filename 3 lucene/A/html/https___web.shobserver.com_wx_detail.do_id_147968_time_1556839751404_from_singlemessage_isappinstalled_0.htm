图说“网红”原梦园：考上研究生的交大宿管阿姨

见识录 2019-05-02 05:45
来源：上观新闻 作者：赖鑫琳
再过几个月，49岁的上海交通大学宿管阿姨原梦园将和儿子一同成为大学研究生一年级的新生。这段励志故事也让原阿姨成了“网红”。

再过几个月，49岁的上海交通大学宿管阿姨原梦园将和儿子一同成为大学研究生一年级的新生。这段励志故事也让原阿姨成了“网红”。

 

上海交通大学宿管阿姨原梦园。

 

原梦园和儿子受邀与大学生一起分享母子一起的考研经历。

 

那天分享活动结束后，儿子给了原阿姨一个大大的拥抱，她笑得很开心。

 

2018年12月，原梦园陪儿子一同考研。最终，母子两人分别考上了广西大学汉语国际教育专业非全日制硕士研究生和复旦大学硕士研究生。

 

原阿姨帮助一位留学生办理入住手续，她能用流利的英语和留学生进行交流。

 

1991年毕业于师范学院的原梦园曾经在河南老家做过老师和银行职员，2011年来到上海“全职”陪儿子读书。

因为酷爱文学，儿子上大学后，她参加了成人高考，考取了复旦大学汉语言文学本科。

 

原阿姨和同事在留学生公寓查房。

 

她总是热心的提醒留学生注意宿舍的卫生整洁，保持窗子通风。

 

原阿姨的热心，让交大留学生公寓的留学生们印象深刻。

 

一个偶然的机会，原梦园应聘到交大留学生公寓当起了宿管阿姨。

作为留学生公寓的前台工作人员，她主要负责留学生开学入住登记、每周例行检查，以及出入境情况报备记录。值班时间从早上10点到晚上10点。

上班前的空挡，原梦园会去旁听汉语言和英语专业的课程，以此锻炼和培养自己系统学习的能力。

 

她还经常帮留学生收快递、接外卖，尽管这些都不是她的分内工作，她说这些留学生在中国求学不容易，就像对待自己的孩子一样。

 

当年为了考大学英语四级，至今原阿姨家的卧室门上还贴着当时记录着英文资料的便签。

 

原阿姨的日常学习笔记和英语考研资料。

 

不仅仅只是为了准备考研，多年来原阿姨一直保持着学习的习惯和热情。

 

她的家中每个角落都摆放着书籍，随手就可以拿来阅读。

 

原梦园考研最大的拦路虎是英语，记忆力差、背诵效率不高。

但她并没有妥协，诀窍之一就是不断重复。她背单词的记录显示，连续700多天背单词，一天都不曾中断。

还有一个诀窍是善于利用工作时间，碰到有留学生来办理登记、寄存等事宜，她就抓住机会坚持用英语交流。她还报名参加了英语和政治的网络课程。坚持在线完成了500余节课，常常回看复习。

最终，考研时她以53分的英语成绩成功过线。

 

旁听大学的课程，是原阿姨来交大工作的动力之一。

 

这是她记录的旁听课程表。

 

原阿姨在食堂吃饭。

 

原阿姨提着热水壶去打开水，她走路都带着风，一来节省时间，二来也能锻炼身体。

 

考研成功对于原梦园来说只是一个开始，她说：“我想学好专业，以后去偏远地区做一名志愿者，教小朋友们语文，带给他们求知的快乐。只要给我一个讲台，我就能尽情施展……”
