依图科技联合创始人林晨曦：打通数据孤岛，上海AI优势将更明显

产业观察 2020-07-09 08:36
来源：上观新闻 作者：刘锟
需要通过“一网统管”等城市级平台把各个委办局的数据打通，才能发挥数据的更大价值，人工智能才有进一步的挖掘空间

谈人工智能无论如何都逃不开芯片、半导体产业，上海在全国领先优势显著，这不是在一两年或者三五年就能做到的。——依图科技联合创始人林晨曦

面对突发疫情，国内医疗机构的临床压力很大，如何利用人工智能技术更准确地评价新冠肺炎患者的病情？上海人工智能“独角兽”依图科技冲在了最前面。“如果试剂盒是定性分析，那么通过CT则可以进行定量分析。”依图科技联合创始人林晨曦说。但如果仅靠人工读片，在疫情爆发初期这是一项无法完成的任务。

今年2月，依图和上海市公共卫生临床中心联合发布业内首款胸部CT新冠肺炎智能评价系统，将原本需要数小时的评估过程压缩到2-3秒，并能自动分析病情的转移和发展，极大缓解临床一线压力。同时，系统肺炎病变检出敏感性达97.3%，特异性达99%，与高年资医生水平相当，显示出高稳定性的检出质量。

如果说几年前人们对人工智能还将信将疑，现在的认知已经发生了深刻变化。“人工智能是百年一遇的技术革命。它不仅仅是一项底层技术，还能促进各个行业的基础技术突破。”林晨曦说。这一轮的人工智能在算法和算力的共同迭代中，真正改变了各个行业。比如，生物制药通过人工智能技术的介入，使得药物在临床前实验的时间可以从3-5年下降到1年。以前制药探索过程很多是靠人的经验去试去碰，不停地靠生物和化学实验来看大分子或者小分子的药，特别是靶向药是不是能打得上靶点。如今，机器能够通过人工智能计算大分子在空间中的折叠形状，把“不靠谱”的直接过滤掉，研发效率大大提升，将加速人类攻克癌症等疑难病症。

现在也有理论界担心，深度学习的算法已经走到尽头。在林晨曦看来，通过将人工智能算法和算力耦合设计，将带来性价比更高的智能算力，深度学习算法就依旧有挖掘潜能。特别在应用侧，现实的生活场景还有大量长尾需求需要通过极致的人工智能算法去解锁，目前的深度学习算法完全可以解决这些问题。比如，现在上海正处于汛期，大雨来时哪些地方会有积水，积水到何种程度，车是不是开得过去等等，都可以通过人工智能对摄像头视频的算法来主动发现。 

林晨曦判断，未来会迎来一轮人工智能算力的变迁。之前更多的是依赖英伟达做的GPU卡，GPU卡在理论中被学术界用得很好，才带来了深度学习这波热潮。而依图也在加紧布局，去年5月在上海发布自研AI芯片求索，在视觉计算维度，求索芯片的单路摄像头功耗仅为英伟达GPU P4的30%。未来一段时间芯片会加速发展，依图正在研发新一代芯片。

身处上海，林晨曦对上海人工智能发展充满期待。在他看来，算力跟算法之间关系的变革，使上海半导体产业深厚的积累在这几年爆发。这一波人工智能的发展上海优势明显，谈人工智能无论如何都逃不开芯片，而上海半导体产业基础在全国领先，这种基础性优势不是一两年或者三五年就能做到的。同时，人工智能的商业化落地需要大量场景，上海这样的超大规模城市具备其他地方无法比拟的丰富应用场景，“这一轮人工智能热潮事实上和互联网公司关系不大，而是由于社会方方面面对人工智能的需求。”

对人工智能发展来说，数据就如同燃料。林晨曦表示，上海虽然有海量数据，但大量的数据还是孤岛，需要通过“一网统管”等城市级平台把各个委办局的数据打通，才能发挥数据的更大价值，人工智能才有进一步的挖掘空间，有产生新价值的机会和条件。
