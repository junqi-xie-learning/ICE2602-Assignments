【视频】上海交大行业研究院揭牌

2018-12-09 15:30:25

扬子晚报网12月9日（记者 罗晓娜 文/摄）12月8日，“纪念改革开放四十周年：中国行业发展高峰论坛”暨上海交通大学行业研究院揭牌仪式在上海交大安泰经济与管理学院举行。

高峰论坛旨在通过对学术内涵和外延的重新界定，把求真、务实和育人更加紧密地结合起来，助力中国行业和社会经济地发展。这将是中国管理教育事业一次重要探索，也是引领中国乃至世界商学院发展的一次开拓性尝试。

上海交大党委书记姜斯宪教授在论坛致辞中表示，今年是改革开放40周年，40年来我国的经济取得了巨大的进步，我国的高等教育，包括管理教育也同步获得了巨大的进步。安泰经济与管理学院在百年的发展中，锐意改革、勇于创新，逐步已经建设成一所国际知名、国内领先的经管学院。成为交大的亮丽名片之一。高校的管理教育也需要破浪前行，要逐渐积累经验，克服陷入学术自循环的怪圈。这需要商学院主动对接行业需求，一方面对国家的经济转型和结构调整提供解决方案和智力支持，另一方面继续向着成为世界顶级商学院的目标不断前进，并且提出具有中国特色的解决方案。
