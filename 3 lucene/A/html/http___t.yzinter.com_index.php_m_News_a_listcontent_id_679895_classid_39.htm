原创|“沿着周总理的足迹”全国微电影大赛颁奖典礼在淮安圆满落幕

2019-03-05 20:06:37

扬子晚报网3月5日讯（记者 罗晓娜）3月5日下午，“沿着周总理的足迹 不忘初心 砥砺前行”全国微电影大赛颁奖典礼在江苏省淮安市举行。本次活动由中共淮安市委宣传部、中国电影评论学会微电影研究会、中共上海交通大学委员会宣传部主办，上海交通大学苏北研究院、周恩来纪念地管理局、上海联红网络科技有限公司联合承办。

2018年，是敬爱的周总理诞辰120周年。用微电影的形式，拍摄和记录周总理的事迹，以及全国各族人民沿着周总理的足迹，不忘初心、砥砺前行，努力建设社会主义物质文明和精神文明，是我们学习周总理、纪念周总理、推进中华民族伟大复兴的有效方式。

大赛从2018年9月21日开启作品征集，南到贵州、广西、海南，北至新疆、甘肃、吉林，全国数百所高校、百余家影视文化工作室，以及数十家企事业单位积极参与，累计收集作品1250余部，最终评选出一等奖3部，二等奖7部，提名奖10部，优秀作品奖10部。北京参赛作品《爷爷的徽章》、同济大学《一个想踢足球的小女孩的故事》、浙江第四监狱《沙果树》获得一等奖。南京中瀚影视文化传播公司《我的铁路我的路》、上海戏剧学院《心履》、云南彝良县政协《山路弯弯》等作品分获二等奖、提名奖和优秀作品奖。

微电影是一种新型媒介传播形式，具有大众性、娱乐性、传播迅速等特点。通过微电影的形式，全面展示各地、各条线学习弘扬周恩来精神的新风貌、新成果，是一种富有创意的文化宣传形式。中国电影评论学会名誉会长章柏青认为，目前全国各种微电影大赛繁多，但是弘扬伟人精神、扣紧时代主题的专项微电影大赛却很少。在周恩来总理诞辰120周年这个时间段，淮安市和国内专业的微电影学术团体联合主办这次微电影大赛，将在我国微电影发展史上创造一个新的种类，具有先声夺人的效果和魅力独具的品味。上海交通大学苏北研究院理事长李建强提出，大学为地方发展建设服务，既要突出科技和人才优势，积极开展创新研发与科技成果孵化、转化和产业化，还要在先进文化引领和推动上作出贡献，这次和地方联合举办主题微电影大赛就是一个有益的尝试。淮安市委宣传部副部长、精神文明办主任杨恒忠表示，淮安是周总理的故乡，本次赛事由淮安市牵头主办，是落实习近平总书记“把周总理的家乡建设好”重要指示，塑造淮安精神，扩大淮安在全国文化影响力的一个重要抓手。摄影：王昊 编辑：李燕
