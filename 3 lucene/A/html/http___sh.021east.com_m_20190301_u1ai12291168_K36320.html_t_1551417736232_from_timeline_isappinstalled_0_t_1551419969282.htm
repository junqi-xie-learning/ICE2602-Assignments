仁济医院宁波医院全面运营 将提供航空救援等服务

东方网 通讯员叶佳琪、袁蕙芸 刘轶琳
2019年3月1日 12:35:54

　　东方网通讯员叶佳琪、袁蕙芸、记者刘轶琳3月1日报道：今天上午，位于宁波市杭州湾新区的上海交通大学医学院附属医院宁波医院全面开业，成为浙东地区首家由百余位沪甬专家常驻的大型综合性医院。医院将正式开设门急诊和病区，首期开放床位600张。

　　仁济医院宁波医院执行副院长陈尉华表示，仁济医院宁波医院的开业是沪甬紧密合作紧跟国家战略，落实长江三角洲高质量一体化发展的医疗新举措。今后，将继续加强沪甬两地合作，克服困难，把最优秀的专家、最强的学科及其骨干输送到这里，把医院建设好。也将继续以“共建、共赢、共生、共享”为宗旨，打造一个优质高效现代化的一流医院，为宁波杭州湾新区甚至长三角地区的老百姓带来最大的福祉。

　　正式启用后，仁济医院近百名医疗骨干常年入驻，为当地百姓提供与仁济医院同质化的门急诊、手术等优质医疗服务。

　　“仁济医院派出的医疗骨干必须是正高职称。要切实适应长三角一体化，同质化高质量发展要求。为此，院方对“大咖”们都有一些硬性规定，比如门诊量、手术量等”，仁济医院副院长王育告诉记者，“今天就要开展2台手术，都是慕名而来的病人，病人如此迫切，也是我们未曾预料的。”

　　仁济医院消化科主任医师马雄向记者透露，宁波地区的自身免疫性肝病患者比较多，原来每次去上海看病至少需要2天的车马劳顿。杭州湾医院全面运营，仁济医院医生到家门口坐诊，病友们都很激动。马雄同时提到，他在当地坐诊有利于直接培养医院的年轻医生，提高该院在自身免疫性肝病的诊治水平。

　　启用仪式上，仁济医院宁波医院与中国人民财产保险股份有限公司上海市分公司进行“直升机救援项目合作协议”签约仪式，根据协议直升机航空应急救援项目将为社会大众提供院前救援、院间转运等常态化的、即时响应的航空医疗救援服务。

　　据悉，2017年5月25日，在“沪苏浙皖闽”四省一市综合医改联席会议上，上海交通大学医学院附属仁济医院与宁波杭州湾新区开发建设管理委员会以及宁波市第二人民医院共同签署了合作办院协议。该院于2014年12月开工建设，作为宁波市新十大医疗卫生基础建设项目，项目总投资14.5亿元，总建筑面积为19.65万平方米，设计床位1200张，以三级甲等综合医院标准建造，是宁波市已建成投用的单体投资额最高的医院。该院同时命名为上海交通大学医学院附属仁济医院宁波医院、宁波二院杭州湾院区、宁波杭州湾医院，于2018年12月26日开业试运营。
