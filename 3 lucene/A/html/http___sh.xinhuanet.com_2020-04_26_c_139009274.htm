7位专家7场讲座，共话后疫情时代的城市治理之路

2020年04月26日 15:33:07 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

    新华网上海4月26日电（吴一航） 突发公共卫生事件如何有效应对？城市治理与应急管理如何实现现代化转型？公共卫生体系该如何完善和构建？在国内新冠肺炎疫情得到有效控制后，城市治理和公共卫生体系建设等问题便成为公众关注的焦点。近日，上海交通大学国际与公共事务学院举行了“新冠疫情与城市治理”系列公益讲座新闻发布会。并且，从4月19日至6月7日，该讲座将举行7场线上直播，针对上述问题，主讲嘉宾们将从不同角度为大家逐一解答。

    上海交通大学国际与公共事务学院党委书记姜文宁在新闻发布会上介绍，本次线上公益讲座将围绕疫情防控与城市治理体系治理能力现代化、公共卫生体系建设、应急管理、基层治理等话题展开，旨在向各界嘉宾学者分享学术研究和资政建言成果，进一步向社会传播温暖有力的“交大声音”。

    4月19日，“新冠疫情与城市治理”系列公益讲座已成功完成第一讲。上海交通大学国际与公共事务学院教授彭勃作为“第一课”的主讲人显得十分激动。“这是第一次在线上做主题讲座，心情比较忐忑。看到‘会议室’显示有近三千人同时在线，感觉很自豪。”彭勃说，这场全民战“疫”的过程中，基层社区发挥了巨大作用。在后疫情时代，基层治理应该加快现代化建设，使之成为社会稳定的坚实基础。

    公益讲座“首秀”便取得了不错的效果，接下来陆续登陆直播间的几位教授纷纷表示信心满满。

    据悉，未来几日，教授们将结合各自在专业领域中多年的研究积累成果，为大家带来“‘互联网+’与应急治理” “新冠肺炎疫情对公共卫生体系的挑战与变革”“新冠疫情防控与非常态治理能力提升”等多场直播讲座。
