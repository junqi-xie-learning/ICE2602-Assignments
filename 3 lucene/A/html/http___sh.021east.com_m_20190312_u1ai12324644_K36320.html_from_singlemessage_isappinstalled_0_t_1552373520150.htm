跨越万里种绿洲！这群交大学生3年防风治沙2000亩

东方网 傅文婧
2019年3月12日 14:42:25

　　东方网记者傅文婧3月12日报道：今天是植树节，植物造林共创绿色家园，是我们每个人的责任。而对上海交通大学绿格防沙公益团来说，他们的“战场”在西部沙漠。每年夏天，这群年轻的大学生都会乘着绿皮火车，将青春热血挥洒在西部的沙漠中，用自己的行动践行着他们的理想。

绿格防沙公益团部分成员

　　作为一支校园公益团队，交大绿格防沙公益团在参与防沙治沙工作中，以制作“麦草格子”的基础，结合高校科研团队的嫁接技术以及社会资金，努力以更高效的方式展开防沙治沙工作，起到解决荒漠化问题兼促进沙产品产业发展的作用。团队曾荣获上海市“知行杯”二等奖，上海市“创青春”公益组金奖等荣誉。

　　每年夏天，团队成员从上海启程，经历一天一夜的火车跋涉后，抵达他们的目的地――甘肃省武威市民勤县的一做小村庄。治理沙漠是一项大工程，但要种出一片绿洲只需从一件小事出发――制作麦草格子。

　　将麦草格子放置在沙漠边缘，可以固定流沙，防止沙漠的进一步扩张。在制作沙障之前，民勤县的沙漠正在以肉眼可见的速度吞噬土地，这里的风沙线长达408km，整个县有91％的面积是沙漠。绿格公益团的同学们每天都要与如此庞大的沙漠对抗。

　　“看着漠漠黄沙，制作座座格子，才明白，原来我们能做的比日行10000步早起收集能量要多得多。”绿格防沙公益团创始人、上海交大环境科学与工程学院2015级本科生刘智卓十分感慨。

　　“绿格”防风治沙公益团创立于2016年夏天，由交大环境科学与工程学院同学倡议发起，参加成员累计350多人次，吸引超过15个高校的同学共同参与。在过去三年中，团队成员利用寒暑假的空闲时间，组队前往荒漠化严重地区进行长期、持续的防沙治沙工作。至今已经连续三年，共计120余人前往甘肃武威组织开展防风治沙公益服务活动，制作麦草格子共计五万个，固定黄沙两千余亩，治沙效果已经初见成效。

　　同时，绿格公益团在制作麦草格子并种植梭梭树的基础上，通过嫁接肉苁蓉获得高经济附加值的药材，也是对当地精准扶贫的一种尝试。目前梭梭苗和肉苁蓉的长势喜人。“在这趟西部实践中有着从未经历过的生活，从未见过的世界。有笼罩着天空的晚霞和星辰交替，有烈日炙烤的大地，还有星空下与大地同眠……”一位参加了绿格暑期防沙公益实践的同学这样形容。

　　此外，在开展治沙行动同时，绿格公益团也在探索，为当地社会发展做出其他贡献。一方面，团队在参与治理我国西北部荒漠化的工作中，不断加强宣传，呼吁更多人参与其中。另一方面，也尝试通过高校宣讲、考察调研、技术引进等方式，为民勤地区带来更多环境、经济、教育等方面的社会效益。

　　热血青春实践公益，用脚步丈量祖国山河，用奋斗践行共同梦想，这群有志于绿色公益的交大学子，在未来将继续行动，为祖国建设贡献自己的光和热。
