建设世界一流数学学科 交大吴文俊数学中心成立

看看新闻Knews记者 周云

2019-05-09 15:24:49

2019年是中国科学院院士、第三世界科学院院士、首届国家最高科学技术奖获得者吴文俊（1919～2017）先生诞辰100周年。5月9日，上海交大吴文俊数学中心揭牌成立。“中心”将致力于数学科学前沿研究，基础数学科学以及数学史领域的重大问题，以数据科学中关键数学方法与技术瓶颈问题为导向，力求解决国家重大需求问题，开辟引领重要研究方向。


吴文俊数学中心成立后，将重点在基础数学、数据科学、数学史领域开展相关研究。



在基础数学领域，中心将重点围绕代数与数论、几何与分析、动力系统与偏微分方程等方向展开研究。代数、数论和几何与分析是数学研究的基础，也是科学创新的支柱，偏微分方程与动力系统是与实际应用紧密关联的基础研究领域。交大数学在上述研究领域有着良好的研究基础和优秀的科研团队。近年来交大研究团队在上述领域取得了许多突破性、原创性的科研成果。我们将以此良好的发展势头为契机，充分利用中心的平台作用，把这些研究方向打造成为交大数学的优势和特色研究方向。


数据科学研究离不开学科交叉。在数据科学领域，吴文俊数学中心将重点聚焦数据科学中的基础算法、统计大数据分析、机器与深度学习、数学与生物医学交叉。2018年9月，上海交大吴文俊人工智能荣誉博士班启动，致力于开展问题驱动的创新性研究模式，为上海交大研究生拔尖人才的培养搭建良好的教学、科研互动平台，培养一批具有宽阔视野、创新能力与社会责任感的人工智能领域领军人才，推动人工智能科技创新。


吴文俊曾说过，“中国数学的道路跟西方欧几里得传统的所谓公理化的数学道路是不一样的”，“不要追求纯而又纯的数学，不要孤立，应与其他学科相联系”。这位在数学研究和人才培养方面倾注了大量精力、为复兴中国古代数学文化做出巨大贡献的科学家，毕生都在探索中国数学学科如何走自己的路、如何与其他学科交叉发展，为世界一流的高水平数学学科建设鞠躬尽瘁。



上海交大校长、中国工程院院士林忠钦表示，世界一流大学建设离不开一流学科的支撑，上海交大数学学科拥有底蕴深厚的传统实力，丰富的理、工、农、医、人文社科等学科群，能够有力地支撑数学学科与其他学科交叉、融合，在这样的基础上成立吴文俊数学中心，为世界一流数学学科建设提供了良好的条件。“上海交大在国家重点建设计划的持续支持下，拥有了若干接近世界一流的学科群，聚集了一批具有国际水准的高水平师资，构建了特色鲜明的拔尖创新人才培养体系，建成了一批实力雄厚的国家级科研基地，形成了‘中国特色、世界一流、交大特征’的办学模式，整体实力接近世界一流水平。而建设世界一流大学和一流学科，是党中央、国务院做出的重大战略决策，对于提升我国教育发展水平、增强国家核心竞争力、奠定长远发展基础，具有十分重要的意义。”


同时，纪念吴文俊院士诞辰一百周年暨数学科学与数学史国际学术研讨会举行，13位院士、近40名国内外知名专家学者齐聚交大，围绕吴文俊先生的成就开展研讨，并分享代数学、代数几何、微分几何、动力系统、数值计算、偏微分方程、数学史等相关领域的最新研究进展，以纪念、缅怀吴文俊先生。
