胡润：中国现有独角兽公司202家，近三月发现21个新面孔，新出上海科技服务机构榜

创新之城 2019-05-26 19:14
来源：上观新闻 作者：徐瑞哲
沪上15家机构分获“领军五强”和“新锐十强”，还有10位上海科技服务技术经理人获评“Top 10”

中国到底有多少估值超过10亿美元的年轻创业企业？作为胡润百富集团董事长，胡润26日现身上海闵行虹桥商务区，在2019浦江创新论坛全球技术转移大会上用中文披露：中国现有202家这样的独角兽公司，而且最近3个月内就新发现了21家，平均不到5天就出现一个新面孔。这些“独角兽”中，人工智能（AI）公司占相当部分，平均创业年限仅为7年。

 

“独角兽”的“角”，正是其核心技术带来的市场竞争力。此间，科技部成果转化与区域创新司有关负责人透露，2018年全国技术合同交易额达1.78万亿，同比上一年增长32%。其中，包括一项大学教授团队专利技术转让，以5.2亿价格许可给企业进行实施，一批这样的成果转移转化的典型案例涌现。同样，上海市科委总工程师陆敏表示，上海用足用好科技创新平台载体，目前申城各类孵化器超过500家，总面积达到320万平方米，服务企业和团队逾2.7万家。

 

为提升全社会对社会化技术转移机构的价值认同，“2019胡润上海科技服务机构榜”同期公布，沪上15家机构分获“领军五强”和“新锐十强”，还有10位上海科技服务技术经理人获评“Top 10”。这十大获奖人物的代表，同时分享了“技术经理人如何利用自身优势助力全球技术流转”的行业经验。

 

对于有别于富豪榜的新榜编制，胡润（Rupert Hoogewerf）解读称，其数据基础来自促进技术转移项目成交数量及金额、行业影响力、财务状况、特色案例这四大维度，并综合参考“以转让、许可、作价投资方式转化项目成交金额”，还关注到机构和企业的团队构成与人才比例，经综合评分后排列出结果。

 

这份新的“胡润榜”显示，脱胎于高校和科研院所的上海交通大学先进产业技术研究院、盛知华知识产权服务等，跻身“机构领军五强”；出现在“机构新锐十强”榜上的华理国家技术转移中心、上海理工大学技术转移中心也来自高校。相应地，这些机构的刘群彦、廖玉清、纵刚，以及来自上海海事大学技术转移中心的莫剑英等，都是“胡润上海科技服务技术经理人”十强。

 

作为机构大平台，国家技术转移东部中心平台、上海闵行国家科技成果转移转化示范区等载体不断发展壮大，也进一步促进了国际国内成果转化的良好生态形成。截至目前，国家技术转移东部中心已建成14个海外渠道，16个国内分中心，集聚218家沪上创新服务机构，涵盖38218项成果，其中包括17973项海外成果。

 

大会现场，作为“走出去、引进来”双向服务渠道，中、新、以、荷、丹等国同行协同，国家技术转移东部中心Brightlands技术转移联盟、国家技术转移东部中心荷兰分中心（欧洲总部）正式设立。
