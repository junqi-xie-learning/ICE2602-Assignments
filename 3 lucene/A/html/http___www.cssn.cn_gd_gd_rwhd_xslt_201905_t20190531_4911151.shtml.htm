上海交通大学倡导构建“科技人文命运共同体”
2019年05月31日 14:34　来源：中国社会科学网　作者：查建国 仝薇 字号

来源：中国社会科学网 作者：查建国 仝薇

“对话与融通：首届科技人文国际学术研讨会”新闻通气会现场 本网记者 查建国/摄　　

　　中国社会科学网讯（记者 查建国 仝薇）5月31日，上海交通大学举办“对话与融通：首届科技人文国际学术研讨会”新闻通气会。上海交通大学党委副书记顾锋致辞。第一届科技人文国际学术会议的发起人、上海交通大学外国语学院副院长、《上海交通大学学报》（哲学社会科学版）主编彭青龙教授在会上发出“构建科技人文命运共同体”的倡议。

　　上海社会科学院文学研究所所长荣跃明，联合国教科文组织哲学与人文国际理事会执委、亚洲新人文网联主席熊炳珍作视频发言。与会学者提出，要直面科技人文学术“融通赤字”、探索构建“科技人文命运共同体”。上海交通大学新闻中心副主任李厚锐主持新闻通气会。

　　顾锋在分析了交大文科发展和文理交叉研究现状的基础上指出，上海交通大学作为一所理工科见长、文科发展迅速的研究型、综合型和国际化的“双一流”建设大学，十分重视科技人文的融合性研究。近年来，学校一直在推动建立文理交叉研究平台，鼓励交大学者开展跨学科研究。最近，交大先后跟国家部委和地方企事业单位合作，建立了多个校级或者院级的跨学科研究机构，如人工智能研究院、心理与行为科学研究院、语言智能中心、健康长三角研究院和城市治理研究院等。立足于理工科的学科优势和多学科背景，上海交通大学主动瞄准世界学术前沿，积极对接国家战略领域和关键领域需求，让科技人文在多学科交叉融合中逐步释放活力、激发动力、提升能力，加快了建成世界一流大学和世界一流学科的步伐，为国民经济社会发展、尤其是进一步对外开放事业以及人类命运共同体贡献上海交通大学的智慧。

　　彭青龙认为，面对“人工智能、基因工程和量子科技”三驾“科技马车”齐头并进、快速发展的新形势，我们发现传统的人文社科研究已经不能适应人类社会和世界发展的新需要，开展跨国界、跨学科、跨领域的“新文科”研究、甚至“融通科技人文”的新文科研究更符合时代发展的需要。这就需要我们要主动打破原有的学科“过度细化”和自然学界和人文社会科学界“过分异化”的局面，消解科技人文老死不相往来的“融通赤字”，倡导构建“科技人文命运共同体”，有效开展“对话和融通”的国际学术活动。他强调，自觉构建“科技人文命运共同体”是对构建“人类命运共同体”有益探索。作为推动人类社会发展的重要力量，科技和人文在全球化的背景下正日趋融合，人类社会和世界发展的重大科技问题和人文社会问题需要全世界的科学家和人文社会学者一起努力方能解决。自觉构建“科技人文命运共同体”是中国从大国向强国转变的必然要求。作为世界第二大经济体和人文底蕴厚实的国家，中国的科技和人文社会科学对世界的贡献有目共睹，但我们在文化软实力建设,尤其是推动科技人文研究方面与我国的国际地位还不相称，因此，构建具有中国特色的“科技人文命运共同体”、推动科技人文理论和实践研究上一个新台阶，既是学科内部融通发展的需要，也是中国和平崛起的需求。

　　荣跃明表示，当今世界已进入了创新引领发展的新时代。无论是科技创新、文化创新、还是制度创新，说到底都是知识创新，是人类认识世界和改造世界能力的不断提升。人类全部知识可分为科技和人文两部分。任何创新，必须有利于提高社会生产力、改进人的生活、推动社会进步，才对人类具有真正的现实意义和价值。因此，科技创新需要人文关怀提供价值尺度、伦理规范和关乎人类前途命运的发展方向；缺失人文关怀的科技创新，在解决人的生产、生活某一具体问题的同时，极有可能在社会层面上带来更为棘手的难题；人们在享用科技创新带来的生产增长和生活改善时，必须直面由此带来的社会冲突和道德挑战。为此，在创新引领发展的新时代，面对科技创新所具有的无限可能性，科技和人文工作者必须携手合作，在创新中加快推进科技知识和人文知识的融合，始终把以人为本作为一切创新的前提，共同承担起创新引领发展的时代使命。从这个意义上说，科学和人文是同一个命运共同体。

　　熊炳珍对即将在上海交通大学举办的科技人文国际学术研讨会充满期待。她指出，纵观世界各国，大家普遍认识到，人文社科的未来在于与科技同行。尽管目前依然有相当多的学者继续从事传统的人文艺术研究，如性别研究、身份认同研究等，但从事医学人文、生态人文、数字人文、管理和评价等领域的学者在其研究内容和方法上越来越呈现出科技人文融合的趋势。我们需要继续搭建国际对话与交流的平台，使得新思想、观点和新范式持续迸发。作为中国、乃至世界知名的高等学府，上海交通大学推动科技人文融合的活动具有重要意义，并将产生重要影响。
