笔墨丹青寄托教职工对祖国祝福

笔墨丹青寄托教职工对祖国祝福

来源：劳动观察

作者：郭翼飞 2019-09-17 08:50

一幅幅构思巧妙、寓意深远的笔墨丹青描绘了祖国风雨兼程的70载，寄托着广大教职工们对祖国的真挚情感。

日前，由上海交通大学工会举办的“翰墨庆七秩 礼赞新时代”庆祝新中国成立70周年教职工书画展开幕。一幅幅构思巧妙、寓意深远的笔墨丹青描绘了祖国风雨兼程的70载，寄托着广大教职工们对祖国的真挚情感。


此次展览由两个篇章构成：由C9高校工会共同发起、西安交大工会牵头的“庆祝新中国成立70周年C9高校教职工书法、绘画作品巡回展”中的部分优秀作品；以及“第六届上海教师书法·板书·钢笔字·中国画大赛”中的部分优秀作品，上海交大学校教工书画协会的部分优秀作品也纳入其中。共展出作品100余幅，含中国画、油画、版画、水彩画等画作以及毛笔、板书、钢笔等书法作品。


西安交通大学工会是此次C9高校书画巡回展的发起单位之一，据该校工会副主席张庆生介绍，此次书画巡回展共收到9所高校91幅作品，其中不乏诸多名家大作，广大教职工将对祖国的热爱诉诸笔端，用情和景展现新中国的70年光辉岁月。她还提到，上海交大工会出版的《书画交大》《诗文交大》等系列书籍，获得了全国高校工会的赞誉，成为宣传展示上海交大教职工文化的新名片。希望两校工会今后继续加强合作，共同搭建C9高校教职工文化艺术交流平台，丰富高校教职工文化生活，提升教职工文化素养。


上海交大党委副书记顾锋表示，学校在去年举办的“纪念改革开放40周年”书画展在校内外得到了热烈的反响，今年在此基础上举办的庆祝新中国成立70周年为主题的书画展，旨在努力讲好中国故事、以书画彰显中国文化、以书画传递中国精神，希望通过活动不断加强师生互动，让学生通过书画艺术的熏陶接受美育教育，让大家在欣赏书画的过程中，深切感受教职工们的爱国之志、报国之情。
