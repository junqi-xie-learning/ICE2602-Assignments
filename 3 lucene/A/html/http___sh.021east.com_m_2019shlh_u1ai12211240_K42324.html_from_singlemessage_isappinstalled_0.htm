中高考加分、买房打折、生育津贴……政协委员“出招”鼓励女性生二孩

来源:东方网 作者:刘理

2019年01月29日 16:31

　　东方网记者刘理1月29日报道：据国家统计局数据显示，二孩政策放开后，二孩生育热情普遍低于预期。为什么政策放开了，人们却也不愿意生了？在今年的“两会”上，上海市政协委员、上海交通大学医学院附属仁济医院副院长狄文建议从医疗、住房、教育方面推出多项政策，鼓励更多家庭生育二孩。

上海市政协委员、上海交通大学医学院附属仁济医院副院长狄文接受东方网记者采访

　　“此前，我在与年轻夫妻交流的过程了解到，大部分人夫妻普遍反应生活成本大，不敢生二孩。比如，生产医疗费用高、买房压力大、教育焦虑等等方面。”狄文委员告诉东方网记者，以上种种因素都影响现代女性生育二孩的热情。

　　对此，狄文委员从政府和企业的角度，提出多项建议，以减轻成本、鼓励更多家庭生育二孩。

　　狄文委员建议，首先，二孩生育服务全程免费或部分免费。“相对来说，生二孩的家庭负担会比较重一点。我说的这个免费，包括备孕前优生检查、怀孕中产检、住院分娩基本服务费、新生儿体检等，通通免费或部分免费。”狄文委员说。

　　目前，家长对孩子教育的焦虑也投射在了“生育”这一层面。狄文委员认为，在中考或者高考等选拔性考试中，应当适当给予“二孩家庭”政策倾斜。“比如，中考或高考的时候，给第二个孩子适当加分、优先录取等。”他还建议，二孩在义务教育阶段以及高中、大学阶段的学费，享受一定减免或者补贴。

　　为进一步降低二孩家庭的生活成本，狄文委员提出，政府可以学习国外的方式，以家庭为交税单位，为有二孩的家庭适当减免个税。在呼吁政府出台相关政策、减轻二孩家庭负担的同时，他也表示，社会应承担一部分政府的负担

　　“大家都在说购房压力大，那企业是不是也应该承担起一部分社会责任。比如，二孩家庭在购房时，企业可以打个折扣，或给予一定的补贴。”狄文委员认为，企业还应为二孩母亲提供较为弹性的工作时间和方式，增加女性职工的生育津贴，给予女性职工带薪休产假的权利等等，以此为女性员工创造一个更好的生育环境。
