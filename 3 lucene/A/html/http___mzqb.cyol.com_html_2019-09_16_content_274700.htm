上海交大众学子参加暑期社会实践——

青春献给祖国是最好的“告白”

中国青年报·中国青年网记者 王烨捷 来源：中国青年报（2019年09月16日  01版）

    9月8日，上海交大。在上万名学生参加的2019级新生开学典礼上，有妙趣横生的暖场快闪、有院士校长的暖心叮嘱，但最受全体新生关注的，当属“女神”学姐梁晴雪带来的一堂“微思政课”。

    与以往“开学第一课”通常由校党委书记或校长开讲不同，这一次，校领导把讲台让给了上海交大船舶海洋与建筑工程学院2016级博士生梁晴雪——一个在上海交大求学10年，3次以新生身份听取“开学第一课”，参加过10多次社会实践活动，被称为交大“女神”的90后。

    “在和交大十年如一日的朝夕相处中，我和周围的同学一样，始终将科学研究和服务社会的交汇贯通融入到大学生活中，不断体验和践行着交大人的使命和担当。”梁晴雪当主讲人的“微思政课”总能与台下的学生找到“共情”，“无论何时，我们交大学子始终具有与祖国同呼吸、共命运的责任担当。”

    中国青年报·中国青年网记者注意到，上海交大学子的这段青春“告白”远不只是“说一说”这么简单。在他们朴实话语的背后，是“脚踩大地”的自信。以梁晴雪为例，她的“微思政课”虽然只有短短几分钟，但这堂小课的背后却是她十年奔赴国内15座城市、海外15个国家进行实践、调研的经历。

    梁晴雪曾是上海交大大学生志愿服务西部计划的一名志愿者，援藏期间，她深入7个地市开展调研，足迹涉及边境地区、偏远农牧区，走近藏民族，学习藏文化，与藏族同胞交朋友，尽可能助力脱贫攻坚、项目建设等工作。

    上海交大的社会实践，在全国各地、甚至是世界各地向来以“实在”闻名。除了调研，他们往往能给对方提供最新、最合适的建议。比如梁晴雪所在的雪域栖居博士生实践团，他们到西藏的目标很明确——研究如何在高原上又快又好地盖起结实的房子。

    平均海拔4000米以上的日喀则是西藏第二大城市，在如今大建设、大跨越、大发展的背景下，这座城市开始寻找改善农牧民住宿条件的好办法。

    两周调研下来，交大团队给出的建议极为细致。比如，他们建议应因地制宜选择合适结构模式，对多层和高层住宅、相关商业办公等项目可采用普钢框架结构体系或预制装配混凝土结构体系；对高档别墅民居、边境小康示范村及低层公建等可采用冷弯薄壁型钢结构体系；对牧区、边境、生态敏感区和抗震救灾等建筑可采用模块化钢结构体系。他们还建议当地政府在政策激励方面，整合运用规划、土地、财政、节能环保、科技创新、产业发展等多方面政策资源。

    今年暑假，由北向南、由东到西，从城市到乡村，从工程现场到行业企业，上海交大809支暑期社会实践团队超过7400人次分赴全国各地和部分境外国家、地区，开展不同形式的实践活动，在实践途中共同喊出“交大，为祖国点赞”，用青春的实际行动，用青春的实践征途，向祖国告白。

    上海交大建筑文化遗产保护国际研究中心主任、建筑遗产保护专家曹永康是2019“拯救传统村落”中意联合工作营的指导老师。这支团队通过让学生深入中国和意大利农村，体验和尝试解决当下需要解决的“乡村振兴”难题。

    曹永康告诉记者，中国的传统村落正在消失，而意大利的传统村落早在30年前就开始消失，农村空心化问题比我国来得“更早、更猛烈”。曹永康所带领的团队，除了在意大利找寻解决方案外，还在浙江丽水的农村“实验”解决方案。

    去年，这支团队为浙江云和县崇头镇沙浦村——一个只剩下老人和残疾人的村落——设计了一整套振兴方案。方案中，从村头到村尾，依次排布了体验工作坊、旅游小径、临时民宿试点、护理中心、旅游中心、品牌民宿、农家体验区和自来水厂。

    “盘活现有乡村至关重要。”曹永康团队的成员以上海交大学子为主，还有来自C9高校（九校联盟——记者注）、港澳高校以及国外高校的学生，他们都必须具备建筑学、城乡规划学、社会学、人类学、农村经济管理类等国家重点学科大三及以上年级资质。

    上海交大学子青春告白的实践足迹甚至走到了“一带一路”沿线国家。今年学校已第二年启动“全球挑战计划”。该计划主要面向东南亚、非洲和中东欧等“一带一路”沿线国家和中国国内部分地区，项目周期为2~4周，由专业教师选定有社会现实意义的活动主题，并与当地合作伙伴共同组织有一定影响力的研究和实践活动。

    上海交大设计学院教授、副院长车生泉已经不是第一次带学生去保加利亚调研了。上海交大2016年就在这里成立了上海交大保加利亚中心，为那里的学子提供教育支持，为当地发展提供智库方案。

    在保加利亚，已开创3个交大与当地院校合作的国际项目，其中两个涉及“乡村振兴”。今年，6名交大师生和4名保加利亚当地学生一起研究当地的乡村文化，并针对由国家总理李克强揭牌的“中东欧16+1农业示范园区”进行初步分析，制定保加利亚版的工作方案。

    在2019年“全球挑战计划”中，上海交大师生分赴保加利亚、罗马尼亚、匈牙利等10余个国家和地区，围绕乡村振兴、公共健康、环境保护等重大问题进行调研和探究，结合当地实际需求，为社会问题的解决提供交大智慧和方案。
