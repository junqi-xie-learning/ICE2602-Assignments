腾讯研究：数字文化正推动中华文化全球传播 

2019年08月28日14:43  来源：人民网-重庆频道
 

　　人民网重庆8月28日电（陈琦、黄军）8月27日，在2019智博会系列活动――重庆数字出版年会暨西部数字内容产业年会筹备会上，腾讯研究院研究中心副主任孙怡发布《西部省区市数字文化及出海年度洞察》，这些洞察来自《2019中华数字文化出海年度观察报告》的研究发现。

　　腾讯研究院《2019中华数字文化出海年度观察报告》指出，当前科技发展打破文化传播的时空界限，网络与数字文化内容出海成为推动中华文化全球传播、提升中国全球文化竞争力的重要途径。中国正成长为全球网络与数字文化领域崛起的新力量，机遇与挑战并存。

　　在全球化浪潮和“一带一路”建设背景下，中国网络与数字文化出海取得了初步成绩，细分领域格局初现，未来全球化发展潜力巨大。报告主要关注中国数字文化内容出海，从现状洞察、战略价值、发展机遇、挑战和建议等维度，试图探寻如何进一步完善中华文化国际传播格局，提升中华文化传播力，增强中华文化影响力。

　　《2019中华数字文化出海年度观察报告》是国家社科重大项目“网络与数字时代增强中华文化全球影响力的实现途径研究（18ZDA312）”子课题三中的增强中华文化全球传播影响力的“文化+科技”实现途径研究成果之一。

(责编：盖纯、张t)
