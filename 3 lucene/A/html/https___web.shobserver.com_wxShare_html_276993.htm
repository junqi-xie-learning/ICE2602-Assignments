有你认识的吗？这11位老师和同学刚刚当选上海教育年度新闻人物！还有提名奖

教育在线 2020-08-06 15:55
来源：上观新闻 作者：徐瑞哲
授予代表上海为新中国70岁生日献礼的东华大学国庆70周年“奋进上海”彩车“花仙子”设计展演团队“上海教育年度新闻人物•特别团队奖”。

解放日报·上观新闻记者6日获悉，“2019上海教育年度新闻人物评选”活动顺利结束。经评选，新一批10位（组）“上海教育年度新闻人物”正式揭晓。他们分别是：中国福利会托儿所所长方玥，上海交通大学教授孔海南，上海市城市建设工程学校学生陆亦炜/上海市杨浦职业技术学校学生徐澳门，上海市电化教育馆馆长张治，华东师范大学第一附属中学教师陈明青，华东理工大学博士生张琦，上海海洋大学教授陈新军，华东师范大学教授季浏，上海市虹口实验学校校长胡培华，上海大学教授彭艳。（按姓氏笔画排序）

同时，上海市奉贤区青溪中学副校长包蓓姹，上海市育才中学副校长李元，上海交通大学辅导员汪雨申/同济大学辅导员李睿，同济大学教授杨贵庆，上海外国语大学附属外国语学校学生胡诗成，上海市晋元高级中学造血干细胞捐献志愿者团队，原宿管阿姨、现在读研究生原梦圆，上海市工商外国语学校教师谢永业，上海市科技艺术教育中心教师葛智伟等9位（组）获“2019上海教育年度新闻人物”提名奖。（按姓氏笔画排序）

评委会一致同意授予中国工程院院士、上海交通大学医学院附属瑞金医院终身教授王振义“上海教育年度新闻人物•特别致敬奖”。授予代表上海为新中国70岁生日献礼的东华大学国庆70周年“奋进上海”彩车“花仙子”设计展演团队“上海教育年度新闻人物•特别团队奖”。

此外，《解放日报》徐瑞哲、许沁，《文汇报》樊丽萍、朱颖婕、姜澎，《新民晚报》马丹、陆梓华、张炯强，《新闻晨报》朱晓芳，上海教育电视台张贤贞\杨扬、方云\申宁、徐晓瑾，“第一教育”吴华，《上海教育》徐晶晶，上海教育新闻网刘时玉等17人，获得“2019上海教育年度新闻人物”记者报道奖；上海交通大学党委宣传部、华东师范大学党委宣传部、华东理工大学党委宣传部、上海海洋大学党委宣传部、上海大学党委宣传部、虹口区教育局等6家单位获“2019上海教育年度新闻人物”组织推荐奖。

上海教育年度新闻人物评选活动由上海教育报刊总社、上海教育电视台和上海市中小学幼儿教师奖励基金会于2008年共同发起举办。经过连续12年的评选，共有700多位个人或团队被推荐参加评选，120位个人或团队当选历年“上海教育年度新闻人物”。2019年，在市文明办、市教卫工作党委、市教委的指导下，上海教育报刊总社、上海教育电视台和上海市中小学幼儿教师奖励基金会继续共同主办“2019上海教育年度新闻人物”评选活动，并由人民日报、新华社、光明日报、中国青年报、中国教育报、解放日报、文汇报、新民晚报、上海广播电视台、东方网、青年报、新闻晨报、劳动报、易班网、上海学习网、澎湃新闻社等作为媒体支持单位。

“上海教育年度新闻人物•特别致敬奖”

王振义院士

上海交通大学医学院附属瑞金医院终身教授

为无数患者点亮生命新希望

2019年，从医77年的中国工程院院士、国家最高科学技术奖获得者、被誉为全球“癌症诱导分化第一人”的上海瑞金医院王振义教授，荣获中国“最美医生”称号，获颁“庆祝中华人民共和国成立70周年纪念章”。王振义是白血病患者心目中的“药神”，他和团队千辛万苦研制出有效药物，将无数患者从死亡线上拉回；他是人们心中的大医，一心救人，对荣誉淡然处之；他是学生心中的明灯，以身立教，倾情传授，只愿为国家和人民多做点事。他坚持每周四上午进行“开卷考试”——由学生提供疑难病例，他用一周时间搜索最新文献，并思考分析制成PPT，为无数患者点亮生命新希望。

上海教育年度新闻人物•特别团队奖

 

东华大学国庆70周年“奋进上海”彩车“花仙子”设计展演团队

东华大学师生

为国庆70周年“奋进上海”彩车设计“花仙子”

2019年10月1日，庆祝中华人民共和国成立70周年群众游行在北京天安门广场举行。其中，“奋进上海”彩车上有7位温婉佳人，她们身着以白玉兰为主要元素的海派旗袍，面带微笑，手执团扇，代表上海为祖国母亲献礼。这些都是由东华大学师生组成的设计团队所负责。从5月接到任务起，团队就积极参加到国庆彩车演出的“花仙子”服装设计、制作以及游行活动的展演工作中，为“奋进上海”彩车在国庆群众游行活动中的亮相提供保障。这既是一次高校优势和特色学科的惊艳亮相，也是学校师生厚植爱国主义精神、不断追求卓越的精神展示。

 

 

 

10位（组）“2019上海教育年度新闻人物”

 

方  玥

中国福利会托儿所所长

（基础教育组）

婴幼儿托育的实践者、探索者、引领者

深耕幼教事业近30年，方玥所长全身心投入于学前教育尤其是0-3岁婴幼儿的教育实践与探索，倡导不教而教，坚持融合教育，努力把中福会托儿所优质教育资源通过出版物、音视频惠及更多人群。2019年主编《上海市0-3岁婴幼儿家庭科学育儿指导手册•相伴成长》一书，为全市家庭提供科学育儿指导。2017年，国家卫健委委托上海市人口早期发展协会开展了《托育服务机构婴幼儿保育教育指导纲要》课题研究工作，方玥作为课题重要负责人，参与了课题研究和报告撰写工作。目前，该课题已经结题，课题成果已被国家卫健委采纳，对于规范全国托育服务机构的婴幼儿保育教育工作发挥了重要的作用。

 

孔海南

上海交通大学教授

（高等教育组）

十余载如一日，扎根银苍玉洱畔

遥远的云南大理洱海是孔海南工作的一线，是他用心守护的一片水域。他主持的项目所取得的成果——富营养化初期湖泊（洱海）水污染控制与生态修复关键技术，被称为全国湖泊治理的生态样本，已经被列为水专项重大标志性成果。到2019年为止，洱海项目“十二五”期间最后一个课题通过国家正式验收，国家水专项洱海项目历时13年，顺利结题。2019年孔海南捐出毕生积蓄200万元成立“上海交大洱海保护人才教育基金”。“老人与海”故事的背后，集聚的是一个个年轻的身影，一趟趟奔驰的列车，是一瓶瓶藻华的水样，一道道山川和河脊。

 

 

陆亦炜\徐澳门

上海市城市建设工程学校学生\上海市杨浦职业技术学校学生

（职业教育组）

勇夺世界技能大赛金牌的上海小伙

在2019年第45届俄罗斯喀山世界技能大赛上，来自上海的选手陆亦炜以他出色的表现获得花艺项目的金牌。而作为上届大赛同一项目金牌选手杨山巍的陪练，徐澳门也终于实现了自己4年多的梦想，获得车身修理项目的冠军。两位上海小伙为国争光、为沪增彩，完成了青年匠人攀登技能巅峰的梦想，激励着广大青年走上技能成才、技能报国之路，更为上海在2021年举办第46届世界技能大赛，进一步在全社会弘扬精益求精的工匠精神，努力将大赛办成一届富有新意、影响广泛的世界技能大赛，起到了积极的榜样示范作用。


张  治

上海市电化教育馆馆长  

（基础教育组）

面向教育现代化，推动教育信息化发展和创新

张治馆长深耕教育信息化领域多年，以推动教育信息化深度发展和创新为己任。由他带领的市电化教育馆作为牵头单位获颁2019年度大数据联合创新实验室，这是本市基础教育领域唯一的一所实验室，也为加快实现数据驱动下的大规模因材施教，进一步促进数据开放共享，挖掘教育数据价值奠定了坚实的基础。在2019世界人工智能大会上，市电教馆携手16家教育信息化企业发布《人工智能助力教育健康发展倡议书》，为广大青少年营造健康、文明、有序、安全的网络空间和学习环境，共筑教育现代化美好未来。

 

陈明青

华东师范大学第一附属中学教师

（基础教育组）

引领学生“读懂中国”的高中思政教师

2019年3月18日，习近平总书记主持召开学校思想政治理论课教师座谈会，华东师范大学第一附属中学政治特级教师陈明青是发言者中唯一的高中思政教师。从教21年以来，陈明青始终坚信，思想政治课是传递党和国家意识形态的主渠道，是学生成长成才的“关键一课”，讲课既要直面学生关切，也要有情感的注入。她立足马克思主义理论学科前沿，不断践行“真学”“真信”“真懂”“真用”，同时准确寻求有效教育资源，让学生们在鲜活的社会实践中“读懂中国”。她坚持做好学生的领路人，让思政课变得有血有肉，连续十几年被学生评为“最受学生欢迎的老师”。

 

张 琦

华东理工大学博士生

（学生组）

用科研报国之心描绘青春梦想

从2018年度上海市青年五四奖章标兵（该奖项设立以来首位学生代表）、第六届“上海市青少年科技创新市长奖”，到第十四届中国大学生年度人物（上海唯一）、2019年全国向上向善好青年，华东理工大学博士研究生张琦收获了多项重量级荣誉。他连续4年获得博士生国家奖学金，并将受邀参加2020年3月美国化学会年会进行20分钟口头报告。读博期间，张琦在费林加诺贝尔奖科学家联合研究中心进行人工分子机器和超分子化学方面的研究。面对失败，另辟蹊径；创新不止，勇立潮头。张琦助力上海科创中心建设，为“强国一代”树立了榜样。

 

陈新军

上海海洋大学教授

（高等教育组）

20年刻苦攻坚，为国家海洋战略研究鱿钓资源

陈新军教授在远洋鱿钓资源与渔场的开拓和我国远洋鱿钓渔业的发展中做出了重要的贡献，被誉为国内远洋鱿钓行业的科技领军人物。作为项目负责人和技术骨干，先后主持并完成国家级和省部级科技攻关课题、农业部公海渔业资源开发等重大项目，及企业委托课题共近40项，获得了重要研究成果，为我国在国际公海海域获取更多的海洋生物资源和优质水产品在科技上做出保障，并获得国际同行的高度认可。作为一个海洋科技工作者，他不断探索新技术，干成事情、拿出数据，更好地助力中国渔业企业，发出更大的中国声音。

 

季  浏

华东师范大学教授

（高等教育组）

“中国健康体育课程模式”提出者

针对青少年儿童身心健康水平持续下降的问题，季浏提出了“中国健康体育课程模式”，这是基于国家《课程标准》构建的中国第一个体育课程模式。该模式目前在全国各实验校推广，有效解决学校体育三大问题，促进学生身心健康全面发展，带动了上千所学校改变落后的体育教学理念，引领了上万名体育教师学习先进的教学模式，使上千万学生受益，有效探索出推动健康中国和体育强国建设的新教育模式。

 

胡培华

上海市虹口实验学校校长

（基础教育组）

构建安全教育体系，呵护每一个可贵的生命

自2008年建校以来，在胡培华校长的带领下，虹口实验学校的安全教育体系日趋完善。学校植安全教育于日常课程之中。如今，虹口实验学校已完成了走进消防体验馆、心理健康教育、生命教育与生涯规划、国家安全教育、第一反应等五大安全教育课程的构建，形成开学第一课、消防应急逃生疏散演练、少年警校、走进博物馆、假期安全教育讲座五大安全主题活动，实现了安全教育全覆盖。2019年9月，教育部印发《关于进一步加强义务教育学校校园安全防范设施建设的通知》。这对于上海市虹口实验学校校长胡培华来说，更是对学校将坚持了十余年的安全教育的一种激励。

 

彭  艳

上海大学教授

（高等教育组）

心系深蓝海洋强国梦的无人艇领军人

上海大学无人艇工程研究院院长、海洋智能无人系统装备教育部工程研究中心常务副主任彭艳教授围绕海洋强国战略，致力于海洋智能无人艇装备、海洋传感器等的研究工作。她率领的无人艇团队用9年的时间，研制“精海”9个系列无人艇，探测南海岛礁海域、南极罗斯海、东海岛礁，成功完成东海“桑吉号”海上撞船事故探测和处置。获国家技术发明二等奖、上海市科技进步一等奖等十余奖项。在实现海洋强国梦的伟大征程中，彭艳教授和无人艇青年团队不忘初心，砥砺前行，彰显巾帼新风采，建功新时代，做海洋强国的建设者。
