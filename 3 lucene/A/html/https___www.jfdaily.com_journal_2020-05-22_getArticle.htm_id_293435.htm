首场“委员通道”举行，政协委员通过网络视频接受媒体采访

“新基建”提升中国经济韧性活力 正在服务全球造福人类

2020年05月22日   03： 要闻   稿件来源：解放日报  
■本报记者 张骏 孟群舒 吴頔 谢飞君 顾杰 王闲乐

昨天下午，全国政协十三届三次会议首场“委员通道”采访活动在人民大会堂新闻发布厅举行，部分全国政协委员通过网络视频方式接受媒体采访。

中医药在抗疫中作用显著

今年2月疫情最严峻的时刻，全国政协常委、中国工程院副院长、中国医学科学院院长王辰赴武汉一线参加抗疫。在谈到抗疫为何能取得显著成效时，王辰从文化、体制、国力三个角度进行了解答。

王辰说，中华民族具有尊重生命、团结一心、共克时艰的优秀文化，医务人员展现了专业水准和高尚的职业精神，为应对疫情发挥了重要作用。同时，我们国家的体制决定了在遇到重大困难时，我们能凝聚思想、充分调动全社会的力量来面对挑战。“最后，是我们的国力。”王辰说，产业链完整、资源调配有力，充分体现了我们作为一个大国的国力，为取得抗疫重大胜利奠定了坚实基础。

王辰表示，这次疫情启示我们，今后应更加重视医学教育、加大医学研究投入、促进“医”和“防”融合。

全国政协委员、中国中医科学院首席研究员王阶说，全世界都没有应对新冠肺炎的特效药，但中国还有传统的中医药。实践证明，中医药在临床救治等方面确实发挥了很大作用。王阶说，到3月30日，金银潭医院南区140例痊愈患者中，88例是纯中医治疗，疗效显著。“我们相信，中医药只要能够传承精华、守正创新，一定可以在临床救治中发挥作用，发扬光大。”

最后一颗“北斗”6月发射

全国政协委员、中国北斗卫星导航系统工程总设计师杨长风在回答记者提问时说，今年是北斗全球系统建设的收官之年，最后一颗组网卫星已经运抵西昌卫星发射中心，计划6月择机发射，届时北斗全球星座的布局将圆满完成。

杨长风介绍，目前，中国70%以上入网的智能手机提供北斗服务，全世界半数以上的国家在使用北斗系统。中国的北斗正在服务全球、造福人类。

全国政协委员、佳都科技集团董事长刘伟分享了一个广州基层社区建立智慧社区防护平台，科技抗疫的故事。刘伟说，在信息时代，以人工智能、5G技术等为代表的“新基建”能力，会极大地推动中国经济的发展。相信随着“新基建”、新技术的不断提升，中国经济会更有韧性和活力。

脱贫攻坚有信心如期完成

今年，不少大学生在求职时遇到了困难。全国政协委员、上海交通大学党委书记杨振斌坦言，受疫情影响，应届生就业压力确实很大，国家和社会都在帮忙解决就业问题。大学生参军入伍、公务员招录、研究生招生、“三支一扶”等项目较大幅度扩招，教育部还组织了48所高校对口支援湖北省高校的就业工作。杨振斌向用人单位呼吁，利用这个机会提前进行人才筹备，为企业发展打下更好的人才基础。

近日，“悬崖村”昭觉县阿土列尔村84户贫困户告别了悬崖爬梯，搬进了县城新居，凉山脱贫攻坚最难啃的“硬骨头”终于被啃下。全国政协委员、四川凉山州人大常委会主任、党组书记达久木甲说，他对凉山实现如期脱贫目标充满信心。

“脱贫攻坚有多难，扶贫干部就有多拼。”达久木甲说，凉山州紧扣脱贫攻坚排兵布阵，调兵遣将，选派上万名优秀干部下沉到农村基层一线。五年来，在脱贫攻坚战场上，有151名干部负伤，有23名干部献出了宝贵的生命。
