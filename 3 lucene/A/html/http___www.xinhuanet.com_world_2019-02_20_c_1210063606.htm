澳大利亚新南威尔士大学与上海交通大学庆祝战略合作十周年
2019-02-20 10:14:31 来源： 新华网

　　新华网悉尼2月20日电（记者 郝亚琳）当地时间19日下午，悉尼古香古色的中国花园——谊园里，澳大利亚新南威尔士大学与上海交通大学共同举行活动，庆祝双方开展战略合作暨孔子学院成立十周年。

　　2009年，两校合作建立了新南威尔士大学孔子学院。十年来，双方在科研项目、学生交流、高层互访、教育管理等多个领域密切合作，涉及工、商、文、法、医等多个学科。双方共同建立了联合创新中心和中澳神经认知中心。50余名学者获得了合作研究基金的支持，发表了218篇合作出版物。超过300名新南威尔士大学学生前往上海交通大学交流学习。

　　十年来，新南威尔士大学孔子学院与25个本地机构建立了合作关系，举办了200余场交流活动，参与人数约3万人，为推动中文教育和中国文化传播作出了贡献。

　　当天的庆典上，新南威尔士大学校长伊恩·雅各布斯表示，该校孔子学院在传播中国文化、融入大学、连接社会等方面作出了突出成就和贡献，希望两所高校能够以十周年为新的起点，让友谊协作之火愈燃愈亮。

　　上海交通大学副校长徐学敏表示，两所高校在各领域的合作取得了长足进步，这样的友好合作关系还将继续传承、不断创新。

　　新南威尔士大学孔子学院中方院长奚俊芳表示，希望孔子学院在新的十年里继续做两校合作的枢纽，推动澳大利亚中文教育的发展。

　　当日，正值元宵佳节，来宾们还一起猜灯谜，赏园林美景，互致新春祝福。

　　中国驻澳大利亚大使馆教育公参宗瓦、中国驻悉尼总领馆教育参赞牛文起，以及当地学界商界知名人士等100余人出席活动。
