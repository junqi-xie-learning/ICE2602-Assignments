AI高端人才如何培养？听听麻省理工学院、新加坡南洋理工大学等高校有何高招

创新之城 2018-09-17 20:49
来源：上观新闻 作者：黄海华
“人工智能是非常耗能的，阿法狗下一盘棋，需要耗掉十吨煤，我们人下一盘棋喝一杯牛奶就行了。”

人工智能时代的高端人才培养如何开展？跨国界、跨学科的人工智能学术研究怎样大有作为？ 9月17日下午，2018世界人工智能大会全球知名高校校长圆桌会议在上海交通大学举行。来自美国麻省理工学院、新加坡南洋理工大学、香港中文大学、上海交通大学等国内外人工智能领域知名高校校长，做了深入探讨与交流。

 

“作为大学掌门人，确实觉得有点恐慌”

 

西安交通大学校长王树国坦言，以人工智能为代表的第四次产业革命影响巨大，而且这一次产业革命可能要有很长的周期，不断渗透到社会各方面，使得社会处于扁平状态，“作为大学的掌门人，确实觉得有点恐慌。”他提出，大学需要变革，现在很多的新成果不是产生于大学。大学现在跟着跑，还有点跟不上。我们需要反思和探寻，大学如何满足社会发展需求。大学的优势在于 基础研究和人才培养，建议大学要主动融入社会，开门办学，如果不参与其中，很难了解到什么才是真正有价值的科研项目。要主动和企业合作，开展校企联合实验室，并让学生参与其中。

 

说到校企合作，香港中文大学校长段崇智很有感触，他们和微软等企业合作，建设了重点实验室，研发了一些人工智能技术。“要把在企业工作的人才引进来做客座教授，刺激学生思维，同时也为学生提供了很好的实习平台。”

 

“我们每年有几十个项目的实习机会，大都由合作企业来提供。””新加坡南洋理工大学副校长林杉介绍，他们在新加坡有许多与企业合作的机会，比如与新加坡最大的电信公司就展开了合作。

 

阿法狗下盘棋，需耗十吨煤，人喝杯牛奶就行了

 

高校是培养人工智能人才的重镇，近年来多所高校谋划和布局了人工智能学科建设。

 

“人工智能成为了第四次产业革命的驱动力，第五次产业革命的‘种子’又会在哪里？这是需要大学来关注的。”上海交通大学校长、中国工程院院士林忠钦表示，以高校学术联盟的形式共同探讨人工智能的当下与未来，具有重要的战略眼光与现实意义。今年1月，学校整合校内相关院系、学科的研究力量，成立了人工智能研究院，并获批人工智能教育部重点实验室。“由于人工智能的发展，很多大学生现在倾向于选择计算机专业，但我们希望在学科建设中，人工智能可以带动其他学科的发展。”

 

“有不少企业来问我，今后5到10年，我们会培养多少人工智能人才？”复旦大学校长许宁生认为，人工智能的推进，不仅涉及科技，也关系到伦理、人文等社会科学，学校正在布局人工智能多学科教育。关于人工智能人才的培养，复旦大学提出“以类脑智能引领人工智能”，将在全球范围内展开相关合作，学校已经成立类脑智能研究院和芯片研究院，启动了大数据试验场，2个月前还成立了上海脑疾病中心。

 

浙江大学代表潘纲教授介绍，上周五学校成立了一个“双脑”计划，把神经医学和人工智能相结合，把人的认知和人工智能相结合。“高校研究应该更多地再往前看，即5到10年后，人工智能会怎么发展，这也是近期企业不太愿意去做的事。”人工智能涉及多学科，浙江大学专门有一批研究生名额给交叉学科，校长基金也会专门资助交叉学科。

 

 “人工智能是非常耗能的，阿法狗下一盘棋，需要耗掉十吨煤，我们人下一盘棋喝一杯牛奶就行了。”中国科学技术大学常务副校长潘建伟说，对大学来说，要针对人工智能两个问题提前布局。第一是降低能耗，第二是计算能力，量子力学奠定了现代信息技术的基础，中科大正在构建国家实验室，希望从量子计算着手为人工智能做好未来硬件的准备。

 

上海科技大学副校长印杰提到，未来哪些岗位会被替代，哪些不会被替代，这在培养人才时必须考虑。要培养学生终身学习的意识和继续学习的能力，比如让学科学的学生研修一些非逻辑的艺术课程。人与人的感情沟通是难以被替代的，因此也要加强道德和伦理教育。

 

“我们会进行竞争，但是不会闭门造车”

 

国外高校的人工智能研究有什么可以值得借鉴？

 

“我们会进行竞争，但是不会关上大门，不会闭门造车。” 美国麻省理工学院荣誉校长埃里克·格里姆森说。“深度学习非常惊人，但接下来会怎么样？”埃里克·格里姆森认为，深度学习需要有成千上百亿信息和云的输入才行，比如说你是一个家长，给一个孩子展示6个例子，他就能自己找到规律。麻省理工学院会继续在神经科学、认知科学、算法等方面做进一步基础研究，才能发展下一代人工智能技术。此外，还要搭建连接人工智能和工程科学的桥梁，才能把成果转化到其他领域。

 

埃里克·格里姆森建议，可以在大学之间以及大学与企业之间交换学生，不同的大学可以共享数据，创造开源的互惠性平台，把一个开源的算法放在一个公共的地方，就能创造出更多的算法。

 

新加坡南洋理工大学副校长林杉说，人工智能不是一个纵向的领域，而是一个横跨多行业的领域，只有和其他领域结合才有意义，比如人工智能+健康医疗。他认为，教育才是立校根本。18个月前，学校成立了一个数据研究中心，把各种教学资源整合到一起，希望把行业中的浪潮能够带进大学，引入到本科生教育，让学生学有所得。
