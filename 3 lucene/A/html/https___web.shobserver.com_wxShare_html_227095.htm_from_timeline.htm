上海教授与欧洲同道共同探讨微创外科疫情应对经验：重组资源，多重防护

医声医事 2020-03-21 10:09
来源：上观新闻 作者：黄杨子
据悉，欧洲医院目前已经在外部搭建帐篷用于分诊，内部病房等医疗设施则进行了结构调整与改造，床位进行了资源的重组和重分布，外科手术仅限于急诊手术和肿瘤限期手术。

近日，上海交通大学医学院附属瑞金医院郑民华教授与法国教授Abe Fingerhut和意大利教授Luigi Boni进行了积极的沟通与交流，就微创外科应对疫情的相关经验进行深入探讨，三位教授的专业性意见已被美国外科学年鉴（Annals of Surgery）接收，并作为社论在线发表。

郑民华指出，在临床医疗方面，疫情对外科系统各专业领域产生了较大的影响，一方面由于各个城市抗击疫情需求，执行了较严格的隔绝措施，减少了人员的异地流动，客观上减少了平诊病人的手术量；另一方面，出于疫情防控的要求，许多外科手术的开展仅限于急诊手术和肿瘤病人的限期手术。因此，如何在确保疫情防控的前提下，不影响正常医疗秩序，特别是外科手术秩序的正常恢复与开展，而作为外科主要技术形式的微创外科，因其特有的腹腔镜技术及人工气腹等技术因素，在疫情防护中更有其独特性。

两位欧洲微创外科的领军人物也将目前在欧洲，特别是意大利与法国在疫情下微创外科开展情况以及经验做了反馈。据悉，欧洲医院目前已经在外部搭建帐篷用于分诊，内部病房等医疗设施则进行了结构调整与改造，床位进行了资源的重组和重分布，外科手术仅限于急诊手术和肿瘤限期手术。在此形势下，对外科病人和参与手术的各类医疗人员而言，相关的防护和管理措施显得尤为重要。

三位来自中欧两地的微创外科专家，结合腹腔镜手术的技术特点，形成了针对疫情时期微创外科在临床实践和病毒防护方面的一系列推荐意见。主要包括：一般防护、气腹相关气溶胶的防护、术中人工气腹的管理、术中手术操作要点、术后手术室和相关设备的处理、医疗机构内手术区域的分区管理、手术室内相关人员的职业防护、疫情后手术病例的术前肺炎相关筛查的常态化等。

据悉，作为中华医学会外科学分会腹腔镜与内镜外科学组组长，郑民华在今年2月已率先通过对疫情下微创外科的临床实践和临床价值进行深刻思考，并对疫情发展至现阶段的微创外科手术的安全开展与学科发展进行了展望，此次与国外同道共同分享经验，引起了广泛关注。
