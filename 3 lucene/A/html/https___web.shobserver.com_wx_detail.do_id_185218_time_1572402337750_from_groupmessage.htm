这个少年宫常客如今成功投入诺奖项目，沈南鹏感觉创投者与科学家的气质高度相似

创新之城 2019-10-30 10:21
来源：上观新闻 作者：徐瑞哲
他们说参加这样的论坛就像参加朋友的婚礼，又能遇见好多过去四五十年来的老朋友。

今天，第二届世界顶尖科学家论坛的第二天。主论坛“未来国际大科学”论坛上来了一位非科学家，第二个发言。不过，他开场白就与科学家很好地“套近乎”。

解放日报·上观新闻记者了解到，这位红杉资本全球执行合伙人沈南鹏，其实也是世界顶尖科学家论坛的全球战略伙伴。这位上海人坦言，自己原本也可能成为“在座的一员”。

“30多年前，我的每个周末，都在市少年宫度过，”沈南鹏说，那些数学和物理公式深刻烙印在科学少年的记忆里，自己也曾梦想着未来成为一名科学家，“但很遗憾，最近20多年来，我做的是创业与投资工作，但科学启蒙与兴趣始终影响着我的职业生涯。”

沈南鹏忆起母校上海交通大学一次校友大聚会，一位已成为美国国家科学院院士的老同学，长期从事天体物理研究，告诉沈南鹏说：“你和我，我们其实有很多共同点。”沈南鹏又想起美国哲学家杜威的一句名言：科学的每一项巨大成就，都是以大胆的幻想为出发点的。他认为，创投与科研具有相当接近的气质特征，走前人未走过的道路，试验成败与产品试错何其相似。

有时候，“诺奖”也是这样“投入”的。2016年时，沈南鹏团队收到其基金有限合伙人的感谢信，感谢他们参与对诺奖大热项目引力波观测的漫长而有力支持，终于证明爱因斯坦百年前的大胆猜想和推测。同样，几周前刚刚揭晓的诺贝尔经济学奖得主，也有红杉相关基金作为其背后的长期资助人，用科学方法解出减贫方法。

“我今天参加世界顶尖科学家论坛，在会场碰到了好几位科学家在交流，”沈南鹏发现，他们说参加这样的论坛就像参加朋友的婚礼，又能遇见好多过去四五十年来的老朋友。他认为，从生命科学到信息技术领域，科学家取得的成就比以往任何时代都能产生更大的影响，从科学到技术再到产品，有了更多更好的方法去实现。而今不论是人工智能、机器人还是量子力学等，它们都是跨学科研究的伟大进步。“我想，这就是办这个论坛的题中之义，让全球科学家擦出思想火花与全新灵感。”

沈南鹏特别期待明日的莫比乌斯论坛，这是一个关注未来科学的畅想型论坛。作为曾经的科学探索人，他表示相信不论科学过程如何艰难，但科学人在其中终究是快乐地工作着，“享受科学，憧憬美好。”
