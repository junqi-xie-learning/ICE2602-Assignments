百年来，心信而行远……他们都成了陈望道的“粉丝”：“我以青春践《宣言》”

教育在线 2020-07-02 19:03
来源：上观新闻 作者：徐瑞哲
我是展示馆的第一批学生讲解员。从2018年5月开始承担讲解工作，不知不觉已经在志愿者岗位上两年多了。

“一周前是端午节，吃粽子的时候会想起陈望道的故事。”做了17年辅导员的“最美高校辅导员”李睿，刚刚到同济大学党委宣传部担任工作。今年是语言学家、教育家陈望道翻译《共产党宣言》首个中文全译本100周年，他和又一批青年师生一起，在“七一”之际走进五角场外、国福路上的陈望道旧居，参观《共产党宣言》展示馆，亲眼见到了陈老用过的笔墨和砚台，对《共产党宣言》的翻译故事感触更深一层。

百年来，心信而行远。他们都成了陈望道的“粉丝”，直言“我以青春践《宣言》”。

【学以致用，“讲”“学”相长】

李睿等人遇上了“金牌讲解员”陆婷婷，她正好也是习近平总书记近日给复旦大学《共产党宣言》展示馆党员志愿服务队回信的队员之一。

“我是展示馆的第一批学生讲解员。从2018年5月开始承担讲解工作，不知不觉已经在志愿者岗位上两年多了。”这位复旦大学马克思主义学院2016级博士研究生说，“星火”服务队的志愿讲解过程就像是跨越时空聆听陈望道老校长谆谆教诲的过程。“我仿佛看到他在早春时节那条件简陋的义乌柴屋中，忘我投入地翻译《共产党宣言》，花了平时译书5倍的功夫，留下了错把墨汁当红糖蘸粽子吃的‘真理的味道’，这段历史深深打动着我。”

解放日报·上观新闻记者了解到，陆婷婷毕业后选择留校担当辅导员，同时也会留在“星火”，以青年教师的身份继续参与讲解。“届时我会邀请我的学生们到旧居参观，为他们讲述理想信念的故事。”她表示，总书记在回信中两次勉励我们“坚定理想信念”。“作为90后青年党员，传道者先明道信道，我学习马院开设的《共产党宣言》等马克思主义经典著作课程，结合‘四史’学习教育，为讲解做好充分的理论准备；同时，讲解工作又激励我进一步学好经典，学以致用、用以促学，可谓讲学相长。”

“2020年的上海中考作文题目是《有一种甜》，而在《宣言》翻译过程中，陈望道心无旁骛、唯见信仰，‘真理味道非常甜’的美谈流传至今。”同来展示馆参观的上海市学联驻会执行主席、华东师范大学河口海岸科学研究院2018级硕士生林聪泳，告诉解放日报·上观新闻记者，在他们华东师大志愿者总队组织下，师大园里也有“华实”校史宣讲队志愿者，不仅协助校史馆开展日常讲解及参观导览，也承担新生入学爱校荣校教育任务，还融入“四史”学习教育整理挖掘校史资料及校园故事。

【火线战疫，精神家园】

来到陈望道沪上旧居，不禁令上海交大医学院青年教师、研究员贺明回想起陈望道的老家。2018年夏，他代表上海市，去浙江金华参加“全国高校青年教师教学比赛”时，在市中心看到了陈望道老先生的塑像和介绍，才知道“真理的味道”的故事就发生在金华义乌。“当年他凭借一本日文版、一本英文版的《共产党宣言》，废寝忘食首译全本；而后，在上海对《共产党宣言》的初印及发行，使共产主义新理论以星火燎原之势席卷全中国，为中国共产党的创立发展奠定了思想理论基础。”

贺明表示，毛主席就曾说过，“有三本书特别深刻地铭记在我心中，使我树立起马克思主义的信仰”，其中之一即是陈望道翻译的《共产党宣言》。而《共产党宣言》毛泽东看了不下百遍，他说“每阅读一次，都有新的启发。”今年疫情之下，贺明非常欣慰地看到，交大医学院医护人员，“特别是年轻党员，很多也是我的学生，一夜成军投身抗疫最前线，全院先后有140人递交入党申请书，火线入党39人。”

同行进馆的华莹奇就是一位火线入党的青年医生。“复旦是我母校，多年前在我入学的第一天，老师们在这里领着我们面向希波克拉底誓词庄严宣誓：健康所系，性命相托。这誓言也成为我作为医生最初的信仰。”今春，这位上海市第一人民医院骨科副主任医师成为上海市第八批援鄂医疗队队员。

雷神山工作期间，华莹奇见到医疗队的党员们，从临时病房建设、严格院感防控到抢救重症患者等，总是冲在第一线，始终保持战斗力，“尤其是好几位90后青年党员，都是同龄人的楷模，并感动着我。”华莹奇坦陈，“那不就是我当年医学院入学誓词的信仰吗？我找到了精神家园。”今年3月，他在前线加入了中国共产党。

【“四史”教材，常读常新】

“我每年都会带着学生读《共产党宣言》，常读常新。”作为思政课一线教师，上海交通大学马克思主义学院副教授周凯也参访了旧居展馆。日常，他承担着毛泽东思想和中国特色社会主义理论体系概论、习近平新时代中国特色社会主义思想概论等课程的讲授，并将马克思主义贯穿在党史、新中国史、改革开放史、社会主义发展史的“四史”教学中。

周凯研究认为，“99年前，中国共产党建党之初，几乎一无所有，当时全国有200多个政党组织。然而，我们党可以用28年的时间，实现由小变大，由弱变强，取得了革命胜利，让苦难的中国换了人间。”他直言，“革命不是请客吃饭，闹革命是要掉脑袋的，可中国共产党人在战争年代冒着掉脑袋的危险、前赴后继、一往无前，这背后支撑他们的是什么？是物质奖励？是制度激励？是想要组织提拔？”他回答了一串设问：“都不是，是理想信念！”

2日下午，来自沪上高校及其附属医院的党员干部以及青年党员教师、学生、医务工作者代表，共赴《共产党宣言》展示馆现场参观学习，还参加了学习习近平总书记给《共产党宣言》展示馆党员志愿服务队重要回信精神座谈。

座谈中，市教卫工作党委书记沈炜透露，当前全系统1.38万个党组织和28万名党员，都在开展“四史”学习教育，“四史”成为坚定理想信念的“活教材”。“心有所信，方能行远。”他大声说：“来吧，上场了!”
