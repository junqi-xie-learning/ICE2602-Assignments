嫦娥落月“背”感骄傲，沪上髙校一起出力让“嫦四玉兔”强筋健骨、耳聪目明

教育在线 2019-01-06 12:33
来源：上观新闻 作者：徐瑞哲
从“超级风火轮”到光学仪器构件，如此轻质高强；“纺织手艺”编制一把伞，“传情达意”嫦娥鹊桥相会……

嫦娥首落月背，“背”感中国骄傲。2019年伊始，“嫦娥四号”探测器成功着陆，“玉兔二号”月球车在月球背面 “登月”，它们通过“鹊桥”中继星传回了世界第一张近距离拍摄的月背影像图，为世人揭开从地球上永远不得一见的月亮背面神秘纱帘。

在此次人类探测器首次月背软着陆，以及月背与地球首次中继通信的任务背后，也有沪上高校材料技术与工程的应用与支撑，让“嫦四玉兔”强筋健骨、耳聪目明。

从“超级风火轮”到光学仪器构件，如此轻质高强

 

“玉兔二号”月球车在月面上开展巡视，面临恶劣空间环境和复杂地形地貌。比如，“昼夜”超过300℃的温差可导致构件产生热应力及热变形；又如，月球车在行走时也要承受各种碰撞、挤压、摩擦、磨损等。“玉兔”直接面对人类从未探知的月球背面，其主要与月表接触的行走机构——棘爪，成为月球车开展月球移动探测的关键，也是难题之一。

在上海交通大学材料科学与工程学院复合材料研究所、金属基复合材料国家重点实验室，张荻教授、欧阳求保教授团队研制出由SiC颗粒增强铝基复合材料制成的行走棘爪，此次成功应用于“玉兔二号”月球车的轮子，堪称它的“超级风火轮”。SiC颗粒增强铝基复合材料具有重量轻、高刚度、高耐磨、耐冲击、尺寸稳定等特点，能够承受月球表面各种苛刻的服役工况条件，为“玉兔二号”漫步月背保驾护航。

上海交大材料学家为“嫦娥四号”构建的“金刚不坏之身”不止于此。记者了解到，团队建立起国内首个航天用铝基复合材料技术标准，研制的这种高性能复合材料还用于“嫦娥四号”4个关键载荷，包括激光测距仪、三维成像仪、红外光谱仪等星载光学仪器中的镜筒、光学底板、框架等12种关键构件。如星载光学仪器，必须具有很高的分辨率和稳定的光学性能，传统材料难以满足其要求，而他们的新型复合材料则同时实现了载荷结构轻量化、不变形、尺寸稳定的需求。

“纺织手艺”编制一把伞，“传情达意”嫦娥鹊桥相会

 

不论“嫦娥月兔”，除了强筋健骨，还要耳聪目明。众所周知，月球的自转和公转周期相同，造成它从来只可能是正面对着地球，地球人自古以来都无法直接望见它的背面。同样，无线电通信在太空里总是“直来直去”，找不到“反射物”也打不了弯。因此在月背着陆的“嫦娥四号”，只能通过绕道月球远端的中继卫星和地球保持联络，这正是中继星取名“鹊桥”的由来。

其实，早在2018年5月，“鹊桥”号就已经先期发射到预定位置，在地面和月球背面之间架起了一座通信的桥梁，等着“嫦娥”来相会。但少有人知的是，两者靠什么来“传情达意”？原来，一把“伞”当作空中鹊桥，“嫦娥”才不会在月宫失联。

记者了解到，“鹊桥”号上最重要的伞状通信天线，使用了东华大学科研团队参与研制的星载天线金属网。有意思的是，这把“伞”正是利用东华最擅长的纺织高科技“编织”出来的。团队负责人、东华纺织学院教授陈南梁介绍，这种可展开天线在形式上正如小伞，卫星发射时它的展开结构是收起来的，卫星进入轨道以后再打开。团队骨干蒋金华表示，天线网所用材料表面镀金，里面是钼丝，在太空中既轻便又稳定，强度也比较好。有了好材料，还用了纺织经编技术，“编”出来的金属网既强又韧，受得住发射飞行外力“折腾”，同时收放自如，可谓“刚柔并济”。
