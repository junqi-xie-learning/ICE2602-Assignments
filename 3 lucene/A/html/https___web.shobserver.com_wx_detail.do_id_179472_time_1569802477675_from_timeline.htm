大学教授黄浦江边“抱笔”写大字，这幅狂草要“飞上天”才看得全

见识录 2019-09-30 08:14
来源：上观新闻 作者：孟雨涵
9月29日下午，“书写上海新高度——徐庆华巨幅狂草现场创作”活动在浦东滨江大道举行。

9月29日下午，上海交通大学媒体与传播学院副教授徐庆华在浦东滨江大道创作狂草苏轼词《念奴娇·赤壁怀古》，将书法艺术融入城市精神，300平方米巨制成就书法史上的传奇。

著名钢琴家解静娴现场弹奏，书法与音乐跨界合作演绎精彩。中外同学也挥笔参与书写，共同礼赞新时代，庆祝新中国成立70周年。

正文每个字约3平方米，草书的线条灵动飞扬，蕴蓄着澎湃的激情，带给观众强烈的视觉冲击力。

浦东滨江大道的广场上铺着长30米、宽10米的白色专用材料，徐庆华两手抱住巨型毛笔，蘸上特别调制的墨汁，一气呵成书写了苏轼词《念奴娇·赤壁怀古》。

在徐庆华挥毫书写的同时，著名旅德钢琴家解静娴博士现场演奏《长江之歌》与《黄河》，音乐与书法两种艺术样式形成跨界互动，传递着绵延古今、通江达海的情怀，共同为观众献上触动心灵的视听盛宴。

中外师生也挥笔参与书写，共同礼赞新时代，庆祝新中国成立70周年。

徐庆华巨幅狂草现场创作活动为2019浦东文化艺术节系列活动之一。

据介绍，浦东文化艺术节自2011年开始举办，今年的艺术节活动还包括“我和我的祖国”浦东新区庆祝新中国成立70周年大型情景歌会、全国十大优秀剧目展演、上海城市公园音乐周、“傅雷杯”首届全国文艺评论征文大赛等。
