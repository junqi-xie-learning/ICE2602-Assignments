让北斗更好地导航百姓生活——记全国模范退役军人、上海北斗导航创新研究院院长郁文贤

来源：新民晚报     记者：江跃中     作者：江跃中     编辑：唐闻宜     2019-07-28 10:21 | |

       前天（26日），从北京传来好消息：在此召开的全国退役军人工作会议上，上海北斗导航创新研究院院长、上海交通大学讲席教授郁文贤，荣获“全国模范退役军人”的称号。

郁文贤长期从事军民融合高新技术研究和产业化，率领研究团队在雷达目标识别、遥感信息处理、融合导航定位技术等方面，取得一系列实践创新研究成果，为我国国防关键技术发展、装备研制和产业化，为北斗更好地导航百姓生活，作出了突出贡献。他先后获得国家科技进步二等奖5项，部委级科技进步一等奖6项等。

岛上攻关近两年

“人生的快乐，很大程度上取决于我们拥有和实现什么样的梦想。”带着对科学技术追求的执著，怀着对国家强盛期待的抱负，郁文贤的求学深造和科技研究等，总是与“国防”两字紧密相连，而这也是他的梦想，快乐人生的梦想。

1981年9月，毕业于松江二中的郁文贤，考入国防科学技术大学无线电测控与数传专业，1985年7月毕业，取得学士学位。此后，他又先后攻读国防科学技术大学通信与电子系统专业硕士学位，国防科学技术大学通信与信息系统专业博士学位。在大学里，郁文贤刻苦学习，努力钻研。1987年夏天，学生们在老师的带领下，来到海军某舰队做实验项目，郁文贤大都是上夜班，天未黑就钻进机房，天蒙蒙亮时才出来，一刻不停地采集数据并调试算法，干了足足两个月。当舰队领导带着摄像记录设备来现场测试和考核时，郁文贤负责整个算法现场的联调和操作，最后取得了100%现场识别成功的成绩，大家欢呼相拥，郁文贤也非常激动。

图说：郁文贤（左二）先后主持和参与国家863计划、国防型号工程、国防预研、国防预研基金等30多个课题。张龙 摄

博士毕业后，郁文贤在国防科学技术大学正式服役和任教，历任学院总工、副院长和大学科研部副部长。他32岁就被破格晋升为正教授，这是部队对他的充分肯定，也激励他不断获取科研成果。郁文贤先后主持和参与国家863计划、国防型号工程、国防预研、国防预研基金等30多个课题。“十五”期间，郁文贤带领的团队，在我国3个舰队的4个观通站开展科研攻关，累计在海岛上的工作时间有近两年，完成了上万次的实际数据采集与识别实验，取得的研究成果及其研究经验是绝无仅有的。

在国防科学技术大学的27年里，郁文贤和他的团队，研发了我国首台套雷达目标自动识别系统，研制了我国第一套空间目标监视战略相控阵雷达目标识别装备等，多次立功受奖。

边走路边想方法

2008年，已经是正师职领导的郁文贤，从国防科技大学转业，他放弃组织行政移交安排，选择到科技创新创业一线工作，并参加了全球院长竞聘，被上海交通大学录用，担任电子信息与电气工程学院院长，由此也开始了作为高新技术背景的退役军人创新创业之旅。

中国北斗卫星导航系统（BDS），是继美国全球定位系统（GPS）、俄罗斯格洛纳斯卫星导航系统（GLONASS）后，第3个成熟的卫星导航系统。北斗卫星导航系统的建设与发展，将提高我国的国际地位，展示综合国力和技术实力，对提升中国能力和推动全球化战略意义重大。郁文贤瞄准北斗卫星导航系统的应用，开展重点研发。

2011年，郁文贤牵头组织论证“北斗卫星导航长三角示范工程”国家重大专项项目，在此后的近3年里，他带领总体论证专家组和十几家科研企业单位做顶层设计方案，经常加班加点，有时候通宵准备各类汇报方案，不少年轻人都撑不住了，而郁文贤还专心致志、精神抖擞地和大家讨论相关技术问题。该工程成为国内第一个通过项目验收并取得重大产业发展突破的全国区域示范工程项目。

郁文贤创建了“北斗导航与位置服务”部市共建重点实验室，致力于解决北斗导航与位置服务中的核心技术问题，技术发展成效显著。针对城市环境存在信号遮挡、多径反射等，导致卫星导航定位精度严重恶化的问题，他主持开发了结合运动情景感知的多源信号融合定位与真实场景库测试验证方法，推动企业产品提升复杂环境下的定位可靠性，典型场景下定位精度和稳定性与国际同期最高水平相当，同时，智能化低功耗多模融合定位终端率先在司法社区矫正、老人儿童关爱等领域推广应用。相关成果获得了2016年度上海市科技进步特等奖。

科研攻关屡屡成功，郁文贤靠的是执著、专注与毅力，在大家的眼里，他不仅是一位科学家，也是一个“铁人”。他在负责一个项目的算法软件技术定型和材料总结时，曾3天3夜没有合眼，到了第4天觉得头有点晕，回宿舍睡了两个小时后，又赶回来加班。在作为算法软件组组长负责一个很重要项目的关键技术研发后，郁文贤整整3个多月，基本上都是在实验室度过的。走路、吃饭和睡觉做梦都和项目有关，最终解决该项目研发的方法，也是他走在路上想起，并立刻返回实验室修改调试才告捷的。他的同事、上海西虹桥导航技术有限公司市场营销部经理朱亚夫告诉记者，郁文贤经常在凌晨一二时，还在和他们通过微信讨论技术问题。

图说：郁文贤（中）创建了“北斗导航与位置服务”部市共建重点实验室，致力于解决北斗导航与位置服务中的核心技术问题，技术发展成效显著。张龙 摄

就是生病住院，郁文贤也不闲着。2014年冬天，他在住院期间，向院方提出建议，能不能通过穿戴式传感器和空间大数据分析，来服务病人康复。目前，这个合作项目双方一直在推进中。

标兵教授做产业

作为退役军人，一名教授，郁文贤不但能在学校成为标兵教授，还能成功走向产业一线，这对很多人来说简直是不可思议的事情，但他做到了。

2011年，上海交通大学开始规划论证建设北斗西虹桥基地，郁文贤发挥高新技术专家型、领军型退役军人的创业优势，率领团队和大家埋头苦干，克服了一个又一个的困难。2013年底一期开园，郁文贤带动和集聚了多个退役军人创业团队入住园区，打造出主题鲜明的、高学历高素质退役军人创业模式的典范，现在有的创业团队吸纳的退役军人就业者高达70%。为推动产学研合作发展，郁文贤同时出任上海北斗导航创新研究院院长、上海西虹桥导航产业发展有限公司董事长。

郁文贤把北斗西虹桥基地建成了国内北斗产业第一园，形成了产业链集群创新与集聚发展的良好生态环境，累计集聚近200家导航产业相关企业，孵化军队转业军人创业运营企业6家，园区从业人数2800人，并形成了完整的北斗导航与位置服务产业链。基地企业综合经济效益也显著提升，产值年均增长率超过50%。

今年5月，为更好地促进退役军人创业就业与产业合作发展，郁文贤和众多退役军人企业家一起共同组织发起成立了上海高新产业融创服务战略联盟，为广大退役军人创业就业搭建各类服务平台、空间和条件环境，目前首批联盟成员有41家，其中退役军人创建、运营的企业占75%，预计2019年联盟成员达到百家。

本报记者 江跃中
