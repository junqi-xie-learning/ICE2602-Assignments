喜看“这么多青年医生表现优秀” “听诊大师赛”人才涌现

东方网 刘轶琳
2019年6月14日 18:36:02

　　东方网记者刘轶琳6月14日报道：今天下午，首届交医心之音听诊大师赛在上海交通大学医学院举行。来自瑞金医院、仁济医院等13家上海交大医学院附属医院的青年医生参赛队，历经院内选拔、晋级赛、半决赛和总决赛的层层考验，最终附属上海儿童医学中心脱颖而出，摘得首届听诊大师赛的赛季总冠军，附属胸科医院与附属瑞金医院凭借出色的表现获得二等奖，附属国际和平妇幼保健院和附属同仁医院获得三等奖，另有8支参赛队获得优胜奖。

　　据悉，本次心之音听诊大师赛由上海交通大学医学院主办，并得到了3M中国的大力支持，面向40岁以下的在职临床医生，以听诊为主题，希望呼吁临床医生重视心音听诊在临床诊断中不可替代的作用，同时帮助提升中青年医生的听诊技能。“上海交通大学医学院一直以来致力于为医学界输送、培养更多卓越医学创新人才。本次大赛的举办也是希望帮助临床医生不断磨练与强化基本技能。”上海交通大学医学院副院长吴正一表示，“我们很高兴能看到这么多青年医生的优秀表现，期待他们日后成为这个行业领域的领头羊，以精湛的技术更好地服务于广大患者。”

　　本次大赛自启动以来便获得了诸多医院的广泛支持与关注。预热阶段，13家医院通过院内选拔，组成了“3名选手+1名导师+10名后援团”的队伍进入晋级赛。在3场晋级赛、1场半决赛的精彩PK之后，最终三支队伍来到总决赛，现场通过电子听诊器进行专业心音听诊与病理分析，角逐首届大赛的冠军；来自交大医学院附属医院的知名教授担任评委，现场点评选手表现，并为参赛医生提供指导。比赛同时通过丁香播咖进行了网络直播。

　　临床上，很多生命因为听诊转危为安。

　　有病查不出？如何是好？老年胸痛患者被怀疑患有冠心病，而冠脉CT以及各方面的心肌缺血的检查均无异样。听诊后，发现患者主动脉瓣有明显的杂音，进而采取了对应的检查。因为听诊，患者疾病诊断得以顺利进行。

　　看似小毛病，真的没有大问题？仅仅感冒发烧的儿童患者，对疾病不以为意。听诊后却发现肺部有音，及时进行了治疗，并应用抗生素。因为听诊，患者的治疗没有被耽误。

　　久治未愈，漏洞出在哪里？被诊断为肺部感染的男性患者，应用抗菌药三天后仍无好转，听诊后发现主动脉瓣区有杂音，怀疑并非患有肺部感染，因此对患者重新进行了疾病检查。因为听诊，及时挽救了临床误诊的患者。

　　在国家贫困的山区中，医院资源紧缺且落后，当地卫生所甚至没有心电图仪器。这种情况下，一个听诊器和一位专业的医生便可以为患者进行疾病的初步诊断。因为听诊，一些特殊环境下的疾病诊断得以实现。
