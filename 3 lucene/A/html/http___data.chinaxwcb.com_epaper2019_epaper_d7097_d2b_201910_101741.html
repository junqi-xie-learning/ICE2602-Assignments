沪上，一场读者爆棚的新书发布会

作者：金鑫 房敏…        来源：中国新闻出版广电报        时间：2019/10/18 9:15:40
分享到：

　　□本报记者 金鑫 通讯员 房敏婕

    黄仁伟为读者签售。

    新书发布会现场座无虚席。

    房敏婕 摄

 

　　10月13日，上海图书馆，降温和秋雨，挡不住300多名读者的热情，260个座位坐满了，加椅子；椅子坐满了，台阶上、过道上，或席地而坐，或站着听完3小时，这都是因为一位主讲嘉宾――复旦大学“一带一路”与全球治理研究院常务副院长黄仁伟。

　　“‘一带一路’共建6年了，用上海话说，‘模子’已经出来了。”《大国坚守：中国外交在十字路口的历史抉择》新书发布会一开场，黄仁伟就迅速拉近了与读者的距离。他说，6年来，共建“一带一路”已完成夯基垒台、立柱架梁，转入落地生根、开花结果的全面推进阶段，成为广受欢迎的国际公共产品，在历史时空中镌刻下深深的中国印记。正如《纽约时报》的刊文，“一带一路”项目的数量和规模令人吃惊，远远超出了科幻作家的想象。

　　《大国坚守》是中外知名外交专家、学者贾庆国、苏格、崔立如、黄仁伟等对中国外交新变化、新应对、新实践的深刻思考和通俗表达，近日由上海交通大学出版社出版。书中，专家学者聚焦中美关系、双边外交、中国主场外交等领域中的重要模块，从历史、现实等不同角度理性观察中国外交政策，在中美关系和“一带一路”等话题上既保留了历史纵深感，又体现了学者的时代关切。

　　黄仁伟多年从事国际关系研究，近两年来他聚焦“一带一路”研究，更多时间用于走访实地。站在巨大的世界地图前，他指点读者观看自己实地勘察过的雅万高铁在建项目。“雅加达与万隆，两座城市之间只有一条高速公路，由于车辆众多，加上施工频繁，140多公里的路程经常要走上四五个小时。等到明年这个时候雅万高铁开通后，单程只需要40分钟，那是怎样的一种体验！”黄仁伟认为，雅万高铁的成功开通不仅能够改变印尼爪哇岛的经济状况，也预示着“一带一路”实质性的推进。

　　黄仁伟认为，“一带一路”是世界资源重新配置的开始。对于新一轮全球化，中国不是通过一种强制力进行改变，而是通过利益交汇点实现利益共享，构建人类命运共同体。这对于世界经济的长期正面影响将日益显示出来。

　　在签售环节，读者们排起长龙，抓住宝贵机会与黄仁伟交流。一位带着女儿从嘉定赶来的读者告诉记者：“当年听黄教授讲座的时候孩子还小，现在孩子都长大成人了，今天特意买了书请黄教授签名。”

　　本次活动由上海图书馆、文汇报社和上海交通大学出版社共同主办。
