奚立峰：将不遗余力的促进日照实现新旧动能转换

2018-10-20 17:25:00　来源: 大众网　作者: 许卉

上海交通大学党委常委、副校长奚立峰在开幕式发表致辞（辛润泽 摄）

　　大众网日照10月20日讯（记者　许卉）10月20日上午，海洋智能装备与未来航运系统创新论坛在日照举办，上海交通大学党委常委、副校长奚立峰在开幕式上发表致辞。奚立峰表示，上海交通大学扎根祖国大地，服务地方发展及产业需求，将不遗余力的促进日照实现新旧动能转换，加快日照向海经济发展，打造“智能日照”品牌做出贡献，助力海洋强国梦早日实现。
　　本次论坛由日照市人民政府与上海交通大学联合主办，日照市东港区人民政府、日照市科技局、上海交通大学船舶海洋与建筑工程学院、海洋智能装备与系统教育部重点实验室共同承办。论坛以“相约活力日照、智创魅力海洋”为主题，二百余位专家学者共聚中央活力区，描绘未来智慧海洋的图景，打造“智能日照”品牌。
　　奚立峰表示，上海交通大学作为中国船舶与海洋工程高等教育和前沿科技的策源地，正集聚融合船舶与海洋工程、信息与通讯工程、控制科学与工程、机械工程、材料科学与工程等多个一流学科，在智能海洋装备与系统等前沿交叉领域深耕精进。学校有决心、有能力打造“国家急需、国际一流”的科研平台，推动提升我国在海上智能装备及系统集成技术领域的原始创新能力和国际核心竞争力。
