沿着周总理足迹砥砺前行 全国微电影大赛在淮安启动

2018年09月21日 16:51:35 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

    9月21日，由中共淮安市委宣传部、中国电影评论学会微电影研究会、上海交通大学党委宣传部联合主办的“沿着周总理的足迹 不忘初心 砥砺前行 全国微电影大赛”举行新闻发布会暨启动仪式。淮安市政协副主席范更生，中国电影评论学会名誉会长章柏青，中央新影集团微电影发展中心主任、亚洲微电影艺术节组委会执行主席郑子等嘉宾出席启动仪式并致辞。

    2018年，是敬爱的周总理诞辰120周年。用微电影的形式，拍摄和记录周总理的事迹，以及全国各族人民沿着周总理的足迹，努力建设社会主义物质文明和精神文明，是新时代学习周总理纪念周总理、推进中华民族伟大复兴的有效方式。淮安市是周总理的故乡。在此重要的时间节点，主办“沿着周总理的足迹全国微电影大赛”，将是落实“把周总理的家乡建设好”的重要指示，塑造淮安精神，扩大淮安全国文化影响力的重要契机。

    本次活动从2018年9月开始，至2019年5月结束。包括作品征集、作品评审、作品表彰、作品推广四个阶段。征集对象为海内外高等院校师生、专业微电影创作团队、工作室，影视传媒公司、文化创作单位及其他创作者。征集作品要求：一是主题积极健康，内容丰富多彩，主要表现和展示新时代广大党员干部、人民群众传承发扬老一辈无产阶级革命家伟大精神，在各自岗位上努力工作、创新创造的各种事迹； 二是形式不拘一格，纪录、剧情、动漫都在征集之列；三是画面音质流畅，字幕清晰，声画配置得当，具有一定的艺术感染力；四是提倡短视频，时间上限不超过30分。

    本次活动的征集对象包括，海内外高等院校,专业徵电影创作团队、工作室,影视传媒公司、品牌公司及其他。将由统一官方邮箱进行报名：zgwdyyjo@126.com。网址http∥/enlai.ussnetcn。

    大赛将邀请电影专家学者对参赛作品进行评审，评出一等奖3部，二等奖7部，提名奖10部，以及优秀作品若干。（钱子达 陈力 颜晓彤）
