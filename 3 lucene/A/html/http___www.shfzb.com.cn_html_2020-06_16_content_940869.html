书写扶贫故事 传承交大精神

上海交大首期“青苗计划”学生骨干培训班开班

□见习记者  张叶荷

本报讯  近日，上海交通大学首期“青苗计划”学生骨干培训班（以下简称“青苗计划”）开班仪式暨导师见面会于线上举行。

据了解，“青苗计划”是共青团上海交通大学委员会创办设立的优秀学生骨干培养计划。作为选调生训练营和“青苗计划”的创办者之一，校学生处处长侯士兵老师在首期“青苗计划”开班仪式上致辞，对首期“青苗计划”全体学员表达真诚祝贺并殷切寄语。

首期“青苗计划”共招募学员44名。他们家乡各异，所学不一，年级不同，但他们都有着共同的理想和追求——“将所思所学投身基层实践、服务人民群众、造福一方水土”。

“青苗计划”将从理论讲座、业务学习、沟通交流、实践锻炼这四个方面对学员进行培养考核。通过系统化培训和实践性锻炼，旨在选拔一批政治素质过硬、综合能力突出、有志于毕业后赴基层部门从事公共服务和国家治理事业的优秀学员，培养一批具有深厚家国情怀、强烈责任意识、堪当大任品质、富有领导力和奉献精神的国家治理青年领导人才。
