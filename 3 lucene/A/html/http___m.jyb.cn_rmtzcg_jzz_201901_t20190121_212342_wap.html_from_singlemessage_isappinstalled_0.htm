上海交通大学成立氢科学中心

作者：董少校 发布时间：2019.01.21
中国教育新闻网

中国教育新闻网讯（记者 董少校）上海交通大学氢科学中心1月18日成立，将聚焦氢能源与氢生物领域，助力中国低碳能源体系建设。

上海交通大学氢科学中心是目前国内首家致力于氢能源、氢医学、氢农学等领域研究的机构，已具备自主知识产权和行业标准，其金属双极板技术和膜电极技术已应用于上汽集团生产的国内首款燃料电池汽车（荣威950）。

未来该中心将致力于“氢制取、氢储运、氢利用”三个方向的研究布局，围绕氢能源和氢生物领域的关键科学问题，包括对光氢催化、光电氢转化、光热氢转化、氢存储、氢燃料电池应用、氢医学、氢农学等共性技术进行系统研究。

“氢科学与技术联盟”筹备成立仪式同时举行，联盟将融合氢能源、氢医学和氢农业等领域的技术研发服务相关单位，探索产学研结合的产业技术创新机制，创建技术创新资源的集成与共享通道。
