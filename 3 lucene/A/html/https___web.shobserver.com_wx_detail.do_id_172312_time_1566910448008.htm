第一届AI本科新生来了！今年沪上两座名校成为全国首批新增人工智能本科专业高校

教育在线 2019-08-27 20:51
来源：上观新闻 作者：徐瑞哲 彭德倩
来自清华大学、香港大学、卡耐基梅隆大学、麻省理工学院、斯坦福大学、加州大学伯克利分校等全球顶尖高校或研究机构的15人，成为“A班计划”首批成员，并将亮相2019世界人工智能大会。

“人工智能（AI）”作为一个独立的专业，正迎来首批科班学子。根据教育部公布的新增审批专业名单，全国现有35所高校获得首批“人工智能”本科专业建设资格。从浙江大学、南京大学，到厦门大学、山东大学，以及沪上的两校上海交通大学、同济大学，这些名校均获批建设这一全新专业。即将到来的新学年，上海的第一批AI本科新生正步入校园。

【六大基础，交叉融汇】

在国家首批人工智能专业建设高校之一的上海交大，人工智能成为其电子信息与电气工程学院在建的第10个本科专业。据悉，首届30多名AI专业本科新生即将正式入学，迎接他们的将是资深教授开设的前沿课程，带其进入AI的“智慧之门”。上海交大人工智能研究院常务副院长杨小康透露，人工智能专业的课程设置与众不同，既强调数学基础，又强调与神经科学等的学科交叉，还强调企业导师加入的应用导向。

同济大学控制科学与工程系副主任、人工智能专业负责人张伟告诉解放日报·上观新闻记者，“不管未来人工智能应用如何变化发展，学生掌握好六大基础，可以‘以不变应万变’。”据了解，同济AI专业首次招生，也是计划30人。在设定好的课程框架中，最底层最基础的，包括数学基础、脑科学基础；其后是控制理论基础、计算机基础、电子技术基础，以及包含机器学习和深度学习等的智能基础。

明明还在暑假，张伟副教授和人工智能专业老师们这些天来依然忙碌。今年，同济对新生采用大类培养方式，一年级下半年才正式选择并进入专业学习。然而，由于人工智能专业培养所涉及的基础课程方向较多，为更有效选拔人才进行定向培养，创新性的“荣誉课”计划应运而生。

2019级新生进校后，根据自身意愿和能力，一开学就开始修读跟数学系学生同样标准的数学基础课程，如数学分析、高等代数等，代替原有的工科数学课程。学生在“荣誉课”上的表现，将成为下半学期双向选择专业的参考依据之一。首次荣誉课计划招收学生60人，目前报名人数已达140人，应者踊跃。

【人才体系，梯队递进】

“成为全国首批人工智能专业，有点摸着石头过河的意思，”张伟表示。在同济，AI专业像金字塔一样建立起教育人才体系。本科层级，主要依托电子与信息工程学院控制科学与工程系办学；上升至研究生，则由智能科学与技术交叉学科承担，充分发挥协同和复合优势；再进一步，则由上海市自主智能无人系统科学中心参与，培养国家亟需的“人工智能+X”人才。

同样，在本科基础之上的研究生教育层面，上海交大设立了以首届国家最高科技奖得主吴文俊院士命名的吴文俊人工智能荣誉博士班，着重培养潜在的人工智能领域领军人才，进一步完善人工智能人才贯通培养的模式。 

师资力量也是梯队递进——目前上海交大在人工智能领域形成一支50人左右的高水平师资队伍，包括3名中科院院士、4名IEEE会士等，其中毛军发院士和杨小康教授分别入选教育部人工智能科技创新专家组的咨询组专家和工作组专家之列。此外，还有入选麻省理工学院（MIT TR35）35岁以下科技创新35人的卢策吾副教授。

身为AI青年科学家联盟执行理事的卢策吾透露，今年4月，AI青年科学家联盟发布了加速孵化人工智能领军人才的“A班”计划，在全球招募20到30岁最具潜质的AI青年，由创业或学术导师进行一对一指导，提供国家重点实验室项目对接，以及孵化期、加速器与资本对接等服务。来自清华大学、香港大学、卡耐基梅隆大学、麻省理工学院、斯坦福大学、加州大学伯克利分校等全球顶尖高校或研究机构的15人，成为“A班计划”首批成员，并将亮相2019世界人工智能大会。“希望通过联盟聚集更多青年科研人员，在一系列‘资源链’的助力下，加快优秀AI人才的高质量成长。”

【大学智慧，服务社会】

“这是智能音箱、智能闹钟，还是智能电话？都是。”上海交大人工智能研究院俞凯教授，向解放日报·上观新闻记者展示了他团队研发的家庭医生智能助理，这个可以人机对话的智能终端，从语音收集、用药提醒，到预约社区卫生服务中心，成为家中“有温度”的AI。 

人工智能领域的大学智慧正服务于社会。徐汇区卫生事业管理发展中心与俞凯团队及AI企业上海思必驰信息科技有限公司合作，探索利用AI语音交互技术赋能居民健康管理的应用场景试点建设，以智能语音结合家居设备，实现慢病患者与家庭医生的实时在线连通。据悉，这一项目先收集居民日常健康相关数据，再针对不同患者进行个性化健康指导，让数字家庭医生不断自主学习，最终扮演家庭医生智能助理的角色。

在张伟看来，人工智能专业应用分为两大分支。其一是机器智能，以智能车、无人机、无人艇、自主教育系统，以及智能制造中具备自主加工能力的单元等为载体；其二是数据智能，也可被称为感知智能，与当今充分发展的大数据技术结合起来，并运用来自物联网的传感器数据等，从中提炼新知新技。因此，“同济牌”人工智能专业结合大学传统优势，通过以机器人、无人车等为代表的自主装置与系统，为人类实现赋予机器智能的长久梦想。

产业协同方面，作为国内最早启动人工智能平台建设的高校之一，上海交通大学牵头共建的上海人工智能研究院有限公司正实现平台实体化运行，与相继成立的上海交大苏州人工智能研究院和上海交大宁波人工智能研究院一起，共同完成了“上海-宁波-苏州长三角人工智能协同创新平台”布局，联合探索人工智能校地之间“政产学研用”合作。
