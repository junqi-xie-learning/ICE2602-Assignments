致敬每一个闪闪发光的你！上海医疗队即将返沪，仁济医生再创新曲《等你》

医声医事 2020-03-30 15:31
来源：上观新闻 作者：黄杨子 袁蕙芸
3月30日，录制好的《等你》飞跃太平洋，来到了武汉，共同奏响抗疫英雄们的归家之歌。

明日，上海第一批支援武汉医疗队即将返沪，上海交通大学医学院附属仁济医院风湿科主治医师、第八批医疗队队员李佳填词的新曲《等你》发布，以此欢迎仁济医院呼吸科主治医师查琼芳所在的第一批援鄂医疗队回家，也致敬每一位抗疫英雄。据歌词作者李佳介绍，《等你》这首歌写于1月30日，这是首批医疗队员们抵达武汉后的第7天，第三批援鄂医疗队员们抵达武汉后的第3天。

当时，李佳看到在武汉医务人员超负荷工作的艰辛，想听到他们每天平安的消息，想问问她的查查老师和朋友们能不能按时吃饭，病房冷不冷、伤口疼不疼、休息够不够、穿一天防护服会不会很难受……可是她忍住了，怕太多的消息会占用他们的休息时间。于是，在1月30日，她写下了自己的心愿——《等你》。

“这场疫情是对每一个人的考验。大年三十的出征，开启了中国历史上规模最宏大的一次医疗驰援，一群热血的白衣战士们从祖国的四面八方紧急驰援武汉。”李佳说，出发的时候，所有留守的医护人员说得最多的一句话是：“保护好自己，早日归来。”在这个春天，平安归来成为最大的心愿。

她的老师查琼芳在大年三十启程前往武汉金银潭医院。在机场，查琼芳说，“武汉这个城市生病了，作为共饮一江水的上海人，我们有责任和义务去帮助她。至于是不是除夕出发，其实对于医务人员来说，我们的工作是不分平时和除夕的，只要有需要，我们随时都可以出发。”此后的日子里，查琼芳用每天的日记来抚慰后方同事们焦虑的心情，而李佳也用实际行动向她学习，现在作为第八批医疗队员，依旧奋斗在雷神山医院。

李佳把歌词发给了在耶鲁神交已久，却未曾在校园里谋面的友人杨斯思，请她作曲。3月22日，曲谱初稿完成。耶鲁大学法学院毕业生、武汉姑娘田园在纽约公寓、南京医科大学黄淑紓在纽黑文的家中分别录好了音，杨斯思用电子钢琴弹了伴奏，然后完成了混音和后期制作。3月30日，录制好的《等你》飞跃太平洋，来到了武汉，共同奏响抗疫英雄们的归家之歌。

据悉，前不久，李佳受全国医务人员抗疫事迹鼓舞，作词创作的抗疫歌曲《勇气》，由专职音乐人袁清谱曲。3月13日，经由上海市委宣传部牵头，在上海市卫健委的关心支持下，上海广播电视台东方卫视以及上海人民广播电台、上海音乐学院以及上海交通大学医学院附属仁济医院联合发布著名歌唱家、上海音乐学院院长廖昌永倾情演绎的抗疫公益歌曲《勇气》以及《勇气》MV。
