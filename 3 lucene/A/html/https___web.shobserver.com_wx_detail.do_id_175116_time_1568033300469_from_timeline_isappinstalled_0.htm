教师节讲讲“我和老师”④ 这对寒门姐弟相继学医上海，不辜负贫困县里共同的老师

教育在线 2019-09-09 20:47
来源：上观新闻 作者：徐瑞哲
第35个教师节，你想你的老师吗？特别的学生，讲特别的老师，还有老师讲讲自己的老师，其中就有这些“我和老师”的好故事。送给全城近30万各级普通学校的教职工，为他们带去节日的问候：“老师，我想对您说……”

9日午后，来自国家级贫困县安徽省阜南县的2019级新生黄家成，刚刚和600多名新同学一起听了“强叔”陈国强院长的第一课，就与一位比他年长3岁的学姐他乡重逢。原来，这就是他最亲的堂姐黄文静。

安徽家中水泥铺地、白漆刷墙，来沪路上都是过夜火车或长途。黄家成、黄文静这对寒门姐弟，从小在农村一起玩到大，一个是班长，一个是学习委员，从同一所中学——阜南一中，考进了同一所大学——上海交大医学院。

因为前后相去3年，姐姐上了初中，他还在读小学；姐姐上了高中，他刚刚进初中。有意思的是，这一次他俩正在同校了。“你们有没有共同的老师？”解放日报·上观新闻记者问道。“有，但只有一位，教政治的冷老师。”

说来也巧，那天姐弟俩同时在家人经营的肉铺里帮忙打理。这时，冷老师来买肉，他们异口同声叫起“老师好”，这才发现这个任课老师的“交集”。

冷老师年届六旬，却随和健谈，也是第一次得知他们是姐弟。冷老师知道姐姐已经考到了上海交大医学院临床医学专业五年制，于是就对弟弟说，“ 你也要加油啊，也考到交大去。” 黄家成记得，中学时冷老师总是把课堂纪律交给他和几个小班干，看似放松地让他们“自治”，“我后来才明白这是锻炼我的管理能力。”

如今，考大学预言真的实现了，黄家成紧随姐姐考进了全院首个儿科学专业“5+3”8年制。他最想对冷老师说的话，就是：“老师，我也做到了！”

更幸运的是，家境贫寒、父母分居的黄家成，到大学没几天就遇见了好老师。在交大为国家专项计划特设的入学教育“筑力营”中，他很想说“谢谢您”的人，是来自护理学专业的辅导员老师卢欣。

就说他身上的这件L号新生报到服装吧。在分发T恤时，全班剩下的只有XXL，对黄家成来说嫌太大。他本想凑活穿了，但卢老师二话没说就去校部调换，顺便还请小黄吃了饭。而且，这不是卢老师第一次帮他打饭了，“新生教育上、下午连轴转，上午课程的场地我还来不及整理，卢老师看在眼里，就为我从食堂捎回了盒饭。”

其实，堂姐黄文静也成了“小导师”。“都说学医这条路很苦很累，当时是因为家里希望能有个医生才选择了学医，”这学年进入临床学习阶段的黄文静，已开始把自己医学院学习的经验教训毫无保留教给堂弟。
