2020年10月13日 要闻

上海滴水湖论坛将于10月29日至31日举办
滴水湖畔群贤毕至 跨界推动创新发展
26位诺贝尔奖得主、17位中国两院院士、18位青年科学家将齐聚临港

本报讯 （首席记者许琦敏 记者沈竹士）金秋10月，滴水湖畔汇聚全球智慧。10月29日至31日，包括26位诺贝尔奖得主和8位沃尔夫奖、拉斯克奖、图灵奖、麦克阿瑟天才奖等世界著名学术奖项得主在内的37位世界顶尖科学家，将与17位中国科学院、中国工程院院士，18位中外杰出青年科学家一起，共赴一场汇聚全球顶尖原创科学思想和科研成果的智慧盛宴——上海滴水湖论坛。

上海滴水湖论坛由上海市人民政府主办，上海临港地区开发建设管理委员会、上海临港经济发展（集团）有限公司、上海市科学技术协会、世界顶尖科学家协会和上海交通大学承办。

本届论坛的主题是“科技，为了人类共同的命运”，分为主题论坛、特别论坛和圆桌会议三大板块。四个主题论坛分别为世界顶尖科学家光子科学与产业论坛、生命科学与产业论坛、创新药研发和转化医学论坛、脑科学与人工智能论坛；特别论坛将为中外杰出青年科学家和世界顶尖科学家搭建深度互动的交流平台；在圆桌会议上，各国顶尖科学家将围绕前沿科技与创新发展畅所欲言。三天论坛交流成果将在10月31日下午的新闻发布会上对外发布。

上海滴水湖论坛旨在聚焦具有全球影响力的科技创新中心建设，搭建国际化、综合性科学平台。论坛将充分发挥诺贝尔奖获得者等顶尖科学家的带动与引领作用，打通科技创新链的不同环节，聚集分散在全球的高端人才、先进技术和优质项目等资源，吸引众多国内有潜质的优秀科学家参与；搭建综合性科创平台，服务上海全球卓越城市建设，力争成为上海建设具有全球影响力的科创中心的新名片，推动上海成为中国原创科技成果的诞生地、世界科技创新发展的策源地。

参加此次盛会的科学家，涵盖了化学、物理学、医学、计算机科学等主要科学研究领域，包括来自美国、英国、挪威、德国、以色列等国家的顶尖科学家，代表了各自领域当前世界最高水准的学术成就和社会影响，具有全球号召力。他们汇聚于上海滴水湖论坛进行智慧碰撞，将对全球科学界和产业界产生深远影响。
