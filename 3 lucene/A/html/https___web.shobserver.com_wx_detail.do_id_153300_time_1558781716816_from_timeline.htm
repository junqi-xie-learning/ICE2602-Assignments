陈竺：我国科技对经济贡献率达58.5%，2020年建成创新型国家目标有望基本实现

创新之城 2019-05-25 18:47
来源：上观新闻 作者：黄海华
创新型国家有一个重要指标，即科技对经济的贡献率要达到60%

“我国科技对经济的贡献率去年已达到58.5%，2020年建成创新型国家的目标有望基本实现。”中科院院士陈竺今天在2019浦江创新论坛上说，创新型国家有一个重要指标，即科技对经济的贡献率要达到60%，今年政府给企业减税降费规模将超过2万亿元，如果其中的10%能够用于研究，政府财政再加一把力，这个目标有望基本实现。

 

我国平均预期寿命达77岁，这在一个2亿人口以上的国家是没有过的

 

科学技术是推进人类文明进步的第一原动力，当前世界各国高度重视科技发展，研发投入稳步增加，中国的研发投入已经稳居全球第二。世界知识产权组织数据表明，专利申请排在前五位的国家分别是美、日、中、德、韩。

 

在一些重点领域，科学技术的学术交叉、组织力提升和国际合作呈现非常活跃的态势。比如，首张黑洞照片的获得、引力波的发现，都是体现这些特征的激动人心的事件。中国科技界也作出了举世瞩目的贡献，比如我国的EAST项目（全超导托卡马克核聚变实验装置）去年获得了1亿摄氏度的高温运行，这个温度是太阳内核温度的6倍；嫦娥四号今年年初在月球背面登陆，这是人类探月工程半个多世纪以来从未涉及的禁区，中国为探索宇宙起源、排除低频电磁波的干扰创造了前所未有的平台，第一次以中国提供的平台搭载了包括德国、瑞典、荷兰等国家的科学测试仪器。

 

科技发展另一个重要特征是和人民福祉紧密结合，在生命健康领域尤其突出。基因编辑技术应运而生，RNA干扰技术获得突破性进展，微生态在疾病防治中已获得非常明确的前景。靶向治疗和免疫治疗在癌症治疗方面有了很大突破，以前的放化疗在杀伤恶性细胞的同时会杀伤正常细胞，也很难清除肿瘤的干细胞，而免疫治疗和靶向治疗为彻底战胜癌症特别是清除干细胞提供了前所未有的可能性。我国婴儿死亡率的城乡区别已经消除，我国平均预期寿命已从2010年的74.8岁上升到了2018年的77岁，这在一个2亿人口以上的国家是没有过的。

 

华为的研发投入一年达200亿美元，其中相当一部分是基础研究投入

 

“基础研究是所有技术问题的总机关，在科技发展中要高度重视基础研究。”当下，我国在基础研究的投入增量是明显的，但其占科技投入的总比值只有5.7%，而一般发达国家这一占比都在15%—25%。基础研究现在需要多元投入，既需要政府投入，更需要企业投入。比如，华为拥有8万研究人员包括700名数学家、800名物理学家、120名化学家和6万多名高级工程师，其研发投入一年达到200亿美元，其中有相当一部分是基础研究的投入。

 

陈竺认为，需要更加完善分类评价体系，如科研论文的引用率、国际同行认可度、造福民众的获得感，这些是从基础研究到应用研究都需要强调的。再也不能把论文作为主要的评价体系，但是作为一个研发机构来说，高水平论文还是需要作为一个衡量指标的。

 

必须携起手来，因为没有任何一个国家能够独善其身

 

“16世纪，哥白尼凭个人兴趣弄一个望远镜就能提出日心说，而进入近代已经是学科交叉的科技了。”陈竺说，面对全球能源危机、气候变化、粮食安全等挑战，必须携起手来，因为没有任何一个国家能够独善其身。中国加大国际合作的决心不变，为国际社会提供更多科技公共产品的努力也不会减少。

 

国际科学技术发展正在酝酿新的革命，中国要从科技大国迈入科技强国行列，只有依靠自主创新创造，舍得投入，发展更加自主完整同时也更加开放的科学理论研究体系，以及具有战略性、通用性的基础技术供给体系，要加强各个领域的转化研究。科技创新必须依靠持续不断的制度创新来释放生产力，尊重知识、尊重人才是基础，建立新时代市场经济条件下集中力量办大事的举国体制是关键。
