努力搭建民企高校与中央国家机关沟通桥梁

来源：海外网 02-27 16:42

海外网2月27日电 2019年2月22日，由国家发展和改革委员会培训（宣传）中心、上海交通大学共同举办的首期“丝路行者”专题研讨班在北京隆重举行。国家发改委培训（宣传）中心党委副书记、副主任魏洪波，上海交通大学党委常委、副校长张安胜，上海交通大学总经济师、海外教育学院院长刘牧群出席开班仪式。

本次研讨班以“高质量发展”为主题，着眼于民营企业家、高校教师最关注的宏观政策、经济形势、幸福产业、战略性新兴产业、精准脱贫和信用体系建设等实务话题，努力搭建民营企业家、高校教师与中央国家机关相关单位的学习交流平台，使高等学校和社会力量进一步聚焦“一带一路”建设、京津冀协同发展、长江经济带发展国家重大战略，共同打赢防范化解重大风险、精准脱贫、污染防治三大攻坚战。

开班仪式上，魏洪波副主任对各位优秀企业家、教师学者的到来表示热烈欢迎。他表示，“丝路行者”专题研讨班是中心今年举办的一个全国性研讨班，也是认真学习贯彻习近平总书记在推进“一带一路”建设工作5周年座谈会上重要讲话精神，充分发挥教育在“一带一路”中的基础性、先导性、引领性作用的总体要求的具体实践。他表示，研讨班要紧紧围绕“一带一路”倡议的原则、理念和核心，加强前瞻研究、培养优秀人才、宣讲政策倡议、促进国际交流。他强调，民营企业家一定要以习近平总书记在民营企业座谈会上重要讲话精神为指导，努力增强创新能力和核心竞争力，将贯彻新发展理念，推动中国经济持续健康发展，造福中国人民和“一带一路”沿线各国人民当作重中之重，为构建人类命运共同体，为实现中华民族伟大复兴的中国梦努力奋斗。

张安胜副校长在致辞中表示，上海交通大学海外教育学院在去年12月提出的构建“全球CEO跨界大联盟”，通过铁三角发展模式，汇聚央企国企、民营企业和港澳台企的合力，共同参与“一带一路”建设的倡议，具有十分重要的现实意义。他强调，此次研讨班也是本着“共建大平台，融汇大智慧”的精神，打造企业积极参与、多重资源整合的企业经营管理人才协同培养机制。他表示，上海交通大学将与国家发改委相关单位一起共商、共建，为“一带一路”建设、脱贫攻坚等提供智力支持、提供高端人才培养，携手开创新时代改革开放的壮丽诗篇。

此次专题研讨充分发挥了国家部委宏观政策解析优势、高校优质教育优势、企业家生产实践优势，是合力开展新型智库建设、应用型人才培养，共同推进国家重点战略落地的尝试。参训人员通过与国家发改委地区司、高新技术司、社会司、财政金融司和宏观经济研究院等部门领导、专家的深入交流和课程学习，进一步加深了对宏观政策的理解、拓展了国际视野、坚定了发展信心。

来自有关省区市的30多名优秀民营企业家和上海交通大学10多名优秀教师参加了研讨班。研讨期间，大家还赴国家博物馆参观了“伟大的变革——庆祝改革开放40周年”大型展览。

责编：侯兴川、王栋
