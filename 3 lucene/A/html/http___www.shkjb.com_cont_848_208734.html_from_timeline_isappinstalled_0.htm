场馆内办科普吸引观众10万人次
——科技节明星项目“科学之夜”国庆节收官记
2018年09月28日  作者:戴丽昕   编辑:ydm   审核:杨冬梅  版面:A3

国庆假期，申城各类科普场馆成为亲子游的好去处，其中，2018年“科学之夜”大型公益科普活动当仁不让地成为 “人气明星”。在嘉定上海汽车博物馆里，经典古董车与竞技机器人、VR体验、科学秀的相遇，给观众带来了别样的科普互动体验，给汽车博物馆带来了开馆以来的最高人气的同时，也为2018年“科学之夜”画上了圆满的句号。

“科学之夜”作为上海科技节的品牌活动，以前只在上海科技馆举办，互动活动多，开放的场次每每爆满。今年，“科学之夜”突破时空限制，相继在上海城市规划展示馆、安徒生童话乐园、长风海洋世界、海洋水族馆、东方绿舟等举办多场不同主题的科普活动。活动场地变了，累计吸引观众近10万人次。

10月5日，虽然天气预报有台风，但是小宝和哥哥两人还是一早就从市区赶到嘉定的汽车博物馆。爸爸从网上帮忙申请到的门票，只要在门口经工作人员验证后就可以轻松进入场馆。哥俩人参观了馆藏的具有年代感的古董车，参与了打造穿越时空的互动体验，全方位地了解了汽车的过去、现在与未来。然后小宝和哥哥戴上了VR眼镜，身临其境地体验在赛道上的驰骋。夜幕降临，科学秀、脱口秀等舞台活动带领亲子家庭参与互动，从了解古董车的历史开始，步入汽车科学探索之门。“卡卡游世界”科普小课堂上，老师给家长和孩子们介绍新奇有趣的“水陆两栖车”与普通汽车的区别；在“汽车工坊”里，孩子们能了解汽车的各种内部构造，与爸妈一起参与模拟驾驶小游戏，边学知识边玩耍。

国庆期间，上海汽车博物馆接待观众同比增长一倍还多。上海汽车博物馆公众教育部主任姚畅表示：“此次活动是我们博物馆这么多年来首次面向公众开放夜场，观众的热情非常高，客流量和礼品店的销售量跟去年同期比都翻番了，10月4日的夜场还一度限流。网上的预约也早已排满了。”而这样的火爆场面不仅在汽车博物馆，在城市规划馆、安徒生童话乐园等场馆都有这样的效果。

“科学之夜”是上海科技节的重要品牌活动，旨在打造一场炫酷好玩的亲子科技嘉年华，寓科学于娱乐，在娱乐中获取知识，激发公众对于科学的兴趣。“科学之夜”里有丰富的科普活动，包括科学实验秀、科学产品秀、科技展示秀、科普剧目表演及科普课堂等形式多样的互动项目。今年，根据《上海市科技创新“十三五”规划》和《上海市科普事业“十三五”发展规划》，市科委启动“科学之夜”全市延展巡演，制定活动执行统一标准，优化完善活动策划方案。通过延展巡演，使更多优质的科普活动和项目资源得以与广大市民见面，进一步推动上海具有全球影响力的科技创新中心建设。

上海科技节组委会“科学之夜”负责人黄飞透露，明年的“科学之夜”活动已开始策划设计，升级后的“科学之夜”魅力值还将提升。黄飞说：“明年的‘科学之夜’活动，将把亲子活动推广到重点实验室、科研院所、高校和世界500强企业中。升级后的‘科学之夜’，主题将更加明确，也将把品牌更深入地推向公众。”
