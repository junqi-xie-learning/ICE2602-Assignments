3400余名上海交大2019届本科生迎来毕业典礼

见识录 2019-07-07 15:05
来源：上观新闻 作者：赖鑫琳
今年的毕业生中有180名2019届致远荣誉计划的优秀学子，其中6人毕业论文获评“上海交通大学优异学士学位论文”、21人获得“上海市优秀毕业生”荣誉称号、52人获得致远荣誉学位，共计166名学生选择继续深造。

7月7日上午，3400余名上海交通大学2019届本科生迎来毕业典礼，获得学位挥别母校，将开启人生新征程。

他们中有的选择继续深造，有的找到心仪工作，也有不少学生投身西部、基层、国防事业或重点行业，到祖国最需要的地方贡献力量，不负时代使命。

毕业典礼现场，上海交大党委书记姜斯宪寄语毕业生，“正处在中华民族伟大复兴的重要时期，既面临着难得的建功立业的人生际遇，也肩负着民族复兴的时代重任，可谓生逢其时。我希望大家在今后的人生道路上，要勇担时代责任，以国家富强、人民幸福为己任，志存高远、脚踏实地。作为交大学子，时代青年，应该具备建设人类命运共同体的胸襟和情怀。既要有全球视野，跨文化的理解能力，也要有促进世界文明交流互鉴、追求共赢的真诚愿望，为全人类的美好未来贡献力量。”

上海交大校长、中国工程院院士林忠钦作精彩讲演，希望毕业生面对机遇与挑战，唯有持之以恒创新图强。

林忠钦说，创新首先要打牢基础，学以致用。通过本科阶段的学习，初步建立了相对完整的专业知识体系，但从“学懂”到“学通”，四年的时间还远远不够。

他希望毕业生继续巩固夯实学到的知识，做到融会贯通、学有所长，并在实际的运用中锻炼创新思维、激发创新潜能，让自己成为善学、善思、善行的强者。 


毕业典礼上还颁发了2019届“上海交通大学优异学士学位论文”奖。经过前期严格甄选，共有40篇聚焦科技、行业前沿，代表学校各专业最高水平的学士学位论文获此殊荣，也是毕业生留给母校的一份厚重的纪念。

今年的毕业生中还有180名2019届致远荣誉计划的优秀学子，其中6人毕业论文获评“上海交通大学优异学士学位论文”、21人获得“上海市优秀毕业生”荣誉称号、52人获得致远荣誉学位，共计166名学生选择继续深造。

近年来，上海交大不断深化就业引导教育，构建了“新生入学-在读期间-毕业阶段”价值引领工作链，整合各方资源，在人才培养过程中，对大学生进行理想信念、社会责任和择业价值观的教育，引导毕业生正确认识时代责任和历史使命，树立科学的就业观和成才观，使他们在择业时做出符合时代使命要求的选择。
