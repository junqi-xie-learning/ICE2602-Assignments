在博物馆散步才是正经事！大佬教你从“看热闹”到“看门道”

东方网 傅文婧
2019年3月9日 08:51:35

　　东方网记者傅文婧3月9日报道：博物馆是跨越时空，汇聚人类文明成就的殿堂。人类的历史长河悠悠5000年，而博物馆作为承载历史的平台只有300年。3月8日晚，美国大都会艺术博物馆（The MET）前任馆长菲利普・德・蒙特贝罗做客“堂[TANG]・世界博物馆馆长文博讲堂”，在上海交通大学文治堂分享了他的艺术见解和对世界博物馆现状的反思。

　　蒙特贝罗是纽约大学美术研究院教授，也是大都会艺术博物馆150年历史中，任职时间最长的馆长。在昨晚的演讲中，他结合自己1977年至2008年，长达31年的执掌经历，带领观众漫步世界艺术博物馆发展的历史长河，又一同思考并展望了未来世界博物馆的发展前景与方向。

　　在活动主持人罗依尔的8分钟幽默“科普”后，蒙特贝罗教授登台，带领观众共同领略“跨越时间空间的人类最伟大艺术成就”。为了帮助观众尽快“入戏”，蒙特贝罗的讲述首先从大都会艺术博物馆中的中国景观、元素开始。模仿中国苏州网师园中“殿春m”设计的中式庭院“明轩”，将观众领进了大都会艺术博物馆――这座艺术宝库的藏品涵盖人类五千年文明史，堪称艺术博物馆中的百科全书。

　　随后蒙特贝罗又向观众们讲述了世界博物馆发展的漫长历程。在人类历史有记载以来的5000年时间长河里，文字、法典、灌溉、城市逐渐产生，而博物馆的发展却只占据了不到300年的时间，诞生于18世纪启蒙时代。蒙特贝罗表示，博物馆诞生的本质，不是为了某种实用性或宗教性目的，而是因为艺术。为了满足审美需求，为了给平凡世界中人们的日常生活带来特殊意义，博物馆因人们对过去、对历史的浓厚兴趣而发展，而兴盛。

　　从最早古希腊藏品的简单陈列，到古罗马的战利品展示，再到中世纪对于古董陈列的青睐，直到后来人文主义的兴起，艺术才慢慢开始真正成为独立于藏品使用功能之外的价值体现。1793年，卢浮宫开启了百科全书式的艺术博物馆时代，自此之后，博物馆逐渐发展成为一种具有代表性的建筑类型。随着时间的推移，世界各大著名博物馆相继落成，形成风格各异、独具特色的建筑风格。接着，当代艺术博物馆拔地而起，成为全新的名胜古迹和文化风景。蒙特贝洛还提到，除了建筑风格和陈展场所的变化，博物馆展览也陆续涌现出新的叙事方式，博物馆对于藏品收藏具有了更高的道德判定标准，新的文化财产法也使得让文物归还等事宜变为可能。走过数百年的世界博物馆，始终源源不绝的挑战中不断向前发展。

　　在纵观世界博物馆发展历程之后，蒙特贝罗也分享了他对于目前博物馆发展现状的反思。对于当代艺术在博物馆发展中应该处于何种地位，他提到，当代艺术自1929年起，逐渐为博物馆界大力推崇。当代艺术博物馆接连涌现，成为全新名胜和文化风景，逐渐成为当今艺术的代表。然而，对于当代艺术的过度重视甚至迷恋几乎已经渗透到当今博物馆的各个环节和部分，这种与艺术对话的方式，是否已经违背了艺术的本质？当前博物馆为了吸引观众而举办的各类边缘性活动，在作为接近艺术的窗口同时，是否也对艺术理解产生了干扰？

　　在当今社会和技术飞速发展，博物馆的观众数量激增，在碎片化阅读的时代，如何令观众集中注意力，用心感受真实的艺术藏品？世界博物馆的发展正面临着更多新情况、新局面，在为博物馆的未来创造机遇的同时，也提出了全新议题和挑战。此外，虚拟现实、人工智能等先进技术手段的发展使博物馆陈展具有了更多可能性，博物馆的交互体验方式日渐丰富，如何处理博物馆展品大众性和真伪性的关系，也为博物馆发展提出了新的命题。

　　最后的互动环节中，台北故宫博物院前院长周功鑫上台和蒙特贝罗一起与观众们进行了进一步交流。交大师生们针对当前博物馆观众的作用、博物馆中当代艺术发展等问题与两位专家展开积极探讨。蒙特贝罗表示，艺术品忠实记录并储存了过去的文明，他希望观众们能够走进博物馆邂逅艺术，体味艺术品的真实存在，从而获得个人信念和向善理念的提升，这也是我们为什么需要博物馆的答案。
