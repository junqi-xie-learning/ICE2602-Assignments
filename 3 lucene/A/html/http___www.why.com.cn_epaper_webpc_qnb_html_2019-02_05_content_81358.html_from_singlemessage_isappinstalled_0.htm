多所申城大学校园开出自己的“春晚”

亲手创造“年味”在实践育人与文化传承中找到最佳结合点

青年报记者　刘昕璐

    留校学生欢度春节。 青年报首席记者　范彦萍　摄

旦角装扮的南湘，走到舞台中央轻抖水袖，跟着悠扬的节奏唱起了京剧《贵妃醉酒》的四平调：“海岛冰轮初转腾……”唱京剧的南湘，是一名来自老挝的留学生。这一幕出现在华东理工大学为寒假留校学生举办的新春联欢晚会现场。青年报记者获悉，不仅是华理，复旦、上海交大、华东师大、工程大等高校也纷纷开出自己的“春晚”，既温暖留校学子的心，亦是让外国留学生对中国年有了切身感受。

青年报记者　刘昕璐

留学生唱京剧，中国传统文化与世界文化交融沟通

春节是阖家欢乐、亲人团聚的日子，由于学习、科研、实习等种种原因，部分学生今年寒假不能回家与家人团聚。为了丰富每一位留校学生的生活，各校纷纷精心策划，组织、安排了一系列迎新春活动。其中，许多留学生“主动”留校，就想真真切切感受一番中国年。

南湘是华理社会学院的博士研究生，来华留学已经有五年多，喜爱中国传统文化的她，不但喜欢吃小笼包、水饺等中国传统美食，更是迷恋上了中国的国粹——京剧。为了在今年小年夜的晚会上表演好《贵妃醉酒》，她还加入了学校的京剧协会，每周末跟着师生票友一板一眼地学唱，她的付出得到了大家的肯定，认真的表情、细致的动作，收获了在场师生的热烈掌声。

很多留学生也加入了交大春晚，歌曲表演《夜空中最亮的星》，赢得了同学们的阵阵掌声。其中，更是有一群提前“报到”的留学生——他们来自美国普渡大学，将于春季学期在交大开始一学期的专业课学习。他们在寒假期间提前来交大，就是为了在春节的中国气息中感受中国文化，作为到中国留学的一部分。他们带来合唱《茉莉花》和功夫表演，和中国学生一起欢度“中国年”。

在华东师大，留学生同学们用中文为大家带来了吉他弹唱《平凡之　路》、独　唱《rolling in the deep》和小组唱《说爱你》，让大家感受到了国际化视野下的汉语魅力，中国传统文化与世界文化的交融沟通。

《说爱你》演唱组合的一员，来自塔吉克斯坦的留学生德明激动地告诉记者：“选择这首歌是因为热爱，我们想对华东师范大学说爱你，想对中国说爱你。”去年9月第一次来到中国的德明，实际上已经结束了一个学期的交流学习，本可以直接回国度假的，但他选择了留在中国——因为很想体验中国最热闹的传统佳节春节。

同样即将结束来华学习项目的乌克兰留学生玛丽感慨：“时间过得太快了！　来到华东师大半年，我已经习惯并爱上了这里的生活。虽然这次只能在中国学习一个学期，但是我很热爱中国文化，以后一定要再来。”

年夜饭菜单精心设计，定点扶贫县大厨亲自掌勺

一边看校园春晚，一边其乐融融吃起年夜饭。今年，华理为寒假留校学生精心准备了冷菜八碟、玫瑰牛肉卷、水晶虾仁、清蒸笋壳鱼、生肖馒头、寻甸烧牛腩、浓汤四宝、五彩寻甸牛干巴爆羊柳、香酥鸭、双色洋芋球、荷塘月色、牛肉水饺、上汤西兰花、枸杞党参竹丝鸡等色香味俱全的美食、点心。而这其中，寻甸烧牛腩、双色洋芋球、寻甸牛干巴则别具特色——这是华理特意邀请了定点扶贫县云南寻甸县的厨师马金涛师徒，用当地食材为留校学生呈上原汁原味的寻甸美食，以此表达对华理助力寻甸脱贫的感激之情。

对于交大来说，一份“定制菜单”亦是煞费苦心。本次留校师生年夜饭，菜单设计上，“桃李天下”“猪年大吉”“诸事顺遂”“如日方升”“鹏程万里”“繁花似锦”等菜名，表达了对交大以及交大学子的美好祝愿。希望新时期的交大，继续走在时代前列，再接再厉，再创辉煌。

不仅如此，本次的定制菜单还加入了很多交大人专属的“小秘密”：“思源锦鲤”“菁菁有戏”“仰思百年”“凤舞南洋”“涵泽垂钓”，众多只有交大人才懂的交大元素，为年夜饭锦上添花，帮助师生共同欢度“新时代交大年”。

以“思政盐”提味年夜饭，让思政教育更加深入人心

在上海工程技术大学，今年邀请了200名寒假留校学子共享“程园家宴”，将“思政盐”融入年夜饭，让留校学生吃出立德树人的“年味”。

清炒虾仁、上海熏鱼蕴涵着学校“立德树人”“德育为先”的育人理念；蚝油双菇、清蒸鳜鱼勉励同学们“品学双优”“蒸蒸日上”；一道酒酿圆子祝福工程大学子“圆梦程园”……26道各色菜式，变成了新年的祝福语，让学生在活动中不知不觉上了一堂思政大课。

来自机械与汽车工程学院的杨同学一边津津有味地品尝着话梅酱鸭，一边感慨：“酱鸭蕴含着厨师们的匠心，自己也要秉持工匠精神，奋发作为。”

与此同时，三十多位参加活动的留校学生还主动报名参加年夜饭的准备工作，从食堂氛围的布置到切菜、配菜到上菜都由留校学生来承担，让习惯在家父母包办的新时代大学生，受到一次深刻的劳动教育。老师们则在准备年夜饭过程中与同学们谈理想、话家常。

与“小猪佩奇”亲密合影，与新春灯谜头脑交锋，与棉花糖甜蜜邂逅，与玻璃宝“猪”斗智斗勇……一个又一个热闹的游园活动为华东师大迎新春师生同乐会做起预热。青年报记者获悉，在长达一个月的筹备过程中，有近70名师大学子自愿报名参与各环节工作，从前期策划到场地布置，从节目彩排到物资采购，到现场服务，志愿者都发挥着主力军作用。校方认为，参与服务，亲手创造年味，是对传统文化的最好的传承，也是对“95后”大学生最好的实践育人过程。
