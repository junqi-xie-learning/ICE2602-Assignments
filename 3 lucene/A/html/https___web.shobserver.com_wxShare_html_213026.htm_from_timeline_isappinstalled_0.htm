疫情当前，沪上高校首次全面“触网” ，老师怎样“主播”学生如何“追剧”？

教育在线 2020-02-18 20:59
来源：上观新闻 作者：彭德倩 徐瑞哲
“‘停课不停教、不停学’是新学期的特殊打开方式，也成为我们主动拥抱教学方式变革、探索教学创新的机遇。”上海交大常务副校长丁奎岭说。这何尝不是沪上多所高校的共同期待。

本月24日，复旦大学将开出本学期的在线第一课——“新冠肺炎防控”，让全校学生从临床医学、流行病学、社会治理等多角度了解相关防控知识与政策，提高科学防控能力。两位授课老师是战斗在防疫一线的重量级专家：上海医疗救治专家组组长、华山医院感染科主任张文宏教授，上海市疫情防控公共卫生专家组成员吴凡教授。

上海市教委18日宣布，由高校根据自身特点和情况，“一校一策”开展在线授课，市教委指导高校综合用好教育部和上海市各类在线精品课程资源和学校课程资源，协调部分公共课程在线资源全市高校共享。

疫情当前，沪上高校首次全面“触网”。平台改变，将为学生和老师的上课体验带来怎样的变化？

【线上教学：“直播追剧”创造全新互动】

大学课程讲求专业性，那么如何将病毒传播、疫情防控等案例知识点融入专业课程？东华大学管理学院每天推送一期“师说”视频：李嘉晨老师从防疫“安全距离”切入，讲解《商业礼仪》中的“商务距离”，周建亨老师以“李佳琦式时尚口碑传播”模式为例，讲述“病毒式传播”……“以疫情防护看品牌七要素”、“以防疫重要物资口罩为例讲解工业品消费品营销”、“疫情期间企业捐赠支出的会计处理”等“师说”视频，也陆续上线。

大批学子“追着看”，工商1601班学生王俊杰说：“满满的干货和正能量，看着老师们精心准备的PPT，听着老师们亲切的话语，仿佛感到老师就在我们身边，里面的内容我还跟家里人一起分享着看呢。”管硕184班张一诺同学则表示：“每段视频最后还有老师们的鼓励和防疫小贴士，让我们在这场防疫阻击战中不再孤单……”

疫情防控和科技热词——“区块链”也有关？华东理工大学信息科学与工程学院黄建华副教授在线上直播“区块链技术及其应用”课程。课上，黄建华从总书记关于区块链技术的重要讲话讲起，介绍了区块链的由来、技术原理、应用和当前面临的挑战，并结合疫情防控介绍了区块链应用。这堂直播课程是华理的“花梨直播间”活动首讲，老师们运用自己的人生阅历、专业知识、业余爱好等，与天南海北的学生们开展“云”对话，为“宅家”学子带去带去知识力量。首次活动中，就有不同学院的90余名学生进班参与直播学习和互动。

近期，“花梨直播间”还将陆续推出“妨害疫情防控犯罪的处置”“大学生居家心理调节”“植物笔记——奉贤校区特色花草速写”等12场直播。

【思政授课：除了“言传”还能“身教”】

“上海交大有数万名学生，他们的青春我们不可辜负。”上好课，从选好人、备好课开始。常务副校长、中国科学院院士丁奎岭向解放日报·上观新闻记者介绍，对课程团队及主讲教师优中选优，选择最具教学经验、讲课效果最好、学生中声誉很高的教师来录制主讲课程。例如，全国优秀教师、全国最美思政课教师施索华教授主讲的《思想道德修养与法律基础》课程等。

高校思政课的在线上法，其实也很专业。中医强调整体观，上海中医药大学发挥医学学科和附属医院优势，通过“木、火、土、金、水”5课，全方位向学生讲好爱国主义、生命健康、心理疏导、网络思想、志愿服务，在线挖掘德育元素，上好思想政治课程。

比如，运用中医理论中“木”代表生机的特性，开展对学生的生命健康教育。公共健康学院辅导员向学生讲述人与自然和谐相处，自觉保护野生动物，尊重地球上所有生命；基础医学院辅导员则线上教学“八段锦”功法，增强学生体质，互动也不断线。

又如运用中医理论“水”滋养万物、润物无声的特性开展心理教育。尤其面对千余名世界各地的国际学生，“不同国家之间有时差，但上中医的关心没有时差”，国际教育学院辅导员们开通24小时网络咨询服务，讲好中国抗“疫”故事，坚定中国必胜信念。来自马来西亚的校友们发来“风雨同心 与你同在”的支持视频，伊朗的国际学生易正也发来太极视频，众志成城、共度时艰。

【质量控制：用好线上独有数据记录】

“‘停课不停教、不停学’是新学期的特殊打开方式，也成为上海交大主动拥抱教学方式变革、探索教学创新的机遇。”丁奎岭告诉解放日报·上观新闻记者。上海交大针对授课教师组织了多轮次、分类别的线上培训和试讲，截至目前已有2700人次参加了培训。同时，学校将分别进行本科生和研究生公共基础课的小范围试讲和全校范围的在线教学测试。

线上课程质量如何把控，是所有高校在课程设计之外的又一全新课题。

“我们有3万多在校本科生，按照原来的教学计划，下学期一共要开3000多门课。”同济大学本科生院院长黄一如说，所有课程上课时间与线下正常上课无差别，以免造成混乱。课程内容方面，同济大学此前储备有200多门自制线上资源。此外，对于教育部提供的精品在线课程，各学院进行选择与匹配。值得一提的是，对于这些已有的视频课程，学校对老师的要求是：学生上课的时候，老师也必须一起看视频，这样可以核对自己的教案及教学目标，拾遗补缺。

上海应用技术大学教务处副主任姜超介绍，即使是选择了教育部提供的精品课程作为授课内容视频，老师依然要在备课时完成“校本化”改造。同时，以两周为一个基本教学单元，两周组织一次作业测验，考核学生学习和老师教学情况。而在线课程独有的数据记录、学生投票等方式，也让学院、学校的督导更为完善。
