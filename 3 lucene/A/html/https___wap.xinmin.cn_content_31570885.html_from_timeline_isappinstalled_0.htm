视频 | “洋”葡萄“落户”上海 上海交大种出圆叶葡萄新品种飞入寻常百姓家
来源：新民晚报   作者：易蓉,孙中钦   2019-08-14 13:48:21

新民晚报“上海时刻”出品

【新民晚报·新民网】经过近10年的引种试种和栽培示范，8月14日，新民晚报记者在上海交通大学葡萄遗传与资源育种圃见到了圆叶葡萄新品种。

新品种单粒均重10克以上，最大达到16、17克，亩产可达2吨，而且风味独特果香清新，抗病性强、丰产性好。

据了解，目前，圆叶葡萄在我国南方各省区推广种植，示范推广面积已达5000余亩，部分果园已经进入盛果期，同时也已在上海“落户”，在崇明、金山、奉贤、闵行等建立的生产基地中生长结果。（新民晚报记者 易蓉 孙中钦）
