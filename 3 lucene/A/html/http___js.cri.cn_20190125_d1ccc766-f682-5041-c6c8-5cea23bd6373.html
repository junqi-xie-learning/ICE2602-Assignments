中国人工智能产业知识产权白皮书正式发布

2019-01-25 15:51:18    来源：中央广电总台国际在线     编辑：罗毅     责编：程茜    
更多

  国际在线江苏频道消息：1月24日上午，由中国人工智能产业发展联盟（AIIA）主办、上海交通大学苏州人工智能研究院以及苏州工业园区科技发展有限公司承办的《中国人工智能产业知识产权和数据相关权利白皮书（2018）》新闻发布会在苏州工业园区召开。本次会议上，《中国人工智能产业知识产权和数据相关权利白皮书（2018）》（以下简称白皮书）正式同步对外发布。

新闻发布会现场 莫涛 摄

上海交通大学苏州人工智能研究院院长王延峰发言 莫涛 摄

  在发布会上，上海交通大学苏州人工智能研究院院长王延峰首先向与会嘉宾致欢迎词，他提出如何加强人工智能领域的知识产权保护、促进创新成果转化，是人工智能技术应用和产业化发展的关键一环。鉴于知识产权领域的争端和诉讼已经频繁发生，大家迫切需要强化知识产权保护意识，这也是白皮书发布的重要背景。

中国知识产权发展联盟秘书长周砚发言 莫涛 摄

  AIIA学术与知识产权工作组组长、上海交通大学苏州人工智能研究院执行院长俞凯对白皮书进行了要点解读。据悉，白皮书主要结构包括知识产权和数据相关权利两部分，并从技术、企业、行业等方面介绍了各个环节涉及人工智能技术的相关知识产权内容。

  中国知识产权发展联盟秘书长周砚表示，希望后续白皮书能够提供更多详实的数据和相关分析，在行业当中慢慢地把这项工作全面性固化下来。

苏州工业园区科技和信息化局局长许文清发言 莫涛 摄

  苏州工业园区科技和信息化局局长许文清介绍，知识产权跟人工智能的结合以及白皮书的发布将对苏州人工智能的发展起到更好的促进作用，目前，苏州工业园区的产业结构在电子信息、高端精密制造方面都有布局，2018年年底刚获批的国家知识产权保护中心也将在2019年落地，这对园区人工智能产业的发展将有重要的意义。（文 莫涛 编辑 韩小强）
