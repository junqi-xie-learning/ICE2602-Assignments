上海高校师生热议总书记讲话 做志存高远青年不负青春韶华

上海高校师生热议总书记讲话 做志存高远青年不负青春韶华飞入寻常百姓家
来源：新民晚报   作者：郜阳 易蓉 张炯强   2019-04-30 16:33:32

来源/东方IC

新民晚报讯（记者 郜阳 易蓉 张炯强）今天上午，纪念五四运动100周年大会在人民大会堂举行。中共中央总书记、国家主席、中央军委主席习近平出席大会并发表重要讲话。上海高校师生通过电视、广播、网络等收听、收看，热切关注大会内容，并结合自身学习、工作和生活实际交流感想。

上海交通大学马克思主义学院青年教师刘伟表示，一百年前，伟大的五四先驱高举爱国民主科学进步的伟大爱国主义旗帜，凝聚民族力量，传播新思想、新文化、新知识，找到了通过奋斗改变国家和民族命运的力量。今天我们共同纪念五四运动，就是要传承伟大的五四精神，接续永久奋斗的伟大传统。作为新时代青年，我们要树立远大理想，用中国梦激扬青春梦，在风雨砥砺中铿锵奋进，勇挑时代重任不畏艰难，为祖国和人民建功立业。作为新时代青年，我们要练就过硬本领，激越生活世界，不辜负伟大的时代，把握人生的黄金时期，增长知识和实践，书写“中国梦”，以真学实干描绘青春最靓丽的底色。

复旦大学团委书记赵强表示，总书记的讲话充分肯定了五四运动的历史作用，充分肯定了青年在救亡图存、国家建设、民族复兴伟大历程中的作用，高扬着爱国主义的伟大旗帜，充满对青年的厚爱和希冀，激励着每一位青年永久奋斗，是对马克思主义青年观的又一次深刻阐释。青年工作，抓住的是当下，传承的是根脉，关系到中华民族永续发展。作为先进青年的政治组织，共青团要始终坚持党的领导，党旗所指就是团旗所向，要始终把为党和国家的事业培养建设者和接班人作为根本任务，要始终谨记巩固和扩大党的青年群众基础的政治责任，要始终围绕中心服务大局，团结带领广大团员青年，投身伟大时代，激扬奋斗梦想，为中华民族伟大复兴贡献磅礴的青春力量。

2019年“全国向上向善好青年”、2018年度上海市五四奖章标兵、华东理工大学化学与分子工程学院博士生张琦说，100年前，青年学生不畏强权，同心协力，为了抵制巴黎和会上中国所遭遇的不公，发起了声势浩大的五四运动。他们用自己的热血唤醒并引导了一代人的觉悟，爱国与进步是基础源泉，民主与科学是核心动力。五四精神弘扬至今，有了新时代的意义。作为新时代的青年，作为一名科学工作者，弘扬五四精神，我们应当以国家的科学进步为己任，要学习国外先进科学技术，同时也要将论文写在祖国的大地上，将核心技术应用到推动国家的可持续发展上。响应总书记对当代有志青年的号召，我们应始终将国家发展的历史使命重任在肩，勇于担当，攀登学术高峰，将科研成果运用在实现中华民族伟大复兴的中国梦中。以青春之我，创建青春之国家，青春之民族。

上海外国语大学国际关系与公共事务学院学生李明炎说，作为共青团员，我们要继续发扬“党有号召，团有行动”的优良传统。作为新时代上外青年，应该能担当、勇奋斗，明辨是非、恪守正道，面对外部诱惑要保持定力，拒绝投机取巧，远离自作聪明，练就过硬本领。拥有“敢为天下先”的拼搏精神，我们要努力为祖国，为上外，为自身的美好生活而奋斗，将自己的“小我”融入祖国的“大我”之中，继续完成实现中华民族伟大复兴的“接力跑”。

华东政法大学传播学院2017级学生滕晓琪说，百年之前，青年学生以自己的信念和热血为刃，斩破封建帝国主义的牢笼，让他们热爱的国家和社会向前迈了一大步；百年之后，无数有志青年回想起“五四”的历史，爱国之情依旧让他们热血沸腾。正如总书记所说，爱国主义自古以来就流淌在中华民族血脉之中，在新时代的背景下，爱国不应该是青年人藏在心里的秘密或者是嘴边的口号，而应密切关注着社会的发展，让自己成为社会大厦建设中扎实的一块砖、一捧土，利用自己所学，切实地为国家、为社会做事。青年人要以国家发展为己任，青年时期就要志存高远，这样才会激发奋进潜力，不会蹉跎青春韶华。

上海海洋大学水产与生命学院青年教师王浩表示，一百年来，五四运动创造的五四精神始终激励着青年人奋斗不息。新时代中国青年要树立远大理想，作为海洋大学的青年人，我们应在五四精神的指引下，站在水产一流学科的平台上，秉承百年来勤、朴、忠、实的校训精神，坚守科研的初心，把科研根植于祖国的水网阡陌，把研究做在祖国的江河湖海。新时代中国青年要锤炼品格，每一个深夜、每一个晨曦、每一次实验，每一个新品种的诞生、每一种养殖新模式的创立、每一次向蔚蓝大海的远航，凝聚着上海海洋大学科研人的汗水和付出。复兴路犹长，奋斗正当时！

上海海事大学商船学院2016级团总支副书记王宜徽说，作为一名航海类专业的学生，国家未来航运事业的接班人，我要把握机会，培养各项能力，学习专业知识，树立伟大理想，明确奋斗目标，在将来学成之时，为国家的“一带一路”建设，为实现海洋强国的梦想，为实现中华民族伟大复兴的中国梦贡献自己的力量。

上海工程技术大学材料工程学院辅导员沈静表示，作为一位青年老师，要继承和弘扬五四精神，担当时代使命，要有爱国情怀，树立伟大的理想，专研好业务，学好本领，为实现中华民族伟大复兴的中国梦而奋斗。

上海健康医学院校团委书记顾斐雯说，百年前的5月4日，一群年轻人英勇地走到反帝反封建爱国运动的前列。今天，我们回望五四百年，正是要明确当代中国青年的使命是为实现中华民族伟大复兴的中国梦而奋斗，激励当代青年把个人理想融入民族复兴伟大理想和中国特色社会主义思想。健康青年生逢其时，重任在肩，在祖国的万里长空放飞青春理想，在复兴的壮阔征程激扬青春力量。
