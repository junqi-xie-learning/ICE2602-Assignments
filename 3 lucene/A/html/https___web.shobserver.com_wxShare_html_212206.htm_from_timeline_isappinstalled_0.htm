交大医学院及瑞金医院“侦查小分队”出征武汉：病因诊断的任务同样光荣而艰巨

医声医事 2020-02-17 11:17
来源：上观新闻 作者：黄杨子
他们的主要任务是进行新冠肺炎病例的病因分析，深入研究疾病的发生发展的病理生理过程，为相关临床诊疗规范的确认提供科学依据，并为后续研究提供重要科学支撑。

2月17日上午8时许，随着D952次动车一声汽笛，上海交通大学医学院附属瑞金医院院新冠肺炎病因诊断专家组踏上征程。该专家组由国家卫健委委派，赴武汉开展新冠肺炎病理学和病因诊断研究工作。瑞金医院病理科主任王朝夫亲自带队，队伍由病理科党支部书记费晓春副主任医师、张衡主治医师、超声诊断科刘振华主治医师、上海交通大学医学院基础医学院蔡军、赵雷老师共同组成。这是一支长期工作在病理和影像学临床和研究一线，具有丰富工作经验和专业技能的精兵强将。据悉，王朝夫曾获得上海交通大学凯原奖，是优秀的病理学专家和备受爱戴的大学教授。

这一次，他们的主要任务是进行新冠肺炎病例的病因分析，深入研究疾病的发生发展的病理生理过程，为相关临床诊疗规范的确认提供科学依据，并为后续研究提供重要科学支撑。

16日下午3时许，接到国家卫健委的通知后，交大医学院院长陈国强院士和瑞金医院院长宁光院士亲自部署，基础医学院和附属瑞金医院迅速筹措队伍，制定增援方案，准备器械和防护物资。短短1小时内，就有12人报名。经过各方面的综合考量，最终形成了6人名单，全部都是共产党员。其中，既有工作经验丰富，既在大学里做过7年病理教师又具有丰富病理诊断经验的王朝夫队长，也有侧重于影像学方面的刘振华医生，还有长期工作在基础病理学领域的蔡军和赵雷老师。 

今天上午7时，交大医学院和附属瑞金医院党政领导为出征队伍送行。瑞金医院党委书记瞿介明主持简短的送行仪式。陈国强院士和宁光院士期待队员们与武汉的研究队伍团结合作，充分发挥交大医学院病理学科方面的优势特点，全力打好这场明确病因的战役。同时嘱托他们注意做好自身防护。基础医学院院长程金科教授也参与送行。

据了解，相比救治病人的一线医护人员，这支队伍的感染风险可能更大，承担了较大风险。为此，他们在16日下午进行了长达4小时的三级防护标准的行前培训。培训结束后，为了防止感染，队员们集体理短头发，为迎接即将与病毒开战的这场战役做好准备。

王朝夫表示，“我对大病理的相关工作很有信心，保证圆满完成任务，为早日打赢抗击新冠肺炎疫情这场硬仗全力以赴。”蔡军说，“基础医学院与临床关系密切的就是病理，病理的老师们最近也一直关注着疫情的发展，只要有任务必须要上。”虽然从接到通知到最终确定成行，准备时间很紧张，但其实也早有心理准备。蔡军的爱人也是一名医务工作者，目前也正在上海市第五人民医院的一线战“疫”，五院也是发热的定点医院，爱人非常了解防控工作的重要性，全力支持丈夫。赵雷则说，作为老师，平时在学校里教书育人，教导医学生救死扶伤、大爱无疆，现在正是言传身教的时候。“作为一名共产党员，更要在关键时刻站出来，冲锋在疫情防控的前线。”
