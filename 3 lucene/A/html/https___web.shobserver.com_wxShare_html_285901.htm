数十万大学生迎开学，送完站就OK？各校“试压”放大招：拉长周期无接触报到

教育在线 2020-09-02 15:15
来源：上观新闻 作者：徐瑞哲
本月底下月初中秋和国庆长假，新老学生是否可以外出，包括离沪返家？

早上6点多，复旦大学2016级临床医学专业五年制班级的班长王政民和同学们已坐在学校食堂吃早餐，准备前往附属中山医院开始新一天的临床实习。“今年新学期开放了堂食，方便很多。”

出校园时在校门口刷卡离校，到达医院后在指定地点刷卡入院，然后去找带教老师报到，离开医院时也需要刷卡，返校时再测量体温、刷卡，宿舍楼下也设置有测温装置实时测温……“我们严格按照学校和医院的要求，‘两点一线’开展活动，尽量减少可能的风险。”

在位于市中心的复旦大学上海医学院，一幢高层宿舍楼入住学生就达2000人左右，校园被马路天然分为两个校区。除了如此高度“聚集”，医学生还要入院实习。这个开学季，复旦上医党委副书记、副院长徐军表示，医生怎么防护，医学生就怎么防护，医院院门作为学校校门的延伸，加强“两点一线”管理。

沪上数十万大学生迎来开学，来自天南海北的家长们送完站就OK了？解放日报·上观新闻记者了解到，各校“试压”放大招：拉长开学周期，“无接触”返校报到。

【开学时限放宽至一个月有余】

今年秋季学期，处于常态化疫情防控下的特殊时期。事实上，9月1日，除了申城中小学开学，仅上海交通大学也有1000多名学子最先一批返校。根据“错区域、错层次、错时、错峰”开学原则，上海交大老生、新生错开时段到校，同时不同学段均拉长注册时间，比如研究生可以放宽到3天之内，本科生可以放宽到6天之内。

在延长报到注册时间方面，有的高校开学季甚至超过一个月。在上海杉达学院，目前在沪学生5800余人、在外省市学生5200余人，在籍学生和今年新生将在4个校区进行返校报到。校长陈以一表示，按照“三分”原则，“分类别、分年级、分校区”地确定返校开学时间。即从9月5日至10月12日，所有年级学生分批次到校。“在物资储备方面，至少有2周使用量的口罩，并将在返校日给每名学生按1周使用量发放口罩。”

而且，由于跨校区办校，不同级的学生还面临升年级、换校区。杉达针对两校区间搬迁学生，按每天500到800人分批，学生可网上预约搬迁时间，由学校安排专车跨校区接送、提供打包行李袋、地图指引“一站式”办理相关手续、两校区教师志愿者接待服务。他透露，“我们还为从嘉善校区到金海校区学习的全体学生3000余人做肺结核筛查，做好秋冬季节呼吸道高发传染病的预防工作。”

【“食堂就餐指数”每10秒刷新】

“无接触经济”的背景下，大学生注册报到也可以“无接触”。同济大学甚至从学校牵头建设的上海自主智能无人系统科学中心中抽调精干科研人员，自主研发“智能返校系统”。学生返校前在系统内填写返校行程备案表，签署严格遵守返校规定的个人承诺书；学院对学生健康状态、上报返校行程审核后，建立返校学生“一人一案”，下发个人专属“返校码”“进校码”。

同济常务副校长伍江透露，在第一版基础上已开发升级版，集“健康上报，进校申请、在校管理、离校申请”为一体，根据返校码对接校园一卡通权限，“凡是离开上海或者连续两天没有健康上报经核实有关情况后即关闭权限，进校需再次申请审批。”

上海交大自主开发的“交我办”APP也增加了“新生”栏目，通过“一个账号+一个APP”实现新生教务、财务、生活服务、网络等多项入校前后事宜畅通。校党委常委、副校长王伟明告诉解放日报·上观新闻记者，今年新生不再到体育馆集中报到，而是可以直接到宿舍。由于新生在线身份核验系统直接与数据库权威比对，通过手机端实时采集人脸数据，可在宿舍内自助完成真实身份验证，不需要大排长队，完全实现非接触。

有意思的是，上海交大还推出“食堂就餐指数”，各食堂总座位数与当前就餐人数、各餐饮大楼楼层区域分布就餐人数一目了然；系统按餐厅空闲状况排序，数据每隔10秒自动刷新，引导师生分流就餐。同时，为支持“教室教学+在线直播”这一新形态，学校暑期将400间教室建设为直播教室，实现本科教学楼全覆盖。

【“十一”长假原则上建议留沪】

与中小学开学不同的是，高校在今年9月份之前并未经历过全面“试压”。可以说，春季学期只是按需来校，秋季学期则是应来尽来。因此，本月起的“满负荷运行”也是对各校治理水平的大挑战。

上海海洋大学党委副书记、副校长闵辉表示，全校为此正在有条不紊推进十个“一”：要求相关人员和相关部门要与每一位学生联系一次，做一次实训演练，与每位学生开展一次谈话，进行一次方案培训，召开一次爱国主义主题班会，召开一次主题团日活动，召开一次主题党日活动，遴选一批“一对一”迎新志愿者，开展一次爱国卫生大扫除，开展一次校园安全风险排摸。比如有了“一对一”迎新志愿者，大到怎么选课、怎么上团课，小到哪里买生活用品更便宜，哪个餐厅红烧肉更好吃，志愿者都会告诉你。

至于本月底下月初中秋和国庆长假，新老学生是否可以外出，包括离沪返家，上海教育部门根据党中央、国务院关于新冠肺炎疫情防控工作有关精神，以及教育部和市委、市政府部署要求表示，目前国庆假期后的防控要求与新学年开学前的要求保持一致。市教委体卫艺科处有关负责人表示，原则上建议学生留校，珍惜在校时间，尽量本市活动。
