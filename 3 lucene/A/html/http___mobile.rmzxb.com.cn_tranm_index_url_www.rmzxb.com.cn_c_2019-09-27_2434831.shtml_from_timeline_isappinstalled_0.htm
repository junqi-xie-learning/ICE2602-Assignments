校地合作打造人才基地 多方共话谱写乡村未来 缪家乡村振兴学院揭牌仪式暨乡村振兴实践高峰论坛盛大召开

2019-09-27 头条号

2019年9月27日，由嘉善县大云镇人民政府主办，上海交通大学海外教育学院承办的缪家乡村振兴学院启幕仪式暨“不忘初心、牢记使命”乡村振兴实践高峰论坛在嘉善县大云镇隆重召开。缪家乡村振兴学院是围绕着“乡村振兴看大云、培训基地在缪家”主题，打造的全国首家实体运营的乡村级振兴学院。

浙江省农业广播电视学校校长应华莘；嘉兴市乡村振兴办副主任、农业农村局副局长贺学明；嘉兴市财政局基层财政处处长沈岚；嘉善县委副书记、政法委书记陈天荣；嘉善县人民政府副县长廖培根；庆元县有关领导；大云镇党委书记沈旭东以及县镇村有关负责人；上海交通大学海外教育学院副院长王良志；中国农村经济专家、中国农业大学原校长柯炳生；上海财经大学旅游管理系主任何建民；上海交通大学农业与生物学院副院长刘志诚；浙江省乡村振兴研究院执行院长潘伟光； 乡伴文旅联合创始人、浙江浙旅乡伴旅游发展有限公司总经理董天姝等专家出席了今天的活动。

校地合作 推动缪家乡村振兴学院深度发展

根据2008年总书记视察缪家村时做出的“走在前列，作好示范”的重要指示，嘉善县大云镇人民政府联合上海交通大学海外教育学院，共建了全国首家实体运营的村级乡村振兴学院。在缪家乡村振兴学院前，省市县镇村各级负责领导与多位嘉宾的共同见证下，依次由应华莘、陈天荣、贺学明、廖培根为乡村振兴学院揭牌并正式发布学院校徽。接着缪家村党委书记、缪家乡村振兴学院院长丁法强先后与王良志、潘伟光进行基地授牌仪式。

在接下里的论坛活动现场，浙江省农业广播电视学校校长应华莘；嘉兴市乡村振兴办副主任、农业农村局副局长贺学明；嘉善县委副书记、政法委书记陈天荣；及参加此次活动的嘉宾柯炳生、潘伟光、王良志相继也发表了精彩讲话。嘉善县委副书记、政法委书记陈天荣表示嘉善县的农村环境、农民收入和土地流转情况居全国前列，是美丽乡村示范县，未来期望打造成长三角城乡融合发展的典范。 浙江省农业广播电视学校校长应华莘指出乡村振兴是总书记的重大战略，缪家村是中组部和农业部唯一的农村教学示范点，期待大云镇继续成为新时代乡村振兴建设的排头兵，更奠定了本次论坛的主旋律。随后王良志、潘伟光分别代表上海交通大学海外教育学院、浙江省乡村振兴研究院，与嘉善县大云镇人民政府、缪家乡村学院展开了院校合作协议签约仪式。为加强校地紧密合作，以高校的人才技术优势助力打造缪家乡村振兴先进示范区、农村一二三产业发展先导区，打造以农为本，三产融合的创新实践基地、开放式的乡村振兴基地和乡村振兴的东部引擎目标。在论坛现场由缪家乡村振兴学院院长丁法强向嘉宾何建民、潘伟光，颁发了特聘专家证书。

解读战略 共话乡村振兴发展意义

围绕着《实施乡村振兴战略的创新路径与改革举措》、《关于全域旅游环境下，乡村旅游品牌打造》等问题，何建民、刘志诚、潘伟光、沈旭东分别进行了深刻解读分析，并与在场与会领导和嘉宾们共同探讨乡村振兴中乡村人才培养工作，以务实管用的理论成果为乡村振兴工作提建议、出方法，共同落实缪家乡村振兴大事。大云镇党委书记沈旭东在发言中表示在农村综合改革试点试验工作中，始终坚持问题导向，以改革创新破解乡村发展难题为工作重点，着眼示范推广，全力探索改革有益经验和模式，不断引领推动改革持续纵深发展。

本次活动对于乡村振兴具有重大的意义，缪家乡村振兴学院的正式成立为总结和传播缪家乡村振兴实践经验，探索乡村振兴教育体系，打造长三角一流、全国领先的开放式乡村振兴教育品牌提供了里程碑式的经验。大云缪家乡村也会继续紧跟时代步伐，探索乡村振兴的规划与实施策略，为乡村振兴的精准实施提供有效的解决方案，助力乡村发展迈向更高台阶。
