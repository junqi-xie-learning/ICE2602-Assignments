听诺奖得主讲解“任意子”，第三类量子粒子揭开神秘面纱

创新之城 2020-05-15 08:01
来源：上观新闻 作者：俞陶然
在二维体系中存在第三类量子粒子，其自旋量子数既不是整数也不是半整数，而是任意大小。

你听说过“任意子”吗？这是诺贝尔物理学奖得主、李政道研究所所长弗兰克·维尔切克教授设想的一种量子粒子，其性质与写入教科书的费米子、玻色子并不相同。昨晚8时，第150期上海交通大学大师讲坛首次采用网络直播模式，维尔切克向上海交通大学、麻省理工学院、北欧理论物理研究所等单位的科研人员和学生讲解了“任意子”。

听众中，有中科院院士潘建伟、中科院院士王贻芳、上海交大物理与天文学院教授郑杭。演讲结束后，维尔切克与这三位嘉宾就“任意子”的前沿研究与未来应用进行了探讨。

“任意子”观测证据登上《科学》

在报告中，维尔切克介绍了“任意子”的概念与来源，并重点介绍了两个实验工作：一个是2016年，中国科技大学潘建伟教授团队在由4个量子比特组成的超导电路中，产生“任意子”激发；另一个工作是来自巴黎大学、法国科学研究中心和雷恩高等师范学校的研究人员在今年4月《科学》杂志上发表的封面文章，阐述了“任意子”的观测证据。

微观尺度中的量子粒子，拥有与宏观物体迥然不同的奇妙特性。在量子力学教科书中，量子粒子被分为费米子、玻色子两类。费米子遵从泡利不相容原理，即两个费米子不能同时处于相同的量子态上；玻色子则相反，可以汇集在相同的量子态中。之所以有这种区别，其深层原因是费米子的自旋量子数为半整数，而玻色子的自旋量子数为整数。

科学家发现，有自旋的全同粒子在交换空间位置时，用来表示体系状态的物理量会出现一个附加的相位——对于玻色子，该相位为0度；对于费米子，该相位为180度。就好像在双胞胎之间放置一面特别的旗帜，当双胞胎位置交换时，旗帜的指向会因两人的内在属性而产生一定变化。如果这对双胞胎是玻色子，两人位置交换后，旗帜指向将保持原状；如果这对双胞胎是费米子，两人位置交换后，旗帜指向则反向。

那么有没有可能有一类双胞胎，两人位置交换后，旗帜指向任意一个角度呢？即除了自旋量子数取整数的玻色子和取半整数的费米子以外，是否存在第三类量子粒子？维尔切克在上世纪80年代提出一个理论：在二维体系中存在第三类量子粒子，其自旋量子数既不是整数也不是半整数，而是任意大小。他将这种量子粒子命名为“任意子”。

李政道研究所实验楼正在建设中

据介绍，上海交通大学大师讲坛创办于2012年4月，是面向全校师生的高层次学术活动，定期邀请国内外学术大师、著名科学家、学者等走进讲坛，以访谈与讲座相结合的形式，讲述学术思想，传授科研方法，弘扬科学精神，激发交大学子的学术志趣。

8年来，大师讲坛已邀请包括17位诺贝尔奖得主在内的一百多位学术大师，与近70000人次交大师生面对面交流。150场大师授业，涵盖数学、物理、化学、生物、工程、经济等各个领域。

解放日报·上观新闻记者从昨晚的大师讲坛上获悉，作为上海市重大工程，李政道研究所实验楼正在如火如荼地建设中。实验楼位于上海张江科学城孙桥科创中心单元中部，总建筑面积约56000平方米，计划今年6月结构封顶，年底基建竣工。

在科学研究方面，李政道研究所探索前沿科学的脚步没有因疫情而停滞，而是积极开展科学研究与学术交流，组织了一批线上学术报告，在包括《物理评论快报》等顶级学术期刊上发表了一系列学术论文。

昨晚的大师讲坛还邀请了麻省理工学院、北欧理论物理研究所、斯德哥尔摩大学等国际知名高校与科研院所共同在线观摩。维尔切克表示，他的一部新著作的写作已近尾声，他鼓励李政道研究所在此特殊时期，静下心来专注于科研工作，为研究所的发展做出积极贡献。
