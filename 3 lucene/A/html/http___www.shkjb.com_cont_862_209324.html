闵行区科委领导调研前瞻创新研究院
2020年03月26日  作者:戴丽昕 汪瑜   编辑:chunchun   审核:刘纯  版面:A2

　　为确保科技创新区级重大项目和区委、区政府重点工作的有序推进，落实重大项目月度推进工作例会机制，抓好重大项目复工及建设工作，闵行区科委党组书记、主任李丽日前带队赴上海前瞻创新研究院走访调研并召开月度推进工作例会。

　　会议聚焦疫情防控措施、企业复工、研发与转化功能型平台建设情况，重点关注疫情影响下前瞻创新研究院全年工作推进的新思路、新举措。赵建龙院长介绍了前瞻创新研究院1—3月份的工作进展和下步工作计划。李丽对研究院在积极复工的同时，所采取的严格疫情防控措施，以及为保证项目顺利有序推进所进行得工作给予了充分肯定，希望研究院再次落实全年工作目标，调整项目组织方式，整合优势资源，推进前瞻技术协同创新；并表示区科委作为区级重大项目的责任部门，将会持续关注前瞻研究院的发展，做好服务工作。

　　前瞻创新研究院作为区政府与市科委联合建立的开放创新的协作平台，作为推进上海前瞻科技深度融合的载体，将在上半年正式揭牌。平台成立后，将加快推进两大重点检测实验室建设，完善市研发与转化功能型平台申报方案，对接大学、科研院所相关资源，积极开展关键技术联合攻关、科技成果转化服务和创新创业赛事活动；与市科委加强对接，发挥集聚资源的作用，超额完成年度工作，助力上海南部科创中心建设。
