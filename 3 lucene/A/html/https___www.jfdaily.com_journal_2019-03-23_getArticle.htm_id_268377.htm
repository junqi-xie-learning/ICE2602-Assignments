瑞金医院专家驰援响水

2019年03月23日   01： 要闻   稿件来源：解放日报  
本报讯（记者 黄杨子）3月21日下午，江苏省响水县一化工厂发生爆炸事故。目前，共有16家医院、3500余名医护人员、90余辆救护车参与救治，接收医治伤员640名。上海交通大学医学院附属瑞金医院在收到国家卫健委应急办指令后，第一时间派出神经外科主任医师孙青芳、灼伤整形科副主任医师张剑和重症医学科副主任医师黄洁赴当地救援，指挥组指派孙青芳为国家卫健委专家组组长。

昨天凌晨约1时20分，3位医生乘坐救护车到达响水县人民医院参与抢救工作，凌晨3时许，又迅速赶往盐城市第一人民医院，该院收治伤员近200人。目前，3位医生在各专业组进行进一步会诊，协助当地医院诊治、排查重症患者，情况稳定有序。
