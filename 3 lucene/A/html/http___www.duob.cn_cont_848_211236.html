怎么把科学原理说得能懂和有趣？
——记2020上海市科协科学诠释者培训与实践活动
2020年07月10日  作者:王毅俊   编辑:chunchun   审核:刘纯  摄影:赵仲弈  版面:A4

　　为进一步加强上海市科普人才队伍建设，提升科普工作者科学传播和沟通表达能力，培养一支传播水平高、互动能力强的科学诠释者队伍及科技志愿者队伍。同时，为市科协“谁是王牌诠释者”大赛、全国科普讲解员大赛、全国科技馆辅导员大赛等发掘和培养参赛选手。市科协日前在科学会堂举办2020上海市科协科学诠释者培训与实践活动。

　　“科学诠释者”一词源于英语 Scientific explainer，最早由意大利欧盟科学诠释者学院启用。科学诠释者除了自身要具备科学知识外，还要学会运用特色的、有效的理念和方式，为公众普及科学知识，用各种演绎手段把复杂难懂的科学理论和科学现象解释给大众。

　　今年是特殊的一年，在市科协指导下，上海市业余科技学院继续以欧盟科学诠释者学院教学资料为蓝本， 邀请“全国科普工作先进工作者”、上海科技馆展教处副主任徐湮，央视“名嘴”董卿的大学老师、上海戏剧学院教授吴洪林，上海戏剧学院讲师张早，上海晟行文化传播有限公司讲师刘健等专家，从科学表演能力提升、科学实验及诠释、语言形象与传播力、科学诠释舞台表现案例实操等方面为选手进行强化培训。同时，采用学员现场表演+专家现场点评等形式让学员学以致用。

　　通过两天的培训，学员们懂得，作为一名合格的科学诠释者，除了拥有丰富的科学知识外，更需要掌握有效传递科学知识应具备的技巧和知识。即用手、用脑、用心，将复杂难懂的科学原理和科学现象解释给普通大众，激发人们对科学的兴趣。

　　用手，是指“每个科学发现都从体验开始”。通常是通过组织工作坊，让参与者共同参与实验或创造新事物，普通公众在边做边学中获得对科学更深入的了解。

　　用心，是指“需要情感的参与”。通过讲述故事或者运用戏剧、表演、游戏等方式，让公众在接触科学知识的时候获得乐趣，从而对科学消除畏惧感，激发更深入探索科学的兴趣。

　　用脑，是指“各种体验必须和谐联系在一起”。科学诠释者透过现象来揭示背后的科学原理，启发普通公众在体验中做出自己的思考，从而获取知识，能够举一反三，提升自身的科学素养。

　　学员Vic表示，非常幸运能参加这次培训，培训第一天上午徐湮老师很直接地剖析了如何有的放矢地去准备一个参赛作品。包括如何选题，怎样呈现，有哪些细节的点需要注意（比如前30秒如何吸引受众眼球等）。

　　下午刘健老师分享一位科普工作者的初心，他是带着怎样的使命感去做科普诠释这件事。他说：“无论你是身在何处，由衷的热爱和使命感，必然是成功的阶梯。”

　　科学诠释者培训与实践活动项目创始于2007年，并为此成立了上海科学诠释者学院。经过多年努力，至今培育了5000多名高素质、高水平、具有社会责任感的科学诠释者精英，为上海市打造了一支高素质的科学诠释者队伍。项目曾获得“第五届世界大都市奖”“上海市推进公民科学素质示范项目”“上海科普教育创新奖提名奖”等奖项，已成为上海普及科学知识、提升全民科学素质的一项有社会影响力的活动。
