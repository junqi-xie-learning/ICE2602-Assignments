“健康”是最好的毕业证书，这3家医学院校长正忙着“嫁女儿”“开药方”

教育在线 2019-06-28 17:30
来源：上观新闻 作者：徐瑞哲
抖音少玩点，技能多学点；外卖少吃点，饮食健康点；熬夜悠着点，锻炼加强点；牢骚少发点，实干多做点。

走进中国科学院院士、上海交通大学医学院校长陈国强的办公室，墙上挂着他自己抄写的《菜根谭》一段话：“栖守道德者，寂寞一时；依阿权势者，凄凉万古。达人观物外之物，思身后之身，宁受一时之寂寞，毋取万古之凄凉。”

这两句，对医者有何深意？陈国强说，这是为了提醒自己：人可以寂寞，也可以凄凉，关键是怎么选择，是一时开心，还是一生安心？他为学生算过了，“1的365次方永远是1，而1.01的365次方是37.78343，可0.99的365次方只有0.02558。”

在这个毕业季，沪上多所医学院校学子正从“学院”正式步入“医院”。作为“过来人”的校长们，对他们想说的话也真是不少。

做医生，从身到心，自身要“健康”，这种全面“健康”才是最好的毕业证书。为此，有校长像嫁自己女儿一样把他们“嫁出去”；有校长用好几味传统中药材为他们开出“方子”。这正合了两对诗句：“浮云游子意，落日故人情”；“临行密密缝，意恐迟迟归”。



【好女儿，也愁嫁】


陈国强，在校内人称“强叔”。学生毕业之际，他也不免想起自己当年读大学：17岁上大学，那是1980年，国家正在进行对越自卫反击战，大力宣传“新一代最可爱的人”；大二，中国女排七战七捷，首夺世界冠军，同学们自发上街庆贺，高呼“为中华崛起而读书”；1984年天安门广场举行庆祝建国35周年盛大阅兵，第一次看到如此规模宏大的阅兵，也看到北大学生在国庆游行时自发打出的横幅“小平你好”……“每一天都是新鲜的，每一天都生机勃勃，我们就是中国，我们就是未来，似乎每一个人都感到家国重担就历史性地落在‘八十年代新一辈’的我们身上。”

而今，“强叔”作为上海交大医学院院长，已是第9年在毕业典礼上演讲。面对医学院2019届本科生、硕士研究生、博士研究生和留学生1857人，他说：“为什么我的心底常怀忧心，那是因为我对你们爱得深沉……”或许，正因为“好女儿，也愁嫁”。

生活中，“强叔”嫁了自己的女儿；校园里，他也把毕业生当女儿嫁。他坦陈，过去8年，每逢初夏，自己就像个在“女儿出嫁”婚礼上唠唠叨叨的老父亲，用心用情将自己成长中的感悟，化作与应届生学兄学姐的道别心语。“今天，是我第九次‘嫁女儿’了，期待也好，忧虑也罢，该讲的过去都讲了，”“强叔”说，“已经老之将至、江郎才尽的强叔，今天依然像个嫁走最小女儿的老父亲，悲欣交集，唠叨唠叨，“希冀所有的毕业学子成为眼中有光、胸中有志、腹中有才、心中有爱的新一代最可爱的人。”


【6种药，多寓意】


医学院校，各具特色，在中医大，6种药名，成为了毕业季最好的相赠。上海中医药大学2019届本专科生毕业典礼上，面对来自我国新疆、云南等地和泰国、马来西亚等国家的家长代表以及920名毕业生，校党委副书记、校长徐建光举出一连串中药材，一语双关，与之共勉。“今天在这里，作为在上中医学习了5年多的校长，我也想有点中医药情怀，在大家又一次启航的行囊里放上三味毕业本草。”

第一味本草是“生地远志”。原来，生地就是地黄，具有滋阴补血，生津止渴的功效；而远志，则具有安神益智、祛痰、消肿的功能。顾名思义，校长希望学子面对未来的世界，始终拥抱梦想、心有远志、志当凌霄。

而第二味本草，“细辛厚朴”。同为草本植物，细辛具有祛风、散寒、行水、开窍的功效；而厚朴，则对食积气滞、腹胀便秘、湿阻中焦等有治疗作用。校长表示，“这是希望你们面对现实世界始终踏实细致、谦卑忠厚、不忘初心。”

第三味本草是“熟地当归”。别称熟地黄的熟地，具有补血滋阴功效；当归根可入药，是最常用的中药之一。“希望你们面对人工智能带来的虚拟世界，始终内心闪光、眼中有美、守正创新。”校长特别告诫，千万不要让网络语言和网上充斥的价值观变成生活的全部——走向社会，必须“当归”。

为此，他还编了个顺口溜：抖音少玩点，技能多学点；外卖少吃点，饮食健康点；熬夜悠着点，锻炼加强点；牢骚少发点，实干多做点。生地远志、细辛厚朴、熟地当归，“中医药烙印是母校最深沉的馈赠，忠实地陪伴你不忘本来、吸收外来、走向未来。”




【做校长，谢学生】


对于沪上最年轻的新兴医学院校，今年的上海健康医学院迎来了2019届也是首届本科生217人，他们来自生物医学工程、康复治疗学、护理学三大专业。28日，作为首家以“健康”命名的医学院首任校长，黄钢教授在毕业典礼上也打出“健康牌”。

首届毕业生与首任校长存在一种创业者之间的伙伴关系，正如黄钢所言，“我真诚地感谢你们，学校在培养你们的同时，你们也在滋养与改变着学校！”

他援引的案例里，满是这四年来学子的“硬核”故事。当人们抱怨大学生眼高手低、动手能力差时，生物医学工程专业的沈佳宜封号“奖项收割机”，揽获国家奖学金、宝钢优秀学生奖、首届上海大学生创客大赛一等奖等数十余项国家级、市级荣誉，并申报专利2项；同样，护理学专业的刘玥被誉为“微笑天使”，在全国护理本科生临床综合技能邀请赛中斩获佳绩；而康复治疗学专业的王顾赟无疑是全能型“康复达人”，不仅是辩论赛最佳辩手，也是创新大赛能手，在康复技能大赛、大学生创业大赛、电子商务三创挑战赛等比赛中屡获佳绩……黄钢说，“四年来的拼搏，你们在各种高层次国内外比赛中创造了多项‘中国第一’的佳绩，因时间有限我不能一一提及在各项大赛中自信展示自我的每一位同学，但你们的优异表现真正诠释了学校高水平应用型人才培养的办学定位。”

校长期冀，做有“温度”的健医人，做有“担当”的健医人，做有“定力”的健医人。“无论黄昏把树的影子拉得多长，它总是和根连在一起。无论这次离别让你们走得多么远，我与学校的心总是和你们紧紧相连。”
