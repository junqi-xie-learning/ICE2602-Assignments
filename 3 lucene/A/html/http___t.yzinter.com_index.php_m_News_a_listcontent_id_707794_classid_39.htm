原创|上海交大李政道科学与艺术作品大奖赛启动

2019-05-19 16:23:05

扬子晚报网5月19日讯（记者 罗晓娜）当严谨理性的科学与浪漫感性的艺术相遇，它们之间会“碰撞”出什么样的精彩火花？上海科普大讲坛第122讲“‘撞’出一片天，‘看’见新世界”活动近日在上海科技馆举办，为观众带来了一场“科艺融合”的知识盛宴。现场，上海交通大学李政道科学与艺术讲座基金还发布了2019科学主题——“光的粒子性”。这一主题由2004诺贝尔物理学奖得主、李政道研究所所长Frank Wilczek教授所选定。上海交通大学李政道科学与艺术作品大奖赛等活动正式启动。

本次活动由上海交通大学、上海科技馆联合主办，邀请了中国科学院院士、中国科学院高能物理研究所所长王贻芳与上海交通大学设计学院常务副院长、创新设计研究院院长胡洁带来专题报告，并由中国科学院上海天文台副研究员左文文担任主持，上海交通大学图书馆馆长、李政道图书馆执行馆长陈进教授代为宣读李政道先生的寄语并发布2019科学主题，共同带领公众穿梭于粒子物理与人工智能之间，畅想科学与艺术之美。

李政道先生在寄语中提到，“科学家以博大的人性情怀拥抱自然之理，艺术家则以宇宙之理倾述人性之情”，号召“广大科学和艺术爱好者能一起探索科学之光，领略艺术之美，挥洒创意，共究真理，同‘看’世界”。

伴随着人类不断探索和进步的脚步，宇宙大尺度观测已大到一百三十亿光年，但是我们仍然没有看到宇宙的“边际”；宇宙小尺度观测已经可以看到千万万亿分之一厘米，但也没寻到“尽头”。中国科学院院士、中科院高能物理研究所所长王贻芳为观众们带来了题为《探索无穷》的科学报告。

陈进教授发布了2019上海交通大学李政道科学与艺术讲座基金（以下简称科艺基金）系列活动的科学主题——“光的粒子性”。李政道研究所所长Wilczek教授专门为此撰写了科普文章，阐述了主题内涵。据悉，2019科艺大奖赛、国际学术研讨会、科艺工作坊等系列活动也将陆续开展：科艺大奖赛的作品征集时间为5月18日-9月16日；5月31日，将在上海交通大学举办暗光子国际学术研讨会，7月16日，科学与艺术工作坊将启动；11月23日开幕，科学与艺术作品展，科学与艺术讲座等活动将揭开帷幕。

上海交通大学李政道科学与艺术讲座基金系列活动重点围绕科学与艺术的融合与创新等问题，为科学家与艺术家提供一个高水平的互动交流平台。
