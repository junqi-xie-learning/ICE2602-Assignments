“不似之似——徐庆华肖形印展”在沪举行

2020年06月29日 16:02:01 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

    新华网上海6月29日电（严曦梦 董少校）既有简约空灵的佛像印，又有“萌”态可掬的小猪佩奇生肖印，日前在上海艺苑真赏社举行的“不似之似——徐庆华肖形印展”上，百余件佛像、生肖题材肖形印以新颖的艺术样式引发广泛关注。

    

上海交通大学媒体与传播学院教授、上海市书法家协会副主席徐庆华个展海报 新华网发

    徐庆华系上海交通大学媒体与传播学院教授、上海市书法家协会副主席。他自上世纪八九十年代开始创作肖形印，在构图上弃繁密而取简约，在用刀和线条质感上弃毛糙而取爽利，形成简拙厚朴的新颖艺术面貌。此次展览以“不似之似”为主题，化用齐白石语“太似为媚俗，不似为欺世，作画妙在似与不似之间”，体现出徐庆华大道至简、遗貌取神、旨归写意的肖形印创作理念。

    此次展览是上海首次肖形印专题个人展览，也是上海“品质生活直播周”系列活动之一，该展览来自徐庆华三十多年肖形印探索积淀，参展作品逾九成为2020年新作，计有原打生肖印蜕32件、佛像印蜕57件，印刷放大印蜕及印面28组，另有2件特大印蜕，展出原石73方，总共包含102个肖形。此外，现场展出徐庆华历年发表肖形印作品集的图录。

    

“不似之似——徐庆华肖形印展”展厅 新华网发

    得益于早年的绘画基础和开放的艺术视野，徐庆华的生肖印从古今中外的动物图像中广泛取法，抓取根本特征，得到一个个抽象又灵动的形象。如站立的鼠参照汉俑，戴着博士帽的猴具有民间艺术的效果，小猪佩奇直接源自时下动画，正面的牛则让人想起中国古代《五牛图》中的一头。

   

 “不似之似——徐庆华肖形印展”部分作品 新华网发

    徐庆华的佛像印同样以简约空灵为特征，人物表情不着一相，不求环境衬托，纯粹以点线面进行巧妙构型，刻制出或站或立、或单人或多人的佛像，造型简单而不失肃穆，凸显现代意识。

    展览期间，徐庆华到展厅进行视频导览，开展现场篆刻创作，向观众介绍篆刻艺术的一般特点，展现他的创作技法。专题节目在十多家平台播出，吸引观众超过150万人次。
