520，全城大学大表白！这33家高校，从今天起到上海解放70年接力拉歌把爱唱出来

教育在线 2019-05-20 17:16
来源：上观新闻 作者：徐瑞哲
爱上一个国，爱上一座城，爱上一所校……从复旦大学、上海交通大学、同济大学、华东师范大学开始，天天都有多所院校师生一起把爱唱出来。

“520”，全城大学大表白：爱上一个国，爱上一座城，爱上一所校！今天起到上海解放70周年纪念日之前的一周，从复旦大学、上海交通大学、同济大学、华东师范大学开始，沪上共有33家高校在线上线下接力拉歌，天天都有多所院校师生一起把爱唱出来。


【校歌融入爱国歌】


这是新中国创造感天动地人间奇迹的70年，这是上海解放、建设、改革和腾飞的70年。“一流城市孕育一流教育，一流教育成就一流城市。”无巧不巧，复旦与上海有一个共同的节日。1950年，复旦大学将校庆日定为上海解放日：5月27日。“复旦复旦旦复旦，日月光华同灿烂……”今天，一曲《复旦校歌》献给即将114岁华诞的复旦园，也献给同呼吸、共命运、齐跨越的上海滩。

复旦学子在邯郸路校区主路上放歌。


复旦大学上海医学院拉歌中。



同样，作为跨过3个世纪的百年老校，上海交大校歌也有新老多个版本。当天，在国旗之下，在钱学森图书馆，在仰思坪，在思源湖，在教学楼，在实验室，在师生守望的学习工作岗位上，交大人唱响《我和我的祖国》《上海交通大学校歌》，礼赞祖国，表白上海，唱响新时代。正如新版校歌的歌词中所唱，“相聚在东海之滨，吸取知识的甘泉”，歌声呼吁同学们牢记“饮水思源，爱国荣校”之校训

上海交大在钱学森图书馆拉歌。


巧的是，5月20日正是同济大学建校112年的日子。同舟共济的同济师生，也与全国高校一道放声歌唱祖国。在作为爱国主义教育基地的同济一二九纪念园，同济第二届青年马克思主义者培养工程“殷夫班”一展歌喉演绎校歌；在同济三好坞，十来位港澳台学生也自发加入老校歌歌唱队列中。第21届研究生支教团成员陈孝锦唱得投入，他说：“校歌印刻下读书救国的光辉岁月，即将奔赴西部支教的我们，在歌声中更加定下了一颗心。”

同济大学师生400人大合唱。


同济大学93岁的孙钧院士与土木工程学院师生共唱。


1950年12月9日，交通大学参加军事干校的同学行进在游行队伍中。



解放日报·上观新闻记者了解到，在此之后，合并组建已4年的上海健康医学院，从一所可追溯校史到1950年的老校，到一所上海市教育综合改革和医疗卫生体制改革的新校，将于5月22日纪念日当天，也在歌唱《我和我的祖国》快闪中融入校歌《健康之歌》：“我们迈步走在健康大道上，追寻着医学之光……未来的白衣天使，未来的医工技师。我们追逐年轻的梦想，担起生命的希望……”



【青春之歌代代传】


各校串起青春之歌，穿越中华时空。回望1949年5月27日，上海解放之初，为巩固新生的人民政权，面对新中国的使命召唤、上海大城市迫切需求，交大“电四讯”毕业班的20多名党团员，赶赴淞沪警备司令部防空处雷达队报到，参与建设上海第一个雷达站，为雷达部队诞生、发展和壮大，做出了不可磨灭的功绩。想当年，交大学子还走上街头开展反银元投机宣传；参与防空袭、反轰炸斗争；台风中奔赴南汇海堤救灾抢险；战斗在严寒的治淮工程第一线……



1950年6月21日，部分雷达兵学子回校，摄于图书馆前。


“爱国是什么？爱国对我们来讲，就是中国人的基因。”拉歌中，上海交大船舶海洋与建筑工程学院刘西拉拉得一手小提琴，这位年近八旬的老教授说：“《我和我的祖国》在我们那时候也唱。很多东西是需要用歌曲把它沟通起来，如果不同年代的人都有这样一个共同的心声，我们就会感到非常欣慰、非常放心。”甚至年已93岁的著名隧道与地下结构工程专家孙钧院士，及其领衔的同济大学港珠澳大桥科研团队，在为国家重大工程建设奔忙之余，也出现在校园歌队之中。

上海交大刘西拉教授拉起心爱的小提琴。


当五星红旗在华东师范大学群贤堂前的草坪上升起，新中国组建的第一所社会主义师范大学就这样宣告成立。上海市教卫系统优秀共产党员、师德标兵杨国荣教授正在丽娃河畔的24小时书吧和同学们交流。作为恢复高考后的第一届大学生，杨国荣表示，“这一代学人特有的对现实的关切、不同程度的理想意识和使命意识，对文化传承、世界变迁、人类命运的关怀，深深影响着其人文研究和学术思考。”

华东师范大学拉歌现场。


致敬七十春秋，师生代代传承。应同济拉歌之邀，在东华大学镜月湖畔，以青春名义为母亲献唱的活动正在火热排演。“我们都是追梦人，在今天勇敢向未来报到，当明天幸福向我们问好……”从《我和我的祖国》《我们都是追梦人》到《夜空中最亮的星》，东华学生合唱团团长、人文学院大四学生杨淇钧唱了一首又一首。今年正值大四，也是他在合唱团的最后一年，“歌声唱给祖国听，这为我大学四年画下完美句号，是一份最有纪念意义的毕业礼物。”

据悉，今年国庆节前夕，沪上还有上海政法学院、上海电力大学、上海出版印刷高等专科学校等12所高校接续第二轮大拉歌。
