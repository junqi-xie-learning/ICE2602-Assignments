一次普通的晨跑差一点要了他的命！三百里加急，年轻生命长三角接力涉险过关

医声医事 2019-09-06 09:52
来源：上观新闻 作者：黄杨子 黄欢 袁蕙芸
尽管天气炎热，平时注重健康的他和往常一样出门开始跑步锻炼。然而，仅仅几个小时后，神志不清的他已经躺在仁济医院宁波医院急救室接受抢救治疗。

近日，39岁的陈先生从上海交通大学医学院附属仁济医院急诊重症监护室康复出院。他可能做梦也没有想到，7月21日一次普通的晨跑差一点要了他的命。更没有想到的是，得益于长三角一体化战略，近两个月内，他虽曾深度昏迷、多脏器衰竭，在死亡线上挣扎徘徊一个多月，但是经过仁济医院宁波医院和仁济医院急诊科医护人员的无缝救治，终于康复出院。

陈先生是宁波一家汽车制造公司的工程师。7月21日早晨6点，尽管天气炎热，平时注重健康的他和往常一样出门开始跑步锻炼。然而，仅仅几个小时后，神志不清的他已经躺在仁济医院宁波医院急救室接受抢救治疗。

来到仁济医院宁波医院的陈先生病情已经万分危急：昏迷，高热，休克，进一步出现抽搐、呼吸衰竭表现。全院上下对这样一个年轻的危重病患者感到责无旁贷，首诊内科ICU主任吕卫星第一时间对其进行了心肺复苏，各种生命支持措施，暂时稳定了病情。

第二天，在患者家属、单位领导与院方共同协商下，决定将患者转至上海仁济医院进一步诊治。对于这样一个已经气管插管，需要呼吸机支持的特殊患者，转运途中的风险可想而知。医院第一时间与上海市医疗急救中心取得了联系。急救中心主任朱勤忠二话不说，以最快的速度调拨到了设备最好的跨省转运救护车发往宁波。

与此同时，宁波医院医务处处长樊翊凌与仁济医院急诊科主任朱长清商讨了患者的转院事宜，最短时间内调配好了急诊科ICU的床位，一场相隔300里的生命接力就此展开。

7月22日下午5时许，在宁波医院当地医护人员的一路护送下，载着陈先生的救护车顺利抵达仁济医院。一路绿色通道，陈先生直接进入了严正以待的急诊科ICU。此时的陈先生依然深度昏迷，相关检验结果还提示患者出现了肝功能严重受损、横纹肌溶解与凝血功能障碍。急诊科ICU主任徐欣晖等医护人员立即开始组织抢救。

仁济医院副院长张继东、医务处处长骆华杰得知该情况后，举全院之力协调一切医疗力量和资源投入到该患者的抢救工作中。由神经内、外科，血液科、消化科、内分泌科等科室组成的第一个院内多学科会诊讨论马上开展。但是，患者的病情依然在进展：持续39摄氏度以上的高热、不断加剧的肝功能损害以及弥散性血管内凝血导致的消化道大出血……

ICU医护人员全勤投入，不分昼夜监测病情，调整治疗方案，仅血浆、红细胞就输注了2000毫升。好在患者的病情终于出现转机，拔除气管插管、脱离呼吸机，意识转清的那一天，陈先生用他仅有的一点肌力，一定要求家属让他写字。展现在医护人员眼前的，是两个歪歪扭扭却饱含深情的“谢谢”。

本以为患者病情开始彻底朝着康复的方向发展，然而事与愿违。陈先生不久后再次出现持续高热，经血液培养为耐碳青霉烯类肺炎克雷伯菌感染，这种高度耐药的细菌使得患者的生命再度蒙山阴影。后经华山医院抗生素研究所会诊，讨论确定了新一轮的抗感染治疗方案，患者病情逐渐趋于稳定。而其实，此后的治疗过程依然是一波三折，包括多耐药鲍曼不动杆菌的导管感染、复苏后的肠功能瘫痪、肢体神经功能受损、营养不良等，陈先生和ICU所有医护人员可谓过关斩将，前后经历了40天的生死搏斗终于顺利康复出院。

急诊ICU专家介绍，陈先生的病因主要来自极度高温的天气，“大家俗称的中暑，就是人体在高温和热辐射的长时间作用下，机体体温调节出现障碍，水、电解质代谢紊乱及神经系统功能也可能发生损害。”医生建议，虽然运动对身体有益，但炎热时节需科学安排工作、运动和出门时间，保证充足睡眠，在高温时段“11时至16时”尤其应该避免强体力活动，在出汗多的情况下及时饮水及补充盐和矿物质。“另外，如果长时间待在空调房，人体的毛孔是收缩的，因此早上出门时也应注意，若到室外温差大但毛孔来不及舒张，也会导致中暑。”

今年3月1日，上海交通大学医学院附属仁济医院宁波医院全面开业运营，陈先生是医院正式运营后首位在院间直接转运救治的极危重症患者。“该患者的成功救治得益于这条长三角生命绿色通道的建立，我们相信，随着通道的不断完善和优化，沪甬合作办医的新模式必将会造福一方，为杭州湾新区乃至浙东地区百姓的健康保驾护航。”张继东表示。
