96岁王振义带上机器人“开卷考试”　

“互联网+医疗”跨越地域空间制约，让更多优质医疗资源辐射全国

2020年05月16日   02： 要闻   稿件来源：解放日报  
本报讯 （记者 黄杨子 通讯员 唐文佳 李啸扬）昨天9时30分，在上海交通大学医学院附属瑞金医院血液医联体会议室里，96岁的中国工程院院士、瑞金医院终身教授王振义端坐在计算机显示屏前，正在对住在同仁医院的疑难血液病患者进行远程会诊。今年疫情之后，王院士“开卷考试”的传统仍在延续。

与以往不同的是，王院士这次还带上了一个高科技助手——移动查房机器人“小雪”。这是一台拥有实时交互功能、能够实现医院内外网安全隔绝下的数据调阅及视频直播功能的智能终端。

“王阿姨，你能听到我说话吗？你有气喘吗？脚肿不肿啊？……”“刘医生，你用手按一下，看看病人腿部有没有凹陷性水肿。被子拉开看看，有出血点吗？”“有的，王院士，您看——”随着刘医生的操作，镜头里“听话”地出现了王院士想要知道的信息。

会诊开始了，同仁医院的主治医生通过计算机客户端向王院士报告了患者的病情，王院士边听边仔细查看病历，并不时提问，整个交流过程非常顺畅。随后，同仁医院血液科的医生和查房机器人“小雪”一起来到患者病床前。王院士通过他的“远程助手”与患者进行沟通，了解患者的发病过程和主观反应，并让机器人充当他的“耳朵”和“眼睛”，对患者进行了详细的体格检查。可以说，通过这个远程助手，除了不能亲自触碰，医生已基本实现跨空间的真实看诊体验，突破了以往远程会诊只能和医生进行交流的局限，让诊疗更为精准。

据悉，昨天的“开卷考试”吸引了全国60余家血液病中心的3000余位血液科医生全程参与，各位血液界后辈实时观看了王院士的查房，认真聆听王院士针对疑难病例的病例分析，并在直播空间里留言，与这位业界超级大咖积极互动。

瑞金医院血液科主任李军民表示，瑞金血液医联体一直致力于将院内的专家查房延伸到医联体医院之间，让专家能犹如在现场一样和异地的患者以及患者的主治医师进行面对面交流。未来，瑞金血液医联体计划将利用更先进的互联网技术，借助人工智能、边缘计算和增强现实等手段，使更多地区覆盖瑞金血液的新知和经验，同时也将借用瑞金互联网医院平台，为更多患者提供远程诊疗等多种个性化定制医疗项目，使血液病患者获得优质高效的居家医疗服务。
