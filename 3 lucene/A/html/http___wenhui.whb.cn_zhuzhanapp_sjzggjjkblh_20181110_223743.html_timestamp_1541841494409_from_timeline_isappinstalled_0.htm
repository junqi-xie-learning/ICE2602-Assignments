凌晨4点不到出发，一天啃6个面包，晚上12点回校……进博会“小叶子”：虽辛苦，但很幸福！
2018-11-10 14:11:27 作者：樊丽萍

今年进博会，在人流如织的“四叶草”中，有一道绝不容错过的青春亮丽风景线，那就是“小叶子”！

来自复旦大学、上海交通大学等34所上海高校的5384名大学生志愿者，成为此次进博会志愿者队伍的主力军。

披星戴月、风雨无阻，全力投入到进博会志愿服务中的“小叶子”，大部分是90后和00后。据说，在“四叶草”的志愿者休息区墙上，有一张每天更新的“步行荣耀榜”。排行榜记载着每日步数前20名的志愿者。很多时候，前10名的步数均超过30000步，位居榜首的志愿者步数接近50000步。

记者获悉，在这群特别能拼、能吃苦的“小叶子”中，有的是服务过多个重大会议、赛事的资深志愿者，也有的是经验丰富、术业有专攻的高校教师。在“小叶子”的岗位上，他们展现了别样的风采，让人难忘。

他们，是一群每天凌晨4点就要出发的“小叶子”！

在临港地区的5所高校，都有学生成为光荣的进博会“小叶子”。由于临港离“四叶草”地理位置相对较远，每天“小叶子”们路上的往返将近4小时。而他们每天出发早上集结的时间是——凌晨4点！

上海海事大学经过3个月的层层筛选，遴选了140位优秀的志愿者成为“小叶子”服务本届进博会。这是“小叶子”在出发前的合影——时间是凌晨4点不到！

本次进博会场地足足27万平方米，声势浩大，免不了产生些许的小bug。上海海洋大学的“小叶子”们细致耐心地发现了展会期间旗帜脱落、挂反等问题，并及时联系组委会解决，保证进博会的顺利进行，避免了国际尴尬，让我们为他们的细致工作点赞！

上海电力学院有134名小电娃，在进博会餐饮消费服务保障组、保障保卫处提供志愿服务。需要行走步数最多的岗位，每天都有步数霸王的小电娃闪现。

上海电机学院和上海建桥学院的“小叶子”，也习惯了披星戴月！

昔日世博会“小白菜”，今日进博会“小叶子”

在“四叶草”，不少“小叶子”都是热心的公益达人。在今年之前，他们服务过很多重大的赛事和活动。上海交通大学博士生罗前程就是其中之一。

8年前的上海世博会，他曾是“小白菜”；现在，他是中国国际进口博览会的“小叶子”，值守在VVIP贵宾室的接待岗位。

“我很荣幸，能先后成为两次盛会的志愿者，一次在我工作的时候，一次在我再次转变成学生身份的时候。”今年35岁的上海交大博士罗前程说。

既是“小白菜”又是“小叶子”，这个“双料”志愿者的身份让罗前程感到无比自豪。

1983年出生的罗前程来自宁夏，在宁夏医科大学完成了本科教育毕业后，怀着对上海这座城市的喜爱与憧憬，于2005年来到上海市第七人民医院工作。工作的第四年，他以医疗急救支援的志愿者身份，参与到上海世博会的保障工作中。

“当时，医院紧急抽调人手前往浦东新区医疗急救中心，我被分配到洋泾分站进行支援，那一急救站点周边人口密度大，又多为老式小区，急救任务重。”他回忆说，2010年4月，他正式上岗成为一名“小白菜”。那时，12小时为一班，经常需要通宵完成急救任务，一天几乎要跑十趟车，“经常是路上吃饭，有时候吃到一半，接到任务又要马上出发。”

当时，原本定下来的支援时间为3个月，但因急救医生紧缺，罗前程主动要求再延长一个月，整整干了4个月。尽管急救任务很重很累，但他认为，这样的机会对他来说是光荣的，对自己的成长经历也有很大帮助。

“我从2001年上大学就一直参加志愿者活动，在金鸡百花电影节、全国少数民族运动会、第九届全球健康促进大会都担任过志愿者，大学期间也作为班级团支书，三年都参与宣传艾滋病防治工作。”回忆起自己的志愿者经历，罗前程很自豪。

在上海世博会结束之后，他前往浦东新区公利医院工作，曾作为组织者和实施者，成立“公利急先锋青年志愿者服务队”，向社区居民、武警部队、学校等推广心肺复苏等急救技能。

2018年，热爱做公益的他，又考上了上海交通大学医学院博士。进博会举办前夕，当接到学院发布的招募志愿者通知时，他第一时间便报了名，并顺利入选。

10月20日进博会志愿者誓师大会后，罗前程便逐渐开始脱产培训，并自10月23日起进入国家会展中心实地培训，每天熟悉场馆。在此次进博会中，他被安排在VVIP贵宾室接待岗位做志愿者，与东方航空的工作人员一起完成接待工作。

忙得一天都在吃面包，回到寝室看朋友圈，还是不忘晒晒幸福感

在上海财经大学，小叶子们的出发时间是凌晨4点半。啃着面包，数着星星，出发！凌晨五点半，当很多同学还在温暖的被窝里熟睡的时候，小叶子们已经走上了工作岗位。

▲这是上海财大一位“小叶子”设的闹钟

据了解，随车志愿者小组是本次活动中志愿者人数最多的组别，负责运送参加开幕式和平行论坛的注册嘉宾。每辆车都配有一位辅导员老师和两位学生志愿者。嘉宾上车前，志愿者们需在车旁进行引导，确认上车人数，向调度处汇报后，发车驶往国际展览中心。在车上，志愿者们需进行语音播报，并实时向调度处汇报情况；抵达后，需引导嘉宾下车，并留在停车场随时待命。

所以，很多小叶子在天还没亮的时候，就接上了一车的注册嘉宾们，和司机与民警一道运送嘉宾前往国际展览中心。虽然战线很长，但他们始终保持良好的精神风貌，全心全意为进博会、为注册嘉宾服务！

吃了一天面包的“小叶子”，在深夜回到学校。他们在宿舍里吃着泡面，在手机上看着朋友圈，虽然腰酸背痛，但有一种满满的幸福感。

每天最晚一批离开的“小叶子”，从到校休息到第二天出发不足4小时

在每天最晚一批离开的志愿者中，总有华东师范大学“小叶子”们的身影。回到学校，经常已经——12点了！

05：15：华东师大闵行校区研究生公寓门口、中山北路校区中江路门口集合。收到来自后勤保障部连夜特制的惊喜早餐和小点心。华东师大闵行校区研究生公寓门口、中山北路校区中江路门口集合。收到来自后勤保障部连夜特制的惊喜早餐和小点心。

06:30  到达国家会展中心，在休息区吃早餐。

07：30-12：00 分岗位投入工作。其中，华东师大有42名志愿者负责核心区四个平行论坛的会务工作，29名志愿者负责其他区域的会务工作，5名志愿者负责市民热线的志愿服务工作。

13：00-18：00 分岗位投入工作。志愿者分别在各论坛现场准备会务工作。

19：00-23：00  分岗位投入工作。志愿者为第二天会议筹备布置中。

24：00启程回校休息。四个多小时后又将迎来忙碌的一天。

最后，小编想说：

各位小叶子们，你们真是好样的！
