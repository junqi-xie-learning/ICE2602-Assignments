中国商学院已没有模仿的榜样！上海交大安泰经管学院成立行业研究院，求解穷理和务实之平衡。
2018-12-08 11:52:19 作者：姜澎

中国商学院已经没有榜样可以模仿，全世界的商学院都面临着同样的问题，重视穷理而忽视务实。在今天首届中国行业发展高峰论坛上，上海交通大学安泰经管学院宣布成立行业研究院，希望通过和行业的结合寻求大学商学院的穷理和务实之间的平衡。

商学院最早诞生在美国发展至今已有近百年历史，出现之初就是为了解决商业实践中出现的问题。但是在上世纪五六十年代时福特基金会对商学院进行了调研后提出，商学院的研究缺乏严谨性，必须要用量化的方法和数学模型来研究管理学。在这一倡导下，商学院才开始追求研究的学理性。但是半个世纪过去了，商学院却从最初的只有“务实”没有学理，走向了另一个极端，缺少了务实。上海交大安泰经管学院院长陈方若教授说：“这是全球商学院面临的问题。在过去几十年中，我们的商学院一直以发达国家的商学院为参照系数，但是现在放眼望去，我们已没有榜样了。即便是被行业内认为是穷理和务实比较平衡的哈佛商学院也越来越像不务实的哥伦比亚大学商学院了。”

他说，作为大学里的商学院，最重要的特点是知识创造，但是目前知识创造的服务对象只是学术界，而服务对象的另一半，实业界并没有受到足够的重视。仅以安泰经管学院为例，学院有10个系，就是10个分隔开的小学科，而商学院的教授甚至从读研究生开始就几十年如一日地在某一个学科，甚至学科之内的狭窄领域里精耕细作，穷理做得越来越好，而离务实却越来越远。

他的看法也得到了业界的支持。中国中东欧金融公司董事长，中国工商银行原董事长姜建清说，目前中国的银行业共有409万人，总资产规模也是世界第一，每天业务笔数超过10亿笔。但是目前全球银行业都进入了发展的战略迷茫期，新的模式尚未形成。但是，在中国的学术界，对于银行业发展实务的研究关照，却少之又少。在这样的情况下，尤其需要学界为实务提供支持。

按照规划，上海交大安泰经管学院将通过行业研究院的模式，重新建立对学术的认识。以行业为载体，研究行业发展的内在规律和未来趋势，通过行业研究打破学科之间的分割。并且建立多元学术评价体系，通过价值评判来改变学术研究的不良的趋势。并且通过行业研究来培养行业大师，平衡商学院务实和穷理之间的关系。
