武汉归来就结婚！上海为援鄂医务人员举行集体婚礼

2020年05月02日 14:20:30 来源： 新华社
  • 评论0
  • 打印
  • 字大
  • 字小

        新华社上海5月2日电（记者仇逸 刘颖）“我愿意”“我愿意”“我愿意”……一句句坚定而深情的誓言回响在婚礼仪式的大厅，现场来宾无不为之感动。

        5月1日下午，上海交通大学医学院附属仁济医院为十对新人举办了一场特殊的集体婚礼。其中，十位新娘和一位新郎是上海市第八批、仁济医院第三批援鄂医疗队队员。十对新人在至亲好友的见证下，在同事、朋友、“战友”们的云祝福中，宣告爱的誓言，喜结爱的契约。

        2020年初，一场突如其来的新冠肺炎疫情打乱了许多人的生活节奏。2月19日，仁济医院156名医疗队员义无反顾地逆行驰援武汉。当爱情遇到疫情时，出征中的十一位准新人毅然决定或取消婚礼仪式、或推迟原定的婚期，报名去抗疫前线。

        十一位准新人（十位新娘和一位新郎）并肩在雷神山医院战斗了整整48天，而九位新郎默默坚守在后方，他们思念、担忧和不舍，然而他们更多的是鼓励对方和守护双方的父母。

        时间和距离阻挡不了彼此的爱恋和支持。1日，新人们穿上洁白的婚纱和帅气的西装，完成了人生中最重要的一场仪式，实现了“武汉归来，我们就结婚”的愿望，兑现了爱的承诺，他们的爱情经历了特殊的考验，彼此之间更加珍惜。

        上海交通大学党委书记杨振斌作为证婚人，向新人们送上了诚挚的祝福。他说：“疫情虽然阻隔了相聚，却见证了爱情最动人的模样，他们将爱埋在心底，将更多的温暖洒向人间。在经历了一段曲折浪漫的爱情之旅后，我从他们如今洋溢着笑容的脸上读出了幸福，更读出了那一份坚定。”

        仁济医院党委书记夏强代表娘家人向新人们表达了祝福，希望这场婚礼连同征战武汉的时光，成为他们镌刻在生命里的美好记忆，也希望新人们能共同携手描绘出人生的绚丽画卷。

        仁济医院护士、援鄂医疗队员李依的妈妈喻秋英感慨，自己的小女孩在仁济医院领导和老师们的关怀和帮助下，迅速成长，学会了担当，李依主动报名前往武汉支援，发挥自己的光和热。如今女儿即将为人妻，为人媳，更成长为有责任心的白衣天使，身为母亲的她感到无比骄傲和自豪。

        仁济医院消化科医生、援鄂医疗队员肖潇作为新人代表发言，她骄傲地告诉爸爸妈妈，“在圆满完成援鄂任务之后，我们已经长大，可以保护好自己，也可以去守护别人了。未来，请你们相信我们也一定可以经营好我们的小家。”
