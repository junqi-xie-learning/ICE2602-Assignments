从小到大的默契 双胞胎姐妹花一分之差双双考进上海交大

来源：新民晚报     记者：易蓉     作者：易蓉 通讯员 皎轩     编辑：钱文婷     2019-07-02 13:37 | |

　　有这样一对“卓越”双胞胎姐妹花，她们经常会同时哼起同一首歌，会同时产生同样的想法，做出同样的动作，互相陪伴共同努力。姐姐叫蔡亦卓，妹妹叫蔡也越，她们是妥妥的高颜值学霸，两人今年高考分数仅1分之差，双双被上海交通大学工科试验班录取。

图说：右边姐姐，左边妹妹  来源/采访对象供图（下同）

　　作为双胞胎，姐妹俩从小就成为了彼此的竞争对手。“妹妹这次数学比姐姐高2分”“姐姐这次语文考得比妹妹好”……在这种好学上进的家庭氛围中，姐妹俩自觉拿起课本，愈发认真求学。虽然有时她们也会因琐事拌嘴，但这丝毫不破坏姐妹俩的“革命情谊”。

　　求学路上，姐妹俩满怀梦想，她们互相陪伴，互相照顾。姐姐考砸时，妹妹贴心温暖的安慰与鼓励；妹妹骨折时，姐姐风雨无阻地推着轮椅上下学以及无微不至的照顾。姐姐坦言，她们一直都在学习与生活的世界里互相搀扶，互相成就。

　　高一时，在学长学姐情系母校的宣讲中，姐妹俩不约而同将目光投向上海交大。向着梦想努力，怀着对数理的热爱，在高中选课中，她们都毫不犹豫地选择了物化生三门学科。三年梓荫华年，她们经历过沮丧，产生过迷茫；但梦想是会发光的，它总能照亮前进的方向。

图说：姐妹俩一起上学，一起放学

　　在选专业的时候，别人都说理科偏理论，工科偏技术。在了解工科之后，姐妹俩发现，这句话也不确切，在她们的理解中，工科是拿起知识和技术的工具，在名为“实践”的高山里磨练技艺，选择工科，意味着要在学习中实践，在实践中学习，将理论用于实际，在一次又一次动手中寻找解决问题的办法。选择工科，就是选择了学习专业技术的道路，用自己的双手和汗水，在未来贡献自己的一份力。

　　如今，高考成绩出炉，凭借着姐妹俩从小的默契，她们也仅以一分之差同时被录取，即将前往上海交大报到。面对梦想之中的大学和充满挑战的大学生活，她们也将拿出小时候“打怪”的勇气，并肩作战，在更大的人生舞台追寻各自的理想。

　　新民晚报记者 易蓉 通讯员 皎轩
