持续创新，“小刀具”助力我国高端制造

上海交大陈明教授团队“高效切削刀具技术”方面成果列入工信部“百项技术创新推进计划”
复杂型线刀具应用于汽轮机转子轮盘榫齿槽高效精密制造

第20届工博会将于19日开幕。高校展区75所高校将带来751个项目，参展高校数量及项目数量创下历年来新高。高校展区紧扣工博会主题，体现高校“智能、绿色”的创新理念，推进高校创新创业发展。同时，围绕国家实施创新驱动发展战略、上海建设具有全球影响力科创中心的定位，借助展台传递高科技创新成果，体现高校的科创能力和服务社会实力。

工博会上有哪些亮点？晨报记者提前探营，推出系列报道，今天第一篇聚焦的是上海交大陈明教授团队的“高效切削刀具技术”。

晨报记者  董川峰

在航空航天、能源和汽车等高端制造领域，难加工零件的高效切削对刀具耐磨损和长寿命有着很高的要求。我国刀具市场总消费规模虽然世界第一，但国产高效切削刀具只占约15%，高端制造领域高效切削刀具严重依赖进口。上海交通大学机械与动力工程学院教授、制造技术与装备自动化研究所所长陈明团队历经20余年，发明了金刚石刀具涂层制备新技术，研制出涂层成套装备，构建了高效切削刀具制备技术链，在国内首次建立了具有自主知识产权的金刚石涂层刀具生产线并在行业内推广。开发出系列化刀具设计与应用专业软件，最终形成7大类18个系列高效切削刀具产品。

成果广泛应用于航空、航天、汽车、汽轮机和发动机等重大装备制造领域。高效切削刀具国产化对我国重大装备研制技术安全性起到保障作用，助力中国高端制造。

勇于挑战，持续创新

切削加工是机械制造基础工艺，刀具是加工系统关键，常常被比喻成机床的牙齿，起到四两拨千斤的作用。

传统刀具在高速切削中精度保持性差、耐用度低，无法满足航空航天、能源、汽车等高端制造领域对刀具的高精度和长寿命要求。据统计，我国刀具消耗全球第一，但高效刀具长期依赖进口，难以满足加工成本进一步降低和重大装备研制国产化水平进一步提高的要求，实现高效切削刀具国产化的需求已迫在眉睫。

陈明教授团队在“973”“863”“04”重大专项等项目的支持下，围绕高效切削刀具的设计、制备与应用等方面开展研究。在研究过程中，他领导的研究团队注意到国产高效切削刀具在设计方法和标准上的缺失问题，提出了形性协同精准设计方法，完成了基础理论创新和关键技术突破。开发出系列化刀具设计与应用专业软件，最终形成7大类18个系列高效切削刀具产品。陈明教授团队针对国产高效切削刀具缺乏关键制备技术的难题，发明了金刚石刀具涂层制备新技术，研制出涂层成套装备，构建了高效切削刀具制备技术链。与传统刀具寿命相比，其开发的金刚石涂层刀具寿命提高了整整10倍；与此同时，他领导的团队开发了系列化大规格复杂型线刀具，为百万千瓦级汽轮机和重型燃机国产化提供大规格型线轮槽铣刀和组合拉刀，打破了国外在该领域的刀具技术垄断。

促高端制造业可持续发展

多年来，在陈明教授的带领下，上海交通大学切削磨削与刀具研究团队，已经在高速切削技术、高效磨削技术、先进刀具开发与超硬砂轮应用研究的路上穿越无数风雨与坎坷，也收获了掌声和鲜花。研究团队在“高速切削技术”方面成果被列入工业和信息化部“百项技术创新推进计划”。

目前，研究团队拥有较为完善的先进仪器设备，并与国内汽车、航空航天和发电设备行业的代表性企业建立了长期的“产、学、研、用”合作关系，研制的高效切削刀具也已在中国商飞上海飞机制造有限公司、上海电气电站设备有限公司上海汽轮机厂等40多家高端装备制造企业遍地开花。

高效切削刀具技术上的成功，在一定程度上会对我国重大装备研制技术安全性起到保障作用，并促进高端制造业可持续发展。
