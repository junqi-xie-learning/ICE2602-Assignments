【青春的故事】唐文涛：用奋斗谱写着青春的华美篇章

2019年04月24日 14:26:41 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

 

    唐文涛积极整合实验室资源，短时间内学习了上千篇文献，尝试了近百次实验，制备了上千块的电池，由于钙钛矿电池通常在导电玻璃片上制作，实验室就戏称他为“片子手”。新华网 严晓瑜摄

    新华网上海4月24日电（吴一航）初见唐文涛，和普通大学生一样，时尚的休闲装搭配着运动鞋，看起来十分精神。看到我们车子停下后，他快步走上来，和我们打完招呼还主动帮忙扛起了设备。“阳光、礼貌、爱笑”这是唐文涛给人的第一印象。

    1993年出生的唐文涛是上海交通大学材料科学与工程学院2015级在读博士生，可以说是一名地地道道的“工科男”。可别小看这位“90后”博士生，在他身上可是有很多“第一”的。先别急，要想了解唐文涛还得从他身上的这几个关键词说起。

    第一个关键词：“钙钛矿模块化研究”。唐文涛所从事的“钙钛矿模块化研究”用更加经济安全的新方法制备出更高效率的钙钛矿太阳能电池，为实现大规模低成本太阳能发电的目标贡献力量。

    唐文涛在精确称量药品 配置钙钛矿前驱液。新华网 严晓瑜摄

    钙钛矿光伏技术凭借其成本低、效率高等巨大优势吸引了世界的目光，成为新一代最有希望平价上网的光伏技术。在唐文涛看来，钙钛矿很有可能真正代替传统能源，可以集成到屋顶、玻璃幕墙、汽车等，一旦实现就可以真正做到“零排放、零补贴、零污染”。

    怎样把钙钛矿电池做大做好，成了他整天思考和挂在嘴边的问题。那时候他一天超过16个小时泡在实验室，一边看文献，一边不停地学习高效器件的制备。“由于钙钛矿器件通常在FTO玻璃片上制作，到现在我已经刷了几千张FTO玻璃片了，实验室同学给我取了个绰号叫‘片子手’”。唐文涛俏皮地说。

    图为实验室中唐文涛制备的钙钛矿太阳能电池，科研项目中每天的工作。新华网 严晓瑜摄

    第二个关键词：“发表《自然》杂志(Nature)正刊”。功夫不负有心人，唐文涛和团队成员经过无数次的尝试和失败后，结合钙钛矿溶液成膜独有的特点和前期的工作经验，创新性地发现通过胺类气体和钙钛矿合成原料间离子-分子间的质子交换和氢键作用原理，实现了大面积、均匀、高质量钙钛矿薄膜的制备。此项研究成果实现了有效面积36.1cm2的钙钛矿电池模块的制备，在国际认证机构日本AIST获得了12.1%的认证效率。

    这一模块的效率值作为首个钙钛矿太阳电池模块效率值被《Solar Cell Efficiency Tables》杂志正式收录，最终也被发表在国际顶级期刊《自然》杂志(Nature)正刊上。那时刚刚24岁的唐文涛成为交大以第一作者（共同）身份发表《自然》杂志（Nature）正刊的最年轻的博士生。

    第三个关键词：“世界纪录”。在研究过程中，唐文涛申请了4项国家发明专利，已通过授权2项。通过对原有方法的研究和新方法的开发，他使36cm²钙钛矿器件的效率进一步提升到了14.2%，这一结果同样获得了国际权威第三方机构Newport的认证。

    平时爱好吉他和网球。青春对于唐文涛来说，不单单是实验室中的科研，还是手中的琴谱和心中的梦想。新华网 严晓瑜摄

    平时爱好吉他和网球。青春对于唐文涛来说，不单单是实验室中的科研，还是手中的琴谱和心中的梦想。新华网 严晓瑜摄

    采访期间，记者问唐文涛在这样一种枯燥、忙碌的科研生活里，为什么他身上似乎有着充沛的活力？“只有强健的体魄和放松的精神状态才能让科研生活更高效、更持久。平时我会打网球，不断跑动、奋力地抽球，不仅能够锻炼身体，也能够让我的精神压力得到释放，也会弹弹吉他什么的。”唐文涛说，自己早在本科期间就组建过乐队，担任主音吉他手，参加过数十次院系、学校和校外演出，这些课余活动也充实了自我。

    唐文涛，一位优秀的“90后”博士生，正用奋斗谱写着青春的华美篇章。
