上海交大师生为垃圾分类贡献智慧

2019-06-26 18:00 光明日报客户端 曹继军

上海生活垃圾分类即将步入法制化轨道，是一项综合性、系统化、长期性的社会工程，如何构建适用我国社区的垃圾分类治理模式呢？6月26日，记者获悉，上海交大学子积极关注社会发展难题，利用自身学科优势和各方资源，借鉴他山之石，为我国垃圾分类资政建言，贡献交大智慧。

新加坡是一个非常注重环境保护的国家，新加坡的垃圾管理也常常被大家称道。“上海交大在六年前就与新加坡国立大学对接，其中一个重要的项目就是垃圾费分类，而垃圾分类最重要的就是管理。”项目组负责人、国际与公共事务学院副院长张录法介绍，此次团队由10余名交大学子、10余名中外教师组成，将与新加坡相关政府机构、大学（智库、研究中心）以及垃圾处理企业交流访谈，并对新加坡普通民众进行问卷调查，深入借鉴新加坡垃圾治理的成功经验，以解决上海垃圾处理过程中存在的主要问题，并提出上海在垃圾回收、资源化利用及最终处置等方面采取的一些对策。

记者了解到，该项目团队源于上海交大最新启动的2019年“全球挑战计划”，除垃圾分类治理项目之外，还有关于非洲用药标准调研、肯尼亚吸引外资法律环境评估、拯救传统村落等13支国际化项目团队。国际合作与交流处副处长姚雪表示，该计划旨在培养青年学子的责任意识和家国担当，鼓励学生拓展全球视野，融入多元文化，提升学生自身的领导力、全球胜任力以及交大的国际影响力。

（光明日报全媒体记者曹继军 通讯员魏娜）
