市科协主席陈赛娟：主要精力还是研究白血病，尽最大努力当好“掌门人”

创新之城 2018-09-25 19:45
来源：上观新闻 作者：黄海华
作为一名女性科学家，陈赛娟认为家庭责任不可或缺，同时也要有“向上的”社会责任感。

今天（9月25日）下午，在上海市科协第十次代表大会上，中国工程院院士陈赛娟当选为上海市科协新一届主席，成为继谢希德、叶叔华之后第三位女性主席。会后，陈赛娟第一时间接受了媒体的采访。

 

要实现25%的公民具备科学素质，还有许多工作要做

 

“我在法国巴黎留学时，就和上一届市科协主席陈凯先认识了，已有30多年。他一直是老大哥，如何做好科协工作，我要向他学习。”陈赛娟坦言，对于市科协主席这一新角色，自己在各方面还很不够，需要不断学习。在谈到谢希德和叶叔华这两位前主席时，她表示自己非常敬仰她们，她们无论是科研工作还是科协工作都非常杰出，希望自己从她们身上汲取力量。

 

第十次全国公民科学素质抽样调查结果显示，上海公民具备科学素质比例达到21.88%，位列全国第一。“这里还有很多可以提升的空间。到2020年末，上海力争要实现25%的公民具备科学素质，市科协还有许多工作要做。”

 

陈赛娟曾在上海血液学研究所发出过“严惩学术不端行为，维护学术诚信”的倡议，在谈到加强科研诚信建设时，她认为科研诚信对于一名科技工作者来说非常重要。要向老一代科学家学习，杜绝不规范的学术问题。对于年轻的研究生而言，更要在一入学的时候就进行相关教育。现在的学生，可能心态比较浮躁，需要及时了解他们的科研进展，给予指导。

 

主要精力还是会放在对白血病的研究上

 

谈到今后工作的重心，陈赛娟表示，她每年都在招收研究生，主要精力还是会放在对白血病的基础研究上，这是她作为一名科学家应尽的责任。

 

瑞金医院血液学研究团队经过四代科学家的努力，创造了治疗急性早幼粒细胞白血病的“上海方案”。陈赛娟带领团队提出并实施了白血病基因组解剖学计划，在国际上率先发现了一批白血病发病相关突变基因和融合基因，揭示了新的发病遗传学机制。该团队发现了由砒霜制成的砷剂的直接药物靶点，它能导致急性早幼粒细胞白血病细胞走向分化和凋亡。这一重要发现，奠定了“上海方案”——全反式维甲酸和砷剂协同靶向治疗的分子基础。如今，“上海方案”使得急性早幼粒细胞白血病缓解率达到95%，五年生存率上升至92%，成为第一个可治愈的白血病。

 

“对于那些治疗后容易复发的高危患者，我们正在研究相关分子标志物，希望也能提高他们的生存率。我们的目标，是让每一个患有急性早幼粒细胞白血病的病人都有机会治愈。”

 

对于年轻的科技工作者，还是要以鼓励支持为主

 

从一名上海纺织女工到中国工程院院士，陈赛娟的经历颇有几分“传奇”色彩。她觉得自己很幸运，离不开改革开放这一历史机遇。初到法国巴黎攻读博士学位时，她对于法文一窍不通，本想先学习法语，但导师希望她一边做研究一边学语言。这无疑是个更大的挑战，她凭借自己的努力，不到4年时间，既学习了法语，还以最佳评分通过了论文答辩。“能吃苦，是我们这一代人的特点，但基础不如现在的年轻人。”陈赛娟说，对于年轻的科技工作者，还是要以鼓励支持为主。

  

作为一名女性科学家，陈赛娟认为家庭责任不可或缺，同时也要有“向上的”社会责任感。她透露，自己年轻时就喜欢田径运动，因此身体素质比较好，这对于女性在平衡事业和家庭的关系时很有帮助。

如何使科协组织成为思想智库？如何进一步发现人才？如何成为上海200万科技工作者的温馨之家？陈赛娟表示，还有很多问题需要思考和研究，她会尽自己最大的努力，当好这一届“掌门人”。

 

陈赛娟个人简历：


血液细胞和分子遗传学专家。1951年5月21日出生于上海市，浙江鄞县人。1975年毕业于上海交通大学医学院（原上海第二医科大学），1981年获医学硕士学位，1989年获法国巴黎第七大学科学博士。上海交通大学医学院教授、博士生导师，上海交通大学医学院附属瑞金医院终身教授。上海血液学研究所研究员、名誉所长，医学基因组学国家重点实验室主任。第十届全国人大代表。获得国家自然科学二等奖、上海市自然科学奖特等奖等国家和省部级科技奖10余项。2003年当选中国工程院院士。
