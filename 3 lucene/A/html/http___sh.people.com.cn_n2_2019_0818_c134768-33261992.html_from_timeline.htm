《大国坚守》新书首发座谈会在沪举行

2019年08月18日22:02  来源：人民网-上海频道
 

人民网上海8月18日电 （曹玲娟 房敏婕） 由文汇报社和上海交通大学出版社联合主办的“走近一带一路，解读大国外交风范――《大国坚守》新书首发座谈会”今天在2019上海书展现场举行。

2019年是中华人民共和国成立70周年，也是新中国外交70周年。1949至2019，中国外交一路走来，从举步维艰，到前所未有地走进世界舞台中心，着实不易，外交智慧和外交定力功不可没。新时代，外交环境风云变幻，黑天鹅事件频发，中国外交仍需坚守，坚守什么？如何坚守？

为更广泛传播中国特色大国外交的内涵，上海交通大学出版社策划出版了《大国坚守――中国外交在十字路口的历史抉择》一书。该书由中外外交专家、学者贾庆国、苏格、崔立如、黄仁伟等领衔解读，以“接地气”的语言梳理总结70年来中国外交经验与教训，同时对中国外交新变化、新应对、新实践等进行深刻思考、通俗表达，引领读者思考。

在该书序言中，中国现代国际关系研究院院长袁鹏评论道，“近年来，继‘国学热’之后，国人似乎又兴起了一股‘国际热’。人们对国际问题加大关注，本身不是坏事，这是中国由大向强转进过程中的一个必经阶段。唯其如此，中国当下尤其需要严肃的媒体人和负责任的学者，于喧嚣浮躁中静下心来，以专业的精神和操守给大众‘传道授业，释疑解惑’。”

具体来看，在《大国坚守――中国外交在十字路口的历史抉择》一书中，专家学者聚焦中美关系、双边外交、中国主场外交等几个领域中的重要模块，从历史、现实等不同角度理性观察中国外交政策，在中美关系和“一带一路”等话题上既保留了历史纵深感，又体现了学者的时代关切。该书既是新时代中国外交的实录，也是学者们解读中国和世界关系的观点引领。

新书首发座谈会现场，上海交大出版社总编辑李广良、文汇报社党委副书记谢海光、复旦大学一带一路与全球治理研究院常务副院长黄仁伟、华东师大俄罗斯研究中心冯绍雷一起为新书揭幕。

(责编：严远、韩庆)
