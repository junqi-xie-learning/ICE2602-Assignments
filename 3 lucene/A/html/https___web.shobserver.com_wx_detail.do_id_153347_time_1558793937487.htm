宁光院士：AI医生查眼珠抓出糖尿病！凭大牌医院26个专病医生怎么筛5万糖尿病人？

创新之城 2019-05-25 22:15
来源：上观新闻 作者：徐瑞哲
在全身唯一能看得到毛细血管的地方，筛查得出患病率超26%

体检时，睁大了眼睛，被拍过眼底照片吗？有过，但那是眼科医生，不会从眼底毛细血管的视角来查验糖尿病。但各科皆通、覆盖全病种的人工智能AI医生可以做到，一旦发现查到眼珠有问题，就把这份情报从眼科给到内分泌科了。

 

今天下午，浦江创新论坛“AI+医疗：赋能健康共同体”未来科学论坛上，中国工程院院士、内分泌专家宁光教授透露，目前全国500家医疗机构接入国家标准化代谢性疾病管理中心（MMC），利用这种AI眼底筛查工具，已总计筛查5.26万人次，检查发现糖尿病视网膜病变（DR）1.38万人次，患病率超过26%。

 

我国糖尿病患者在6000万人规模，并且仍以每年2%的速度增加，这一增幅高于可比的美国。常有病人抱怨，检查了一大堆项目，化验报告厚厚一叠，结果给医生看时，医生就对了对名字，其他几乎看都不看。医生若被问道，回应一句：“查了又查，不就是血糖高吗？”

 

宁光坦陈，当下医生资源和医疗资源不足，比如某大牌医院总共只有26个糖尿病诊疗的医师，不包括急诊，每天需要问诊“千量级”的病人，并管理“百量级”的病床，这种工作量没有人工智能辅助，难以实现人民满意的医疗。

 

全身唯一能看得到毛细血管的地方——眼底，成为突破口。此前，内分泌科有大量慢病患者，病症也都反映在眼部上。其中一部分眼病比较轻微的患者在过去往往被忽视：眼科认为他们的病情不足以在眼科治疗，内分泌科又不懂眼科这些检查，也无法追踪病人。

 

而通过眼底彩照，体素科技发明的“视网膜全病种解决方案”不仅反映眼部疾病，还能反映高血压等慢性病的累积，而用眼底检查实现所有全病种检查及慢性病评估是被大大低估的。由于现有医疗机构的眼科人力资源不足，只能管理到应该筛查的慢病患者人群1/10左右，还远远没有发挥对高危人群的早防早治作用。

 

宁光介绍，目前基于AI的全病种眼底筛查解决方案，成为国家标准化代谢性疾病管理中心主要的眼底AI筛查工具，在以三甲医院内分泌科为主的分中心应用，上线第1年内就筛查糖尿病患者超过28000人次，月筛查量超过4000人次，帮助中心更好管理糖尿病患者的眼底并发症，大大提高了早期视网膜病变的筛查率，从而也很大程度提升糖网病人的治疗效果。

 

作为研发方，体素科技首席执行官丁晓伟博士告诉记者，按工作流程，先采集眼底照片，上传MMC糖网筛查平台，通过AI+人工眼底阅片，先由AI自动读片，进行病变检测，并分级按轻、中、重度来描述病变，再由眼科医生阅片复核。整个过程，只要1到5分钟出结果，远远少于从眼科到内分泌的奔波时间。

 

 从上海市一院、六院到瑞金医院，通过筛查大数据接入，中心实时可见筛查结果。宁光表示，希望以此平台管理1000万糖尿病病人，相当于全国约1/6病人。同样，眼底人工智能的阅片技术不仅可以为缺乏眼科专科专家的偏远地区、社区卫生中心、体检中心服务，更可以为除眼科之外的内分泌科、肾内科、神经内科等跨科室服务。

 

作为上海交大人工智能研究院副院长，王延峰表示，AI作为“超级医生助手”，利用这一类多学科综合诊断（MDT）系统，以病人为中心，通过多科室现场协作，获得最佳诊疗方案。

 

他说，全国平均每天超过1万人确诊为肿瘤，上海现有肿瘤患者36.7万人左右，而上海某顶级三甲医院每年对单病种进行多学科综合诊断仅200例，“没有AI帮忙，现有医疗资源无法满足这种多学科综合诊断要求；而经多学科综合诊断后，对照之下可提高患者生存率约10%左右。”
