陈豪阮成发会见上海交通大学党委书记姜斯宪

发布时间：2019年04月28日 18:16:29  来源：云南网

陈豪阮成发会见上海交通大学党委书记姜斯宪

共同见证签署相关合作协议

4月28日，省委书记陈豪、省长阮成发在昆明会见上海交通大学党委书记姜斯宪一行，并共同见证签署相关合作协议。记者 张彤 摄

　　云南网讯（记者 瞿姝宁）4月28日，省委书记陈豪、省长阮成发在昆明会见了上海交通大学党委书记姜斯宪一行。双方就进一步深化高原湖泊保护治理、科研平台建设、脱贫攻坚、医疗卫生、人才培养等方面合作进行了深入交流，并共同见证签署《省科技厅、上海交通大学 、大理白族自治州人民政府关于建设云南洱海湖泊生态系统野外科学观测研究站的合作备忘录》《上海交通大学昆明理工大学战略合作协议》。

　　上海交通大学常务副校长、中国科学院院士丁奎岭，云南省刘慧晏、陈舜、杨杰参加。
