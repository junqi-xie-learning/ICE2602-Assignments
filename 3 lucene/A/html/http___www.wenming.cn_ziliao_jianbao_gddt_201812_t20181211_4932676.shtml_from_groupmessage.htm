【高校文明校园创建调研座谈会发言③】上海交通大学：以大学文化构建培育时代新人大格局
发表时间： 2018-12-11来源： 中国文明网
 
 
 

　　上海交通大学在文明校园创建过程中，着重突出以文化人以文育人，探索构建世界一流、中国特色的大学文化内涵体系，牢固树立师生的文化自信，努力构建培育时代新人大格局。

　　筑大学精神之魂，底色鲜明“举旗帜”。上海交大在办学过程中始终注重红色基因传承，把培育和践行社会主义核心价值观作为大学精神之魂，将立德树人根本任务抓牢抓实。一是加强和改进思想政治课程，推进马克思主义理论学科建设与思政课教学深度融合，书记、校长带头上好思政课，带动更多的校院领导及教学名师走上讲台。二是实施浸润式思政教育，积极推动“思政课程”向“课程思政”转变，使各类课程与思政课同向同行。如，定期组织“全国道德模范事迹报告会”等先进典型学习宣传活动，营造“崇尚英雄、学习英雄、捍卫英雄、关爱英雄”的浓厚氛围。三是理想信念教育贯穿始终，从“选择交大就是选择了责任”到“思源致远 砥砺前行”，坚持党委书记讲授“新生第一课”，扎实组织开展毕业生远航教育，引导大学生“领悟为学之道，秉持做人之本，树立报国之志”，坚定为祖国建功立业的决心。

　　培优秀传统之根，思源致远“兴文化”。将中华优秀传统文化融入学校文化，让学生领悟其中蕴含的思想观念、人文精神和道德规范。将优秀传统文化“引进来”。开展传统艺术和经典艺术教育，每年引进京剧、昆曲、评弹、越剧、沪剧等戏曲的经典剧目展演，承办教育部“礼敬中华优秀传统文化项目评选”活动，创立“菁萃·大家讲堂”，邀请人文艺术名家走进交大开设分享课，推广“唐调”“摄画”等非遗项目，引导大学生感受美、理解美、传播美。让学校历史文化资源“活起来”。挖掘交大自身“校训文化”“老建筑文化”“校友故事”等资源，通过人物传记、校史编述、回忆录等多种形式，让交大往事跃然纸上。整合高校博物馆资源，创办“堂——世界博物馆馆长文博讲坛”，邀请单霁翔、王旭东等文博专家开设专题讲座。学校连续五届获评全国高校校园文化建设优秀项目特等奖。

　　抓文化育人之核，强化担当“育新人”。一是突出杰出校友的引领作用。先后出版钱学森系列学术著作20余种，创作校园话剧《钱学森》并进行全国巡演，设置钱学森图书馆、钱学森路、钱学森班，让“爱国、求真、奉献、创新”的“学森精神”成为交大人的“精神内核”，增强学生的使命担当。二是树立“立德树人、教书育人”的价值导向。建立教书育人、科研成果和管理服务三大奖励体系，通过“我心中的好老师”、“教书育人楷模”事迹宣传、“校长奖”、“凯原十佳教师”等活动，树立师德师风典型，激励师生见贤思齐，对有违师德的教师，实行“一票否决”。三是打造环境育人“大课堂”。将立德树人、文化传承、学术探究、科普教育、公益实践等融入环境建设中，探索环境育人“大课堂”模式，打造学以致用的“情境实验室”，使校园环境成为师生立身做人的“无言之师”。（摘自《未成年人思想道德建设工作简报》2018年第22期）
