第五届君子文化论坛在沪举行 聚焦“家国情怀与君子文化”
2019年08月24日 20:01   来源：中新网上海  

　　中新网上海新闻8月24日电 (记者 许婧)第五届君子文化论坛24日在上海交通大学举行，来自北京大学、清华大学、中国社科院等全国六十多所高校、科研机构的百余位专家学者齐聚一堂，围绕“家国情怀与君子文化”主题展开研讨和交流。

　　会上宣布，著名哲学家、国际形而上学协会主席、哲学国际研究院(IIP)院士、教育部长江学者特聘教授杨国荣出任上海交大中华君子文化研究中心主任，余治平教授为执行主任。

　　君子文化是中国传统文化的重要组成部分。近年来全国的君子文化研究大热，每年都有许多不同层面的君子文化研究学术论作发表出版，有各种层次的君子文化研究学术会议(论坛)召开，诸多高校、科研院所纷纷成立君子文化学术研究机构。

　　本次论坛聚焦于君子文化中的家与国、个体与族群、小我与大我之间的历史实践和义理关系，深入挖掘君子文化蕴含的思想观念、人文精神、道德规范，结合社会现实，予以积极继承和创新发展，让君子文化展现出永久魅力和时代风采。

　　上海交通大学党委副书记顾锋在致辞时表示，创建123年以来，交大的创建、发展和兴盛都紧跟了国家的命运和时代的脉搏。历经淬炼，百折不挠，而浴火重生。交大著名校友、中国“导弹之父”钱学森先生不畏艰难，决意回国，“造次必于是，颠沛必于是”，其可歌可泣的感人事迹就是对“家国情怀”的最好诠释。

　　复旦大学上海儒学院理事长谢遐龄教授、北京大学哲学系郑开教授、浙江大学何泽华教授、安徽省君子文化研究会会长钱念孙研究员、复旦大学杨泽波教授、中国人民大学韩星教授、中国社会科学院罗传芳研究员、清华大学方朝晖教授等围绕君子的存在结构、家国一体、儒家思想中的君子、忠孝德行、新时代君子文化的传承与超越等内容作主题报告。

　　本届论坛由上海交通大学中华君子文化研究中心承办，安徽省君子文化研究会、湖南省君子文化研究会、浙江大学君子文化研究中心、安徽省社会科学院君子文化研究中心、江苏省社会科学院君子文化研究中心共同协办。(完)
