时间反演多重演绎 科艺结合以美育人
2018李政道科艺大奖赛获奖作品揭晓
数字交互作品《上一秒，光》和坍缩态宇宙模型《先知的梦魇》获一等奖

2018-11-23来源：中青在线 作者：魏其濛

　　中青在线上海11月23日电（中国青年报·中青在线见习记者 魏其濛）以“时间反演”为主题的李政道科学与艺术作品展将于11月24日在上海交通大学李政道图书馆展出。记者在23日的探营活动中看到，参展作品涵盖雕塑、油画、水墨画、多媒体等多种艺术形式，其中获2018李政道科艺大奖赛一等奖的是上海视觉艺术学院谭壹建的数字交互作品《上一秒，光》和鲁迅美术学院吴彤雕塑工作室韩韬的坍缩态宇宙模型《先知的梦魇》。

　　据介绍，这是上海交通大学举办的第五届科艺活动，每届科学主题皆与当年的研究热点相契合，并得到李政道的认可。今年的科学主题是“时间反演”，正是当今物理学的研究热点之一。据了解，早在1956年，李政道和杨振宁就提出弱相互作用下宇称不守恒，即空间反演对称性的破缺，打开了微观物理研究的另一扇大门；此后，物理学家对于微观下时间反演的对称性进行了探索和研究，并最终证实了时间反演对称性的破缺。李政道在本次活动的致辞中写道：“时间之河的流动是不可逆转的，但是在想象的空间中，我们可以令时间之河倒流，并进一步研究微观环境下物理规律的时间反演对称性。”

　　本次展览的主体画《宙》由刘海粟研究中心主任朱刚与科学家反复沟通交流后创作。“宙”意指古往今来所有的时间，作品用天地、日月、四象、八卦等元素诠释了时间的无限循环，用炫丽的色彩展现了科学世界的神奇与美丽。另外，本届大奖赛共收到了全国147人的169件作品，46人获奖，49件获奖作品将展出，作品覆盖范围比往年有显著增长。其中交大师生共提交了25件作品，有10件获奖，如杨清枫的作品《时间反演对称》，展现了用相近色、几何体模糊彼此的界限，表达画面时间反演的对称性；余辉的作品《人生》旨在体现时间反演对称性及其破坏的相关理论。

　　据悉，本次展览由上海交通大学、中华国际科学交流基金会和上海科技馆联合主办，是上海交通大学李政道科学与艺术讲座基金系列活动之一，该基金设立的初衷是以艺术的手段呈现科学之美，体现“做好美育工作，弘扬中华美育精神”。展览为期30天，将持续到12月24日。
