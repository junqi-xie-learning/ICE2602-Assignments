上海交大吴文俊数学中心成立

2019年05月09日 19:00:37 | 来源： 新华网

　　5月9日，上海交通大学吴文俊数学中心揭牌。当日，“纪念吴文俊院士诞辰一百周年暨数学科学与数学科学史国际学术研讨会”在上海交通大学举行。研讨会上，上海交大吴文俊数学中心正式揭牌，该中心将致力在基础数学、数据科学、数学史领域开展相关研究。新华社记者 刘颖 摄
