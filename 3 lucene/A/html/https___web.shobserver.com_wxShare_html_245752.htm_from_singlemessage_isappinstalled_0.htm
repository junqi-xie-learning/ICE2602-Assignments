别忘病毒源，聚焦“全健康”！中英名校合作解决“人—动物—环境”复杂问题

教育在线 2020-05-08 18:58
来源：上观新闻 作者：徐瑞哲
“全”字，纳入“人”“土”“王”三要素，可以概括健康离不开管理的统驭之势——人类健康、动物健康和环境健康三者统合为一个健康整体。

新冠病毒的源头，究竟是什么？这正是一个从环境到动物到人的复杂健康问题，需要从“人—动物—环境”整体视角加以解决。

“疾病，是全人类共同的敌人，‘健康融入万策’是大势所趋。”正如《同享一世界，共筑全健康》倡议所言，“期盼弥合人体医学、动物医学、环境科学和公共卫生学、健康管理学乃至法律学科之间的‘空白点’和‘盲区’……”

北京时间8日傍晚，作为中英老牌名校，上海交通大学与创建于1583年的爱丁堡大学联合建立“全健康研究中心”，并向在线接入来自十多个国家和地区的同行发出中英文倡议。对此，各国际组织代表跨越时差，通过视频连线方式回应，包括世卫组织（WHO）、联合国粮农组织（FAO）、世界动物卫生组织（OIE）等专家。

那么，在全球新冠肺炎疫情之下，“全健康”究竟意味着什么？

新冠肺炎疫情，短短两个多月便发展为全球大流行。中国科学院院士、上海交通大学副校长、医学院院长陈国强表示，此次新型冠状病毒溯源仍是未解之“谜”，但可以肯定的是，它最初来自人畜共患病，或者，动物至少是它的中间宿主。同时，多耐药微生物频繁出现，食品因添加有毒有害物质引起的食源性疾病，也不断威胁着人类健康。生活方式改变、城镇化加速、环境污染、全球变暖等对人类健康的长远影响，更难以评估。

进入21世纪，“One Health”理念不断形成，迄今中文尚无统一翻译。经反复讨论，陈国强建议将“One Health”译成“全健康”。他说，“全”字纳“人”“土”“王”三要素，可以概括健康离不开管理的统驭之势——将人类健康、动物健康和环境健康三者统合为一个健康整体，涉及人类和动物健康、环境卫生、食品安全和农业生产等方面，也理应包括“全球健康”概念，正所谓“一个世界，一体健康”。

事实上，去年6月10日，博鳌亚洲论坛全球健康论坛大会召开之际，全国人大常委会副委员长陈竺院士、海南省省长沈晓明、国家疾控中心主任高福院士以及陈国强院士等，会同WHO前总干事陈冯富珍、OIE、FAO等国际组织代表召开会议，讨论了One Health研究工作。

当前，任何单独学科或单独机构、组织甚至国家，都无法承担以解决复杂公共卫生问题为宗旨的“全健康”重任。中国面对严峻复杂的国际疫情，党和国家领导人发出“携手构建人类卫生健康共同体”的积极倡议。上海交大党委书记杨振斌表示，探索与实践“全健康”理念贯彻这一精神，是提升我国乃至全球传染病防控能力的现实需要，也是上海建设全球最健全公共卫生体系的重要支撑。

但不可否认，我国“全健康”实践尚处起步阶段。此次中国上海交大与率先联合医学院和兽医学院的英国爱丁堡大学共建中心，是推动“全健康”研究行动的发力点。新中心的中方主任由上海交大医学院—国家热带病研究中心全球健康学院副院长、世界卫生组织热带病合作中心主任、中国疾病预防控制中心寄生虫病预防控制所所长周晓农教授担任；英方主任则由爱丁堡大学农业与食品安全系主任、农业与食品安全国际研究院主任、爱丁堡大学助理校长杰夫·西蒙教授担任。此前爱大在全球率先设置了“全健康”硕士和博士研究生项目，今年联合研究中心也将率先在我国开展全健康双硕士联合培养。

解放日报·上观新闻记者了解到，不仅中英双方签署“全健康”合作框架协议，两校、新中心还与海南博鳌乐城国际医疗旅游先行区管理局签下合作备忘录。中心计划依托我国第二大岛海南岛和第三大岛崇明岛建立全健康实践与培训基地，作为“试验田”促进全健康产学研合作和成果转化，推动全健康体系在中国落地实施。

中国科学院院士、全国人大常委会副委员长陈竺通过视频祝贺，海南省副省长王路进行现场连线，上海市副市长陈群出席并讲话。
