【视频】毕业前，他用“最后一课”兑现援鄂承诺 ：瑞金毕业生学习穿脱防护服

医声医事 2020-07-03 16:35
来源：上观新闻 作者：黄杨子
“我相信，后浪们一定会像前浪一样，不仅成为合格的白衣天使，也将是优秀的白衣战士。”

特殊的毕业季，一份特殊的“毕业礼”——今天下午，上海交通大学医学院附属瑞金医院援鄂医疗队中的20名队员，再一次穿上了严密的防护服。这一次，他们不是作为医疗队员，而是作为一名临床教师，为即将踏上自己毕业征程的医学生们，带来毕业前的“最后一课”。

2020年，对于每一位毕业生来说都十分特别，对于医学生来说更是如此。平常在临床工作时带教自己的老师，转身即成为了身披盔甲的白衣战士，学生们深受感动、深受鼓舞。在医疗队出发时，不知道能做些什么的医学生们，为老师们制作了一张海报，海报上写着“今天你们出征，明天我们接棒！”“老师，平安回来！”这张海报，曾被打印出来，贴在了援鄂医疗队战斗过的武汉同济光谷院区展示栏上，激励老师们更有动力去战斗。

而今天，在瑞金医院毕业典礼的前一天，老师们也为毕业生们准备了特殊的“最后一课”——手把手传授毕业生们抗疫级别的防护措施。这既是老师们对同学们深深的护犊之情，更是殷殷的嘱托和期盼。

上海市第6批援鄂医疗队总领队、瑞金医院副院长胡伟国在活动开始时，对所有毕业生动情地回忆道：“今天，其实是我们兑现承诺的日子。你们还记得吗？医疗队的老师们在武汉抗疫时，仍谨记自己临床教师的责任，通过云端授课的形式，为大家录制了‘开学第一课’。今天，我们再为大家特别安排了毕业前的‘最后一课’。”

胡伟国告诉记者，“我当时在武汉跟他们说，其实我们心中有愧，因为在医学基础教育中，灾难医学几乎是空白的，导致面对疫情时底气不足。这一次，我们也是头一回学习，因此当时我承诺他们，回来了一定得补上这一课，让青年医师们未来能更有底气响应国家的号召，真正做到零感染的院感防护。”

上海市第6批援鄂医疗队队员、院感科90后公卫医师石大可为学弟学妹们分享了来自武汉抗疫前线的故事，并向大家介绍了重要的院感知识，此后，20位援鄂医疗队的老师与近60位毕业生一共分为10组，近距离地演示了防护服的穿脱方法，保证所有参与学生都能掌握。

 “这是我第一次穿防护服，感觉很闷热，这才几分钟已经汗流浃背、呼吸困难了。”今天的毕业生之一、上海交通大学瑞金临床医学院2015级本科毕业生彭昊自我“吐槽”，“感觉穿成这样只能写病史了。”他告诉记者，老师们教导的重点要领中，他尤其记得“肌肤不能暴露”，“不过我现在脖子和下巴当中好像还是有缝隙，这样的实战训练以后要多多参加，快速熟悉穿脱流程。今后国家若有需要，我也一定会第一时间选择上前线。”

最后一天，补上最后一课。胡伟国为毕业生们提出殷切期望，“希望大家能在今后的学习和工作中拥有优秀的硬核技术和充分的心理准备，同时也不要忘记，自己是瑞金培养的医生，要有广博慈爱这颗人文关怀的心，也要有追求卓越的科学创新精神。”他说，师生之间并非只是单方面的教导，学生们也是鼓励、鞭策老师的动力。“这一次，正是医者和教师的双重身份，让我更坚定地踏上前线，他们制作的海报照片，一直在我手机里。”胡伟国笑道，“我相信，后浪们一定会像前浪一样，不仅成为合格的白衣天使，也将是优秀的白衣战士。”
