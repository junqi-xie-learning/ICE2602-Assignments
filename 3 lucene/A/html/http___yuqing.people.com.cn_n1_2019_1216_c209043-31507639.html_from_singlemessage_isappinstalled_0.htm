“我眼中的中国”

首届在华留学生短视频大赛颁奖典礼在上海交大举行

2019年12月16日09:12  来源：人民网-舆情频道
 

12月14日，“我眼中的中国”――首届在华留学生短视频大赛颁奖典礼在上海交通大学媒体与传播学院演播厅隆重举行，来自全国各地的21位获奖留学生，与各界嘉宾，专家学者和交大同学近300人共同见证了这一盛典。“‘我眼中的中国’首届在华留学生短视频大赛”由上海交通大学媒体与传播学院主办，联合中国传媒大学、中国人民大学、复旦大学、北京大学、清华大学、同济大学、北京外国语大学等兄弟院校合办，上海报业集团、上海文广集团、东方网、腾讯等媒体支持。来自41个国家或地区的留学生的224个短视频进入评审程序。这些参赛作品视角独特、制作精美、情感真挚，表达出参赛者对经济发展、山川秀美、社会和谐的今日中国的赞叹和喜爱。

上海交通大学党委常委、副校长王伟明在致辞中表示，在华留学生亲自了解、亲身感受，通过短视频的形式，讲好中国故事，传播中国声音，能够给世界提供独特的视角，展现真实、立体、全面的中国。本次“我眼中的中国――在华留学生短视频大赛”充分激发了在华留学生了解中国国情、中国文化和民众生活的热情，为他们展示新媒体传播的创新、创意能力提供了平台，也客观真实地展现了他们眼中的中国。

东方网总裁徐世平代表东方网，对这次短视频大赛的成功举办表示热烈的祝贺，对参赛同学、获奖同学的辛勤付出表示衷心的感谢。他认为短视频是时下最受欢迎的传播形式之一，而且仍处在高速增长的阶段。短视频不仅短平快，在表达上也更具趣味性，能够激发并满足观众的好奇心，在增进东西方文化交流、弘扬中国传统文化和加强对外传播等方面发挥着重要作用。徐世平总裁认为在华留学生的短视频作品题材丰富、内容多元，可以更有效地让世界各国了解中国的方方面面，他对获得特等奖作品给予了高度的评价。

腾讯视频总监史少楠致辞表示，短视频大赛给参赛者提供了一个表达自我、展示自我的机会。从摩登现代的城市到历史悠久的古都，从体验新奇科技到庆祝传统节日，从学习汉字到歌唱流行歌曲，在华留学生用自己的视角解读着城市文化和中国文化，通过生活再现和情感传播，实现了自我与更广阔世界的对话。腾讯视频立足青春正能量，通过不断拉近与年轻圈层的情感距离，记录更多当代青年在奋斗追梦、在奋斗圆梦的故事，以正能量价值观为导向，继续做好主流舆论的宣传阵地。

教育部“长江学者”特聘教授、上海交通大学媒体与传播学院院长李本乾在致辞中表示：文明因交流而多彩、文明因互鉴而丰富。举办“我眼中的中国”在华留学生短视频大赛，是上海交通大学媒体与传播学院推进多元文化互学互鉴、构建人类命运共同体的有益探索和实践创新。短视频大赛启动以来，来自不同国家的在华留学生拿起手中的相机，亲身体验、全心融入，以沉浸式、体验式的方式近距离感知和触摸古老悠久、充满活力的中华文明，对绚丽多姿、丰富多彩的中国文化和群众生活加深了认识和了解。留学生们用独特视角真实客观地表达了他们“眼中的中国”，展现了不同文明之间碰撞交融的独特魅力，给予了我们更多的智慧启迪。当今世界，文明交流交融是主旋律和最强音，我们将以此次大赛的成功举办为契机，积极搭建交流互鉴的平台，进一步加强与来自不同国家、不同文明的留学生同学沟通交流、增进共识，并通过你们向世界展示真实的中国、宣介美丽的中国，构筑起世界了解中国的又一窗口和桥梁。

随后，来自学界、业界和政府部门的嘉宾为获得本次大赛各级奖项的留学生分别颁发了获奖证书和奖杯。

在获奖作品点评环节，北京外国语大学国际新闻与传播学院的章晓英教授认为，本次在华留学生短视频大赛开创了高校先河，首次以外国留学生拍摄视频的形式，来向世界展现中国的多样文化。这些参赛作品情真意切的表达了对中国的热爱与赞叹，观众很容易就产生了共鸣与共情；通过短视频的媒体形式，留学生展现了自己在中国的所见所思，生动有趣、令人信服，这是吸引海外民众广泛了解中国、深入理解中国的有效途径。

国际媒介与传播研究学会(IAMCR)国际传播分会副主席、中国传媒大学姬德强副教授在点评大赛获奖作品时，用多元化、差异性和对话这三个关键词来总结。 “一千个观众心中就有一千个哈姆雷特”，每位留学生观察和思考中国的视角必然不同，特殊的文化背景和成长环境造就了多元化和差异性，也造就了精彩纷呈的“我眼中的中国”。

典礼还举行了本次大赛获奖作品腾讯视频上线仪式。这些优秀的作品会被更多人看到，将为中华文化的传播，中外双方的交流，做出更大的贡献。腾讯政务舆情部总监高大为表示，腾讯在2020年会有一个新的尝试，通过大数据分析对“美好”这个主观的感知进行量化衡量，引导更多的人发现生活中的美好，用大数据能力更精准地进行服务，让活动的效果更加可见、可感，助力活动品牌影响力提升。

最后，在现场观众的欢呼声中，6位嘉宾共同启动了第二届“我眼中的中国――在华留学生短视频大赛”。

本届“我眼中的中国――在华留学生短视频大赛”充分激发在华留学生了解中国国情、中国文化和民众生活的热情，为他们展示新媒体传播的创新、创意能力提供了平台，客观真实地展现了他们眼中的中国。我们期待第二届在华留学生短视频大赛，能够涌现出更多、更精彩的作品。

据悉，本次大赛由上海交通大学媒体与传播学院发起并主办，北京大学、清华大学、中国人民大学、中国传媒大学、复旦大学、武汉大学、华中科技大学、南京大学、浙江大学、暨南大学等三十余所大学参加，得到中国新闻社、上海报业集团、上海文广集团、腾讯视频、腾讯指数、东方网等媒体支持。大赛面向具有外国国籍、并正在中国就读的全体留学生，鼓励他们用手机或其他设备拍摄短视频，生动展示他们眼中中国的模样。

据悉，此次大赛的参赛者来自清华大学、中国传媒大学、上海交通大学、武汉大学、华中科技大学、北京外国语大学、苏州大学、重庆大学、同济大学、北京师范大学、华侨大学、上海体育学院、浙江大学、西北大学、东华大学、上海师范大学、厦门大学、西安交通大学、上海戏剧学院、上海外国语大学、北京理工大学、西交利物浦大学、北京交通大学、首都体育学院、华中师范大学、天津师范大学等高校留学生积极参赛。参加比赛留学生的国籍遍布：澳大利亚、俄罗斯、西班牙、法国、美国、加拿大、波兰、丹麦、韩国、吉尔吉斯斯坦、马来西亚、日本、乌克兰、乌兹别克斯坦、新加坡、也门、伊朗、意大利、印度尼西亚、缅甸、泰国、阿富汗、摩尔多瓦、科特迪瓦、菲律宾、保加利亚、埃及、安哥拉、哥斯达黎加、巴基斯坦、刚果、圭亚那、柬埔寨、老挝、尼泊尔、尼日利亚、喀麦隆、印度、土耳其、越南、墨西哥等41个国家。

特等奖获得者Jacob Thomas认为：今天的颁奖典礼非常非常棒！这个比赛是个非常好的机会，让来自世界各国的同学能够分享他们在中国的经历。其实我是一个学商学的学生，这次机会可以说给我的思考打开了更多扇门，如果我毕业了，除了经济类的职业，做一些视频制作的工作也会很有趣。一等奖获得者Anish Kumar Saurabh表示：感谢上海交通大学媒体与传播学院组织这样的比赛给留学生提供创新创业的机会，从留学生的角度来了解中国。二等奖获得者Park Nuree认为：颁奖典礼很酷很高大上，我见了爱中国、爱拍视频的人感觉很开心。

(责编：邱越、袁勃)
