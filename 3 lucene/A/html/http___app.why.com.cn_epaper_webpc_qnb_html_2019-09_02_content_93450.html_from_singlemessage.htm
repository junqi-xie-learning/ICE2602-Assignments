上海交通大学“交龙”战队登上《开学第一课》 为中小学生埋下科技的种子

除了上海“跳绳小囡”亮相央视《开学第一课》外，上海交通大学“交龙”战队和华南理工大学“华南虎”战队联合带来了第九个节目——互动节目“机甲大师救援赛”。

节目组设置了一个巨大的沙盘，模拟一个自然灾害之后的情景。沙盘设置上下两方路线，红蓝两支队伍要在这个微缩的世界上完成从搜索到营救的全部任务。

众所周知，一般灾后救援只有黄金救援72小时，对于埋在废墟里的伤员来说，三天的时间可能是抢救他们的最佳时机。此次，对抗赛中，把这三天浓缩成为现场的三分钟。

两支战队并不是直接用成品机器人来比赛，而是现场组装，而其实这也是他们要挑战的一部分，在最短时间内装出实际可用的机器人，并选择同一时间出发完成救援行动。

整个互动节目让许多中小学生目不转睛。而参与其中的大学生表示，最大的意义不仅仅是学到了技术，得到了锻炼，磨练了意志，而是在于真正能够影响到身边的人。

上海交通大学“交龙”战队的指导老师、上海交大IT创新中心主任冷春涛告诉记者，希望这样的开学一课让中小学生埋下科技的种子，也让身边更多的人对科技产生兴趣。青年报记者 刘昕璐
