“上海发布”第一，全国分三个梯队，《2018中国政务新媒体发展白皮书》发布

互联网观察 2019-01-16 21:30
来源：上观新闻 作者：刘璐
由于资源、能力、领导重视程度的不同，国内政务新媒体之间的发展差距进一步拉大。

1月16日下午，首届主流政务新媒体发展论坛在上海交通大学举行。论坛由上海交通大学媒体与传播学院与上海交通大学政务新媒体研究中心主办，发布了《2018中国政务新媒体发展白皮书》。

 

“有些技术刚出来，你还没搞明白，人家已经玩腻了！”“有些规定动作年年有，今年的脑洞反响不错，但明年怎么突破？”在论坛的“模式创新者说”环节，来自上海多家主流政务媒体的代表们，结合各自实际情况提出的一些困惑，引发了现场同行们的共鸣。

 

“大家能坐在一起总结教训、分享经验，这样的交流机会非常难得。”一位区县政务新媒体账号负责人告诉解放日报·上观新闻记者。

 

平台格局发生改变

 

过去，政务新媒体的主要发展平台集中在“两微一端一网”。根据这份白皮书的数据，进入2018年以后，客户端的价值不断下降，几成“鸡肋”；网站总数也在缩减，截至2018年6月，全国政府网站总数较2015年12月减少70.1%，。

 

整个政务新媒体平台格局，逐渐向“1+2+N”发展——“1”指网站，“2”指微博、微信，“N”则指以今日头条、抖音、澎湃、喜马拉雅等为代表的新平台。其中，到2018年6月，各级党政机关共入驻政务头条号账号74934个；截至2018年9月11日，全国有2795家党政机构开通政务抖音号。

 

全国“政务小编”约有60-80万

 

政务新媒体账号背后，经常加班、偶尔加鸡腿的“政务小编”，究竟有多少人？

 

通过对上海地区政务新媒体从业人员的平均数进行推算，白皮书得出如下保守估计——全国政务新媒体从业人员大约在60-80万人，包括专职和兼职。这个数字是基于上海地区平均每个政务新媒体单位4-5个人的情况看，参照微博的整体数据（共计137677个），推算而来的。

可以看出，这是一个庞大的群体，需要从体制、机制、政策、资源等领域加以综合考虑。

 

发展差距拉大，全国分为三个梯队

 

过去这一年，由于资源、能力、领导重视程度的不同，国内政务新媒体之间的发展差距进一步拉大。白皮书将它们分成了三个梯队：第一梯队是高速发展的账号，如长期保持全国第一的“上海发布”、高歌猛进的“杭州发布”、“青春上海”、“南京发布”、“苏州发布”、“滁州发布”等；第二梯队是维持现状的账号，由于各种原因，进退两难，勉强支撑；位于第三梯队的一些政务新媒体账号，由于无法支撑持续发展，直接放弃了内容生产。

 

在第一梯队中，一些优秀政务新媒体的品牌化逐渐成型，用户黏性随持续提升。它们的共同点是，普遍重视自己的社群建设，除信息发布外，积极发挥政务服务的功能。

 

问题不少，专业化将成趋势

 

不过，政务新媒体面临的问题也不少。

 

从全国范围内来看，一些核心服务数据接口因为数量较少成为“香饽饽”，各级各类政务新媒体都希望接入，然而僧多粥少，在少数主流政务新媒体大号以外，大多数基层账号无法获得类似资源；与此同时，同一地区重复建设高频应用的接口，造成资源浪费。

 

一些由于管理不善导致的问题，在2018年通过部分舆情事件暴露出来，如安徽省全椒县的@全椒发布账号在“漫威之父”去世当天连续发布6条相关新闻，引发网友争议；四川自贡市环保局的官方微信号回复“不说话没人把你当哑巴”，激怒市民。

 

具体到团队内部，还有一些共性的问题。比如人员紧缺，服务外包，拥有事业编制或公务员身份的人员少之又少，如何提升团队的积极性和稳定性？比如，没有采访权，只能通过资料汇集进行信息整合，造成原创不足、内容同质化问题比较突出。领导期望高、阅读量却上不去，小团队往往承受着技术、传播、创新等多重压力。

 

据上海交大政务新媒体研究中心分析，未来1-2年，政务新媒体机构建设将进入分类发展阶段，呈现以下几个趋势：一是机构主导、固定部门主导、体制内外包三种方式并存，但会进一步整合与聚合；二是运营团队将产生大量专业岗位，从业者的专业素养将得到提升，这种专业素养包含制作能力以及数据素养能力。此外，这份白皮书还指出，在实践中如何合法地利用外部资源推动可持续发展，可能会成为今后需要直面的一个重要话题。

 

与白皮书同时发布的还有《政务新媒体的模式创新》一书，系该研究中心主任禹卫华教授“走进新型主流媒体”系列丛书的第一部。
