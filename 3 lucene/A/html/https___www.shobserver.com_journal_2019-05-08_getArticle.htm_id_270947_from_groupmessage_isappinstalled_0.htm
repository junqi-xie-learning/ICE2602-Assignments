《战上海》作者带着交大学子实地还原上海解放史

史家视角讲述70年前的共产党人

2019年05月08日   02： 要闻   稿件来源：解放日报  
■本报首席记者 徐瑞哲

在一座百年老校园里，师生们从双烈士墓前、经大操场外围、到工程馆楼下，一路走过了3个地标，仿佛一起穿越70年时空。

昨天下午，“中国好书”《战上海》作者、上海交通大学历史系教授刘统，带着人文学院等院系学子“边走边讲‘战上海’”。在本月纪念上海解放70周年系列实景对话的上海交大专场，上了一堂行走着的思政课。

不占师生宿舍度过驻沪首夜

师生们行至工程馆建筑门口，刘统戴着耳麦做起了“导览”。这座建筑，当年正是华东局机关进驻上海第一天的驻址。

1949年5月25日夜到26日晨，华东局机关和接管干部队伍由丹阳乘火车到南翔，又由南翔分批乘汽车进军上海。陈毅问周林：“接管上海市政府的队伍先进驻哪里？你熟悉。”周林说：“先进驻徐家汇交通大学，然后分头接管。”当时下着细雨，道路泥泞。车队到达交通大学，没有占用师生宿舍，度过了进入上海的第一个夜晚。

上海交大党委书记姜斯宪表示，《战上海》以史学家独特的视角，用一个个生动的历史细节，讲述了70年前一群意志坚定、充满智慧的共产党人，怎样成功接收和管理这座远东第一大都市。“共产党人在上海的一系列创新性探索，为新中国各项建设提供了宝贵经验。”

解放第三天烈士遗体回校园

徐汇校区大草坪南侧的史霄雯、穆汉祥烈士墓，方尖碑下，春暖花开。

1949年4月，国共正在举行和谈期间，交大举办了一场“真假和平辩论会”，分管学生自治会工作的学生党员穆汉祥担任正方主辩人，党领导的秘密宣传小组成员史霄雯主持。因为在公开场合发表反蒋言论，穆、史二人被列入“黑名单”。

4月30日、5月2日，两人分别被捕，被关在警察总局死牢内，受尽刑讯逼供也没有妥协。刘统考证发现，各方为营救他们想尽办法，交大校长王之卓亲自给毛森打电话，但毛森矢口否认有这两个人。事实上，5月20日他俩在闸北宋公园（今闸北公园）被秘密杀害。

至5月29日，也就是上海解放第三天，交大学生自治会在工人指点下找到了两位烈士的遗体，安葬在徐汇校园中。1950年5月，陈毅市长为二烈士墓碑题词：“为人民利益而光荣就义是值得永远纪念的。”

本报讯（记者 徐瑞哲）昨天，在上海交大、上海世纪出版集团主办的《战上海》座谈会上，社会各界人士与专家学者共同回顾70年前上海从百废待兴到重获生机的“大解放史”，热议中国共产党人“战上海”精神及其当下的现实意义，纪念新中国成立70周年、上海解放70周年。市人大常委会副主任沙海林出席座谈并交流。
