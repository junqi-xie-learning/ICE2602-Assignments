上海交通大学与新南威尔士大学战略合作暨孔子学院成立十周年庆典在悉尼举行

2019年02月20日12:14  来源：人民网-澳大利亚频道
 

双十周年庆典现场（摄影 沈诺）

人民网悉尼2月20日电 2月19日下午，上海交通大学与新南威尔士大学战略合作暨孔子学院成立十周年庆典在悉尼谊园举行。新南威尔士大学校长伊恩・雅各布斯、上海交通大学副校长徐学敏、中国驻澳大利亚大使馆教育处公参宗瓦、中国驻悉尼总领馆教育组参赞牛文起等应邀出席活动。

新南威尔士大学校长伊恩・雅各布斯在十周年庆典上致辞（摄影 沈诺）

伊恩・雅各布斯在庆典上发表致辞称，十周年这一里程碑见证了两校在教育、创新和创业等领域的密切合作以及取得的突出成就。正如中国谚语所说，“众人拾柴火焰高”。合作共进是推动科研成果进步、造福社会的最佳方式，它使我们能够在面对时代新挑战时，迸发出无穷力量。

雅各布斯回顾了十年间两校的各类合作成果。此外，他高度赞扬新南威尔士大学孔子学院在传播中国文化、融入大学、连接社会等方面做出的贡献和成就，并衷心期望两校以十周年为新的起点，让友谊协作之火愈燃愈亮。

上海交通大学副校长徐学敏出席庆典并致辞（摄影 沈诺）

上海交通大学副校长徐学敏表示，正值中国元宵节，与新南威尔士大学的同仁们共度佳节，这份家人般的感情弥足珍贵。在过去的十年中，从学生交流到联合学位项目，从机构基金到联合研究实验室，两校在广泛的领域取得了长足的合作和进步。

徐学敏称，未来两校间的友好合作关系和战略伙伴关系必将继续传承、不断创新。她对新南威尔士大学孔子学院为两校所带来的荣誉和成就表示感谢，为两校在新的十年再创辉煌表示衷心祝愿。

庆典现场两校嘉宾交换礼物（摄影 沈诺）

庆典现场两校嘉宾交换礼物（摄影 沈诺）

随后，现场嘉宾共同观看了为此次双十周年庆典特别制作的纪录短片。短片以2009年新南威尔士大学孔子学院建立为起点，回溯了两校合作的十年历程，在科研合作、学生交流、高层互访和教育管理等多方面乘风破浪、并肩远航。而新南威尔士大学孔子学院作为合作纽带，成为两国民众友爱理解的桥梁。两校嘉宾还交换了礼物，共同期盼掀开交流合作的新篇章。

新南威尔士大学孔子学院中方院长奚俊芳发表致谢词（摄影 沈诺）

为表彰新南威尔士大学孔子学院中方院长奚俊芳在两校交流、建立合作关系以及孔子学院建设方面的杰出贡献，雅各布斯在庆典上为其颁发奖杯。奚俊芳在致谢词中对国家汉办和两校对孔子学院的大力支持及帮助表示感谢，并期望孔子学院在新的十年继续作为两校合作的枢纽和骄傲，推动澳大利亚当地中文教育的发展。（沈诺）

(责编：王泉骄、雪萌)
