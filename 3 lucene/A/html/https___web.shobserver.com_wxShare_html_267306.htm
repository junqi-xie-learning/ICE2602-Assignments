世界人工智能大会花开申城，“白玉兰开源”共建AI开源生态，趋势不可阻挡

教育在线 2020-07-08 15:19
来源：上观新闻 作者：徐瑞哲
《关于建设人工智能上海高地构建一流创新生态的行动方案（2019-2021）》，重点提出了“联动国际资源，打造开源开放社区平台”。

7月9日起3天，2020年世界人工智能大会云端峰会花开申城。人工智能之花如何开放？为有源头活水来。

在IT界，简称“开源”的开放源代码业内皆知，用户基于源代码可以进行修改和学习，程序员能对源代码二次开发利用，普通终端消费者也受益于开源系统，比如非苹果手机的用户最熟悉的开源软件就是天天在用的安卓操作系统。同时，苹果系统布局的全球开发者生态同样很开放。

因此，开源开放是一种潮流，也是全球人工智能发展不可阻挡的重要趋势之一。解放日报·上观新闻记者获悉，就在世界人工智能大会（WAIC）之际，以东道主上海市市花——白玉兰命名的开源开放联盟性机构诞生了。

掌控开源生态，已成为国际产业竞争的焦点。2017年国务院发布《新一代人工智能发展规划》，明确将“开源开放”作为四大指导原则之一，推动人工智能技术和应用发展创新。同年，上海市出台了《关于本市推动新一代人工智能发展的实施意见》，加快建设上海“创新策源、应用示范、制度供给、人才积聚”人工智能高地。上海市人工智能产业工作领导小组办公室编制《关于建设人工智能上海高地构建一流创新生态的行动方案（2019-2021）》，重点提出了“联动国际资源，打造开源开放社区平台”。

为更好落实国家和地方人工智能开源开放战略部署，助力上海人工智能高地建设，由上海交通大学牵头，联合中国电子技术标准化研究院、北京大学、机器之心、华东师范大学、开源社、上海人工智能研究院有限公司等单位，共同成立上海白玉兰开源开放研究院（即“白玉兰开源”）。

据悉，7月11日2020世界人工智能大会云端峰会·开发者日主论坛上，中国科学院院士、上海交通大学人工智能研究院院长梅宏将以“参与融入、蓄势引领”为主题，阐释白玉兰开源核心理念，来自北京大学、同济大学、华东师范大学、华为、腾讯、阿里、Intel、OpenI理事会、机器之心、开源中国等专业代表均参与共建开源社区与生态。

上海交大人工智能研究院总工程师金耀辉说：“开源开放既不是一个新名词，也不是一个新行动，在当今的人工智能时代，开源开放是AI与社会发展深度融合的有力工具，要想获得先发优势，开源开放生态的建设至关重要。”开发者日主论坛上特设“开源的中国声音”圆桌，并计划发布一部《人工智能开源开放发展报告》，在对国际、国内人工智能开源开放发展深入、广泛调研基础上，结合权威部门和机构的相关资料和数据，对开源开放的内容范畴进行细致梳理和汇总，提出开源开放生态的发展特点和建设建议。

据上海交大人工智能研究院常务副院长杨小康透露，目前上海交大发掘AI相关学科内涵，汇聚校内外力量和资源，布局11个重基础、重交叉的研究中心。包括：数学基础研究中心、机器认知计算研究中心、视觉智能研究中心、智能语音与自然语言处理研究中心、集成电路EDA技术研究中心、操作系统研究中心、智能网联汽车研究中心、云边端协同制造研究中心、智能金融科技研究中心和人工智能治理与法律研究中心，涉及前沿基础、关键技术、应用场景和治理规则4个方面，搭建跨学科、开放式、国际化的研究与合作平台。
