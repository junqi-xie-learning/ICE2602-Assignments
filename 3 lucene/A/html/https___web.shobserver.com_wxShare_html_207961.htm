危中有机！这位政府参事称疫情分三阶段影响各行业，短期影响一年半载，长期影响一年后

教育在线 2020-02-07 16:12
来源：上观新闻 作者：徐瑞哲
新型战略性行业如生物医药、人工智能等会继续得到资本青睐而持续发展，甚至房地产业也会在一定范围内得以增长。

“在相当一部分城市，绝大部分传统服务业，如餐饮、快递、外卖、家政等从业人员来自城外，在这期间这些行业难以全面恢复运行。”作为上海市人民政府参事，中国管理科学与工程学会副理事长、上海交大行业研究院副院长、上海交大安泰经济与管理学院教授陈宏民，从正在发生着的“第一阶段”开始，分析当前新冠肺炎疫情对行业发展的不同影响。

解放日报·上观新闻记者获悉，7日，20多名学者以上海交大行业研究院团队名义发表《新冠肺炎疫情对若干行业的影响分析》，共结集19个行业分析，可谓“危中有机”。

陈宏民

【困境从下游逐渐向上传递】

“熬过严冬是春天。”这是陈宏民的基本判断。站在行业层面，对于疫情导致行业发展出现困境的主要原因，首先是整体需求萎缩，当前服务业面临的主要问题正是如此，如酒店旅游餐饮等；其次是生产要素（如劳动力资源）大量减少或成本上升，如快递、家政服务等；再者是政策性影响，暂时限制人员流动也形成许多企业困境。

制造业的困境，大致是从下游逐渐向上传递的。陈宏民举例，比如疫情造成国内服装需求减少，于是服装行业面临困境；然后困境传递给纺织业，再传递给纺织机械等。其他行业也大致如此。此外，疫情还会显著影响我国的出口贸易，从而进一步加剧许多下游企业的困难，并逐渐向中上游企业传递。

上海交大安泰经管学院院长、行业研究院院长陈方若表示，不同于已有的宏观或微观层面的论述，众教授行业研究的分析集试图从另外一个角度来分析疫情影响，并提出应对策略。“这种行业的中观视角有益于政府在制定相关政策时，了解行业之间的不同，从而制定出更加精准的产业政策，也有益于企业家明晰本行业发展趋势，看准方向，及时找到适合自己的策略。”

【疫情影响行业分三个阶段】

陈宏民认为，疫情对各行业的影响大致可以分为三个阶段。

第一阶段是疫情高峰期。从疫情引起全社会关注重视到全线严防、全力死守，直到疫情高峰过去。“这个阶段可能在三个月左右，从今年春节前夕到之后的两三个月里。”他说，“相当部分企业还不能正常运行，相当部分企业员工在家办公。”

第二阶段是疫情缓解期。疫情高峰确认拐点，各类病例减少，传染大风险过去。“这个解冻缓解阶段大概也要半年时间，估计也会延续到今年年底。”

第三阶段是疫情消除期。这时疫情已经远离人们生活，各种针对疫情的防范措施已经解除，社会秩序和人们心态恢复正常。

那么，各个阶段对行业发展有何不同影响？

陈宏民表示，第一阶段正在发生着，可称为“对行业的当期影响”。在这期间，传统服务业几乎停摆，酒店旅游业几乎停业，交通餐饮业运行越多亏损越大，处在被动运行阶段。“有一部分企业难以为继。”制造业也会因为用工困难、成本上升、订单减少，而陷入困境，勉强开工维持或者处于半停滞状态。

当然，也有些例外而得到迅猛发展的。如医药医疗和医疗器材等行业，会得到短暂刺激性发展。

他认为，第二阶段大概在两个月左右之后出现，为期在半年到一年，可称为“对行业的短期影响”。在这个阶段，一方面政府对人员流动的管控还适度存在，另一方面大部分国人依然保持着谨慎心态，虽然工作生活开始正常，但一时不会恢复原先的习惯。比如不愿意上街聚餐，抑制旅游冲动等。所以这时期绝大部分传统服务业处于恢复性状态，但业务规模在三个月到半年里还难以恢复到往年同期水平。

同时，需求萎缩波及到更多的服务业和制造业，“政府会出台一系列刺激经济的举措，全国范围内的基础设施建设会继续进入高潮，与之相关的行业会得到激励；”陈宏民提到，新型战略性行业如生物医药、人工智能等也会继续得到资本青睐而持续发展，甚至房地产业也会在一定范围内得以增长。

第三阶段，大概会在一年后出现，可称为“对行业的长期影响”。这时绝大部分需求都恢复正常。不过许多行业在过去的一年里经历了重新洗牌，好的企业弯道超车，脱颖而出。一些行业在疫情中凤凰涅槃，通过转型升级获得新的发展。

【两大行业预见大规模兴起】

危中有“机”。陈宏民教授分析称，有几个动向值得关注。

一是在线教育可能大规模兴起。由于之前两个阶段的影响，在线教育的优势进一步凸显，无论新型办学机构还是传统办学机构如全日制大中学，教学双方越来越适应各类线上教育的互动形式，从而有可能使在线教育成为主流教育模式之一。

其二在于，产业互联网平台可能获得大规模发展，进而改变一些行业的价值链和商业生态形式。面对未来一年里许多行业里中小企业生存困难的严峻挑战，会有一批先行者搭建互联网平台，为这些中小企业解决采购、销售、信息乃至融资等一系列问题。在资本市场的支持下，这些平台可以（至少在短期内）大大降低那些中小企业的运营成本，提高经营效率。政府也会乐见其成，积极扶植这些平台的发展。

“十多年前非典过后，中国的互联网经济乘势而起，电子商务等一大批平台如雨后春笋般出现在许多行业里，又何尝没有那场疫情逼迫的呢？”因此陈宏民表示，今天面对新的疫情，从中央到地方，各行各业众志成城，“只要我们撑过眼前这一段难熬的日子，就一定能看到许多行业的春天！”
