“沿着周总理的足迹”全国微电影大赛在淮安揭晓

2019-03-05 19:22:29

 扬子晚报网3月5日讯 （通讯员 李鸣鸣 钱子达 陈力 记者 朱鼎兆）3月5日是周恩来周总理诞辰日，当天下午，由淮安市委宣传部、中国电影评论学会微电影研究会等单位主办的“沿着周总理的足迹 不忘初心 砥砺前行”全国微电影大赛颁奖典礼在淮安市举行。最终评选出一等奖3部，二等奖7部，提名奖10部，优秀作品奖10部。主办、承办、协办方等相关单位领导、专家以及来自全国各地的高校师生、文化工作者、获奖团队代表300余人参加颁奖典礼。

 
大赛从2018年9月21日开启作品征集，南到贵州、广西、海南，北至新疆、甘肃、吉林，全国数百所高校、百余家影视文化工作室，以及数十家企事业单位积极参与，累计收集作品1250余部，最终评选出一等奖3部，二等奖7部，提名奖10部，优秀作品奖10部。北京参赛作品《爷爷的徽章》、同济大学《一个想踢足球的小女孩的故事》、浙江第四监狱《沙果树》获得一等奖。南京中瀚影视文化传播公司《我的铁路我的路》、上海戏剧学院《心履》、云南彝良县政协《山路弯弯》等作品分获二等奖、提名奖和优秀作品奖。
