上海交通大学：创新劳动育人载体 践行后勤育人使命
上海市学校后勤协会
2020年07月16日 15:13

劳动教育是中国特色社会主义教育制度的重要内容。为弘扬劳动精神，营造育人氛围，更好发挥后勤劳动育人功能，上海交通大学后勤保障中心围绕立德树人根本任务，组织后勤员工技能比武，开展沉浸式劳动教育，组织特色劳动文化活动，与思想道德教育、专业知识教育相配合，促进大学生德智体美劳全面发展。

一、培育劳动育人理念

后勤服务渗透校园生活的方方面面，与学生学习生活息息相关。后勤人员是后勤服务的主体，在服务的每一个环节，每个人的言行都与服务对象发生关系，产生作用。上海交通大学后勤保障中心历来重视劳动与服务潜移默化的育人作用，重视育人理念的培育，并以此为共同的价值引领，建设后勤队伍。

一是统一思想认识，加强思想引领。通过举办“心怀师生，了解交大，服务交大”主题沙龙活动等形式多样的活动，多角度多层次开展教育思想大讨论，提炼后勤文化精神，统一劳动育人认识，增强劳动育人理念。

二是弘扬工匠精神，开展技能比武。近年来，中心举办厨艺、电力、绿化、急救、幼儿保育等行业技能比武，幼儿教师教学比武；同时，中心还开展后勤工匠奖的年度评选，引导广大后勤员工学习劳模精神，争当后勤工匠，提高劳动育人技术水平，筑牢服务根基。

三是倡导做精做实，组织业务培训。在餐饮、绿化保洁、交通运输、能源保障、物业等各个行业组织安全生产培训、服务礼仪培训、垃圾分类培训等专业素养培训，提升员工综合素质，营造“做事有态度、服务有温度、站位有高度”的后勤劳动育人氛围。

二、开展沉浸式劳动教育

一是组建后勤管理服务学生志愿者团队。组建学生红十字分会，在学校每年组织的六次主题志愿献血活动中，同学们总是全程做好服务；他们定期巡查分布在校内近50台心脏除颤仪（AED）运行情况，和指导老师一起拍摄原创情景短剧《“救”在身边！看交大学子如何“狙击”心脏骤停》，形象动态地科普AED用法，被中央电视台报道。在开学、校庆等重要时间节点，垃圾分类志愿者积极参与校园生活垃圾分类宣传。在军训期间，“军训督查小分队”主动到集中用餐的食堂执勤，引导参训师生有序就餐。

二是在后勤设立勤工助学岗位。结合后勤服务的行业特点，中心设立了吃吃督查（食堂督查）、节能督查岗、校医院一门式服务岗、保健辅助岗等勤工助学岗位，招聘学生上岗。学生在参与后勤管理服务的过程中，加强了对后勤的了解，树立了正确的劳动观。

三是在后勤服务活动中大量招募学生志愿者。在连续多年开展的“德国美食节”（后改为“德法美食节”）等活动中，大学生志愿者作为翻译进入食堂后厨，协助德、法、中厨师之间的交流；在每年学校校庆季开展的“后勤服务一条街”、校运会期间的“美食嘉年华”等活动中，组织学生志愿者参与“美食与健康”、“拒绝舌尖上的浪费”、“我与交大同生日”等环节的策划与实施，在服务保障过程中感受劳动的乐趣。

四是为学生志愿服务西部支教团队组织“临行送锦囊，生活更安心”烹饪培训。结合同学们的需求和支教地的实际情况，厨师们精心设计烹饪教学内容，提前准备好各种食材，让同学们快速掌握一些基本的烹饪知识和技巧。同学们感慨地说：“师傅们讲解非常仔细，我们学会了这几道家常菜，对支教生活更有信心了”。

三、拓展互动式劳动体验

一是在传统节日举行互动式文化活动。与学校学生工作指导委员会协作，在学生年夜饭上增设师生共同包饺子环节，学生与校长一起包饺子，既有大家庭的温暖，又烘托了节日氛围。清明节举办“食堂阿姨教你做青团”特色活动，在食堂阿姨的帮助指导下，学生们现场制作青团，吃得格外香甜。端午节，在中心职工党支部与电子信息与电气工程学院博士生电子第一党支部、本科生微电子党支部结对学习交流，举行“棕叶飘香话端午”包粽子活动，让后勤党员和学生党员在手把手传授包粽子的过程中交流互动。

二是开展专题救护技能培训。每年军训期间组织参训学生学习应急救护基本技能，提高救护技能水平；组队参加上海市高校应急救护比赛，并多次获得团体与个人奖项。2019年1月20日，全国首个急救日，中央电视台报道了上海交通大学学生团队的急救活动。

三是与学生社团联合开展各类文化特色活动。后勤保障中心第三党支部与SK Sunny大学生志愿服务社团联合举办“剩食派对”活动，倡导节约粮食、爱惜粮食的“光盘行动”。

四、搭建实践式育人载体

一是依托学院专业特长打造绿色校园。委托环境与工程学院专业团队的技术力量提升思源湖水质，在团队师生的共同参与下，思源湖变身美丽“魔镜”，学校获评上海市“蓝天碧水保卫战”先进集体。

二是学生参与后勤文化设施建设。在学校“双一流”建设项目的支持下，设计学院的学生参与校园巴士候车亭设计及食堂标识、文化墙的设计等，使校园文化更深地融入后勤服务。

三是与学院合作共建节能育人平台。与电子信息与电气工程学院积极开展合作共建：提供科研培训场地，开放电力集控中心-KB05站及能源保障部试验厅，对电院本科学生进行供电、配电和电气技术的参观培训实践；将电院师生的新技术、新产品应用于学校供电网络，改善供电质量；通过联合开展智能用电、安全用电等研究，解决校园供电安全及推动校园节能降耗。

积极参与大学生劳动教育，后勤责无旁贷。后勤保障中心还将与学校党委宣传部、学生工作指导委员会、教务处等部门通力合作，更深入、广泛地开展劳动教育，形成育人合力，共同培养热爱劳动、奉献社会的交大人。
