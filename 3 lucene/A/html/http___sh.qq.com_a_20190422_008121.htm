上海交大大师讲坛：点击化学在生物方面的应用

要闻上海交大2019-04-22 14:41

点击化学（Click chemistry）是一种实用的哲学概念在合成化学上的映射，已被广泛应用于有机化学、聚合物材料合成、生物分子标记、抗体修饰、药物开发等一系列重要的研究和生产领域。

4月22日，不对称催化和点击化学领域的奠基人，诺贝尔化学奖、普里斯特利奖、沃尔夫奖、美国富兰克林奖章获得者，美国科学院和艺术及科学院双院士，卡尔巴里 夏普莱斯（Karl Barry Sharpless）教授做客上海交通大学大师讲坛，作题为《SuFEx点击化学在生物方面的应用》报告，向交大师生分享其近期在研究具有抗癌活性的小分子方面的工作。

热爱中国文化，认为《道德经》道出“点击化学”哲学真谛

夏普莱斯教授于1941年4月28日出生于美国费城。据悉，夏普莱斯喜欢中国文化，特别是可以团结很多人的家族观念。他也非常欣赏老子《道德经》里的一句话，“故有之以为利，无之以为用”，认为这句话道出了“点击化学”中的哲学真谛：平时不会反应的碳-杂原子的结合，看起来似乎没什么用处，可当带上了特殊的官能团后，就会具有很高的选择性。比如夏普莱斯教授正在研究的第二代“点击化学”反应中，带上硫氟基团的小分子，可以精准识别出能与之结合的蛋白质，并形成相当稳固的共价键。

点击化学应用前景广阔，SuFEx反应助力发现抗癌活性小分子

夏普莱斯不对称双羟基化反应与夏普莱斯环氧化反应，是现代有机合成中最重要的反应之一，夏普莱斯教授因这一贡献获得2001年诺贝尔化学奖。点击化学是一种实用的哲学概念在合成化学上的映射，它的核心理念是——合成化学的灵魂在于其实用性，要以分子功能为导向。

最近在点击化学的核心理念上又发展出来新一代的点击化学——基于“硫氟交换（SuFEx）”的高价硫氟类化学。夏普莱斯教授在报告会中介绍了他于2014年发现的这一点击化学反应——SuFEx反应。该反应以商业化生产的工业原料硫酰氟（Vikane，SO2F2）和乙烯磺酰氟（ESF）来构建多种含有有S（VI）-F键的分子结构，如含-OSO2F（ 氟代硫酸根）和-SO2F（磺酰氟根）等基团的分子。自开发以来，SuFEx反应已在材料科学、化学生物学和药物研发方面得到广泛应用。

在有机碱的存在下，酚类化合物和硫酰氟气体（SO2F2）可以通过SuFEx反应高效地合成芳基氟代硫酸酯类化合物。该类化合物在生物环境中通常是稳定的，但当特定蛋白质侧链存在可以与芳基氟代硫酸酯官能团反应或促进氟离子离去的基团时，氟代硫酸酯官能团会被该蛋白质活化。这种蛋白质与芳基氟代硫酸酯官能团之间的相互作用实现了探针分子与蛋白质配体之间的共价连接。在演讲中，夏普莱斯教授重点介绍近期使用这种新方法在发现具有抗癌活性的小分子方面的工作。

交大师生与大师进行热烈互动

慕名前来倾听大师教诲的交大师生坐满了报告厅，在场师生在互动环节中就夏普莱斯教授的报告踊跃提问，反响热烈。

上海交通大学化学化工学院博士生张嘉诚讲到：“去年有幸参加滴水湖举办的世界顶尖科学家论坛，当时就得到了夏普莱斯教授的指导，我还清晰的记得，当我汇报了我在C-C键构建领域的研究后，教授评论说，‘大自然本身就是研究复杂C-C键的能手，我们应该学习生物体内合成大分子的方法研究C-杂原子键的构建’。教授的话给了我很大的启发，我们的研究本就是探索自然的奥秘，未来也应服务于人与自然的和谐共生。”

上海交通大学化学化工学院张书宇特别研究员这样评价点击化学。点击化学是通过小单元特定官能团的的高效拼接，在温和的反应条件下快速地完成目标分子的化学合成。它尤其强调开辟以碳-杂原子键（C-X）合成为基础的组合化学新方法，并借助这些反应（点击反应）来简单高效地获得分子多样性。点击化学的概念对化学合成领域有着巨大的贡献，在药物开发、生物医用材料、化学生物学等诸多领域中得到了广泛的应用，并极大的推动了这些领域的发展，点击化学目前已经成为最为引人瞩目的合成新策略之一。

感知大师思想，启迪科学人生，寻求钱学森之问解决之道

据悉，上海交通大学大师讲坛创办于2012年4月，是面向全校师生的高层次学术活动。大师讲坛定期邀请国内外学术大师、著名科学家、学者等走进讲坛，讲述科学知识、科学道德、科研精神和研究方法，与交大师生面对面交流，以成功者的典范指引学生的学术航向，激励学生投身学术道路，为祖国科技事业的兴旺发展贡献力量。首期大师讲坛由“国家最高科技奖获得者”王振义院士主讲。截至目前，大师讲坛已经邀请了120余位学术大师，包括13位诺贝尔奖得主、4位菲尔兹奖得主、3位图灵奖得主等，来访交大并与近60000人次交大师生面对面互动交流，成为交大学术文化建设中一道亮丽的风景线，产生了良好的社会反响。

一直以来，大师讲坛的讲座以高水平嘉宾、高质量组织和高热情参与而受到校内外广泛认可。大师讲坛陆续推出“科普三分钟”、“大师午餐”、“大师寻访”、“与大师同行”等特色项目。“科普三分钟”是在每次讲座开始前向观众放映一部时长约为三分钟的视频，向现场观众介绍本次讲座的学科背景。“大师午餐”是交大学生可以通过在线预约的方式争取与学术大师共进午餐的机会。“大师寻访”是组建实践团并利用寒暑假时间实地拜访大师，实践团曾经先后拜访袁隆平院士、黄伯云院士等。

钱学森曾问：为什么我们的学校培养不出杰出的人才？一直以来，国人都在苦苦思索这个问题。作为国内一流高校，上海交通大学一直在针对钱学森之问寻求解决之道。
