上海交大加强实践育人体系建设成效显著
2018-11-22 11:56
来源：科学网 www.sciencenet.cn

 

11月22日，记者从上海交通大学获悉：该校将实践育人质量提升体系纳入学校“双一流”建设整体规划，完善机制，坚持理论教育与实践养成相结合，不断加强学生价值引领，成效显著，得到了相关部门的好评。

强化理论课程建设，凸显理想信念教育。联动马克思主义学院，全面推进社会实践学分化，在《通识教育实践活动》课程基础上，建设《新时代社会认知实践》实践课程，纳入“两课”必修课板块，不断增强课程的针对性和吸引力。强化思想引领，校领导从“如何认识新时代变化”“如何进行认知和实践”“如何做好实践的准备”三个维度给全体学生上好《新时代社会认知实践》导论课；聚焦问题导向，专家教授、思政工作者、辅导员围绕社会调查研究方法、实践报告撰写、网络宣传技能等专题合力上好《新时代社会认知实践》导论课分论课。

优化实践板块设计，加强实践项目管理。学生工作指导委员会加强统筹，积极构建“大实践”格局，系统设计学生党支部“共行计划”专项、就业引领专项、通识实践专项三个板块，规范实践立项、管理及考核。“共行计划”专项面向学生党员群体，鼓励以学生党支部为单位组队，强化与地方基层或重点行业单位党支部结对共建、互学互访，增强社会责任感和历史使命感；就业引领专项面向高年级本科生、研究生群体，鼓励以院系专业为单位组队，集中到国家重要行业关键领域的企事业单位调研实践，倡导学生在择业就业中为国家和人民做选择；通识实践专项面向低年级学生群体，鼓励学生根据兴趣自由组队，通过国情调研、社会服务等多种形式，多走多看，常思深悟。

整合各类实践资源，精心设计实践内容。以“行万里路，知中国情”为主题，依托533个社会实践基地、213个就业实践基地、32个研究生挂职锻炼基地等平台，每年组织800余支团队近万名师生奔赴祖国各地开展实践活动，教育引导师生在亲身参与中增强实践能力、树立家国情怀。例如，组织“红色之旅”主题实践，结合“学习实践党的十九大精神”，推动一批优秀学子走进延安、服务革命老区；组织科研转化主题实践，鼓励学生联动青年教师，结合专业课题研究，到地方一线去解决具体问题；组织“助飞计划”主题实践，利用在校师生回家返乡的机会走访贫困新生，介绍国家社会学校的帮困助学政策，送上学校入学无忧礼包；组织“三下乡”主题社会实践，鼓励学生聚焦科技下乡、教育扶贫，以实际行动助力扶贫攻坚，其中2018年“情系新疆，践行使命”实践团赴南疆开展教育援助服务的事迹，受到中央电视台《新闻联播》专题报道。

2018年上海交通大学学生社会实践成果丰硕，在2018年“知行杯”上海市大学生社会实践项目大赛中，获2个特等奖，5个一等奖，取得历史最好成绩，问鼎“知行杯”；在2018年全国大中专学生志愿者暑期“三下乡”社会实践活动中，共青团上海交通大学委员会获评全国优秀单位，“显微知著”西部农村中学生命科学实验室援建实践团获评全国优秀团队，赛福鼎?阿力木同志获评全国优秀个人。

据悉，上海交通大学不断加强实践育人体系建设，围绕“立德树人”根本任务和学校“四位一体”育人理念，以“行万里路，知中国情”为主题，号召全校学生深入基层、深入行业、了解国情、增长才干、明确方向，推动学生价值引领工作落细落小落实，在实践中不断增强“四个意识”、坚定“四个自信”、涵养家国情怀、培养学术志趣，树立在新时代勇担民族复兴大任的远大理想和使命认同，实践育人，鼓励学生到祖国和人民最需要的地方建功立业。
