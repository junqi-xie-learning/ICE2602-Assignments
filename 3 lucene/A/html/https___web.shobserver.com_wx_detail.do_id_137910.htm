理工科高校如何运用大数据技术优势提升党建质量，看看上海这所知名大学的做法！

上观学习 2019-03-19 06:26
来源：上观新闻 作者：全林
上海交通大学运用大数据技术优势切实提升党建质量

党的基层组织是党的肌体的‘‘神经末梢’’，是党的全部工作和战斗力的基础，务必成为宣传党的主张、贯彻党的决定、领导基层治理、团结动员群众，推动改革发展的坚强战斗堡垒。在我国一些具备条件的理工科高校，如何运用大数据人才与运用优势，夯实高校基层党建，提高高等学府党的建设质量，解决理工科高校党建质量一般不高的现实问题，是基层党建需要研究与关注的一个重要方向。近年来，上海交通大学在这方面做了一些有益的尝试，值得总结分析。

 

以大数据技术建立健全基层党建工作的科学评价体系

 

如何建立健全基层党建工作的科学评价体系，是新时代高校党的建设的题中应有之义。依托大数据思维和技术，收集、追踪和分析基层党建工作相关数据的方式方法，构建基层党支部‘‘党建质量指数’’，为学校党委及组织部门掌握基层党建工作动态，及时规避各种风险，切实推动基层党建实践创新，实现一流党建引领一流大学建设，提供科学严谨的实证探索和智力支持，以实现基层党建的科学化、智能化和常态化管理。

 

一流党建的评价体系应注重五个方面的探索实践：一是组织力建设，增强基层组织的凝聚力、战斗力和向心力；二是强化政治功能，做好與情分析研判引导，宣传党的主张、贯彻党的意图和执行党的决定，体现纯洁性；三是完善服务功能，求实情、办实事和解难题，体现先进性；四是突显精细化管理功能，建立健全制度建设、常态化管理、时效性与长效性并举等；五是加强创新实践，教育引导以及日常活动的方式和方法创新等。

 

党建工作的评价系统是党建工作进入到业务与数据之间的关系、数据变化的趋势价值等更为深入的信息模型之中。上海交通大学注重建立健全基层党建工作的科学评价体系，全力推动院系政治建设、思想建设、组织建设、作风建设，把制度建设贯彻其中从严抓落实，扩大基层党的组织覆盖和工作覆盖。强基固本，从建章立制上保障基层党的领导，建立健全和完善党建工作组织体系、治理体系，责任体系、评价体系。在院系等单位成立二级党组织同时成立机关工作党委、学生工作党委、联合党工委，加强党建工作指导。先后修订、制定《基层党组织按期换届选举工作实施细则》、《学院党政联席会议工作条例》，研究学院党组织工作规则和会议议事规则，实施基层党建工作责任制和基层党建工作督导实施办法，坚持抓基层党建工作述职评议，试点推行教工党支部书记述职评议，把全面从严治党的要求延伸到基层组织，并系统地评价党建工作。

 

用大数据为基层党建工作的管理和决策提供支持

 

高校要重视提升党建工作的决策管理能力，就要把一批政治素质优、管理能力强，懂业务、有修养的优秀干部选派到院（系）党委领导岗位上，建齐配强院系领导班子。持续推动基层党建工作的闭环思维和模式创新，练好善于获取数据、分析数据、运用数据做好工作的基本功。

 

高校党支部处于教学、科研、管理和服务的前沿，与党员联系最紧密，充分发挥党支部主体作用和党支部书记的头雁效应。为促进和保证学术繁荣发展、人才培养质量不断提高提供保障，充分发挥党支部作用，上海交大全校644个党支部每年开展形式多样的主题党日活动数千场，搭建了线上线下交流平台。定期组织“支部书记沙龙”交流合作，联建共建。依托学校“一门式服务”平台，建设交大党建APP，方便各支部自主记录分享组织生活内容。并加强检查督导，对落实“三会一课”制度每季度进行汇总检查，定期抽查，确保党支部活动质量。党支部书记在推动学校发展、加强人才培养和学科建设中发挥了骨干作用，促进学术、党建双向提升，有效地提升了党组织吸引力和党员积极性，扩大了党支部活动覆盖面和影响力。

 

大数据技术如何持续推动党建工作创新发展

 

作为党的肌体的“细胞”和中坚力量，党员作用能否有效发挥直接关系到高校能否在办学治校实践中坚持社会主义办学方向、切实落实立德树人根本任务。需要创新发展，引领示范，构建党员发挥作用的有效平台，从导向上、机制上推动和保障每一位党员发挥先锋模范作用。以大数据驱动高校基层党建工作还处于一个探索与实操的创新发展期，其目标定位在构建四大平台：1、智能决策平台，为决策者在工作目标制定、过程管理(风险防范)、结果评价方面提供智能化的分析和决策支持。2、日常管理平台，问题导向，通过数据更加精确、及时反馈实际工作和目标之间的偏离，实现日常管理工作的精细化。3、党建服务平台。求实情、办实事和解难题，解决党建工作中的实际问题。4、创新联动平台。一流党建工作的创新举措，引领一流高校建设。通过这四大平台交流窗口，展示党建工作的思维、模式、技木创新，揭示党建工作对于业务发展的引领价值。从而达到习近平总书记指出的那样：“不断提高对大数据发展规律的把握能力，使大数据在各项工作中发挥更大作用”。
