上海自然科学基金、扬帆计划等将投入翻番，人工智能是投入重点之一

创新之城 2018-09-18 20:58
来源：上观新闻 作者：黄海华
支持各类企业创新，促进龙头企业勇攀高峰，促进中小企业创新创业

上海自然科学基金、扬帆计划等将投入翻番，人工智能是投入重点之一。9月18日下午，2018世界人工智能大会人工智能前沿论坛在上海交通大学举行，市科委主任张全透露了这一信息。

 

 

人工智能将对人类社会的生产和生活方式产生颠覆性变革，众多落地的人工智能技术正在创造万物智能的新时代。上海交通大学积极响应国家和上海市的发展要求，充分发挥优势，率先成立了人工智能研究院，人工智能重点实验室也于近日通过教育部论证，着力打造人工智能多学科交叉融合的研究高地，为上海市人工智能科技与产业发展积极献计献策。

 

张全说，上海市科委将在五个方面支持上海交通大学和参与合作的各方，以更高标准集聚和培养国际一流研究团队，为上海科创中心建设造就更多卓越的人才，做出更大的贡献。市科委将继续加大力度支持人工智能基础技术研究，为人工智能应用与发展提供不竭动力；打造上海人工智能功能型平台，支撑产业发展和企业培育；支持各类企业创新，促进龙头企业勇攀高峰，促进中小企业创新创业；加大投入壮大人才队伍，上海自然科学基金、扬帆计划等将投入翻番，人工智能是投入重点之一；加强政策支持，体制机制创新，为发展增活力，让保障更给力。
