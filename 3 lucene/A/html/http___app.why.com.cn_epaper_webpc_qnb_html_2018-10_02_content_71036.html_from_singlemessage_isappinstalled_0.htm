交大升旗仪式献礼祖国六十九周年华诞

青年报记者 刘昕璐

本报讯 昨天是祖国六十九周年华诞。当日清晨，伴随着上海交大国旗护卫队铿锵有力的步伐，鲜艳的五星红旗在交大闵行校区仰思坪冉冉升起。来到现场的有交大师生，上海交通大学第二附属中学、上海交通大学附属实验小学和华东师大二附中紫竹双语学校的师生代表，还有周边社区的居民。

现场气氛欢快而热烈，仪式简单而庄重，大家相聚于此，共同表达对祖国母亲的祝福与热爱，追忆共和国一路走来的光辉历程，展望国家更加灿烂精彩的明天。

不少交大学生在此驻足，共同见证国旗升起的时刻。“升旗仪式不仅是一个仪式过程，更重要的是其中传达的精神。在这过程中能最直观地感受到大家对祖国的热爱与敬仰，是一个很有意义的活动。”他们的脸上写满敬意，也洋溢着热烈而崇高的爱国之情。

前来参加升旗仪式的还有一位曾经参加过抗美援朝的老战士，他在现场激动地说：“今天是国庆节，是我们国家六十九周年的大庆，我作为一个抗美援朝的老战士，我感到非常的激动，也很荣幸，今天，我想对在场的同学们说：你们是国家未来的栋梁，待到100周年国庆的时候，你们将看到一个更为富强的中国！”

现场除了为数众多的观众，还有在仪式准备和进行阶段辛勤付出的志愿者和国旗护卫队的朋友。“大家的爱国热情非常浓厚，都积极前来参加升旗仪式，我们切实感受到了大家对祖国母亲的热爱。”现场一位志愿者说。
