上海交大：一堂穿越70年的思政课

来源：光明日报客户端2019-05-07 21:39
X

矗立在交大徐汇的“史霄雯、穆汉祥二烈士纪念碑”背后有着一段怎样可歌可泣的解放英雄故事？华东局机关进驻上海第一天住址为何选交大工程馆？5月7日下午，《战上海》作者、著名军史专家、上海交大教授刘统在工程馆前为学生们上了一堂穿越70年的思政课。

1949年4月，国共正在举行和谈期间，交大举办了一场“真假和平辩论会”，穆汉祥担任正方主辩人，史霄雯主持会议。因为在公开场合发表反蒋言论，交大总支委员会组织委员、中共徐龙区委委员穆汉祥和新民主主义青年联合会会员史霄雯二人被列入黑名单，两人分别于4月30日、5月2日被捕，5月20日他们在闸北宋公园（今闸北公园）被秘密杀害。两位烈士现安葬在交大徐汇校区校园中。

在让学生行走中观察，与历史对话，这堂穿越70年的思政课意义非凡。刘统教授坦言，写这本书就是为了展现70年前的风风雨雨，从前辈们跌宕起伏的经历中寻求他们留下的遗产和精神财富，告诉年轻的大学生们，要发扬70年前共产党人战上海的精神，从他们留下的精神财富中汲取奋斗的力量。

回顾写作过程，刘统教授表示，该书最注重的是保留原始史料和记录，些重大问题的叙述上，他都尽量引用原文，尽可能体现历史的原貌。本书追求叙述的特色和细节，把意志坚定、百折不挠、勇往直前的共产党人一个又一个还原在我们的面前。

值得一提的是，近日，《战上海》入选了2018年度“中国好书”。上海交通大学党委书记姜斯宪认为，《战上海》一书在今天具有十分重要的历史意义和现实意义，其所蕴含的历史遗产和民族精神，是一笔极其宝贵的财富，具有深刻的启迪作用。（光明日报全媒体记者曹继军 通讯员 魏娜）
