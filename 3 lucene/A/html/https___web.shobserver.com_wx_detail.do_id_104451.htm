附视频|新生报到高峰，从汶川地震羌族孤儿到终身残疾山村女娃，贫穷限制不了想象力！

教育在线 2018-09-09 12:50
来源：上观新闻 作者：徐瑞哲
上海已有17所高校新生开学，近4万新生到校报到，2582名家庭经济困难新生通过绿色通道顺利办理入学手续。

今天，上海交通大学新生开学了，当贺宇轩与陈恩桃抱在一起，这是相隔15年两个特困生的拥抱。十年前在汶川大地震中痛失父母双亲的羌族学子贺宇轩，在校勤工助学办公室主任陈恩桃关照下，8日正式通过绿色通道向上海交通大学农业与生物学院报到。他是全校这一批数百名家庭经济困难学生中的一员，更是全校万余名2018级新生中的一员。

 

 

投桃报李，从交大一路升学并毕业留校的“桃子老师”，告诉自己的学弟：根据资助政策，不算地方和社会上各种资助，他每年可获国家和校方资助至少1.4万元，上完大学绝对不成问题。

 

来自上海市学生事务中心的最新统计显示，截至9月4日的来沪报到高潮中，上海已有17所高校新生开学，近4万新生到校报到，2582名家庭经济困难新生通过绿色通道顺利办理入学手续。与贺宇轩一样，寒门再也限制不了他们的能力。

 

 

“我争取能直博，不行也要考博”

 

来自北川的羌族男孩贺宇轩展示自己的录取通知书。

 

或许是师生两人的缘分，15年前，“桃子老师”陈恩桃与贺宇轩一样也是特困生，也是农生学院新生，而且也是四川老乡。这个“校园打工王子”，曾经打过6份工并帮助各地6个孩子读书，自己则如期完成本硕学业；如今，从受助到助人的角色变化持续着，这位学校勤办负责人在母校迎来了一批批自强不息的励志学子。

 

在体育馆前广场的报到大棚里，农生学院迎新学长对于即将开始的大学生活，纷纷向贺宇轩“面授机宜”。“放心师姐，我不用手机导航，就已经可以找到教学楼、实验室、餐厅、宿舍啦，”原来，贺宇轩9月1日就受邀加入了全校的“筑力计划新生体验营”，吃住行费用均由校方承担。“营长”陈恩桃带着他以及200多个家庭经济困难学生，提前熟悉校园方方面面。期间，贺宇轩等营员各自选出10张照片，由学校寄回老家。“过几天，爷爷奶奶在家就能见到我这个大学生了，” 贺宇轩说，爷爷奶奶让他“吃好”，别担心家里。

 

2008年“5·12”，只有二年级的贺宇轩从北川当地小学飞奔出教室，逃过命中一劫。而爸妈却与他的多数师生一样，被大地震夺走。他与爷爷奶奶相依为命，从绵阳八一帐篷小学，到重建的北川中学一直坚持学业，终于以661分高考成绩如愿考入上海交大。这个暑假，他带着录取通知书，重返震后的老县城，告慰父母：“安心宽心”。

 

 

爷爷奶奶70多岁了，身体不支，无法陪他来校。他只身前往江苏，与家中在外打工的亲戚相会，再一起到沪。他告诉记者，爷爷奶奶其实并不希望他拿国家和地方助学贷款，甚至要卖掉家中林子，只为不让孙子背上什么经济负担……

 

但陈恩桃让贺宇轩先放下“包袱”、轻松上阵，自助者天助。他想起当年，自己进校时资助额度为七八千元，如今小贺这届已翻了一番。根据发展式、造血型的资助育人理念，“桃子老师”带他们加入累计已有数千名成员的“爱立方”自强自立联合会。在这个互帮互助的公益社团中，寒门学子不仅勤工助学、志愿服务，还接受能力拓展培训，包括外语、计算机甚至领导力训练。“我要一直读到博士，争取能直博，不行就考博。” 贺宇轩自信满满。

 

来自广西仫佬族的姑娘李想是上海交大医学院的一名新生，因为母亲生病她立志做一名医生救死扶伤。

 

市学生事务中心负责人向解放日报·上观新闻记者表示，在农村贫困地区定向招生专项计划中，各校招录的贫困新生在专业课以及综合能力水平上参差不齐，让他们大学四年之后交一份满意答卷，成功踏入社会，这是目前申城“发展型”学生资助的重点之一。

 

 

“我有自理能力，更有学习潜力”

 

全国新生“家访”地图。

 

不让任何一个学生因家庭经济困难辍学，这是承诺，也是行动。在上海理工大学开学前，管理学院辅导员就建起了贫困生咨询群，发现来自贵州山村的新生杨才琴曾在12岁时因脊椎高度弯曲做过一次大手术，仅手术花费就约20万元，还不幸落下终身残疾。

 

杨才琴父母均为农民，还有弟弟要抚养、奶奶要赡养，家庭主要收入源于耕种，并不稳定。小杨至今仍受“后遗症”困扰，腿一长一短，走路不正常，不能提重物更不能剧烈运动。

 

在杨才琴已经办理生源地贷款的基础上，辅导员专门指导她准备贫困生认定的材料，让她解除顾虑按时来校报到。入学当天，学校先给予1000元的临时生活补助金，再将优先安排适合她的勤工助学岗位，满足她为家庭减轻负担的一片孝心。“我高中学业水平考试11门考试中有6门获得A，今年高考成绩在学校文科生中排名前20，”杨才琴流露出阳光般的笑：“我有自理能力，更有学习潜力。”

 

华东理工大学“绿色通道”。李强 摄

 

同在8日，4200余名2018届新生向华东理工大学奉贤与徐汇校区报到，而通过“绿色通道”报到的新生们都收到了一份看似薄薄的特殊礼物——“十二卡牌大礼包”，内含华理新生欢迎卡、生活保障爱心卡、防寒防潮毛毯领取卡、绿色出行交通卡、馥郁芬芳咖啡券、新生助学金、能力发展卡、校内工作岗位卡、技能学习卡、导师陪伴卡、创新创业实践卡、青春公益卡。

 

华理12张资助卡包。

 

从缓交减免学费到预支勤工助学工资，如此细分资助只为背后全方位支撑。校党委副书记宋来介绍，12张卡牌中，爱心卡内含有价值1000元的爱心币，学生凭卡到校内慈善爱心屋选购所需的生活、学习用品；导师陪伴卡，则是学校选聘出经验丰富的老教授，为学生学习和生活答疑解惑、指点迷津；绿色出行交通卡，则是学生设计的限量专属本地公交卡，首期充值50元。

 

新生领受资助卡包。李强 摄

 

上门排摸、精准资助、个性帮扶，沪上高校各具特色。复旦大学“腾飞计划”等专项计划今年招生500余人，开通“绿色通道手机热线”，专职老师轮值接听，实现“学校有暑期，资助无假期”； 同济大学依托学生社团“挚友社”，社团化开展新生挚友系列活动，以朋辈支持、朋辈领袖示范等模式进行新生适应性教育，覆盖近半数新生……目前，各大高校校长都成了新生入学资助工作第一责任人，各校建立了“奖、助、勤、贷、补、减”的全方位帮扶体系，阻断贫困代际传递、促进资源优质均衡，“让每个孩子都能享有公平而有质量的教育”。
