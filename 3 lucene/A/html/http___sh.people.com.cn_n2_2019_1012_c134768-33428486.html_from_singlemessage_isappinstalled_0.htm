感恩军旅生涯 退役大学生士兵：此生不悔入伍！

2019年10月12日13:54  来源：人民网-上海频道
 

人民网上海10月12日电（葛俊俊 实习生 竺嘉茹） 青春有很多种颜色，唯独军旅的颜色，过目不忘。前不久，在新中国成立70周年阅兵仪式上，英姿飒爽的年轻官兵们雄赳赳气昂昂地走过天安门广场，坚毅的面庞、整齐划一的步调，给全国观众留下了深刻的印象。

告别年少青葱的纯白岁月，满怀参军梦想踏上征程，手握钢枪保家卫国。成为军人后的他们青春洋溢、正气凛然。

为了找寻这些年轻人入伍初衷，探索军旅历程，记者找到了曾经赴朱日和训练基地参加重大演习任务的退役男兵以及曾与国庆受阅女兵共同受训过的退役大学生女兵，揭秘他们曾经“不平凡”的军旅生活。

巾帼不让须眉 在“润物细无声”中成长

“自从初中接受了一场阅兵仪式的洗礼，自此，心中便埋下了一颗想参军的种子。”现就读于上海杉达学院市场营销专业学生张艳芳，2015年9月至2017年9月曾服役于武警126师通信连，期间荣获优秀士兵荣誉。

高中毕业后直接报名参军，但由于竞争激烈而落选。但她没有放弃，大一再次报名，幸运的是，这次她成功入选，成为同批20名入伍女兵中的一员，为此她倍感自豪。

当兵后新奇的感觉，仅维持了三天。

队列训练、机械训练、三公里跑体能训练、手榴弹的投击训练、打靶训练……高强度的训练让张艳芳度日如年，有时想家想的鼻头发酸。

部队规定，每周可以打一通5分钟的电话。短短5分钟的通话时间，便成为她一周的精神寄托。

 

左一为张艳芳 受访者供图

张艳芳回忆，入伍后的第一个春节，她在部队里参加报务训练。因为她妈妈的生日在年前的三天，她特意向班长申请给妈妈打庆生电话。班长提醒她，如果背不完号码，就没有打电话的可能。可她紧赶慢赶，还是迟迟没完成训练。

让张艳芳所料未及的是，在军营春晚上，当着所有人的面，部队现场连线了她的妈妈。后来她才知道，这个情节是班长早就策划好的，只有她一人被蒙在鼓里。那一刻久违的感动，让她至今记忆犹新。

从军三载，数不清的磨砺，数不清的付出。“部队给我带来的成长是‘润物细无声’的，良好习惯的养成令我受益匪浅。”从小女生成长为兵味十足、综合素质过硬的优秀女兵，在张艳芳看来，只有当过兵的人，才能理解这种感情。

她说道，穿上军装那一刻起，此生不悔入伍。

学霸携笔从戎 复学后投入校国防教育

李荣祥，上海交通大学媒体与传播学院在读硕士研究生，2014年9月至2016年9月曾服役于原陆军第十四集团军。期间，他赴朱日和训练基地参加了重大演习任务，被评为“优秀义务兵”。

朱日和演习 受访者供图

知名高校大学生参军入伍，虽说不算稀奇事，但也吸引了许不少人的目光。作为连里为数不多的大学生之一，李荣祥直言心理压力大，训练完成得出色是理所当然，若出了差错则会被其他士兵唏嘘。

“学习不分年龄，但当兵需要年少。”当李荣祥决定参军入伍时，周遭的亲友颇为不解，他却坚持自己的决定，参军是他从小的梦想，经过部队的锻炼，自己定有所获，定有所成。事实亦是如此，军旅生活让他褪去了幼稚，懂得了责任，懂得了感恩，学会了担当。

 

左一为李荣祥 受访者供图

退役不褪志，退伍不褪色。退役复学后，李荣祥加入了上海交大退伍学生们成立的“上海交大退伍军人协会”。

为了扩大退伍学生身为学生榜样的影响力，激发同学们的国防意识和报国热情，李荣祥通过“交大戎耀”公众号，定期发布学校退伍军人的动态讯息、退伍感悟文章等，把入伍期间学到的知识、技能，还有入伍经历教给他的荣誉、责任、坚强等精神，毫无保留地传授给同学们。

此外，李荣祥及“上海交大退伍军人协会”的其他成员积极配合学校武装部参与国防教育，并在军训期间担任连队副连长协助参训。

2009年起，党中央、国务院、中央军委着眼建设一支强大的人民军队，作出重大战略安排，全面展开大学生征兵工作。10年来，累计征集大学生150余万人。国家和军队为鼓励大学生参军入伍，相继出台一系列激励政策，助推一批批高学历青年走进军营。

军营是一个大熔炉，选择了军营就注定青春不平凡。同张艳芳和李荣祥一样的有志青年不在少数，他们有抱负、有担当，为建设我国的强军事业贡献了自己的一份力量。他们说，建功军营才不负青春。

(责编：葛俊俊、轩召强)
