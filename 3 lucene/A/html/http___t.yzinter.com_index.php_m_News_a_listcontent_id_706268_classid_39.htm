原创｜新增人工智能专业，上海交大电院“一体两翼”格局培养人才

2019-05-15 12:36:40

扬子晚报网5月15日讯（记者 罗晓娜）教育部日前公布了全国首批35所人工智能专业建设高校名单，上海交大新增人工智能专业，成为电子信息与电气工程学院（下简称电院）在建的第十个本科专业。作为交大创建最早的学院之一，电院历经百十年风雨，与中国信息科学的发展同频共振，缔造了诸多中国第一。在新一轮的人工智能浪潮中，百十电院积极迎接人工智能时代的使命与挑战，在科学研究、人才培养等方面布局设点，以崭新姿态重装起航。
电院是上海交大规模最大的学院，也是办学实力最强的学院之一，吸引了全国最优秀的学生在此求学。电院2019年开设人工智能特色班，形成了一个大平台+两个特色班的“一体两翼”人才培养新格局。
一体即一个大平台。电院实施信息技术引领、强弱电结合的大电类专业宽口径培养新机制，特色鲜明的电子信息领域人才培养模式荣获2018年国家教学成果一等奖。
两翼即两个特色班：IEEE试点班和人工智能特色班。电院积极探索和实践国际一流创新人才培养新模式，2011年获得电子信息领域最权威的国际学术组织IEEE授权批准，成立了国际上首个IEEE试点班，创新的国际化办学模式于2017年获上海市教学成果特等奖。2019年，电院正式招收人工智能特色班本科生，面向国家需求，致力于成为人工智能基础研究拔尖人才和交叉应用高端人才成长与发展的“摇篮”。
交大建设人工智能专业具备得天独厚的优势。上海正加快向具有全球影响力的科技创新中心进军，智能上海行动“AI@SH”口号响起，上海人工智能产业蓄势待发。据相关报告显示，全国已有1/3人工智能人才集聚上海，上海的人工智能核心企业已超过1000家，泛人工智能企业超过3000家，人工智能相关产业规模超700亿元，位居全国第一梯队。上海正致力于打造 “徐汇滨江—漕河泾—闵行紫竹”人工智能创新带，上海交通大学的两个主校区恰好位于徐汇区与闵行区，坐落在这条AI创新带上。上海已初步建成了有全球影响力的AI生态圈，为交大建设一流人工智能专业提供肥沃的土壤。编辑：李燕
