“选择交大，就选择了责任” ——上海交通大学2020级研究生的“开学第一课”

2020年09月08日 12:25:21 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

    9月6日，上海交通大学党委书记杨振斌为2020级研究生上主题为“选择交大，就选择了责任”的“开学第一课”。全校10000余名研究生在闵行校区霍英东体育中心主会场和30多个线上分会场及线上平台同上一堂课。

    

图为上海交通大学“开学第一课”现场 新华网发

    从124年前盛宣怀秉持“自强首在储才，储才必先兴学”的理念，创办交通大学的前身——南洋公学，到今天坚定自信迈向世界一流，杨振斌向新生们讲述了上海交大跨越三个世纪的历史与底蕴，以及他对新一代交大人所寄予的厚望，希望大家牢记肩负的责任与使命，传承、发扬交大人的优秀精神品质，砥砺奋进，全面成长，与祖国同向同行。

    “‘起点高、基础厚、要求严、重实践、求创新’是交大流传至今的特色传统。”杨振斌通过鲜活生动事例带领大家回首百年历史，希望新生们传承“图强”“改革”的精神，争做“第一等”人才。“上海交大为国家培养输送了各类人才30多万，校友群星璀璨，栋梁辈出，一大批治国英才、科学大师、产业巨擘和文化精英，为民族振兴、国家富强做出了卓越贡献。”杨振斌说，现在的上海交大名师大家荟萃、学科建设成效显著、科研实力雄厚、国际合作日益广阔，使同学们深深感受到今日交大集综合性、研究型、国际化之大成，正坚定自信迈向世界一流。

    “一个人能不能成为人才，主要并不是看他的智商有多高，最终要看他对国家和社会的贡献大小。最能为国家和社会做出贡献的岗位肯定就是自己最能发挥作用的岗位。”课堂上，杨振斌分享了他对人才的理解，并向新生提出了三点寄语。他说，“‘一等’是交大的文化基因，也寄寓着‘储才兴邦’，成就强国的梦想。矢志一等，理应成为当代交大学子的理想追求。”

    同时，杨振斌希望同学们在校学习期间，除了提升个人能力，努力感悟学校的精神和文化，还要认知自我，把个人成长与国家发展结合起来，探寻并坚定人生方向。保持勤奋专注，做惜时如金、与日俱进的交大人。“在交大，勤奋是一种文化，一种氛围。”杨振斌说，“要珍惜当下学习的黄金时期，树立梦想从学习开始、事业靠本领成就的观念，让勤奋学习成为青春远航的动力，让增长本领成为青春搏击的能量。”

    杨振斌勉励大家在未来的学习生活中秉持为人之本，不仅要学做学问，更要砥砺一等品格。“要学会用辩证思维看待问题，正确认识社会现象和问题。要大气，有开阔的眼界、思路和胸襟。”他说。

    “白日放歌须望远，青春作伴好读书。” 电子信息与电气工程学院2020级新生王博认真记下了杨振斌书记讲课中的这句话。“作为一名研究生新生，我的责任就是潜心学术，锐意进取，而我也会认真求学，努力在计算机视觉领域取得新的成绩和突破。”

    来自船舶海洋与建筑工程学院的新生张栗铭同学表示，“作为交大船舶专业2020级研究生新生，我感到十分的幸运与荣幸。今年23岁的我处在学习的黄金时期，走进交大学习和做科研，不忘初心、责任在肩，希望因为有了我们的奋斗，祖国能够顶天立地。”

    

图为上海交通大学“开学第一课”现场 新华网发

    “由学校党委书记为全体新生上入学第一课，是交大坚持多年的传统。”正如杨振斌在开课时所讲，这堂内容生动的入学教育“第一课”是交大从2005年起就形成的良好传统。通过第一课，同学们在感动与震撼之余，收获心灵的洗礼。爱国奉献、创新图强的崇高追求沉淀在每个交大人的心中，让同学们在成为交大人的那刻起，就烙下家国情怀的深厚印记，砥砺强国之志、实践报国之行。
