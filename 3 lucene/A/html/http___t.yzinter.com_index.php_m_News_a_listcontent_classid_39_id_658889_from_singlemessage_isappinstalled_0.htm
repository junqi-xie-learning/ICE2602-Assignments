原创|国家社科基金重大项目上海交大开题

2019-01-04 16:19:27

扬子晚报网1月4日讯（记者 罗晓娜）上海交大的“精神障碍人群语料库建设及面向脑科学和人工智能的语言研究”项目获国家社科重大项目立项，2019年1月5日和6日在上海交大正式开题。

该项目将以语言学为立足点，联合临床医学、计算机科学与技术、病理语言学、心理学等学科，旨在通过建设精神障碍人群语料库,对患者的言语语言及神经行为数据进行标注和分析，探索障碍人群言语语言大脑加工机制，为相关临床研究提供理论依据，为我国实现“脑计划”的国际领先地位奠定基础。同时项目还将利用人工智能对数据进行基于深度神经网络的多模态联合建模和统计，初步实现智能诊断，有效促进精神障碍等疾病的早期筛查和预防。

语言是人类沟通的桥梁，是心灵的窗户。言语听力能力受损不仅会给患者的日常交流造成阻碍，还易导致社交孤立、焦虑、抑郁等心理认知问题。而且，言语障碍人群不在少数且类型多样，亟需社会各界的广泛关注和共同努力。然而，我国言语康复事业起步较晚，存在专业服务人员稀缺、评估工具尚不完善、干预手段效果有待提升等问题。

为有效解决这一问题，上海交大外国语学院整合院内语言学与语料库等相关交叉领域的团队，创立了跨语言处理与语言认知基地（现更名为语言、智能与神经科学研究基地），并在原有语言认知实验室和语音学实验室、外语教学与学习研究所基础上组建言语-语言-听力研究中心，以进一步推动我国言语-语言-听力科学发展，向语言研究服务国家和社会重大需求这一战略目标前进。

目前，言语-语言-听力研究团队共有4名教授、2名美国一流大学合作教授、4名副教授、3名讲师。该团队与国外著名大学的言语认知神经科学团队开展密切合作，并携手上海交大医学院附属上海市精神卫生中心、上海市聋儿康复中心、上海交大医学院附属上海儿童医学中心、上海交大电子信息与电气工程学院计算机系、瑞士专业机构等，立足病理语言学学术研究和社会需求，联合申报课题，定期举办专题沙龙和学术会议，联合出版著作、研发产品，合作发表学术论文，推动言语障碍研究、治疗和教育体系建设，打造系列言语-语言-听力科学暑期培训项目，以其前沿性、正规性、科学性为广大科研、医疗和康复机构的人群提供最好的继续教育机会，努力建成国内一流的言语-语言-听力科学研究基地。 编辑：李燕
