【视频】医疗机器人：当你卧病在床，它会来按时喂药！

2018-12-20 19:38:55

扬子晚报网12月20日讯（记者 罗晓娜 文/摄）在进博会上备受瞩目的医疗机器人，让人们相信机器人从实验室走向临床指日可待。12月20日上海交大医疗机器人研究院国际学术论坛上，国内外医疗领域大咖齐聚，医疗机器人再次“站到了聚光灯下”。
 
喂药时间到，谁喂？“他”！

扬子晚报记者在论坛的展台看到，一种轻量型机器人，相当于为患者提供了可以自由活动的手臂，能“够着”药瓶和水杯。

病人需要定时地吃药和护理，如果身边一下子没有人怎么办？医疗机器人可以来喂药，拿起药丸喂到病人嘴里，再举起杯子放到病人口边，这些事情由机器人就可以安排妥当。上肢残疾的病人，尤其需要这样的帮助和护理，以及在紧急时或按动一些需要的开关。据展位专家介绍，它作为医疗机器人，还可以做移动的检测。另外，它的设计完全是按照人的腕部来设计的，所以病人接受起来更容易些。它的末端带三个指的抓手，拿取东西的负重是1.5公斤。

此外，一种胃肠道疾病精准无创诊察微型仿生机器人，可用于功能性胃肠病检查、消化道溃疡、出血检查、消化道肿瘤检查等；喉部微创手术机器人，适用于多种喉部手术。可完成病变组织精准咬切、电凝吸引、深腔空间缝合打结、激光多种刀具治疗；柔性手术机器人，可用于喉部微创手术，在狭小空间内做精准缝合。

多家医院签下临床研究协议

论坛期间，上海交大医疗机器人研究院分别与附属九院、附属仁济医院、附属胸科医院签订了临床研究中心合作协议。上海交大医疗机器人研究院同时还与多家企业建立战略合作伙伴关系，致力于医疗机器人研究的国际化发展。交大医疗机器人研究院联手各附属医院及国内外协作院所，针对肿瘤、心血管、脑卒中等重大疾病微创精准医疗，发展个性化、智能化手术及康复机器人。论坛上，各国医疗机器人专家还分享了手术机器人最前沿的报告。
