【新时代新作为新篇章】民生建设成果更多惠及全体人民飞入寻常百姓家
来源：新民晚报   作者：王蔚 杨玉红 左妍 易蓉   2019-11-04 11:00:00

“治国有常，而利民为本。”连日来，上海教育、卫生、房管等系统，掀起了学习贯彻党的十九届四中全会精神的热潮。广大党政干部和一线党员职工纷纷表示，对坚持和完善中国特色社会主义制度、推进国家治理体系和治理能力现代化问题作出重大决定，充分体现了以习近平同志为核心的党中央深远的战略眼光和强烈的历史担当。着眼于上海的发展，务必进一步加强普惠性、基础性、兜底性民生建设，不断增进人民福祉。

造就更多优秀人才

市教卫工作党委书记沈炜说，上海教卫系统承担着“幼有所育、学有所教、病有所医”的重要民生责任，要使改革发展成果更多更公平惠及全体人民；要构建服务全民终身学习的教育体系，完善覆盖全民的社会保障体系，强化提高人民健康水平的制度保障；要按照中央部署进一步加强和改进大中小学校党的建设，全面完成中央交办的教育综合改革和深化医药卫生体制改革国家试点任务。

    复旦大学党委书记焦扬表示，要把学习贯彻四中全会精神与中央要求的主题教育专项整治工作结合起来，加快转化主题教育专题调研成果、落实各项整改任务，健全党对学校全面领导的体制机制，真正把党的领导内嵌到大学治理各环节。上海交通大学国际与公共事务学院教授陈尧说，四中全会指出中国国家制度和国家治理体系具有多方面的显著优势，其中包括坚持德才兼备、选贤任能，聚天下英才而用之，培养造就更多更优秀人才的显著优势。社会主义教育制度是能够充分体现这一优势的。

推动健康中国建设

上海市卫生健康委员会党组书记黄红提出，要以四中全会精神为指引，把学习贯彻全会精神转化为精准对接人民群众健康服务高质量发展、推动健康中国建设的新动能，以更大的力度深化医疗卫生体制改革，以更实的举措推动卫生健康民生保障制度进一步完善，让改革发展成果更多更公平惠及人民群众，不断增强人民群众的健康获得感、幸福感、安全感，奋力开创新时代卫生健康事业发展的新篇章。

上海市第一人民医院党委书记冯运说，要把党的政治优势和组织优势转化为推动医院高质量发展的强大力量，建设让患者放心、让人民满意的公立医院。上海中医药大学附属曙光医院院长周华认为，要在“长三角一体化”国家战略中进一步发挥专业和学科优势，着力提升长三角地区中医临床和科研水平，还要做好“一带一路”的健康使者，传播中医文化、讲好中医故事。

廉租住房托底保障

“要不断增强住房困难家庭的获得感和幸福感。” 上海市房管局党组书记、局长王桢表示，要牢记为人民谋幸福的初心，房管系统将把住房保障工作放在保障和改善民生的大局中进行谋划，围绕“房住不炒”的定位要求，肩负起实现人民群众“安居梦”的使命责任。王桢说，要进一步提升精细化管理、便利化服务和高质量发展水平，以群众需求为导向，聚焦上海市户籍中低收入住房困难家庭和来沪就业新市民两类人群，不断优化完善各住房保障品种的准入条件、审核机制和供应管理。上海将继续强化廉租住房“应保尽保”托底保障功能，通过公共租赁住房持续常态化解决青年职工、引进人才、来沪务工人员等各类新市民的阶段性住房困难。

新民晚报记者 王蔚 杨玉红 左妍 易蓉
