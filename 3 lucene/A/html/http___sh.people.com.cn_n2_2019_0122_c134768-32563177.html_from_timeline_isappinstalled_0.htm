聚焦人工智能 上海交大荣获分类话题最佳案例奖

2019年01月22日21:06  来源：人民网-上海频道
 

人民网上海1月22日电（葛俊俊） 近日，由上海交通大学上海高级金融学院金融硕士项目（简称“MF”）主办的2019年国际青年领袖金融高峰会在沪落幕。最终，来自上海交通大学、西安交通大学、北京大学、麦吉尔大学和伦敦政治经济学院的优秀参赛选手组成的五个代表队分别摘得分类话题的最佳案例奖，来自西安交通大学的代表队获得全场团队一等奖。

据了解，本届高峰会在设计主题时围绕上海筹建科创板的战略方向，将科学技术创新和金融服务支持紧密结合，确定了“人工智能+资本运作”的峰会主题。同时围绕金融硕士学生的就业偏好、能力特长从行业研究、建模评估、股权融资等方向要求参赛者选定行业及标的公司，在深入调研后用AI赋能，并制定融资方案向投资者模拟融资。

“本次高峰会不仅囊括了清华大学、北京大学、中国人民大学、南京大学、复旦大学以及上海交通大学等在内的C9及名列前茅的985院校，也得到了来自剑桥大学、伦敦政治经济学院、萨里大学、多伦多大学等学子的积极响应。”据高金金融硕士项目负责人杨琳透露，本次共有来自全球1200多名知名学府的学生申请参加竞赛，最后来自世界各地22所顶尖高校的120余名优秀本科生通过层层选拔挺进决赛。

峰会项目涵盖了电商消费、教育、大数据应用（金融/交互平台）、智慧农业及智慧城市、医疗医药等多个话题。在实案竞赛环节，来自高金的教授们与来自红杉资本、启明投资、熊猫资本、景林投资、领中资本、美国信安集团等知名企业的业界嘉宾，一同担任峰会竞赛的评委，现场对项目进行评选和点评。

“学生们能在如此短暂时间内给出如此精彩的表现是超出我们预期的。希望通过此次高峰会他们能更了解市场需求，争取将更多创新的想法落地。”业界评委们如是评价。

(责编：葛俊俊、韩庆)
