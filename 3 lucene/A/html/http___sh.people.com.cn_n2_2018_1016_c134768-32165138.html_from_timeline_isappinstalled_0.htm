系统总结长三角城市治理实践经验 为全球城市治理提供范本

王奎明 上海交通大学中国城市治理研究院助理研究员

2018年10月16日21:08  来源：人民网-上海频道
 

今年是我国改革开放四十周年，同时也城镇化快速推进的四十年，四十年来我国探索出了一条具有中国特色的城镇化发展道路，同时在城市治理领域也形成了诸多宝贵经验值得系统提炼和广泛推广。长三角地区作为我国城镇化发展基础最好的城市群之一，同时是“一带一路”与“长江经济带”的重要交汇地带，未来更是建设面向全球、辐射亚太、引领全国的世界级城市群。深度、系统总结和提炼长三角地区城市治理的宝贵经验具有极其重要的价值和意义。

从方法论的角度而言，对于典型治理案例的深度分析与比较研究是对于治理经验总结最为有效的方式。今年7月份开始，首届“长三角城市治理最佳实践案例评选”活动在上海市、江苏省、浙江省、安徽省范围内展开。从案例申报情况情况来看，充分展现出了长三角地区在城市治理实践方面的引领性。

一、遍地开花的探索性实践。案例申报广泛分布于沪浙苏皖的市、区、街道、社区、居委会等多个层级，同时也包括来自科研机构、社会组织、外资与本土知名企业的申报，长三角地区的城市治理正在朝着多主体的协作性路径大步迈进。

二、制度性改革与机制性调试并举。长三角地区的城市治理实践进行了诸多创新性的有益探索，既包括了突破性的制度性改革，特别是在市一级政府管理层级方面的改革，如有的城市对于街道层级的撤销，也包括了机制性的调试，如政府机构人员的整合、服务职能的提升等等。

三、丰富多元的涉及领域。长三角地区在城市治理的探索性实践丰富多元，涉及到党建引领、社会治理、社区治理、空间规划、环境治理、政务服务、智慧治理、文化治理等若干领域。关键是每个领域内的实践机制也呈现出因地制宜的独特性。

四、“服务理念”根植于实践活动中。“城市 让生活更美好”是城市发展与城市治理的根本要义，长三角城市治理的相关实践充分体现了这一点。本次案例申报的多元领域中都深刻体现了“服务”、“人本”的理念，民众的生活满意度成为了实践活动的出发点与落脚点。

五、“智能化”助力城市治理的创新性实践。随着信息技术的进步与智慧城市建设进程的加快，“智能化”推动城市治理实践进入了新的阶段。小到如“二维码”成为了民众参与城市治理的新方式，大到如整个城市智能网络的构建。

六、“城市治理协同化”推动“长三角发展一体化”。随着交通系统的便捷与信息技术的普及，长三角地区的城市治理协同化水平日益提高，特别是在“党建引领”的作用下，长三角地区的资源整合与资源共享的层次不断提高，为长三角将来更深层次的一体化发展奠定了重要基础。

(责编：陈晨、轩召强)
