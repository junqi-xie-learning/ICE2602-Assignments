黄震：引育并举培养青年人才

2019-03-09 19:25:01 来源： 新华网

　　新华网北京3月9日电（李萌） 3月9日，全国政协常委、上海交通大学副校长黄震做客新华网2019全国两会特别报道。他认为，在青年人才培养上，一定要坚持引育并举，双向发力。

3月9日，全国政协委员、上海交通大学副校长黄震做客新华网2019全国两会特别报道。新华网 毛嘉炜 摄

　　以下为访谈主要内容：

　　主持人：上海交通大学作为我国著名高等学府之一，在人才培养方面一定有独到之处。您认为如何让高层次人才成为学科发展新“引擎”？

　　黄震：上海交通大学的目标是建世界一流大学。在人才培养方面，我认为核心就是要打造一支与世界一流大学相适应的师资队伍。近些年，我们已经在建设高层次的师资队伍上花费了很多精力。

　　主持人：高等学校青年教师是高校教师队伍的重要力量，在青年人才培养上，如何提供适应他们发展的事业空间？

　　黄震：在青年人才培养上，我有一个非常鲜明的观点，就是一定要引育并举：“引”就是海外引进的人才，“育”就是自己培养的本土人才。要坚持引育并举，双向发力。

　　此外，我建议要特别关注本土培养的青年教师、青年人才的生活状况和工作状况。

　　主持人：在人才队伍建设方面，您如何看待本土培养的青年教师和海外引进的人才？

　　黄震：我认为这两者应该是相辅相成。海外引进的人才有得天独厚的国际视野，而本土培养的人才有扎根祖国大地的优势，这两者都是青年人才队伍建设不可或缺的。交大在实践中提出“英雄不问出处”，青年人才不论是海外回来的，还是国内培养的，我们更看重的是他们自身的学术能力和水平。

　　主持人：上海交通大学在人才引进方面做了哪些方面的努力？

　　黄震：在人才引进方面我们主要做了三个方面的努力，第一是面向全球招聘，第二是同台竞技，第三是择优入职。

　　主持人：现在怎样让广大教师安心从教、热心从教，解除他们的后顾之忧？

　　黄震： 对高校来说要加强对人才的服务理念。不论是海外引进的人才，还是其他高校过来的老师，我们学校的管理部门都要提供良好的服务。说到底本土人才也好、招进来也好，重要的是要留得住、用得好，让人才在这样的环境中真正发挥自己的聪明才智。

　　主持人：在推动“双一流”大学建设的过程中，您有哪些经验分享给全国高校的管理者？

　　黄震：对高校管理者来说，一定要高度认识人才是第一资源。创建世界一流大学最核心的是人才，对待人才一定引育并举，双向发力，以此来建设一流的师资队伍，为进一步发展高等教育，创建世界一流大学打好基础。（文字：李萌 剪辑：王生元 摄影：毛嘉炜 摄像：黄璐璐 韩磊 导播：王安 主持：孙博 速记：曹莉娜）
