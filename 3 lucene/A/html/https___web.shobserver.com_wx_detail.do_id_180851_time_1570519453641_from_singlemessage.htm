这位拿全校最高荣誉的特聘教授“全学段教书”，做过中小学、本专科老师，以及硕导博导

教育在线 2019-10-08 15:22
来源：上观新闻 作者：徐瑞哲
他每次约见同学的时候都要求他们把近段的研究工作进展明确写出来，然后通过投影一条一条地仔细讲给他听。

2019年，对罗守贵教授而言，从教40个春秋。获奖无数的他，终获上海交通大学最高荣誉——教书育人奖。这40载，罗守贵担任过5年小学老师、4年初中老师、4年高中老师、5年大学专科老师、16年本科老师、13年硕士生导师和9年博士生导师——教师身份变化了那么多，但他教书育人的初心不曾改变丝毫。


罗守贵教授在浦江创新论坛上。
图片来源：上海交通大学


【一周工作时间“887”】

——“社会越是浮躁，教师越是要沉静下来，只有这样才能让学生沉静下来。”

罗守贵出生于河南省大别山区一个非常偏僻的乡村，15岁开始他在村里大庙小学当老师。当时的农村地区教师奇缺，有些教师自己仅受过小学教育，最好的教师也仅仅是1976年以前毕业的初中及高中生，八成教师不会解一元一次方程。

目睹着教育落后状况，1983年时罗守贵将一个班级从一年级带到五年级小学毕业，并考入中等师范学习。在这之后，他陆续完成了大学专科进修以及硕士和博士阶段的学习。在他硕士毕业、博士毕业和博士后出站时，有多次到其他行业在别人眼中都是很好的工作机会，但他都毫不犹豫地回到了他心爱的讲台。

解放日报·上观新闻记者了解到，2003年罗守贵刚到上海交大时，一学年承担近300学时的教学工作。那时他家还住在五角场附近，每天早晨必须乘第一班地铁，再转公交或校车到闵行校区上八点钟早上第一节课，晚上十点多才能回到家里，每天在公交车上就要近5个小时。

如今，已经50多岁的他，仍然不知疲倦地忘我工作在教学一线。近6年来，在研究生课程以外，他承担了742学时的本科教学工作，加上面向宁夏大学和西藏大学的远程教学，总共近900学时，并且教学效果优良。与职场上的“996”相比，罗守贵的工作时间是“887”：每天早晨8点前就到办公室，晚上8点后乃至更晚才回去。除了外出开会或调研，双休日和寒暑假也是天天如此。


罗守贵教授与学生们在调研中。
图片来源：上海交通大学



【为每个研究生建单独文件夹】

——“教师讲课用口，但真正能影响学生的是一身正气，真正能感染学生的是一颗真心。”


罗守贵曾修读过两轮教育学课程，无论是本科生还是研究生，他都特别重视因材施教。不仅乐于指导成绩好的同学，也将指导和帮助学习困难的学生视为责无旁贷的事。他先后4次获得安泰经济与管理学院“最受研究生欢迎的教师”和“最受本科生欢迎的教师”称号。

这5年来，作为本科生论文指导教师和志愿导师、硕士生和博士生导师，他共指导本、硕、博学生40名。他常常在各种场合说:“我们招进来的都是千里挑一，乃至万里挑一的优秀学生，我们有责任把他们培养得更加优秀，不然我们就对不起学生，也对不起家长，更对不起国家。”

每当遇到有学习困难同学的棘手问题需要解决，教务部门老师第一个想到的就是罗守贵，把学生交给罗老师他们特别放心。曾经有一位同学由于各种复杂原因面临退学的境地，罗守贵不顾繁忙工作主动帮助这位同学。不仅与教务办老师、各门课程的任课老师密切沟通，共同督促这位同学，还亲自担任他的毕业论文指导工作，最终使他顺利毕业。

同样，罗守贵指导研究生平均至少两周举行一次组会，为每位同学分类指导每两周不少于一次。他每次约见同学的时候都要求他们把近段的研究工作进展明确写出来，然后通过投影一条一条地仔细讲给他听。就这一点，罗老师的学生都比较“怕”他，因为“空口无凭”，偷懒在罗老师面前根本行不通。

每一位研究生入学，罗老师都为其建立一个单独的文件夹，里面不仅详细记录每位同学的论文开题、中期和最终稿，而且包括学生在研究过程中数据搜集、研究方法、学术论文投稿修改等各种问题及其解决方案。到毕业时，每位同学的文件夹内通常能有三四十个文件。据统计，他近5年指导的学生共获包括国家和全市的校级一等奖以上奖励或荣誉38次。


罗守贵教授与学生在一起。

【逐章配套中国经济案例】

——“教书育人是润物细无声，是绣花中的精工，关键是绵绵用力，久久为功。”


罗守贵担任本科生《经济学原理》课程的责任教授。这是一门经济学基础课，不仅要培养学生学习经济学的兴趣并建立基本的经济学专业思想，而且要为学生后续学习经济学建立一整套的基本概念、基本原理和初步的知识体系，地位十分重要。罗守贵带领十余位青年教师集体备课，建立试题库，将每一个知识点都落实到位，其团队承担了学校教改项目《使用国外教材配套系统中国经济案例的经济学原理教学改革》。为解决使用国外教材脱离中国经济发展实践的矛盾，他花费大量精力组织编写了一套逐章配套的中国经济案例。

在近4年的课程教学中，本校3572名学生使用，加上对西藏大学和宁夏大学的远程教学，超过5000名学生受益，效果非常明显。该项目今年也获得了上海交通大学教学成果一等奖。

丰富的教学内容、前沿的学术知识和鲜活的教学案例，来自于罗守贵大量的科研和社会服务活动。他先后主持了国家社会科学重大项目、国家自然科学基金项目、上海市软科学重点项目、上海市决策咨询重点项目、上海市哲学社科基金项目等国家和省部级科研项目30余项，并为包括上海、江苏、浙江、山东、河南、湖北、四川、贵州、新疆等东、中、西部近30个城市，提供了大量区域经济与产业经济发展战略的决策咨询研究。

作为国家社科基金重大项目首席专家、上海市软科学研究基地首席专家、世界知识竞争力中心亚太分中心主任，罗守贵在国内外发表学术论文（论著）100余篇（部），先后获得教育部高等学校科学研究优秀成果奖（人文社科）、全国青年优秀图书奖、上海市决策咨询研究成果奖、上海市哲学社会科学优秀成果奖等省部级以上奖励11次，产生了广泛学术影响和社会影响。


罗守贵教授。
图片来源：上海交通大学


【名师名片】

罗守贵，上海交通大学安泰经济与管理学院特聘教授、应用经济系主任。2001年在南京大学获得博士学位，2003年于复旦大学应用经济学博士后流动站出站。受学院“海天计划”资助于2007-2008年赴英国谢菲尔德大学访问研究，受国家留学基金资助于2013-2014年赴美国加州州立大学北岭分校访问研究。在交大，他获得教书育人奖以及优秀教师奖、“三育人奖”、优秀共产党员等多种奖励及荣誉。
