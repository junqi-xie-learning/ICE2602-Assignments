上海交大首届“筑力营”助专项计划新生 提前进入大学状态

2019年08月31日07:21  来源：人民网-上海频道
 

人民网上海8月31日电 人工智能机器编程、生物植物有趣实验、探秘美丽校园多角度“抢鲜体验”；钱学森图书馆、上海飞机设计研究院、瑞金医院等多领域参观；垃圾分类、急救常识、艺术欣赏多维度拓展……如何在人才培养中让“教育增值”最大化？如何全方位为学生成长成才打下坚实基础？

为进一步响应国家号召，全方位为专项计划生的成长成才打下坚实的基础。2019年，上海交大启动首届“专项计划生筑力学习先修营”项目（以下简称“筑力营”），为专项计划新生量身打造“助力工程”。

首届“筑力营”面向被学校录取的2019级国家贫困地区专项计划、思源计划、新疆内高班、西藏内高班、南疆计划以及预科转正批次录取的本科新生，共吸引各专业400余名新生参营，新生专业以医科、理工科为主，其中，医学院88人，电子信息与电气工程学院85人。专项新生主要来自河南、安徽、贵州、四川、湖南等地。

第一课堂“学业筑基”，新生营员每天进行6节课堂学习和2节自主学习，课程有数学、英语、物理、化学、计算机基础与应用，还有阅读与写作、编程基础和实践课程，通过提前接触大学课程，营员体会到高中学习与大学学习的差异，积极适应与转变。在晚间和周末进行丰富多彩的第二课堂活动，包括：“中国书法与跨文化交流”、观看纪录片等名师大咖讲座和活动等。

记者了解到，“筑力营”虽为首届，但交大在满足“经济解困”――无一名学生因经济困难而辍学的基础上，强调“精神致富”――培养学生感恩、责任与自立自强的意识，构建了“以勤工助学为主要手段，以奖助学金为激励方式，以助学贷款为重要渠道，以学费减免和困难补助为辅助措施”的全方位帮困助学体系。2018年，除国家助学贷款外，家庭经济困难本科生获得学校直接发放的各类资助人均超过10000元/年。

首届“筑力营”自8月13日开营，9月6日结束，427名新生共分为9个临时行政班级。结束后，营员将与新生共同开始新的生活。

(责编：葛俊俊、韩庆)
