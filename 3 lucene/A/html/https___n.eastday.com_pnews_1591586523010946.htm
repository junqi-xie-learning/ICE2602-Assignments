云端相聚携手设计抗疫 上海交大2020 FourC Challenge 24小时设计挑战赛开幕

2020-06-08 11:22:42 作者：傅文婧 来源:东方网 选稿：贺江敏

东方网记者傅文婧6月8日报道：上海交通大学“FourC Challenge 24小时设计挑战赛”日前在线上火热开幕。本次活动由设计学院IIDE（国际工业设计工程）项目中心主办，邀请到来自哈佛大学、西北大学、新加坡国立大学、悉尼大学、清华大学等52所境内外高校的两百余名学生参加。

FourC Challenge代表创造（Create）、联结（Connect）、相通（Communicate）、合作（Collaborate）。FourC Challenge 24小时设计挑战赛旨在通过活动倡导学生关注人类社会发展趋势，在实践中创新思路，寻找解决问题的最优设计，同时要有勇气去打破和颠覆，打造具有社会和产业价值的作品，将学科理论和实践相结合，共同服务于可持续、健康、美好的人居环境营造。

86400秒=1440分钟=24小时，可以是平凡的一天，也可以是充满创意及激情的一天。本设计挑战赛旨在有限时间里激发参赛者的想象力与创造力，在有限的24小时内，凝聚不同国籍及背景的青年，以不同的视角共同探索和解决社会问题，并提出具有创新性、科学性、前瞻性、实践性的设计方案。

2020年新年伊始，一场突如其来的新冠疫情悄然肆虐，全球疫情的发展超出了所有人的预期，人人在经历着前所未有的挑战。长时间的隔离、封闭增加了人与人之间的隔阂。在这样一种心情的驱使下，我们倡议海内外高校能联合起来，以线上设计的形式联动，用设计力量凝聚人心，凭设计理念携手抗疫。同时，希望不同文化、不同背景的青年能够增进理解、加强合作，为共同面对全球危机提出新的理解与方案。本次竞赛的主题定为 “ReConnect”，意为重拾因为疫情而分开的联系，也希望能用设计重拾人们对疫情后生活的向往。

竞赛以Make-a-thon的方式举行，各参赛小组将在24小时内围绕主题开展线上直播创作设计，全程使用英语交流。参赛所需设备（计算机及程序软件等）及Prototype制作物料由参赛选手自备。

评选过程采用成果演示视频展示和线上评审的方式，由竞赛评委根据作品的创新及创意性、作品的完整性、技术合理性和作品应用前景等因素综合评审。竞赛奖项分设特等奖一组，一等奖两组，三等奖三组，优胜奖若干，奖金金额共计近10万元。

上海交大设计学院院长阮昕教授表示：“作为设计师、设计学生，学习设计的第一课就是共情，从疫情爆发到现在，我们目睹了很多让人焦虑、难过、也很感动的事，会让人有‘想做点什么’的意愿。在现在线上教学已经成为新常态的状态下，线上竞赛依然有很多挑战，但我们相信，在重新建立联系的这一过程中，我们对沟通的愿望会跨越这些障碍。我们将来自不同国家、地区、院校、专业的同学们、评委们混合编组，大家跨越空间，集合在24小时这一个短暂的时间内，沟通、交流、创作，通过思想碰撞和创新探索，用设计特有的创造性力量，可以携手做一些相互关怀、让人对未来具有更多向往的事。”
