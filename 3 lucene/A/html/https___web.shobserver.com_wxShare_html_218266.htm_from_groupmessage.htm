新学期第一课在线讲“时间与生命”，全球5万师生重聚上海交大一起上“云课堂”

教育在线 2020-03-01 16:30
来源：上观新闻 作者：徐瑞哲
包括200多门用非中文讲授的课程，涉及 100余位任课教师，其中 55位在国外的外籍教师也将同步开展线上教学。

“今天我很高兴以一名公共卫生战线医务工作者站在这个讲台上，为大家带来一课。”1日下午2时，上海交通大学2020年春季学期在线教学第一课开讲。在疫情防控特殊时期，在校长、党委副书记林忠钦院士主持下，授课名师来到课堂现场，与全球100多个国家的5万多名交大师生“云端”重聚，通过网络空间同上一堂课。

【时间就是生命，向这五六百人致敬】

时间就是生命。在第一课上，校党委书记姜斯宪向目前仍在湖北省的交大师生致以真诚祝福。自除夕夜以来，上海交大各附属医院先后有560位医护人员挺身而出、驰援武汉，占到全市援鄂医疗队总人数的三分之一以上，用行动续写出交大人家国情怀和责任担当。

同时，全校科研人员急国家之所急，在病毒核酸检测、疫情排查机器人、医疗污染物处理、医用防护器具，以及专家咨政建言、防控知识普及等方面也贡献颇多。

“120年前战火纷飞中，秉承‘自强首在储才，储才必先兴学’理念，毅然接应照料北洋大学堂师生赓续学业……”姜斯宪深情回顾了交通大学124年发展历程中，在建校、抗战、改革开放之初等3段艰难困苦中读书治学的往事，以“众志成城战疫必胜，学在交大不负青春”为题为全校师生上课。

中科院院士、上海交大化学化工学院樊春海教授在线教育中。

事实上，开展全覆盖、大规模的在线教学在交大百年校史上也尚属首次。对此，姜斯宪语重心长提了两点要求，“希望同学们无论身处何地，都要惜时如金，专注勤奋；也希望大家以此为契机，独立思考、激浊扬清，在共同应对疫情中实现理想信念的升华和人生的成熟成长。”

【公共卫生之战，提高免疫力干货满满】


“目前我们正经历着一场没有硝烟的战争，这次战争让我们每个人都成为了战士！为生命而战！” 上海交通大学医学院公共卫生学院院长王慧以“防控疫情，从我做起”为题作报告。

专家之言，干货满满。关于如何提高免疫力，她向师生给出均衡营养、健康生活和心理调节三点建议。

王慧专题报告。

其中科学合理的营养膳食能快速有效改善营养状况、增强抵抗力。目前，国家卫健委已经发布了新冠肺炎防治营养膳食指导，对于一般人群，归纳其重点包括：食物多样，谷类为主以保证能量供应；多吃蔬果、奶类、大豆来补充足够的微量元素；适量吃鱼、禽、蛋、瘦肉确保摄入足够的优质蛋白；少盐少油控糖，保证充足饮水。她说，一个特别重要的提醒在于：要坚决杜绝食用野生动物。

王慧与学生们分享希望，“希望同学们追求热爱自然情怀，真正懂得尊重生命、敬畏自然、热爱自然、与自然和谐共处；也希望大家携起手来，提升全民公共卫生素养和健康意识，众志成城、守望相助，在与病毒的斗争中早日凯旋。”

上海交大-巴黎高科卓越工程师学院法方院长Frédéric TOUMAZET（杜伯泽）在线教学。

【3000余门次课程上线，包括外籍在内千人任课】

解放日报·上观新闻记者了解到，伴随在线教学第一课，从3月2日起，上海交大将全面启动2020年春季学期的“线上教学”，依托在线教学平台，针对绝大部分理论教学课程，开展直播授课、录播授课、慕课授课、研讨授课等多种形式的线上教学活动。

留学生托马斯。

上海交大自2月初就开始筹备在线教学工作。据悉，这学期3000余门次课程的1000余位任课教师均积极响应、全情投入，包括200多门用非中文讲授的课程，涉及 100余位任课教师，其中 55位在国外的外籍教师也将同步开展线上教学。

留学生张筱妍。

仅从有统计的次数看，期间开展了10多次zoom和canvas等教学工具的培训，以及20多次示范教学培训，确保实现“停课不停教，停课不停学”。姜斯宪强调，“延期开学、在线上课”是新学期的特殊“打开方式”，也是学校、老师和同学们一起拥抱变革、探索创新的机遇。

校长林忠钦院士主持第一课。

“今天，学校专门组织这次特殊的‘第一课’，既是向大家发出新学期的‘动员令’，也是希望给大家带来战‘疫’之中的现场教学。”林忠钦表示，线上教学是从“面对面”到“键对键”，全民战疫也是毕生难忘的“人生一课”。
