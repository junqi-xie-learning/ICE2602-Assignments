三亚与上海交通大学合作共建深海科技城
海南日报客户端 2019-04-09 19:45:28

编者按：

据海南日报客户端丨记者黄媛艳

4月9日，三亚与上海交通大学在沪签署合作协议，双方将围绕招商引智、科研合作、人才培养等开展合作，共建三亚崖州湾深海科技公共平台，设立“上海交通大学崖州湾深海科技研究院”，旨在推动鹿城深海科技基地和国家深海基地南方中心建设，助推城市高质量发展。

 三亚与上海交通大学签署合作协议 

根据协议，双方将着眼于海洋尤其是深海领域发展，着力将三亚崖州湾深海科技公共平台打造成为全国深海科技领域研发的开放式共享基地，力争形成国家深海科技人才集聚地和深海科技研发高地。该公共平台将以超前规划布局深海装备和技术发展，探索构建集科技与产业连接器、科技成果转化助推器、创新创业孵化器和应用型人才哺育器等功能为一体的发展格局。 同时，双方将抢抓新一轮产业革命发展机遇，共同在三亚深海科技城设立“上海交通大学崖州湾深海科技研究院”，为城市和产业发展提供人才和智力支持。 上海交通大学党委书记姜斯宪透露，此次与三亚的合作是该校与海南战略合作内容的一部分。学校将充分发挥在深海科技领域办学、科研和服务产业的优势，主动对接国家战略，把握海南快速发展的机遇，在南海海洋牧场、智能渔场、海上养殖工船、海水综合利用等多领域开展技术装备设计、研发，全方位支持三亚面向深海科技的前沿发展规划落地。

“三亚因改革而兴，因开放而强，该市将充分发挥地理、资源、政策等优势，深化与上海交通大学等方面的海洋科技合作，力促高新技术产业发展，三亚在教育、医疗、人才服务等多方面提供全方位支持。”省委常委、三亚市委书记童道驰表示。 据悉，上海交通大学是我国船舶与海洋工程人才培养和科学研究的发源地之一，拥有船海工程A+学科，涉海学科群布局完善;拥有船海领域唯一的国家重点实验室—海洋工程国家重点实验室等多个国家级科研基地和一大批先进的仪器设备与配套设施，海洋装备试验能力国际领先;长期致力于深海探测与海底资源开发装备技术研究，承担船海领域多个国家重大涉海任务，取得一批涉海重大成果。此次校地深度合作是双方在成立附属儿童医学中心三亚妇女儿童医院、附属仁济医院与三亚市人民医院结成医联体等工作后的又一次牵手同行。
