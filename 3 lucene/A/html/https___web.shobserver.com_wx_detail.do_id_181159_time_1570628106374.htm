诺贝尔牌锂电池起争议！差20岁两位诺奖得主与中国常有交集，争相上演“战争与和平”

创新之城 2019-10-09 21:27
来源：上观新闻 作者：徐瑞哲 裘雯涵
锂离子电池终究会拿诺奖，但可能没有人能够单独拿下，冠以“锂电池之父”的约翰·B·古迪纳夫也不能例外。

北京时间9日傍晚揭晓的2019诺贝尔化学奖，颁给了几乎人人都用的锂电池。对锂电池本身而言，这座诺奖并无争议；但对依旧“3人成组”的新科诺奖得主而言，尤其是年龄相差20岁左右的两位美国教授97岁古迪纳夫和78岁威廷汉，确是各有粉丝与拥趸，排位先后存在一定争议，有人甚至出过一本相关书籍《锂电战争》，讨论锂电池发明者的是非功过。

但不论如何，他们都与中国、与上海、与本土专家常有交集，在中外能源电力领域中相安无事、合作甚好。

【“诺贝尔牌”电池呼声高】

锂电池打赏“诺贝尔牌”，毫无争议。“今年颁奖给锂离子电池是实至名归。”上海科技大学物质科学与技术学院林柏霖教授告诉解放日报·上观新闻记者。锂电池投入市场近30年来，学界对这项发明获得诺贝尔奖的呼声一直很高。为何今年诺贝尔奖才垂青于锂电池？其推测，这是由于近年来锂电池在新能源汽车等移动式产品中得到广泛应用。此外，可能也和全球气候变化导致的二氧化碳减排压力有关。“我们团队曾在《焦耳》杂志上发表论文，预测2040年前的这段时间将是人类通过二氧化碳减排，避免灾难性气候变化最后时间窗口。锂离子电池，则是低碳排放新能源储存方式中必不可少的一种。”

在林柏霖看来，奖项颁给3位科学家让人信服，“他们三人接力做出了商业化的锂离子电池。”其中，M·斯坦利·威廷汉（M. Stanley Whittingham）是锂离子电池最早的“开拓者”。1970年，他就采用硫化钛作为正极材料，金属锂作为负极材料，制成首个小型化锂电池，但由于没有选对材料，最初的锂电池极易爆炸。 

1980年，约翰·B·古迪纳夫（John B Goodenough）提出，使用金属氧化物制成电池的正极，大为提升锂电池的性能。不过，他还是没能开发出商业化的锂离子电池。等到1985年，当诺奖第三人——吉野彰（Akira Yoshino）将金属锂和碳材料组合在一起用作电池的负极，才真正解决了锂离子电池的安全问题。此后，锂电真正走进并改变了人们的生活。 

诺贝尔奖官网

“我们都是在大树下乘凉的人，这三位科学家则是种树人。” 不过，上海理工大学能源与动力工程学院苏林教授也提出，目前锂元素在地球含量十分有限，属于相对微量元素，“如果把目前地球所有的锂都只用来生产新能源电动汽车，也只能满足几千万辆新能源汽车使用。”因此，在锂离子电池广泛应用的同时，也需要同步进行回收处理等方面的研究。此外，新的能源形式还需要不断进行开发。

【老爷子一点不像年近百岁】

在80后教授江浩眼中，古迪纳夫则被尊称为“老爷子”。从新加坡求学从研，到回国工作从教，江浩见过“老爷子”不止一次。“他常跑中国，在国际会议上做报告”，作为华东理工大学超细材料制备与应用教育部重点实验室副主任，江浩最近一次见古迪纳夫也是他90多岁之时，“不仅身体好，而且很敬业。”

“古迪纳夫教授一点也不像是个90多岁的人，看上去特别精神。”上海科技大学物质科学与技术学院刘巍教授告诉解放日报·上观记者，4年前她曾经在美国加州伯克利大学和古迪纳夫教授有过一面之缘。当时，教授做了一个多小时的报告，会后刘巍与他交流时，不禁感叹：教授的思路仍比一般学者要清晰得多。直到目前，古迪纳夫教授仍在美国德州大学奥斯汀分校授课，他的学生中有不少都是中国留学生，他们也将成为国内新能源储能领域的生力军。

尽管古迪纳夫最初并非电化学专家，但他当年一下子就提出了两种锂电池的正极材料：钴酸锂、磷酸铁锂，与负极材料——金属锂相配合，形成从高到低的电势差，“就如同水往低处流一样，让锂离子在正负极之间往返跑，完成充电和放电过程。”因为两种含锂化合物特性不同、各有短长，分别主要用于电子产品和电动车等领域。

事实上，特斯拉最新款电动车采用的，以及包括江浩本人正在研究的，都是新型化合物“镍钴酸锂”的正极材料，降低贵金属钴的比例，而将镍的比例提高至80%甚至更高。但不可否认，锂电池没有根本性地突破古迪纳夫提出的基本架构。江浩认为，古迪纳夫后的数十年至今，锂离子电池性能不断发展和完善，包括此次同获诺奖、实现商用的日本科学家吉野彰在内，让锂电池“多快好省”并更加稳定和安全，不论对于手机还是车辆，正朝着更快充电、更长续航方向进步。

【中美教授每年要见好几次】

听闻化学诺奖公布，马紫峰教授多年前的预言也被验证了，他告诉解放日报·上观新闻记者：锂离子电池终究会拿诺奖，但可能没有人能够单独拿下，冠以“锂电池之父”的约翰·B·古迪纳夫也不能例外。

马紫峰作为上海交通大学化学化工学院特聘教授兼能源研究院副院长，也是中国化工学会理事兼储能工程专业委员会主任委员、上海电化学能源器件工程中心主任，不仅在海内外合作研究中与这位最年长的诺奖得主接触过，最近半年还邀来此次诺奖名单上的第二位美国人M·斯坦利·威廷汉来沪，并共赴绍兴，参加第14届中美电动汽车与电池技术信息交流会。会议由美国能源部与我国科技部合办，宾主双方和平友好、坦诚相见。

“威廷汉和我每年要见好几次。”在马紫峰看来，论资历，威廷汉确实在古迪纳夫之后；但论成果，并不一定。他表示，威廷汉上世纪70年代就在《科学》杂志上发表锂电池原型论文，只不过他当时只是初入美孚石油公司的实习员工，而古迪纳夫已是麻省理工学院副教授。事实上，在日本人商用开发后，古迪纳夫直至1990年代才正式申请并获得相关专利。这两个时间点，与两人的年龄差一样，也相去20年光景。

吉野彰也曾应邀探访上海，当时他就感慨于车用锂电池和新能源汽车发展迅猛之势，也对中国锂电池制造商评价甚高。他笑言，借助于日本科技力量，锂电池在研发上有了重大突破，而中国企业也正与全世界公司一道，让锂电池重新定义未来，“尽管技术千变万化，然而与之相对应的科学研究并非一日之功，在快慢之间寻求平衡与突破是科技的真正要义所在。”

不论锂电界的“战争与和平”，诺奖新得主对锂电池发展贡献颇丰，同样中国锂电池也为世界做出贡献。马紫峰教授团队完成的 “磷酸铁锂动力电池制造及其应用过程关键技术”项目，今年荣获国家科学技术进步奖二等奖。他表示，无论是在新能源汽车、轨道交通、物流车等电动运输工具，以及智能电网储能系统等，中国在磷酸铁锂电池应用领域的研发与生产已经走在了世界前沿。“一项新技术并非诞生于灵感产生的瞬间，而是在广大研究人员不断探索和实践的基础上逐渐发展起来的。”
