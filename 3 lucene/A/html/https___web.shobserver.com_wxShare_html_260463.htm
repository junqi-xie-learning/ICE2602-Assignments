源自双一流的“大海洋”，前浪后浪奔涌“环交大”，新平台泊入“大零号湾”

教育在线 2020-06-18 11:40
来源：上观新闻 作者：徐瑞哲
在基础理论突破上跑出“最先一公里”和在创新成果转化中跑好“最后一公里”。

“双一流”建设高校的世界一流学科建设，让“环交大”的“大环”进一步外溢，前浪后浪在“大湾区”奔涌而来。

6月18日，上海交通大学“大海洋科研创新平台及产业化基地”落户闵行开发区。5年前的今天，“零号湾”全球创新创业集聚区在此启动；5年后的今天，上海交大、闵行区政府、地产集团在这个首批国家级双创示范基地的基础上，合作共建 “大零号湾全球创新创业集聚区”及“大海洋科研创新平台”。

2019年12月2日，上海市委主要领导等来到上海交大调研表示，上海交大要为上海强化科技创新策源能力，在基础理论突破上跑出“最先一公里”和在创新成果转化中跑好“最后一公里”。

截至去年年底，零号湾在孵项目总数超过630项、在孵化企业超过470家、项目融资总额超过 12亿元、获投融资企业超过40家。另有100多位创业导师、91位创投导师、48位法律顾问提供相应服务，“小零号湾”创新创业园区初现规模效应，初步建成双创生态体系，成为上海南部科创中心建设重心之一。

解放日报·上观新闻记者了解到，“大零号湾”将实施功能开发、产业开发、形态开发等三大核心开发，紧紧依托上海交大、华东师大，推进成果转化和企业孵化，打造医疗机器人、人工智能、海洋装备、新材料等四大主导高科技产业，并建设组团化的全球科研创新区、新兴产业引领区、现代商业与文创教育服务区、高品质国际生活社区4个功能区，实现各功能区在产业、功能、布局、形态上的交织配套、交互融合和集合发展。

据悉，为进一步服务海洋强国战略和上海科创中心建设，上海交通大学制定“大海洋”专项战略，2020年3月正式成立海洋装备研究院。海洋装备研究院以教育部深海重载作业集成攻关大平台为依托，对标国家实验室建设的要求，同时实施“深蓝计划”主动谋划海洋科技创新，为海洋装备领域自主创新提供源头供给。

新研究院按照“三步走”发展规划，到2022年，依托闵行开发区文井路基地探索产学研工科新特区、新模式，建成海洋装备战略新兴产业创新基地。到2025年，建成上海海洋装备研究中心，成为上海具有全球影响力的科创中心的重要单元。到2030年, 作为主体之一承建海洋领域国家实验室，成为国际海洋研究的杰出人才聚集地。

今天，海洋装备研究院首批遴选了深海重载作业装备研究中心、船舰装备可靠性评估与智能诊断研究中心、海洋智能信息技术与系统装备研究院、焊接与激光制造研究中心、水下专用特种装备研究中心等来自校内多个院系的9个团队入驻闵行开发区“大零号湾”，以国家重大任务为抓手开展集成攻关。

与此同时，上海交大海洋装备研究院、教育部深海重载作业装备集成攻关大平台、国家大学科技园（闵行开发区基地）、科技创新与成果产业化基地、上海交大-闵行开发区产学研创新合作中心均告设立。
