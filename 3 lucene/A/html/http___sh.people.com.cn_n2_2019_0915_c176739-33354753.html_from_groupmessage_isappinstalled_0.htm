搭建文化体验“桥梁”，上海交大举办中日青年精英夏令营

2019年09月15日17:46  来源：人民网-上海频道
 

人民网上海9月15日电 今年是“中日青少年交流促进年”，9月14日，中日青年精英夏令营开放日暨东京大学・大阪大学宣讲会在上海交通大学徐汇校区凯原法学院东方会堂举办。这也是上海交通大学首次面向日本一流大学的大学生举办夏令营。

此次夏令营共有东京大学28名、大阪大学17名，共45名大学生来沪参加。开放日活动作为中日青年精英夏令营的重要组成部分，扩大了夏令营的社会影响力，为中日青年交流搭建“桥梁”，助力了双向留学。

本次中日青年精英夏令营的课程经精心设计，颇具特色：注重理论学习和实践、文化课程的巧妙结合，不仅有喜闻乐见的八卦掌、书法和乒乓等兴趣课程，还有与曾留学东京大学或大阪大学的交大教师校友交流、深入虹桥街道与市民交流垃圾分类、参观青浦区精神卫生中心了解中国社会对弱势人群的关爱、上海中医药大学针灸推拿体验等特色实践活动。

夏令营的教学语言为中文，前期根据日本学生的中文水平做好分班准备，设计教学方案和内容，将中文教学与考察实践有机融合：夏令营日程一般为上午在课堂中文教学，下午外出考察交流。中文教师根据外出考察交流的内容，设计开展教学；班主任跟班外出，现场指导日本学生在考察交流中练习课堂中学到的知识点，通过学以致用提高学习兴趣和效果。

文以修身，武以养德。素有“行走的艺术”之称的八卦掌，是一种以掌法变换和行步走转为主的中国传统拳术，在治病、内功、技击和涵养道德方面，也有明显的效应。此次活动邀请到了尹派宫式第四代八卦掌传人徐培根先生授课，日本学生们学得有模有样，体会到了中国传统武术的博大精深和乐趣，中日学生在共同体验中国文化中也增进了交流。

为探索讲好中国故事，深入推动中日人文交流，9月20日结束在上海的活动后，其中12名东京大学学生将赴上海交大大理研究院实地学习考察，他们将从中日比较的角度，围绕垃圾分类、教育、环保等主题开展调研并撰写研究报告。同时上海交大孔海南教授将从环境保护等国际共通的热点话题入手，与日方学生座谈交流，讲好、传播好中国故事。

上海交大党委副书记周承认为：“国之交在于民之亲。在两国国民的交往当中，青年人代表着未来。希望通过举办活动，在中日两国青年层面、特别是在代表着中日两国未来的一流大学大学生的心中播下互相了解对方，互相愿意跟对方交往的种子，从而为进一步推动中日两国的相互理解、友好交流做出贡献。”

(责编：葛俊俊、韩庆)
