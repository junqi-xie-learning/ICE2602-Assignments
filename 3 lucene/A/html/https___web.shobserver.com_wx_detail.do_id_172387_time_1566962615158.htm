认识一下中国队学历最高的这对上海学霸组合：网络安全“夺旗赛”勇夺世界技能亚军！

教育在线 2019-08-28 11:19
来源：上观新闻 作者：徐瑞哲
肖子彤从2016年开始担任上海交通大学0ops安全战队队长，冯柱天是战队核心成员之一，两人也是学校网络安全俱乐部的成员。

Capture The Flag（CTF），直译“夺旗赛”。CTF竞赛发展成为全球范围内网络安全圈里的流行比赛，也成为刚在俄罗斯喀山落幕的第45届世界技能大赛新增的网络安全比赛项目之一。

当地时间8月27日晚，北京时间今天凌晨，各项赛果在颁奖式上揭晓：来自上海交通大学的两名研究生——肖子彤、冯柱天代表中国夺得这项新赛事的世界亚军！

这对学霸组合是中国代表团参赛63名选手中学历最高者，以国内选拔赛第一名的成绩入选国家队，在上海10名赴俄参赛选手“两金一银”中拿下唯一银牌——来认识一下？

【世赛年龄放宽3岁，95后一生只赛一次】

被称为“技能奥林匹克”世界技能大赛，按其规则所言，每位选手一生中只能参加一次世界技能大赛。同时，此项目与其他世赛项目要求年龄22岁以下不同，选手年龄范围扩大到25岁以下，这样就使两位1996年出生的上海交大研究生才得以参赛。

比赛现场，“网络双子星”冯肖组合坐在模拟工作间的电脑前，眼睛紧紧盯住屏幕，双手在键盘上飞快舞动。网络安全项目两人一组，分为A模块、B模块、C模块3个不同的任务模块，安排在4个比赛日进行。其中，A模块为基础设置和安全强化，B模块为网络安全事件响应、数字取证调查和应用程序安全，C模块是夺旗行动（CTF）挑战。

自备战以来，从市赛、国赛到世赛，一路过关斩将，两人就一直保持着这种紧凑、高强的实战状态。他们说，“世界技能大赛是一个世界级水平的大舞台，能代表中国与来自世界各地的选手华山论剑，不仅可以开拓视野，更是终身难忘的经历。”

【同门4年并肩征战，以赛促学获奖无数】

解放日报·上观新闻记者了解到，肖子彤、冯柱天均来自上海交大电子信息与电气工程学院的网络空间安全学院，堪称同门兄弟。“寓学于赛、以赛促学”，上海交大十多年来在网络空间安全领域坚持自身办学和科研特色，在国内形成了明显的学科优势和高地效应。

其实此次大赛前，两人早已成为黄金搭档。肖子彤从2016年开始担任上海交通大学0ops安全战队队长，冯柱天是战队核心成员之一，两人也是学校网络安全俱乐部的成员。

本科时，他们就因参加全国大学生信息安全竞赛相识。近4年并肩作战，相互学习一路合作，在众多比赛中战绩斐然——2018年“强网杯”全国网络安全挑战赛线下赛二等奖、2018年国际“*CTF”比赛第一名、MeepwnCTF季军、第十一届全国大学生信息安全竞赛一等奖；DEF CON CTF第四名；WCTF亚军；XCTF联赛总决赛亚军；全国高校网安联赛2018赛季（国家级）预赛第一名、决赛第一名等。

【一动一静一外一内，优势互补配合默契】

赛训之余，两个小伙伴又恢复了阳光大男孩的本色。肖子彤酷爱旅游，还喜欢看历史书、读四大名著；冯柱天则喜欢跑步、健身。可以说，那些一起读过的书、一起聊过的天、一起攻克的难题、一起打过的比赛，一路拼搏，共同战斗，既是知己，又互为榜样。

现实生活中，肖子彤比较开朗、健谈，自觉担当了学霸组合的“对外发言人”；冯柱天比较内敛、沉稳，心有成算，却不宣之于口。而在竞技场上，一个灵动跳脱、随机应变，一个注重细节、精益求精，两人优势互补、配合默契，兼备广度、深度、灵活度、准确度。

在历经各种大大小小的CTF比赛后，他们认为，在赛程中，除了队员的默契配合与良好的应对策略，最大的挑战是，如何在落后情况下保持良好心态，就算是错失先机也要重整旗鼓。正是从每一次的比赛中吸取经验，不惧“被虐”，才让这支中国上海战队快速成长，最终高举国旗登上世界技能最高领奖台。
