优势叠加，跑好科研成果转化的最后一公里

上海交通大学“大海洋科研创新平台及产业化基地”落户闵行开发区

2020年06月18日17:52  来源：人民网-上海频道
 

人民网上海6月18日电（龚莎）6月18日上午，零号湾全球创新产业集聚区建立五周年暨上海交通大学“大海洋科研创新平台及产业化基地”落户闵行开发区启动仪式在位于闵行开发区文井路217号的基地建筑内成功举行。

启动仪式上，林忠钦校长、冯经明董事长、倪耀明书记分别代表上海交通大学、地产集团、闵行区致辞。 

上海交通大学校长林忠钦在致辞中表示，2015年4月，闵行区政府、上海地产集团、上海交通大学三方签署合作备忘录，整合区内人才服务及政策优势、国家级开发区运营及管理优势、高校领先科研力量及技术优势，开启“零号湾全球创新创业集聚区”建设。“零号湾”当前阶段建设的成效，得益于三方共同的目标，以及优势互补的投入、携手同心的配合、及时到位的沟通，为下一轮发展积累了宝贵经验，探索了有效机制，奠定了良好基础。此次落户闵行开发区的大海洋科研创新平台及产业化基地，正是“大零号湾”建设产业引导的核心任务之一。相信以此次启动仪式和一系列合作项目签约等为契机，在零号湾全球创新创业集聚区现有优势基础上，三方将进一步整合优势、携手同行、紧密合作，打造海洋装备产业新高地，共同迈向大零号湾建设新征程。

地产集团董事长冯经明在致辞中表示，五年前闵行区、上海交大和地产集团合作，零号湾创新创业集聚区项目正式启动，通过整合市场、大学、国企三方力量，对标上海科创中心建设，实现优势叠加，项目取得很大成绩，为助推科技成果转化、升级区域产业、推动大众创业、万众创新做出了积极贡献。如今上海交通大学“大海洋科研创新平台及产业化基地”落户闵行开发区，是三方在科技成果转化方面深度合作的又一次成功典范，也必将加速闵行区国家科技成果转移转化示范区建设与融入长三角一体化发展战略步伐。地产集团将进一步深化“生态、生产、生活”高度融合的城市更新平台发展战略，充分发挥地产闵虹在空间资源、产业承接、运营管理等方面优势，有效利用上海交大在科技创新策源、科研集成攻关和科技成果转化等方面优势，深度合作推进“大零号湾”建设，持续推动形成科创可持续性发展生态体系，努力把“大零号湾”打造成为首批国家双创示范基地，成为闵行开发区未来转型发展“新引擎”，为上海加快建设具有全球影响力的科技创新中心战略目标作出新的贡献。 

闵行区委书记倪耀明在致辞中表示，“大海洋科研创新平台及产业化基地”落户闵行开发区，这标志着闵行区在建设上海南部科创中心核心区、国家科技成果转移转化示范区的道路上又迈进了坚实一步。在零号湾创立五周年之际，作为建设“大零号湾”的实际行动之一，交大将大海洋科研平台及产业化基地落户闵开发，并与开发区企业签约，助力企业技术攻关。零号湾“老东家”之一的地产集团，也积极参与其中，我们热烈欢迎其他经济主体都积极参与进来，携手共进、共享机遇、共创未来。

仪式上，围绕大海洋科研创新平台及产业化基地正式启动，进行了一系列重点项目的集体签约。一是上海交通大学副校长王伟明、闵行区副区长吴斌、地产集团副总裁李钟分别代表上海交通大学、闵行区人民政府、上海地产集团进行了三方签约。二是上海交通大学副校长毛军发，地产闵虹执行董事、总经理冯晓明分别代表上海交大和地产闵虹签约共建产学研创新合作中心。林忠钦校长，倪耀明书记，冯经明董事长，朱嘉骏总裁，李钟副总裁，吴斌副区长，奚立峰副校长，毛军发副校长，王伟明副校长，孟光校务委员会副主任等领导见证签约。三是地产闵虹执行董事、总经理冯晓明与交大海洋水下工程科学研究院院长杨启及其他海科院产业项目代表进行集体签约。此外，还进行了上海交大海洋装备研究院与上海交大科技园、海装院和海装项目团队，上海交大科学技术发展研究院与闵行开发区企业圣戈班研发、米其林（中国）、ABB高压电机、ABB船舶、亨斯迈研发中心、凯士比泵、三菱电梯等集体签约。

启动仪式上还为“上海交通大学海洋装备研究院”、“教育部深海重载作业装备集成攻关大平台”、“上海交通大学国家大学科技园”、“上海交通大学-地产闵虹科技创新与成果产业化基地”以及“上海交通大学-闵行开发区产学研创新合作中心”进行揭牌。

近年来，地产闵虹在地产集团的领导下，按照地产集团城市更新平台总体发展战略，主动融入上海科技创新中心建设，坚持科技创新引领，走科技创新园区发展之路。2015年4月，按照地产集团部署，地产闵虹与上海交大、闵行区人民政府政府共建“零号湾全球创新创业集聚区”。三方各自发挥优势、协作高效推进、合作成效显著，“零号湾”获批成为首批国家级双创示范基地。经过五年发展，已形成较好的创新创业生态体系和行业品牌影响力。截止2019年12月，累计孵化项目超过620个，在孵企业超过460家。“零号湾”已经成为上海南部科创中心的重要组成部分。2018年起，地产闵虹和上海交大以“零号湾”的合作共建为基础，明确将“大海洋科研创新大平台”（交大称为“深蓝计划”）引入闵行开发区，双方合作共建“上海交大――地产闵虹科技创新与成果产业化基地”，进一步发挥双方在平台、资源、资本以及智力、科技、人才、信息等方面的资源优势，致力于推动打造我国海洋装备领域集应用研究和产业化于一体的高端示范平台。未来，地产闵虹将围绕“科创闵虹”的“十四五”发展新主线，在地产集团的领导下，在闵行区委、区政府的支持下，携手上海交大，从零号湾全球创新创业集聚区的良好合作出发，共同谋划推动“大零号湾”市级科创示范区的建设与发展，对标国际最高标准、最好水平，以培育经济发展新动能为目标，以创新服务能力提升为着力点，以“智造+”与“+智造”双向融合为手段，拓展强化科技成果转化、科技企业孵化等功能，形成以科技创新为特色的闵行开发区转型升级新引擎，以此加快园区创新升级步伐，提升创新策源能力。力争再用5至10年时间，以二次创业的精神实现再造一个闵行开发区的目标，将闵行开发区进一步打造成为高端制造、智能制造名园，努力创造国家级开发区转型发展的新传奇。

(责编：龚莎、轩召强)
