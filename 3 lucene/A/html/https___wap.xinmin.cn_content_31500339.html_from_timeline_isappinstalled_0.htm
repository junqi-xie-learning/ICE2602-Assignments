他们在沙漠里种出一片绿洲 上海交大公益团科技治沙防沙

他们在沙漠里种出一片绿洲 上海交大公益团科技治沙防沙飞入寻常百姓家
来源：新民晚报   作者：易蓉 通讯员 皎轩   2019-03-12 13:39:26

来源／视觉中国（图文无关）

甘肃民勤，一个坐落于腾格里沙漠和巴丹吉林沙漠中间的县城。这里的沙漠以肉眼可见的速度吞噬土地，风沙县长达408km，整个县有91％的面积是沙漠。而上海交通大学绿格公益团的同学们要对抗的，就是这样庞大的沙漠。

“绿格”防风治沙公益团创立于2016年夏天，由交大环境科学与工程学院学生倡议发起，共累计350多人次，超过15个高校的同学参与了这项治沙行动。在过去的三年，公益团一直通过空闲时间，组织团队前往荒漠化严重地区进行长期、持续的防沙治沙工作。至今已经连续三年共计120余人前往甘肃武威组织开展防风治沙公益服务活动，制作麦草格子共计五万个，固定黄沙两千余亩，治沙效果已经初见成效。团队也获得上海市“知行杯”二等奖，上海市“创青春”公益组金奖等荣誉。

漠漠黄沙，是从未经历过的生活和世界

每次从上海出发，经历一天一夜的火车之途，抵达甘肃省武威市民勤县的一个小村庄里，团队的目标很简单，只为做好一件事情：制作麦草格子——麦草格子放置在沙漠的边缘，固定流沙， 防止沙漠的进一步扩张。

“看着漠漠黄沙，制作座座格子，才明白，原来我们能做的比日行10000步早起收集能量要多得多。”绿格防沙公益团创始人、上海交大环境科学与工程学院2015级本科生刘智卓说道。

绿格公益团在制作麦草格子并种植梭梭树的基础上，通过嫁接肉苁蓉获得高经济附加值的药材，也是对当地精准扶贫的一种尝试。目前梭梭苗和肉苁蓉的长势喜人。“在这趟西部实践中有着从未经历过的生活，从未见过的世界。有笼罩着天空的晚霞和星辰交替，有烈日炙烤的大地，还有星空下与大地同眠……”一位参加了绿格暑期防沙公益实践的同学这样说道。

创新进取，用初心践行公益事业

绿格公益团在社会实践的同时，也很关注自身公益行动带来的社会效应。除了防沙治沙公益服务，绿格公益团在开展治沙活动启动之前，还与另一个学生团队“松鼠抱抱”合作，结合“六五”世界环境日，开展废旧纸盒兑换草方格的活动，解决治沙资金问题，提升活动影响力。

在实践的过程中，他们也发现自身开展治沙行动的能力是有限的。因此，绿格公益团一直在思考，除了自身参与到防风固沙的公益实践之外，还能够为当地做出什么样的贡献和帮助。一方面，绿格公益团紧抓宣传工作，期待通过宣传的方式，能让更多人了解到中国西北部地区荒漠化现状，有更多人关心荒漠化问题。另一方面，绿格公益团通过自身行动，不仅仅为民勤地区打造一片新绿洲，更通过高校宣讲、考察调研、技术引进等方式，为当地社会带来更多的环境、经济、教育等方面的社会效益。

用热血青春实践公益，用脚步丈量祖国山河，用奋斗践行共同梦想，用一个不平凡的夏天，经历一场难以忘怀的公益实践。未来，他们继续在行动。

新民晚报记者 易蓉 通讯员 皎轩
