听！这5位赴京观礼的上海院士和名师，第一时间向全校师生发来“现场感”

教育在线 2019-10-01 13:35
来源：上观新闻 作者：徐瑞哲
每个队伍走过，大地震颤，威风扑面，气势恢弘，好一个强大自信的中国！我深刻感受到中华民族政治、经济、军事、科技的迅猛发展所带来的灿烂文明。

今天，此时此刻，天安门广场与上海校园拉得如此之近。解放日报·上观新闻记者了解到，包括两院院士、全国名师在内的上海交通大学5位老师应邀赴京观礼国庆大典，当庆祝中华人民共和国成立70周年大会、阅兵式和群众游行刚刚结束之时，亲临庆典现场的他们迫切地通过手机，将“现场感”带给在闵行菁菁堂主会场、徐汇分会场等一起观看典礼的全校师生。


作为2019年获评的中宣部、教育部“最美高校辅导员”，上海交大学生处副处长汪雨申表示，“我爱你，中国！这是我最想大声呼喊的一句话。”他说，在这秋高气爽的日子里，在这普天同庆的时候，有幸能够在天安门广场亲身经历这一伟大的历史时刻，要说心情，那真是无以言表的激动。“在这里，我能感受到祖国的脉搏，与祖国同频共振！祖国强大，我的责任！滴水可以汇成江河，每一个中国人都团结一心，在中国共产党的领导之下，阔步前行，为早日实现中华民族的伟大复兴，贡献自己的青春和力量！”汪雨申表示，作为一名高校的青年教师，一个思想政治教育工作者，将在今后的工作中破浪前行，接续奋斗！并努力为祖国培养一代又一代有能力、有担当、值得信任的可靠接班人！

上海交大物理与天文学院教授、中国科学院院士李家明，作为上海交大无党派人士代表，有幸在新中国成立70周年国庆盛典的历史时刻登上天安门观礼台，分享祖国母亲的激情与荣光，见证激动人心的一刻，心中涌动着无限荣耀与自豪。“看着眼前随风飘扬的五星红旗，看着我们的威武之师、文明之师、胜利之师气势磅礴地走过天安门广场，我身在其中、心在其中；看着祖国历经70年峥嵘岁月后的硕果累累，回想起一代又一代中华儿女浴血奋战、团结一心、开天辟地的勇气和决心，我热泪盈眶。”李家明表示，七十年，是一个传承，更是一个新的起点，祝福祖国母亲，期待未来能见证中华民族的统一，自己将一如既往与祖国同向同行！

作为上海交大Bio-X研究院院长、遗传生物学家，中国科学院院士、发展中国家世界科学院院士贺林表示，中华人民共和国成立70周年阅兵仪式是一场举国同庆的盛典，它是新中国魅力在中国共产党励精图治下的一次完美展示。自1949年以来，我国的综合国力逐渐稳步提高，各行各业百花齐放。“我长期从事生物遗传学研究，充分感受到了与党和国家共同成长的欢喜与压力。多年来，自己在党和国家提出的‘科教兴国，科技强国’发展战略指引下，始终奋斗在科研前线，揭示了第一例孟德尔常染色体显性遗传病A-1型短指（趾）症致病机制，发现了国际首例以中国人姓氏命名的‘贺-赵缺陷症’，直至如今依然忙碌于为人类健康谋福祉。”贺林说，正是党和国家作为坚实的后盾，自己与同仁们才能坚守在科研道路上，期待祖国的明天更加繁荣富强，我国的科研行业更加人才济济，硕果累累。

作为国家级教学名师、全国模范教师、全国先进工作者，上海交大制冷与低温工程研究所所长、教育部太阳能工程研究中心主任王如竹表示，新中国成立70周年之际，有幸亲临北京首都阅兵式现场，怀着无比崇敬和激动的心情观看了阅兵仪式：整齐的步伐、抖擞的精神，徒步方队15个、装备方队32个、空中梯队12个，总接受检阅近1.5万人，各型飞机160余架，装备580台套……“每个队伍走过，大地震颤，威风扑面，气势恢弘，好一个强大自信的中国！我深刻感受到中华民族政治、经济、军事、科技的迅猛发展所带来的灿烂文明。”王如竹表示，“建国创业”“改革开放”“伟大复兴”三个篇章的群众游行活动，体现了中国人民同心共筑中国梦，中华民族的伟大复兴已然到来，“我心潮澎湃，热血沸腾，一种强烈的爱国情和民族自豪感油然而生。”他告诉师生们，我们处在一个生机勃勃不断变革和创新的伟大时代，作为一名高校教授，我们担负着教书育人、科学研究、社会服务的几大重任，培养爱国奉献和勇于创新的国家栋梁之才是我们的基本任务。“百年大计，教育为本，我热爱我的教育事业！我热爱我的中国！我愿意为你的繁荣昌盛而奋斗终身！”

对于上海交大致远学院党总支书记、交大民族联常务副会长洪梅而言，今天是她第一次参加国庆观礼，内心的激动久久不能平复，“这是难得的荣耀，也将成为我一生中最美的印记。”作为全国少数民族参观团的上海代表，洪梅参加了上午在天安门广场举行的庆祝中华人民共和国成立70周年大会，聆听了习近平总书记的重要讲话，每一句都铿锵有力、催人奋进。随后举行的盛大的阅兵式和群众游行展示了全国各族人民的大团结，彰显了国家的整体实力。当看到阅兵式和彩车游行方阵从眼前而过，感受着祖国在国防、科技、教育、文化、工农业等多个领域取得的让世界刮目相看的伟大成就，见证着祖国在世界面前展示的中华文明和强大自信，内心感到无比喜悦、无比振奋、无比自豪。“中华民族一家亲，同心共筑中国梦。”她说，“国庆观礼带给我的不仅是心灵的震撼和感动，更给了我锐意前行的动力。我将立足岗位，守初心、担使命，为学校的发展、祖国的强盛，做一块坚强的基石！”

“十一”庆典结束后，师生们还围聚在一起不愿离开，热议习近平主席的讲话和这一盛世典礼，激动的心情久久难以平息……
