【附视频】和中国青年面对面，世界顶尖科学家说：搞科研，这三点最重要……

见识录 2018-11-01 06:00
来源：上观新闻 作者：解放视觉
2018年10月31日是“世界顶尖科学家论坛”最后一天，中外科学家与中国青年和青年科学家代表进行了一场面对面的交流……

在本篇报道视频的末尾，来自华师大二附中的樊悦阳同学告诉记者，和他交流的科学家告诉他：对于科学研究而言，“好奇心、倾尽全力和敢于挑战”是最重要的。

当日下午，在上海滴水湖畔举行的“世界顶尖科学家论坛”举行了中外科学家与青年科学家面对面的对话交流活动。

中外科学家与青年科学家进行对话，倡导开放包容的全球性科学创新网络，将带来更具可能性和值得信赖的未来。对全球青年科学家的帮助和支持，鼓励和支持更多年轻人加入到创新性教育和公众科学活动中，将推动人类科学事业发展，为科学创新发展增添年轻的血液，注入新的动力。
