诺贝尔化学奖得主夏普莱斯来沪演讲：《道德经》里藏着“点击化学”真谛

2019-4-22 17:18:42

来源:东方网 作者:傅文婧 选稿:潘丽娟

诺贝尔化学奖得主夏普莱斯教授

　　东方网记者傅文婧4月22日报道：“故有之以为利，无之以为用。”是老子《道德经》中的重要思想，也被不对称催化和点击化学领域的奠基人，诺贝尔化学奖得主卡尔・巴里・夏普莱斯（Karl・Barry・Sharpless）教授视为“点击化学”的哲学真谛。4月22日，夏普莱斯教授做客上海交通大学大师讲坛，作《SuFEx点击化学在生物方面的应用》报告，分享其近期在研究具有抗癌活性的小分子方面的工作。

　　夏普莱斯教授于1941年4月28日出生于美国费城，是不对称催化和点击化学领域的奠基人，诺贝尔化学奖、普里斯特利奖、沃尔夫奖、美国富兰克林奖章获得者，美国科学院和艺术及科学院双院士。夏普莱斯教授非常喜欢中国文化，他也十分欣赏老子《道德经》里的一句话，“故有之以为利，无之以为用”，认为这句话道出了“点击化学”中的哲学真谛：平时不会反应的碳-杂原子的结合，看起来似乎没什么用处，可当带上了特殊的官能团后，就会具有很高的选择性。比如夏普莱斯教授正在研究的第二代“点击化学”反应中，带上硫氟基团的小分子，可以精准识别出能与之结合的蛋白质，并形成相当稳固的共价键。

　　夏普莱斯不对称双羟基化反应与夏普莱斯环氧化反应，是现代有机合成中最重要的反应之一，他也因这一贡献获得2001年诺贝尔化学奖。点击化学是一种实用的哲学概念在合成化学上的映射，它的核心理念是――合成化学的灵魂在于其实用性，要以分子功能为导向。受到自然界少数合成砌块就可以构建复杂功能的启发，夏普莱斯教授自90年代中期开始积极探寻一种方法来发现新的反应性和化学连接，直到他的小组发现CuAAC反应（铜催化的叠氮化物- 炔烃环加成反应）之前，点击化学并没有获得很大的发展。点击化学对反应条件和反应对象有精准的选择性，这使得它具有在各交叉学科的广泛而高效的适用性，已被广泛应用于有机化学、聚合物材料合成、生物分子标记、抗体修饰、药物开发等一系列重要的研究和生产领域，拥有广阔的前景。

　　最近在点击化学的核心理念上又发展出新一代的点击化学――基于“硫氟交换（Sulfur(VI) Fluoride Exchange，SuFEx）”的高价硫氟类化学。夏普莱斯教授在报告会中介绍了他于2014年发现的这一点击化学反应――SuFEx反应，并重点介绍了近期使用这种新方法在发现具有抗癌活性的小分子方面的工作。

　　演讲现场，慕名前来的交大师生与夏普莱斯教授积极互动。上海交通大学化学化工学院博士生张嘉诚表示：“去年有幸参加滴水湖举办的世界顶尖科学家论坛，当时就得到了夏普莱斯教授的指导，我还清晰的记得，当我汇报了我在C-C键构建领域的研究后，教授评论说，‘大自然本身就是研究复杂C-C键的能手，我们应该学习生物体内合成大分子的方法研究C-杂原子键的构建’。教授的话给了我很大的启发，我们的研究本就是探索自然的奥秘，未来也应服务于人与自然的和谐共生”。
