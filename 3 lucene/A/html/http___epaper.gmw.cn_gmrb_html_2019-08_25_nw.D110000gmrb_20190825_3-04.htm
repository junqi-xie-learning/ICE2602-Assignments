    从君子文化中汲取爱国主义基因

    第五届君子文化论坛在上海交通大学举办

    作者：本报记者 刘江伟 曹继军 《光明日报》（ 2019年08月25日 04版）

        本报上海8月24日电（记者刘江伟、曹继军）由光明日报社和上海交通大学联合主办的第五届君子文化论坛24日在上海交通大学举办。为期两天的论坛将以“家国情怀和君子文化”为主题，围绕“社会主义建设与君子文化”“改革开放与君子文化时代内涵”“新时代的君子人格风范”等话题展开深入讨论。

        2014年6月13日，《光明日报》头版头条刊发了安徽省社科院研究员钱念孙的文章《君子文化与社会主义核心价值观》，在社会上引发广泛关注。以这篇文章为发端，全国不少地方成立了君子文化研究的专门机构，举行了以君子文化为主题的各类会议。今年恰逢新中国成立70周年，再次举办君子文化论坛，目的是大力弘扬爱国主义精神，推动君子文化的创造性转化和创新性发展，为实现中华民族伟大复兴的中国梦提供共同精神支柱和强大精神动力。

        光明日报社副总编辑李春林在论坛开幕式上表示，家国情怀是君子文化中最具代表性的人格标识，是中国传统文化最宝贵最活跃的精神资源，是每个中国人最真挚的情感归宿和最鲜明的精神底色。当前，全党正在深入开展“不忘初心，牢记使命”主题教育。新时代践行初心和使命，更需要重视君子文化，自觉扬君子之风、倡君子之道，把君子的言行标准融入实际行动，胸怀祖国、心系人民，努力在实现中华民族伟大复兴的征程中有更大的作为。

        上海交通大学党委副书记顾锋从上海交通大学的办学历史阐释了“家国情怀和君子文化”的主题。他指出，上海交大的人文教育旨在培养新时代的新君子。作为中国第一流的高等学府，学校始终致力于为国储才，为国家发展和社会建设培养具有爱国精神、科学精神与创新精神的“新君子”，而这就需要从优秀传统文化中充分汲取精神养分。

        清华大学历史系教授廖名春、方朝晖，北京大学哲学系教授郑开，中国人民大学国学院教授韩星，南京大学中国文化研究院副院长徐小跃，复旦大学上海儒学院理事长谢遐龄，山东大学儒学高等研究院副院长黄玉顺，复旦大学哲学学院教授杨泽波，安徽省君子文化研究会会长钱念孙，浙江大学君子文化中心主任何泽华，湖南省君子文化研究会会长李利君等百余名专家学者在论坛上交流。华东师范大学哲学系教授杨国荣受聘为上海交通大学中华君子文化研究中心主任，上海交通大学哲学系教授余治平为执行主任。

        论坛由上海交通大学中华君子文化研究中心承办，安徽省君子文化研究会、湖南省君子文化研究会、浙江大学君子文化研究中心、安徽省社科院君子文化研究中心、江苏省社会科学院君子文化研究中心协办。
