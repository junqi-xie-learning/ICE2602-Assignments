视频丨“战地”生日许愿：愿花开疫散，早日凯旋！

2020年02月20日21:00  来源：人民网-上海频道
 

人民网上海2月20日电 （龚莎）2月20日，是上海市第六人民医院第三批援鄂医疗队来到武汉的第2天，也是援鄂护士于树婷、朱江英的生日。 

往年的生日，她们都是跟亲朋好友一起庆祝。而今年的这一天尤为特殊，奋战在抗疫阻击战一线的她们，与队友们一起度过了一个难忘的“战地”生日。赋予这个生日特殊意义的，还有她俩特别的生日愿望：51个人能够平平安安回去和家人团聚。六院援鄂医疗队队长范小红亲手为两位过生日的队友下生日面，并送上了暖心生日祝福。上海市第六人民医院的职工们也纷纷通过电话、微信视频等方式向她送上生日祝福。

在防疫的最前线，队友和医院同事们送上的祝福无比温暖。对于树婷和朱江英而言，这个“战地”生日虽然简单，却深深地感受到来自家庭、队友、以及后方同事的深切情意，她们感到无比温暖，同时这份温暖也化作抗击疫情的信心和动力。手捧着蛋糕的她们哽咽道：“愿花开疫散，早日凯旋！”

(责编：龚莎、轩召强)
