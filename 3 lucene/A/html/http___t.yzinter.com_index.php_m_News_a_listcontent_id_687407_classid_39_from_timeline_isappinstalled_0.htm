原创|寿司之神的店是世界上最难预定座位的餐厅之一，为什么？

2019-03-25 20:45:14

扬子晚报网3月25日讯（记者 罗晓娜）3月25日，上海交大安泰经管学院举办深度思考论坛“日本企业的工匠精神及启示”。本科和硕士就读清华大学，在北海道大学取得博士学位，现为札幌大学综合研究所所长、经营与会计学系教授汪志平先生担任主讲人。

生产者将其情怀、态度、信念倾注于手中的产品

汪志平教授说：中国目前正处在令人兴奋的转型时刻，在探索高质量发展的过程中，日本的经验和做法值得关注和研究。从日常生活用品感受日本企业的工匠精神。智能马桶，日本用了20年时间，将智能马桶，从最初的小众产品变为全民性产品，1991年普及率14%，2012年普及到76%，现在已经96%了，基本上都是智能马桶。现在日本使用智能马桶已经成为一种共识了。是舒适、卫生、人性化的现代产品。电饭煲也是日本发明的，真空加压、保鲜。同样一个电饭煲，日本的工匠精神不断精益求精的基础上搞的花样很多，根据你的需求选择适合你的产品。

在企业经营领域，日本工匠精神享誉全球，日本长寿企业数量也是居世界之最。东京商工会议所的一项调查显示，截至2017年创业超过100年的日本企业家有33069家。仔细分析研究老企业的发展历史发现，他们对产品反复研究、精雕细琢、精益求精、持之以恒的一种理念。生产者将其情怀、态度、信念倾注于手中的产品，极度注重细节、不断追求完美和极致的信仰，表现对职业敬畏、对工作执着、对产品负责的精神。

寿司之神的店是世界上最难预定座位的餐厅之一

讲到工匠精神，日语的意思是自我深度探求技术，对于自己的作品拥有相当的自信，不因为金钱或时间的限制而妥协，而只要是答应接下的工作，即使最终导致利益上的损失，也会全力完成。把简单的事情研磨到极致，让普通的事物升华到艺术。工匠精神关于小野二郎的传说，他是寿司之神，他的寿司店在东京办公大楼的地下室，没有洗手间，十个座位，却连续八年被评为米其林三星餐厅，被认为是世界上最难预定座位的餐厅之一。需要提前数月预约才能有位置，但是价格不高。他是全世界最年长的米其林三星大厨，现在93岁还在做，已经做了70多年寿司。他的寿司和别人不一样，他的大米是指定供应商，米饭本身也经过了专业的蒸煮，保存的温度也是很讲究，各种海产品也是亲自鉴定新鲜程度和质量，自己觉得好的才会买，不好的不会买，客人去店里是不能点餐的，在市场上买了什么鱼就要吃什么，你要预定提前两个月，根本不知道吃什么东西。他要每天去市场挑选应季味道最好的食材，因此菜单变化不定，这种做法严格地保证了寿司的口感和品质。他自己在记者采访中说：我从来不曾厌恶这份工作，我爱我的工作，我一生投入其中，即使到我90多岁年纪，我仍然不认为自己已臻至完善，能用上等的金枪鱼捏寿司，我感到意气风发，我就是爱捏寿司。认定一个职业并且全身心投入，通过不断训练做到极致，沉浸其中享受不为人知的美妙，是这位寿司之神传达人生的态度。

200年以上的企业在日本占56%

日本七家企业是一千年以上的，全世界最老的企业都是在日本。200年以上的企业在日本占56%，德国占15%，荷兰4%，法国4%，其他国家21%。跟精益管理和工匠精神问题有关的是垃圾问题，现在很多城市很多新的关于垃圾分类的规定，希望要普及，但是估计会遇到很大的困难。日本在垃圾分类方面可以说是全世界做的最好、最精致的。垃圾焚烧场像城堡一样的设计。垃圾分类就要从小孩子开始抓起，进了学校就要说瓶盖怎么分，到了四年级亲自带孩子去垃圾处理厂进行实践学习。以后回到家垃圾分类按照规定分类，能用的尽量循环利用，减少被烧掉的垃圾的量。设计理念是“融技术、生态、艺术为一体”，不仅是孩子，普通的游客去参观也体验垃圾处理的过程，学会垃圾分类处理等环保知识，实现人与环境和谐相处。数据全部都公开。编辑：李燕
