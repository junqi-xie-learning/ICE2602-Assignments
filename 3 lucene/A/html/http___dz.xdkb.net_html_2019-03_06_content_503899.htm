“沿着周总理的足迹”全国微电影大赛颁奖
《爷爷的徽章》获得一等奖

  3月5日下午，“沿着周总理的足迹 不忘初心 砥砺前行”全国微电影大赛颁奖典礼在江苏省淮安市举行。《爷爷的徽章》《一个想踢足球的小女孩的故事》《沙果树》获得一等奖。《我的铁路我的路》《心履》《山路弯弯》等27部作品分获二等奖、提名奖和优秀作品奖。

  本次活动由中共淮安市委宣传部、中国电影评论学会微电影研究会、中共上海交通大学委员会宣传部主办，上海交通大学苏北研究院、周恩来纪念地管理局等联合承办。

  

  张海峰 钱之达

  用微电影缅怀周总理

  2018年，是敬爱的周总理诞辰120周年。习近平总书记在《纪念周恩来同志诞辰120周年座谈会上的讲话》中指出：“周恩来同志是近代以来中华民族的一颗璀璨巨星，是中国共产党人的一面不朽旗帜。周恩来同志的崇高精神、高尚品德、伟大风范，感召和哺育着一代又一代中国共产党人。周恩来同志身上展现出来的中国共产党人的崇高精神，是历史的，也是时代的，将激励我们在新时代坚持和发展中国特色社会主义征程上奋勇前进。”用微电影的形式，拍摄和记录周总理的事迹，以及全国各族人民沿着周总理的足迹，不忘初心、砥砺前行，努力建设社会主义物质文明和精神文明，是我们学习周总理、纪念周总理、推进中华民族伟大复兴的有效方式。

  参赛作品超过1250部

  大赛从2018年9月21日开启作品征集，南到贵州、广西、海南，北至新疆、甘肃、吉林，全国数百所高校、百余家影视文化工作室，以及数十家企事业单位积极参与，累计收集作品1250余部，最终评选出一等奖3部，二等奖7部，提名奖10部，优秀作品奖10部。北京参赛作品《爷爷的徽章》、同济大学《一个想踢足球的小女孩的故事》、浙江第四监狱《沙果树》获得一等奖。南京中瀚影视文化传播公司《我的铁路我的路》、上海戏剧学院《心履》、云南彝良县政协《山路弯弯》等作品分获二等奖、提名奖和优秀作品奖。

  淮安市委常委、秘书长李森，中国科学院院士、上海交通大学党委常委、副校长毛军发，上海交通大学党委常委、党委宣传部部长胡昊，中国电影评论学会会长、中国文联电影艺术中心主任饶曙光，主办、承办、协办等相关单位领导、专家以及来自全国各地的高校师生、文化工作者、获奖团队代表300余人参加颁奖典礼。

  淮安市委宣传部副部长、精神文明办主任杨恒忠表示，淮安是周总理的故乡，本次赛事由淮安市牵头主办，是落实习近平总书记“把周总理的家乡建设好”重要指示，塑造淮安精神，扩大淮安在全国文化影响力的一个重要抓手。许多获奖者纷纷表示，通过这次主题大赛进一步感受、领悟了周总理等老一辈无产阶级革命家的革命精神和崇高品质，思想上受到深刻洗礼，精神上得到极大鼓舞，更加坚定了不忘初心、奋勇前行的信念。
