这549校刚上了“中国最好大学榜”，13家非双一流高校跻身百强，医科大学排名首发

教育在线 2019-01-23 07:00
来源：上观新闻 作者：徐瑞哲
今年共有22所上海高校上榜中国最好大学排名，前50名中有7所高校，数量在各省市中位于第二，仅次于北京。

北京时间23日今晨，作为全球四大高校排名系统之一，拥有16年历史的“世界大学学术排名（ARWU）”，最新发布2019“中国最好大学排名”，除我国台港澳地区之外综合实力最强的549所大学上榜，“清、北、浙”三甲位次没有变化，十强之中唯有北京和上海各有2所高校在列。【上海市高校名单附后】

 

 

值得注意的是，基于更为细分的学科数据，这一排行系统还首次披露了中国最好医科大学排名，共有71所医科大学榜上有名，而综合性大学的医学院不在其中参与排名。此次，北京协和医学院高居榜首。【后附全榜单】

 

 

13所非“双一流”高校跻身百强

 

 

 

从中国最好大学排名评价体系看，共设人才培养、科学研究、服务社会和国际化四个维度，采用最体现大学办学水平的关键指标，例如新生高考成绩、毕业生就业率、论文数量与质量、成果转化收入等，以便排名结果客观反映大学的竞争性地位。

 

 

在清华大学、北京大学、浙江大学之后，上海交通大学、复旦大学、中国科学技术大学、华中科技大学、南京大学、中山大学、哈尔滨工业大学依次排在4到10名。其中，华中科技大学比去年同期上升1名，与南京大学并列第七。

 

 

在各种标准的考量下，名校发挥基本稳定。“双一流”建设高校在2019中国最好大学排名中保持绝对优势：排名前50位的高校中，有49所是“双一流”高校；同样，百强大学中，“双一流”高校有87所。与此同时，也有13所非“双一流”建设高校凭借过硬的素质表现跻身百强。

 

 

究其特点，这13所学校大多数来自沿海发达地区，包括广东（南方科技大学、深圳大学、汕头大学）、福建（华侨大学）、江苏（江苏大学、南京工业大学、扬州大学）、浙江（宁波诺丁汉大学、浙江工业大学、浙江师范大学、杭州电子科技大学）等。

 

 

解放日报·上观新闻记者了解到，今年共有22所上海高校上榜中国最好大学排名，前50名中有7所高校，数量在各省市中位于第二，仅次于北京。百强之中，沪上九校跻身，除复旦、交大外，依次还有同济大学、华东师范大学、华东理工大学、上海大学、上海财经大学、东华大学、上海外国语大学。其中，排在第41名的上海大学是地方高校中表现最好的三所大学之一，另两所分别是苏州大学（29名）、南方科技大学（35名）。

 

 

 

医科大学启动单科性大学独立排名

 

 

“大学排名的挑战之一是不同类型学校之间的可比性问题，分类排名是提高大学排名参考价值的重要方法。”中国最好大学排名创始人程莹博士向解放日报·上观新闻记者表示，比如医科大学是比较独特的单科性大学，医科大学和综合性大学界限分明，几乎没有医科大学独立发展成为综合性大学的先例。因此，今年选择从中国医科大学开始，启动单科性大学的独立排名。

 

作为大学分类排名，2019年入选中国最好医科大学综合排名的学校数量共71所。除北京协和医学院领跑之外，首都医科大学排名第二，南京医科大学、南方医科大学和天津医科大学位列前五。之后，依次为：中国医科大学、哈尔滨医科大学、上海中医药大学、重庆医科大学、大连医科大学。

 

 

医科大学也是人才培养目标和学生就业去向最清晰的单科性大学。据了解，这一医科大学排名在中国最好大学排名指标的基础上，又引入了“造就医学人才”指标。程莹认为，在医科大学的毕业生中成为医疗卫生行业的高水平专家和杰出管理者的数量，是检验医科大学人才培养质量的重要体现。

 

 

从上海医科高校看，作为“双一流”学科建设高校的上海中医药大学在排名中位列第8，是十强之中唯一中医药院校，也是沪上唯一一座上榜高校。需要说明的是，沪上海军军医大学（第二军医大学）作为军事院校，不在此次医科大学排名范围之内。

 

 

上海交大高等教育研究院院长刘念才教授认为，与大多数排名相比，这一排名系统的显著优势不仅详细说明各项指标的统计方法和数据来源，还直接公布各项测量的原始数据，使得排名完全透明，对排名结果负责，主动接受外界检验，方便他人核实验证和个性化分析，提高了排名结果的参考价值。
