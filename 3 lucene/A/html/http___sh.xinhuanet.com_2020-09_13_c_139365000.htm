上海交通大学举行2020级本科生开学典礼现场。新华网 发

        13日，上海交通大学在闵行校区举行2020级本科生开学典礼。4000余名本科新生通过线上线下结合的方式，共同参与了本次开学典礼。

        上海交通大学校长林忠钦在开学典礼上表示，在今年“加长版”的高三备考阶段，大家克服困难分秒必争，最终以优异的成绩如愿加入上海交大这所百年名校。“你们的到来，对学校而言，既是荣幸，也意味着责任。大家把人生的梦想托付给交大，学校的老师们一定会用心用情培养好、呵护好每一位同学，让大家在充满创新活力与人文关怀的校园里成长成才。”

        大学是个人成长中的黄金时期，迈入大学，意味着将有更多机会去发掘自身的潜力和探索世界的奥妙。面对美好而又未知的大学生活，新生们该如何更好应对？

        为此，林忠钦给新生们提了三点建议：树立远大理、胸怀报国之志；精勤求学、筑牢基础；坚韧进取、乐观豁达。

        “本科四年是打牢知识基础的重要阶段，大家一定要在公共基础课和专业基础课上下足功夫，同时还要以开放进取的姿态，努力培育创新能力。唯有立报国之志，才能具备大气恢弘的胸襟；保持乐观豁达的人生态度和坚韧进取的精神，才能在中国的强国之路上留下属于你们的独特印记。”林忠钦说。
