婉拒名校只为救死扶伤之志，医学院迎来“00后”：领到人生第一件白大褂，希望成为被需要的人
2018-09-08 16:39:27 作者：唐闻佳 杨静

领到人生第一件白大褂，让他们兴奋不已。今天是上海交通大学医学院的新生报到日，600多名新生的到来也意味着医学院首次与“00后”集中相遇了。他们中不少人婉拒了许多名校，只为一个救死扶伤的理想。

以613分高分（上海卷满分660）考入交大医学院临床医学八年制的茅天誉说，最初是因为对人体奥秘感兴趣而想读医，看了一系列医疗纪录片后，看到医生们哪怕只有一丝希望都会拼尽全力去救治，被感动之余，更坚定了学医的想法，也希望成为那个被需要的人，可以拯救无数生命。

“起初父母朋友不是很理解我的选择，告诉我这个职业很辛苦，压力很大，但看到我的坚持，他们也表示了尊重。高考出成绩后，我放弃清华、北大的邀请，父母很支持我坚持自己的理想。”茅天誉说。

与茅天誉一样来自复旦附中的王佳琦，也是高考600分以上的“学霸”，他也选择了临床医学八年制。这个“00后”把中国工程院院士、上海交大医学院附属瑞金医院血液学家王振义视作偶像。

▲王佳琦和振义路路牌合影

“没有像样的实验室，没有任何精密的仪器，但王院士以其执着与坚守攻克了急性早幼粒细胞白血病，治愈了无数患者，并在很大程度上推进了血液病学的发展。更令我心生敬仰的是，王院士没有为此申请专利，使治疗该种白血病的费用能被大部分患者的家庭承受。他的医者仁心使我敬佩！”报到当天，王佳琦在交大校区看到以王振义院士命名的“振义路”路牌，立刻开心地与之合影。得知随后的新生入学教育还有机会聆听王老的教诲，他更是大呼期待。

婉拒了香港大学生物医学工程专业，坚定选择上海交大医学院临床医学八年制法文班的何咏元也有自己的医学偶像，就是被称作“斯坦福天才医生”的保罗·卡拉尼什。卡拉尼什的遗作《当呼吸化为空气》一度成为全球畅销书，感动了很多人。此书包含了他对与生死命题的思考、对生命终结的感悟。“医生不仅需要医治好疾病，更重要的是能帮助病人寻找到生命的价值和意义，”何咏元受此书启发，也希望自己能在今后的学习中得到更多成长。

来自新疆的新生迪娜是“医二代”，她的父母同为医生，她从小在医院的家属楼里长大。她说，自己的医学偶像就在身边，这些长辈里有的年纪古稀却依旧认真钻研医术；有的奔赴各地，向新疆偏远地区带去医疗支援；有的精通各民族语言，为广大患者热忱服务；有的飞行千里，学习更先进的医学前沿知识；有的为改进医疗设备及器械，四处奔波求索。

“小时候，只能看到长辈们匆忙的背影，成年后，我很想参与其中，一同担当起减轻他人痛苦的责任。”迪娜充满期待地说，八年后，希望自己成为值得被相托性命的温暖医生。

近年来，上海交大医学院录取分数线呈持续攀升态势，越来越多优秀的高中生热衷投身学医队伍。这批抱持“被需要”初心的年轻人的到来，也让医学院倍感振奋。据了解，9月10日新生入学教育当天，交大医学院还将举行白袍宣誓仪式、入学第一课和舞台思政课《清贫的牡丹》公演等，学校希望用更有温度的医学教育，为这批医学“新鲜血液”灌注灵魂，助力成才。
