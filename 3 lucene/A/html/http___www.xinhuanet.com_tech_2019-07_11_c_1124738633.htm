上海交大水热资源化技术处置湿垃圾工业化试验成功
2019-07-11 10:05:57 来源： 新华网

　　新华社上海7月11日电（记者仇逸）在没有臭味的情况下，1小时内就能将湿垃圾处理完毕，余下的“废水”和“废渣”可用于农业肥料……上海交大环境科学与工程学院金放鸣教授领衔团队经过多年技术研发，“湿垃圾水热氧化技术”项目迈出重要步伐，第一台连续式水热资源化技术处置湿垃圾装置投料运行测试成功，设计日处理量为100吨。

　　湿垃圾水热资源化技术又称水热氧化技术，是指通过模拟地球碳循环过程，在特定的高温高压环境中，可以加速湿垃圾中的有机物分解，把它们转化为腐殖酸、甲酸、乙酸、乳酸等高附加值产品。

　　湿垃圾中的有机物经过水热氧化技术转化后，产生的“废水”含有大量有机质和其他营养物质，是高品质的液肥，能够直接用于农作物施肥或水产养殖；固体“废渣”是腐殖酸原粉，经过加工后可以用于土壤污染修复、沙化地修复、重金属污染土壤修复、大气污染修复、水体污染修复等，也可以直接用于农业施肥。腐殖酸类物质是一种净化生物圈、改善生态环境的宝贵资源。

　　据介绍，湿垃圾水热氧化技术处理速度快，无毒无害无三废，相比于目前应用广泛的湿垃圾厌氧发酵技术，水热氧化可以做到高度集成化，厂房占地面积大大缩小，日处理100吨湿垃圾处理能力的核心部件只需占地60平方米左右；技术规模化程度高，可以根据需要建造大、中、小型装置，应用于城镇、商场、学校等不同场所。
