高校开学季：所谓“封闭”能否正常出入校园，新生家长可否进校？上海讲清楚了

教育在线 2020-09-02 12:17
来源：上观新闻 作者：徐瑞哲
师生员工凭有效身份证明、接受体温测量后可正常出入校门。原则上建议家长不进校，但不搞一刀切。

上海全城高校在读学生规模约70万人。眼下迎来开学季，针对所谓“封闭式”一说，上海市教委2日通报称，目前没有封闭校园，而是重在“加强管理”。

解放日报·上观新闻记者了解到，当前各校正把好校门关，严格管理入校人员。其中，师生员工凭有效身份证明、接受体温测量后可正常出入校门，鼓励学校通过楼宇门禁、校园出入卡口系统等信息化手段溯源轨迹。

值得注意的是，如果特殊情况不得不离沪，则必须报备，并按照开学返校口径做好相应卫生防护和健康管理措施。

至于校外人士，在疫情彻底结束之前，严控校外无关人员进校。校外人员因工作需要进校，应提前报备并接受身份查验、体温测量和出示健康码等。同时，校内服务设施原则上暂停对校外人员开放。

此外，对快递、食堂主副食品配送、校内设施设备维保等涉及师生基本民生和学校日常运行的服务人员，通过企业和人员信息提前报备、指定校门出入口、人员身份核查、登记和管理、规定进校时间和路线等措施，在满足师生和学校需求同时，切实维护好校园管理秩序。

除了“老生”返校，还有“新生”报到。对于新生报到家长是否进校，上海教育部门表示，应因地制宜、因校制宜，原则上建议家长不进校，但不搞一刀切。总体上，高校充分考虑有可能造成校门口人员高度密集和车辆阻塞，严重影响社会道路通行和管理秩序的情况，充分考虑在高温、大风、暴雨等恶劣条件下，老生返校和新生报到的人性化、科学性处理措施。

值得一提的是，结合劳动教育做好大学新生独立迈好大学校园第一步的宣传教育工作，并安排志愿者协助新生办理报到手续。学生公寓或生活区实施严格管理。

上海市教委后保处处长张旭告诉解放日报·上观新闻记者，关于开学后校园管理和后勤保障有关工作的基本原则，可以用一个成语概括：张弛有道。

他认为，“张”就是紧绷疫情防控红线不放松，坚持底线思维，坚持必要的、有效的、科学的防控措施和手段。“驰”就是依托全国、全市疫情防控形势总体向好的大环境，相关管理和服务措施充分考虑实际情况，以合理需求为导向，科学防控的同时体现人性化管理。

还有“有道”，即立足“可溯可防可控”的应急处置机制——包括“可溯”：就是掌握和可以快速复盘所有人员信息和活动轨迹；“可防”：提高全体师生员工防护知识和防护能力，各种技防、人防和物资到岗、到位，把各类疫情风险降低到最小；“可控”：万一出现不确定状况或突发病例，最快速度切断外部输入渠道或者把影响面控制到最小范围，可控还包括对各类不实信息和负面舆情的及时引导和消除。
