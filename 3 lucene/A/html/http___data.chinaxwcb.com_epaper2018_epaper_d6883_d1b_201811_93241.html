《平易近人：习近平的语言力量（外交卷）》出版座谈会在沪召开

作者：金鑫        来源：中国新闻出版广电报        时间：2018/11/28 9:25:15
分享到：

    本报讯 （记者金鑫）《平易近人：习近平的语言力量（外交卷）》今年10月正式出版，图书首印10万册，当月便重印3次。11月27日，上海市委宣传部、上海交通大学共同在沪主办《平易近人：习近平的语言力量（外交卷）》出版座谈会。上海市委常委、宣传部长周慧琳，上海交通大学党委书记姜斯宪出席会议并讲话。

    《平易近人：习近平的语言力量（外交卷）》由外交部中国国际问题研究院编撰、上海交通大学出版社出版发行。该书以语言力量为出发点，选取习近平总书记在外交活动中使用的特色语言，分开篇、形象比喻篇、俗文俚语篇、诗文引用篇4个部分，对党的十八大以来中国外交的总体布局、理论创新、实践成就等进行解读，有助于读者更好地理解蕴含其中的思维方法和外交艺术。

    会上，周慧琳要求上海出版界认真总结《平易近人――习近平的语言力量》系列图书的策划、编辑、出版、发行经验，继续按照习近平总书记致《大辞海》出版暨《辞海》第一版面世80周年贺信的要求，坚定文化自信，坚持改革创新，打造传世精品，紧紧围绕党和国家大局，高质量实施重大主题出版工程，持续推进思想精深、艺术精湛、制作精良的精品力作，推动上海主题出版朝着精品化、专业化、精良化方向迈进。同时，在构建中国话语体系中体现上海作为，要深入学习总书记的语言艺术，全面加强国际传播能力建设，为讲好中国故事、传播好上海精彩作出新的更大贡献。

    该书主编、中国太平洋经济合作全国委员会会长苏格介绍了编撰过程及图书特点，上海交通大学出版社社长李芳介绍了选题策划及出版过程。

    据了解，上海交通大学出版社近年来在主题出版、学术出版方面打造精品。该社策划出版的《平易近人――习近平的语言力量》和《平易近人――习近平的语言力量（军事卷）》已重印20余次，销量超过90万册，并输出日语、英语等5个语种。
