“祝你成为富婆、百万富翁、总裁！”于漪发现的这些小学毕业赠言，也惊到了教育专家们

教育在线 2019-01-04 06:00
来源：上观新闻 作者：徐瑞哲
一些地方一些学校把学生当作手段，初三学生做功课做到凌晨1点，“做眼保健操都会睡着。”

“祝你成为富婆！”“祝你变成百万富翁！”“祝你当上总裁！”这些毕业赠言竟都出自小学生之手。

 

于漪作为基础教育界唯一代表，刚刚成为“改革先锋”称号百名获得者之一，而这位九十高龄的老教师今天（3日）与教育专家们交流了一种现象：她在某校五年级学生的毕业纪念册上发现了这些“祝福”。“全班几乎找不到‘祝你成为科学家’的留言，”于漪遗憾表示，班上只有个别祝愿某同学将来成为医生，因为孩子妈妈是医务工作者。

 

 

十余年寒窗苦读，可到底“为谁辛苦为谁甜”？于漪当天的发问，在引发第十三届上海民进教育论坛与会师者学者的惊诧之余，也带出一连串反思。

 

教育部高中课程改革委员会专家委员、北京十一中学校长李希贵

 

越跑越快、越跑越累，可别忘了从哪里出发

 

今年于漪从教67年，自视为“草根教师”，笑称“顶大的长处就是年纪大。”这位上海“教育事业杰出贡献奖”得主说，之所以作为代表获得“改革先锋”称号，其实是国家和百姓对1700万基础教育教师的肯定和赞扬。

 

“当老师的这最近40年，常有恍如隔世之感。”于漪想起，曾经上海学校一节课只能发给老师两支粉笔，而上个月她去一所小学所见的教学设备之先进，“与从前比简直是天堂，我是连做梦也想不到的。”

 

 

今非昔比，从都市到山村，中国义务教育巩固率达到93.8%，如此大的受教育人口为世上所无，只有中国办成了办不成的大事。“教育是国家的教育、民族的教育，要坚持什么、又要改什么，应当一清二楚。”于漪坚定强调。

 

“教育失败，是一种根本性的失败。”民进上海市委主委、上海交大副校长黄震援引了总书记振聋发聩的论断。专家学者们都关注到，日前教育部、国家发展改革委、公安部、民政部等九部门联合印发《中小学生减负措施》，共出台“减负三十条”。黄震教授也发现一些地方一些学校把学生当作手段，初三学生做功课做到凌晨1点，“做眼保健操都会睡着。”他直指，绑上疯狂战车，结果越跑越快、越跑越累，但别忘了从哪里出发，必须扭转教育功利化倾向。

 

民进中央委员、湖南卫视主持人汪涵

 

少人追问学生为谁而学，学习动机和动力何在

 

学业压力之下，“学生为谁而学”的问题恰恰少人追问。于漪表示，教育工作者在学生学习动机和动力方面要多下功夫。她认为，动机和动力一方面是来自兴趣，另一方面是来自志向，而且两方面可以结合在一起。但当前，不少学生在18岁成人之前胸无大志，更别说志存高远。

 

于漪之前赴京参会，接受表彰，碰到陈景润夫人格外激动。因为她当年在读到徐迟报告文学《哥德巴赫猜想》时，作为语文老师与数学老师搭档开课，分别从语文和数学角度给学生讲授陈景润的“猜想”。她表示，陈景润在锅炉旁6平方米之内，验证哥德巴赫猜想，仅仅“从1到2”，却至今意义非凡。与现今的计算手段相比，那时全靠笔算，稿纸一麻袋一麻袋地装，“这是什么动机和动力在起作用？”

 

 

如今，这种“动能”何在，是考试吗？上海市教委副主任兼市红十字会副会长倪闽景坦言，他听到过不少老师面对学生问题，都冒出一句口头禅般的回应：“不考，不要问。”而他知道，当年复旦大学钱文忠恰恰因为高二时历史老师多说的一句“废话”，走上学术研究之路。

 

钱文忠高中成绩平平，历史课上听班上郝老师讲课，说“全中国只有一个人懂梵文，他就是季羡林“。课后，钱文忠向老师求证，对学习梵文大发兴趣，下决心求学季羡林门下，并写信给季先生。结果次年，他以上海文科第二名的高考成绩成功考入北大，如愿以偿成为梵文传人。

 

香港大学原副校长、教育政策研究中心主任程介明

 

“智”字以外，从“头”到“尾”都是素质学习

 

与学习动机和动力相关的，是学什么？跳出头脑的第一印象往往是智育，仿佛“德智体美劳”全面发展的5个字中，“智”字最大。但与会教育专家对此都不认同，其实从“德”开始，从“头”到“尾”都是素质学习。

 

 

这第一个字“德”，在倪闽景看来其结构可以拆分：双人旁代表“行动”，右侧的上、中、下部分，分别是：“十”代表方向，“目”代表眼光，“心”代表思想。按古代字体解释，“德”表示“目正、行正、心正”；用现代的话说，其实正是“有理想、有本领、有担当”。

 

作为中国教育学会第八届理事会学术委员会顾问，朱永新教授表示，德育是共通语言，日本文部省规定“学校的道德教育，以通过学校整个教育活动进行为准则”，并设立了以德育为中心的“综合课”；美国的社会课、公民课等作为非常重要的基础课程，其实也是德育课。

 

 

“从《共产党宣言》到《道德经》，从《量子史话》到《几何原本》……”殊不知，这4本书，正是华东师范大学4门经典通识课的2.0版课程。华东师大副校长兼基础教育集团主任戴立益表示，他本人除了分管教学，还分管后勤，在自己眼中，“后勤”是在后台的“勤俭”“勤快”“勤恳”“勤勉”，也是服务育人。比如，华东师大著名创新菜“玉米炒葡萄”的发明人，就是一位获得“上海市五一劳动奖章”的食堂大厨师。因此，戴立益倡导大学生与后勤工作者一起劳动，从植树造林到厨艺展示，把“劳”也纳入育人范畴。
