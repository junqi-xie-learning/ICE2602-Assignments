C919大飞机隔热材料竟成了保暖服，本埠外埠高校上交会首展“高科民用”衣食住行

教育在线 2019-04-18 16:41
来源：上观新闻 作者：徐瑞哲
经酵母菌种配套研发，果酒酿造器攻克难关，苹果酒、梨子酒、蓝莓酒、菠萝酒等都能酿造。更神奇的是，酿造工艺周期创纪录地从全程7天缩短到3天。

看似一卷卷大地色系的毯子，却是C919大飞机正在使用的隔音隔热玻璃纤维棉毡，而这样又轻又强的材料正在转化为消费级产品，做出了绝不怕冷的防寒保暖服。

 

 

18日，解放日报·上观新闻记者走进刚刚开幕的第七届中国（上海）国际技术进出口交易会，在世博展览馆西区的高校展区内，外埠名校首次参加上交会，与本埠高校同台展示“高科民用”的衣食住行。

 

 

【衣】飞行外衣变保温外套

 

 

这种航空级隔音隔热用的玻璃纤维棉，就是南京航空航天大学带来的“名特优”项目。来自南航技术转移中心的科技成果推广部部长刘华翔告诉记者，这种棉毡材料的基本原料是直径不超过2微米的超细玻璃纤维。可以这么说，废旧玻璃同样能被循环利用，加工成为超细玻璃棉。然而，目前世界上仅有美国一家企业有能力提供类似产品，不过价格是自主研发的国产产品两倍左右。

 

于是，这种“棉”首先被应用于C919大飞机机体，同时在高铁保温降噪领域也具有很强优势，目前已为中车长客提供部分样品进行装机试验，还成为冰箱、冷藏车等现代冷链物流的保温节能关键材料。更值得终端消费者期待的是，通过长三角校地合作，学校已在企业生产线上完成中试，可以将其制成特别能保暖的外套，以及可以防火的冲锋衣等。

 

 

【食】三日家酿各种果酒

 

 

在家自己用机器做豆浆、做酸奶、做面包，可能都不算稀奇了。这回，最新鲜的酿酒机来了，一次最多可酿成4瓶标准葡萄酒。

 

此前中国上海工博会上，上海应用技术大学香料学院冯涛教授拿出自己主导研发的果酒酿造器，当时展出的还只是项目雏形，却被相中成了上海应技大技术转移有限公司自己孵化的第一个项目。副总经理张钰说，研究食品发酵的冯涛教授团队和研究机械自动化控制的张慧敏教授团队，两个跨学科团队走到一起，从食品发酵菌种、水果发酵工艺、自动化设备、智能控制等方面一步步完善项目。经过一年多改进，这台柜式机酿造出的果酒通过检测，各方面指标均达国家标准，并完成3C认证。

 

作为酵母，红酒发酵的菌种市面上很多，但其他水果的发酵菌种就是空白。经配套研发，果酒酿造器攻克难关，苹果酒、梨子酒、蓝莓酒、菠萝酒等都能酿造。更神奇的是，酿造工艺周期创纪录地从全程7天缩短到3天。

 

 

【住】打开家门沐浴芬芳

 

 

作为上海应用技术大学老校长，市教委主任陆靖也到应技大展台追踪成果转化有无“下文”。上交会就是合作会，他特别提示应技大的特色学科香料专业，可到隔壁的兄弟院校展台，一起寻找协同创新机会。原来，上海中医药大学研发的“精油MASTER大师”刚获专利，成为参展专业观众的热门体验处，机器后台的“材料库”正是出自各种天然草本的香料。

 

“想不想刚打开家门，就能在房间里沐浴阵阵花草芬芳？”中医大研发人员王健英向记者模拟了这台“精油大师”的应用场景。这台小冰柜大小的智能机，可以选装各种成分明确、符合国标的植物精油，通过物联网设计，接入手机终端，就能远程操作和监控精油制备，做好喷雾吹肤准备。

 

譬如，中医有春、夏、秋、冬加之长夏的“五季”之说，每一季适合不同精油养护。在下班途中，挤在地铁的主人，就可结合天候变化，根据个人体质、劳累程度、情绪状况等，进行复方精油的自动化配液，定时在家中实现香薰、加湿、净化等功能。经年累月，其内置的自学习系统，还能更精准地提供个性配置方案与芳香保健原则。

 

 

【行】二氧化碳空调上高铁

 

 

从大型客机到高速铁路，上面的高新装备都具有民用前景。就在高校展区入口处，上海交通大学研发的“轨道交通二氧化碳列车空调”，直接将高铁车厢顶部的一部分中央空调搬上展台。按规划，中国将从2024年冻结使用HFCs等含氟制冷剂。因此，这种高铁空调系统就采用纯天然的工质——二氧化碳，与传统使用“氟利昂”等作为制冷剂的空调本质不同，对脆弱的臭氧层毫无伤害。

 

上海交大科研院科技合作办副主任华黎明告诉记者，学校制冷与低温工程研究所陈江平教授团队，经与高铁设备主力供应商合作，联合研制出了我国首台时速400km及以上高铁用的二氧化碳空调机组，解决了二氧化碳制冷剂在空调系统中运行压力高、排气温度高等技术难题，提高了制冷系统全工况的稳定性和运行效率。

 

“这项成果还将辐射应用到德国联邦铁路双层客车改造项目上。”预计今年针对性完成空调设计及性能调试、明年在德国铁路客车上装车运营试验，使环保型二氧化碳列车空调系统在产业化道路上更进一步。
