明年消除疟疾，2030消除血吸虫病，全球健康学院从上海铺开这条“健康丝绸之路”

教育在线 2019-06-16 12:43
来源：上观新闻 作者：徐瑞哲
作为上海交通大学医学院-国家热带病研究中心全球健康学院学术委员会主任，陈竺院士今天发来长段视频……

以各种寄生虫病为主的热带病曾造成我国数千万人到上亿人感染，现今这样的疫情仍在一些国家和地区持续肆虐；而全球人口大流动带来的输入性热带病越来越多，近年来西非埃博拉等致命病毒也不时威胁包括亚洲在内的各个大洲。

中国经验、中国人才如何加入卫生健康领域全球治理？16日，上海交通大学医学院-国家热带病研究中心全球健康学院在沪成立，成为首家以学院命名的全球健康机构。中国科学院院士陈竺担任全球健康学院学术委员会主任，中国科学院院士、中国疾病预防控制中心主任高福担任院长。

全球化的快速发展，国家与国家之间界限的模糊，共同催生了全球健康 （global health）的概念，全球健康关注伴随全球化而出现的全球性问题。自19世纪热带医学学科开创至今，热带医学一直与国际健康（后称：全球健康）息息相关，是其最为重要的构成部分。作为新兴国家和发展中国家的代表，中国已经成为世界卫生领域的重要行为体。新的上海交通大学医学院-国家热带病研究中心全球健康学院，正是定位于推动我国全球健康学科发展、加快全球健康人才培养。

根据此前国家卫生部和上海市“部市合作”协议，双方共同筹建“国家热带病研究中心”，即以中国疾病预防控制中心寄生虫病预防控制所为核心单位，联合在沪高校及相关单位资源，形成集科学研究、教育培训、国际交流、疾病临床治疗与防控于一体的国家级热带病研究中心。

今年，中国CDC寄生虫病预防控制所加挂“国家热带病研究中心”，并上海交通大学医学院合作共建，定标于2025年、2035年分别成为亚洲领先、世界一流的热带病研究机构，最终可与英国伦敦热带医学与卫生学院、瑞士热带病与公共卫生研究所等机构相提并论。目前，这一全国寄生虫病预防控制业务指导中心正继续推动血吸虫病、疟疾疫情降至历史最低水平，促进2020年消除疟疾，2030年消除血吸虫病进程，并开展包虫病流行病学调查与试点等，推动2030年基本控制包虫病危害。

解放日报·上观新闻记者了解到，医学名校与国家中心强强联合而成的全球健康学院，将重点聚焦热带病与突发传染病、全球卫生政策与标准、全球妇幼卫生、全球营养与食品安全、全球精神卫生这5个发展方向。运用全球分布的国际合作网络，促进中国方案、中国产品更好为全球人类健康服务，也为我国参与全球健康治理提供有力支撑。上海交大医学院院长陈国强院士透露，全球健康学院除拥有医学学科优势外，也将利用上海交大多学科综合优势，促进经济学、政治学、人类学、社会学、法学等学科交叉复合、共同建设，还将从国际化（international）迈向更大范围、更高水平、更深层次的全球化（global）。

“全球健康不仅是地域的概念，因为疾病包括传染病和慢性非传染性疾病从来不遵守国界，它还是一个生物圈的概念。”作为全球健康学院学术委员会主任，陈竺院士今天发来长段视频，他表示：近些年来，在世界卫生组织推动下，在联合国千年发展目标和联合国2030可持续发展议程的推动之下，全球卫生合作风起云涌、势不可挡。他期望上海交通大学医学院-国家热带病研究中心全球健康学院取得成功，为“健康中国”建设，为“健康丝绸之路”和构建人类命运共同体做出不可替代的贡献，让中国模式、中国智慧在全球化进程中更好地给全人类带来福祉。之前，国家卫健委副主任曾益新院士等各方代表为全球健康学院揭牌。
