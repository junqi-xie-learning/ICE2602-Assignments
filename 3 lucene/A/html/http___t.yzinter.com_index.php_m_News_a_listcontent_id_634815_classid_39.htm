原创｜数百名专家学者为知识产权“把脉”

2018-11-01 11:12:08

 

扬子晚报网11月1日讯（记者 罗晓娜 文/摄）大数据、区块链、云计算、人工智能等高科技的迅猛发展，一方面为公共行政和市民生活带来的巨大便利，但同时为政策法律的制定、监管和司法适用提出了大量的挑战，高科技产业的进一步发展亟需明确相关的政策法律规定。11月1日，“知识产权与竞争法交大论坛”暨“数字治理的法律与政策国际会议”在上海交大举行，数百名专家学者为知识产权“把脉”。

论坛以“数字治理的法律与政策”为主题，围绕互联网、云计算、大数据、人工智能为特征的数字时代的相关政策法律问题展开了多场学术研讨与交流，邀请了近300名来自国内国外司法审判界、政策监管界、学术界、与产业界的重量级专家学者出席。

在中国国际进口博览会召开之际，知识产权保护也受到了社会各界越来越多的关注。良好的知识产权保护体系，对于上海建设具有全球影响力的科创中心、助力创新大国建设、乃至持续深化改革开放都有着十分重要的意义。

论坛凝聚了国内外有关行政和司法部门、各大高校、研究机构的智慧力量，对推动当前的数字治理工作具有重要意义。

高端对谈中，各位专家分别针对大数据、云计算、人工智能等最前沿的问题发表了观点。主要包括：数字经济时代背景下，知识产权与竞争法的发展趋势如何？知识产权的地域性原则是否已经被突破？标准必要专利纠纷中，法庭能否就全球许可费率做出裁定？

分论坛谈到人工智能的政策法律问题。荷兰海牙上诉法院高级法官Rian Kalden等围绕人工智能的发展给司法审判所带来的挑战、应对及机遇、人工智能的民事主体资格、专利授权、伦理应对、政策监管等问题发表了各自的看法，认为人工智能技术能够为司法、行政决策带来便利，但是人工智能产业的长远健康发展需要政策法律的提前布局。编辑：李燕
