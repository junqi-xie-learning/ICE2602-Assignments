大学生创新创业团队走进崇明 用行动开启“红色筑梦之旅”

上海市“青年红色筑梦之旅”启动仪式现场。

    7月22日，第六届中国国际“互联网+”大学生创新创业大赛上海市“青年红色筑梦之旅”（简称“红旅”）启动仪式在崇明举行。大赛自开放报名以来，上海赛区累计报名两万余个项目，其中红旅项目超过两千项，参与人数近1万人次。

    据悉，本次红旅活动是“互联网+”大赛的重要组成部分，今年的主题是“青春领航脱贫攻坚，红色筑梦创业人生”。活动为期两天，期间，主办方安排参赛团队参观走访了崇明烈士馆、崇明规划馆、乡村振兴示范村、智慧农业示范地等，希望青年学子们能多了解崇明乡村振兴的发展历程，学习“艰苦创业、敢为人先”的崇明垦拓精神。

    上海交通大学副教务长吴静怡表示，高校作为培养青年学子的摇篮，是大学生创新创业教育的重要载体，学校要引导青年学子，积极贴近国家和上海的发展需求，用自己的聪明才智，为国家富强和民族复兴做出应有的贡献。

优秀红旅项目进行现场路演展示。

部分大学生创新创业项目与崇明相关单位现场签约。

“青年红色筑梦之旅”活动设置的直播带货区。

部分参赛团队参观崇明革命史迹展览馆。
