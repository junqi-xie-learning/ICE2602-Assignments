上海民主广场在哪里？华东局机关进驻首日去了哪？《战上海》作者实地还原大上海解放史

教育在线 2019-05-07 17:43
来源：上观新闻 作者：徐瑞哲
陈毅问周林：“接管上海市政府的队伍先进驻哪里？你熟悉。”周林说：“先进驻徐家汇交通大学，然后分头接管。”

你可知道，上海的“民主广场”在哪儿？你可知道，华东局机关进驻上海首日去了哪？

 

在一座跨越3个世纪的百年老校园里，师生们从双烈士墓前、经大操场外围、到工程馆楼下，一路走过了3个地标，仿佛一起穿越70年时空，历经了解放上海时“黎明前的黑暗”、“民主广场”上的呐喊，以及华东局机关进驻上海第一天驻地的旧址。

工程馆如今是自修教室。徐瑞哲 摄

 

7日下午，“中国好书”《战上海》作者、上海交通大学历史系教授刘统，带着人文学院等院系学子“边走边讲‘战上海’”。在本月纪念上海解放70周年系列实景对话的上海交大专场，上了一堂行走着的思政课。

刘统教授与学生交流。徐瑞哲 摄

 

 

 

没想到自己的校园那么红

 

 

菁菁校园，莘莘学子，朗朗书声，梧桐树下是日常。然而，在行走课堂中，当刘统为大学生“导航”数十年前的老校园如何上演民主革命，同学们纷纷感喟：没想到自己熟悉的校园一直那么红。

 

众所周知，1921年7月中国共产党在上海诞生，而1925年底，中共交大党支部就成立了。这不仅是交通大学历史上第一个共产党组织，也是全国高校中最早建立的共产党组织之一。

解放前的上海交大校园。（资料）

在解放战争时期，交通大学被称为上海的“民主堡垒”，“不是交大人自封的，也不是后来的党史研究者加上去的，而是在上海的爱国民主运动中自然形成的，是当时上海的革命学生、群众推举并公认的。”

 

原来，1948年时，美国为了利用日本反苏反共，背离《波茨坦宣言》有关彻底摧毁日本战争力量的基本原则，援助日本复兴和军国主义复活，交大就成为上海反美抗日运动的主战场。刘统说，“民主堡垒”就是这样喊响的，交大的大操场当时也被上海学联命名为“民主广场”。

 

作为入选“中国好书”人文社科类的9本图书之一，《战上海》用历史考据方法书写党史和军史。回顾全书写作过程，刘统最注重保留原始史料和记录。他认为，当年共产党的政策和指示都是开诚布公的，政府的公告、首长的讲话都在《解放日报》上，批评和处分的决定也都公开。“当年没有官话套话，都是开门见山的大实话。”因此，在一些重大问题的叙述上，他都尽量引用原文，尽可能体现历史原貌。

 

 

 

不占师生宿舍度过驻沪首夜

 

 

师生们行至工程馆建筑门口，刘统又戴着耳麦做起了“导览”。这座竖立着无线电报发明人马可尼纪念柱的工程馆，如今是交大学子最常去的自修大教室，当年却正是华东局机关进驻上海第一天的驻址。

工程馆原貌。（资料）

如何接管好大上海，是对解放军的更严峻考验，吸取南京等城市接收的经验教训，华东局、华东军区和三野前委制定了《入城守则》《城市纪律》和《外交纪律》等文件发到各部队。陈毅对部队入城后的纪律提出了极为严格的要求，最基本的一条就是“不入民房”。他说：“这一条要无条件执行，说不入民宅，就是不准入，天王老子也不行！这是我们人民解放军送给上海人民的‘见面礼’！”

 

刘统实地还原大上海解放史的关键一夜。1949年5月25日夜到26日晨，华东局机关和接管干部队伍由丹阳乘火车到南翔，又由南翔分批乘汽车进军上海。陈毅问周林：“接管上海市政府的队伍先进驻哪里？你熟悉。”周林说：“先进驻徐家汇交通大学，然后分头接管。”当时下着细雨，道路泥泞。车队到达交通大学，没有占用师生宿舍，度过了进入上海的第一个夜晚。

工程馆及马可尼纪念柱。徐瑞哲 摄

上海交大党委书记姜斯宪表示，《战上海》以史学家独特的视角，用一个个生动的历史细节，讲述了70年前一群意志坚定、充满智慧的共产党人，在没有现成经验和各种困难面前，怎样成功接收和管理这座远东第一大都市。“共产党人在上海的一系列创新性探索，为新中国各项建设提供了宝贵经验。”

马可尼纪念柱碑文。徐瑞哲 摄

 

 

解放第三天烈士遗体终回校园

 

 

方尖碑下，春暖花开。在这堂走着上的思政课开始时，学生们还与刘统教授一起瞻仰了位于徐汇校区大草坪南侧的史霄雯、穆汉祥烈士墓。

 

两位年轻的英雄为何长眠于此？上海解放前的一个月，是黎明前最黑暗最艰苦的时期。1949年4月，国共正在举行和谈期间，交大举办了一场“真假和平辩论会”，分管学生自治会工作的学生党员穆汉祥担任正方主辩人，党领导的秘密宣传小组成员史霄雯主持会议。因为在公开场合发表反蒋言论，穆、史二人被列入“黑名单”。

两烈士墓前。（资料）

解放日报·上观新闻记者从《战上海》书中了解到，事实上，4月26日凌晨1时许，大批国民党武装军警以装甲车为前导，突然冲入交大，在校园内大肆搜捕。在“四·二六”大搜捕中，“黑名单”上有108人，实际被捕的交大师生有50余人。由于事先得到消息，列入黑名单的重要“共党分子”大部分没有被抓到。

 

但不幸的是，两人分别于4月30日、5月2日被捕，被关在警察总局死牢内，受尽刑讯逼供也没有妥协。刘统考证发现，各方为营救他们想尽办法，交大校长王之卓亲自给毛森打电话，但毛森矢口否认有这两个人。而事实上，5月20日他俩在闸北宋公园（今闸北公园）被秘密杀害。

 

至5月29日，也就是上海解放第三天，交大学生自治会在工人指点下找到了两位烈士的遗体，安葬在徐汇校园中。1950年5月，陈毅市长为二烈士墓碑题词：“为人民利益而光荣就义是值得永远纪念的。”
