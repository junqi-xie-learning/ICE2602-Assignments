世界顶尖科学家论坛在上海滴水湖开幕

2018-10-29 12:01:00来源：央广网

　　央广网上海10月29日消息（记者吴善阳）据中国之声《央广新闻》报道，为期三天的世界顶尖科学家论坛今天上午在上海滴水湖开幕。26位诺贝尔奖科学家汇聚上海临港，与17位中国两院院士同台论剑，掀起世界最强头脑风暴。

众多诺奖科学家出席大会。（央广网记者 吴善阳 摄）

　　本届论坛的主题是“科技，为了人类共同的命运”。在今天上午的开幕式上，上海市市长应勇致词时表示，上海建设科创中心，将进一步夯实基础支撑，拓造光子科学、生命科学、海洋科学大科学设施群，欢迎全球优秀科学家来开展研究工作，并提供完善的知识产权保护。世界顶尖科学家协会主席、2006年诺贝尔化学奖得主罗杰・科恩伯格，表示希望加强基础科学家之间的交流，也希望更多的基础研究成果能够进入人们的生活。他还希望这样的论坛能够作为世界顶尖科学家协会的年度会议，每年举办一次。

　　这次参加世界顶尖科学家论坛的有包括26位诺贝尔奖得主在内的37位世界顶尖科学家和17位中国两院院士。与会的26位诺贝尔奖得主有着非常多的科学传奇：如诺奖历史上第五对夫妇诺奖、2014年诺贝尔生理学或医学奖得主爱德华・莫索尔、梅・布里特・莫索尔，他们都是当前世界上脑科学研究的前沿领袖。此外还有几对同年获奖得主联袂出席：2014年诺贝尔化学奖得主威廉・莫纳、斯特凡・黑尔；师生同年获得2004年诺贝尔物理学奖的诺奖的戴维・格罗斯和弗兰克・维尔泽克。

　　四个主题论坛分别聚焦光子科学与产业、生命科学与产业、创新药研发和转化医学、脑科学与人工智能四大科技前沿领域；世界顶尖青年科学家论坛将为中外杰出青年科学家和世界顶尖科学家搭建深度互动的交流平台。上海交通大学校长、中国工程院院士林忠钦表示，参会的各位学者代表了人类科技前沿，也肩负着引领国际学术交流进步、创新人才培养的重要使命。相信通过世界顶尖科学家的智慧碰撞，必将对上海科创中心建设产生强大的推动力。
