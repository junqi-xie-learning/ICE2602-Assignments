中国能否诞生“极地大片”

2019年03月19日   09： 思想周刊/观点   稿件来源：解放日报  
■黄庆桥

中国第35次南极科学考察队安全完成考察任务，近日乘“雪龙”号返回上海。这次南极科考活动取得丰硕成果，实现了多个“首次”突破，为我国极地事业发展再立新功。

公众对极地缺乏了解

南极因特殊的地理位置、独特的科学价值、丰富的资源禀赋、特殊的主权冻结状态等，受到世界各国的极大关注。虽然南极条约体系暂时冻结了南极地区的主权归属、资源开发等问题，但该条约体系的有效期是有限的。有鉴于此，一些国家不断加大南极考察力度，大力建设南极考察基地，提高南极考察能力，以增强在南极治理上的国际话语权。

中国是南极条约体系协商国之一，也是南极条约体系的坚定维护者。1984年，中国政府首次派遣考察队赴南极考察并建立考察站。第35次南极科学考察队于去年11月2日从上海出发，先后4次穿越西风带，总航程30800余海里，历时131天。其间，遭遇船撞冰山的困境，但最终克服种种困难，完成各项预定任务。

然而，社会公众对中国极地事业的发展还是缺乏了解，极地科普能力也有待提高。特别是，极地科普场馆的公众参与程度、定期的极地科普展览、中小学及大学的极地科普课程、极地科普网络资源的数量、极地科普网络资源的影响力与利用率等，都不尽如人意。

中国是一个有着近14亿人口的大国，也是一个网络大国，但国家南极官方网站的流量数据和访问频次居然排在阿根廷、挪威、智利、乌拉圭、厄瓜多尔等国家之后。这种状况与中国极地事业的巨大成就和发展需要是极不相称的。

我也曾在本科生公共课上组织过一次关于极地科学知识情况的调查。结果显示，即便在重点大学，人们对极地知识的掌握也不容乐观。尤其是，大学生对极地科学知识以及人类认识、保护、利用极地的发展历程，了解非常有限。

降低极地科普场馆入馆门槛

在建设“海洋强国”的时代背景下，加强极地科普教育具有重要的现实意义。

第一，加强极地科学知识科普力度，增强公众的认同感，在“硬科普”上下功夫。

南极、北极是地球之冷源，对全球气候变化和环境变迁可以产生关键影响。开展相关科学考察与研究，对于人类深入了解地球的发展演变及其外层空间，积极应对全球气候和环境变化，具有极其重要的科学价值。同时，极地各类资源丰富、利用潜力巨大，和平利用极地为人类服务具有深远的战略意义。下一步，要提升公众的极地科学水平和认知水平，就必须在极地“硬科普”上下功夫，进而为极地事业发展营造厚实的社会氛围。

第二，加强极地科考成就宣传力度，增强公众的自豪感，在展示“硬实力”上下功夫。

改革开放以来，中国南极事业从无到有、从有到大、从大到强，国际影响力日渐增强。特别是，经过几代人的不懈努力，我们形成了“爱国、求实、创新、拼搏”的极地精神。这既是科普工作的重要内容和鲜活素材，而且是进行爱国主义教育的生动案例，体现了科学研究与思想教育的完美结合。

同时，大力宣传中国极地科考发展历程和成就，还有利于廓清中国政府的极地事业主张，澄清国际上的某些误解，展示中国政府为维护极地和平、发展与稳定所作出的努力与贡献。

第三，加强极地科普能力建设，提升极地科普水平和效率，在开展极地科普任务上下功夫。

我国现有专门的极地科普场馆不足10家，涉及极地内容的综合性海洋科普场馆也仅有10余家。在场馆数量偏少的情况下，有的场馆超负荷运行，有的场馆却利用率不高。有鉴于此，加大极地科普场馆建设与投入，努力降低入馆门槛，应当纳入有关地区和部门的常规工作内容。

极地因特有的神秘性、趣味性，非常容易与互联网结合，“新媒体+极地科普”理应成为极地科普的一个重要发展方向。着眼未来，需要解决的一个关键问题是把“新媒体+极地科普”优质平台、优质活动、优质内容展示好、运营好、维护好，而不能一次性投入、一次性重视、一次性使用，一阵风过后无人问津，成了“网络烂尾工程”。

第四，加强极地科普分众化供给，打造优质文化创意产品，在对接公众需求上下功夫。

前不久热映的电影《流浪地球》，极大地引起了人们对地球、太空和环保的重视。在这个基础上，公众层面的需求就会进一步释放，相关科普工作也就比较好推进。

就极地科普而言，也不乏成功案例。2005年初，法国生态纪录片电影《帝企鹅日记》的上映就曾引起热议，很多人看了这部电影后成了南极的企鹅迷。这部纪录片为什么会火起来呢？关键在于做到了将环保主题、公众需求、市场拓展的有机统一。

应当看到，极地科普文化创意产品的生产是一个大有可为的领域。极地有鲜活、吸引人、感动人的素材和内容，关键是采用什么样的方式、通过什么样的途径呈现给公众。这是对中国极地科普事业的考验，更是一大机遇。我们期待，在极地科普领域也能诞生像《流浪地球》那样有影响力的“大片”。

（作者为上海交通大学马克思主义学院副教授）
