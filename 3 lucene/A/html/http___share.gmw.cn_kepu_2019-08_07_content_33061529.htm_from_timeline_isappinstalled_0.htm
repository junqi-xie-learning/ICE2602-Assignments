2019中国智能船艇挑战赛“海上争锋”赛将在山东日照举办

来源：光明网2019-08-07 16:05
X

    8月6日，中国智能船艇挑战赛发展研讨会暨海上争锋赛发布会在北京举行。会议宣布，2019中国智能船艇挑战赛“海上争锋”赛暨第二届海洋智能装备与系统创新论坛将于今年10月10日至13日在山东日照举行。

    “海上争锋”赛是我国在实际海域海况条件下举办的首个无人智能船艇竞赛，参赛团队将使用具备自主决策、自主航行能力的船艇参赛。竞赛由中国造船工程学会、上海交通大学、山东省日照市人民争锋、中国船级社、中国海洋学会、中国航海学会与中国人工智能学会联合发起主办，专业成都高、参与范围广、资助力度大，同期还将举办“第二届海洋智能装备与系统创新论坛”，发挥“赛事+论坛+展会”的叠加效应，推动智能船艇技术转化和规范形成。

    热潮冷思考：以竞赛形式推动智慧船舶与智能航行技术快速发展

    智慧船舶与智能航运是人工智能浪潮在船舶行业的场景化表达。中国智能船艇挑战赛在设立伊始就确立了要贴近任务需求和实际场景的竞赛原则，通过制定任务要求引领智能无人船艇的动力、控制、通讯、船型、岸电和管理技术协同发展，促进智能船艇技术创新和成果转化转化，推进相关规范形成。比赛构画从运河出发，再到海上争锋，最终实现环球航行，分三阶段九环节逐步实施，对实际航行场景任务需求进行解构和实现。

    中国智能船艇挑战赛设立两年多来，已在扬州江都举办了两届运河段的比赛。中国造船工程学会秘书长林宪东介绍说，智能船艇挑战赛在扬州古运河的航行竞赛是智能船艇在内河航行的生动实践，依托竞赛举办的船舶与航运智能化国际论坛、海洋智能装备与未来系统创新论坛，有助于更深入理解国际船舶和航运智能化发展动态，促进我国船舶工业智能化转型发展。

    经过学会与专家组多轮的走访和研判，在前期大量工作的基础上，一致认为举办海上竞赛技术成熟、时机恰当。今年4月，中国造船工程学会与上海交通大学、中国船级社和山东省日照市人民政府论证并完善了竞赛任务和选址方案，签订框架协议书联合发起启动海上争锋竞赛。

    兼具竞技与趣味：规则设定更贴近船艇实际航行任务需求

    《2019中国智能船艇挑战赛“海上争锋”赛竞赛规则》已对外发布，这份规则是7月15日由竞赛委员会主席、中国工程院院士邱志明院士组织召集领域专家审定通过，既关注无人智能船艇航行前沿技术，又兼具了竞技性和趣味性。

    规则规定，本次竞赛针对参赛任务不同分为四个组别，遥控绕标赛、自主绕标赛、穿越险阻赛与海上争锋赛。最为引人瞩目的是海上争锋赛。竞赛要求参赛团队使用具备自主智能决策能力的竞赛船艇，按照规定的出发时间、航线和要求在近岸竞赛区内完成航行任务，总航程约为10km。航行任务包括“智绘2019”“智寻标的”“海上追梦”，完成所有航行任务得分最高的船艇将被授予海上争锋杯，并将获得15万元的奖励资助，单项科目第一名也将授予3万元的奖励资助。

    “智绘2019”要求船艇需按照路径绕过虚拟点标，整个路线形成“2019”字样，航行过程包括了Z字形循迹、直线循迹、定常回转等航行要点。该项目考察船艇的快速性、操纵性以及实海域抗风浪流扰动能力。

    “智寻标的”科目则在指定海域内随机散布5个浮标，浮标颜色各不相同。船艇需搜寻经抽签决定的相应颜色浮标，第一时间上报其坐标后，抵近相应浮标，与浮标真实坐标相差10m内视为定位抵近成功。该项目考察船艇目标搜索能力与环境感知能力

    “海上追梦”科目要求船艇需追随目标靶船在指定海域内做20分钟随机机动航行。期间，参赛船艇需搜索并在距离1km范围内跟踪，同时保持在一定的安全距离。比赛期间设有干扰船只穿越妨碍，船艇需自主避碰。该项目考察实海域目标感知、识别与追踪的能力。

    易宏教授介绍说，此次“海上争锋”竞赛重点考核各参赛船艇的自主航行与障碍规避、目标搜索与识别、任务规划与协同策略等指标的发展水平，同时设立新型智能船艇功能展示竞赛，考察参赛单位在智能船艇的设计理念、功能实现、场景应用和结构材料等方面的最新研究及开发成果，既关注了海上智能装备演进的前沿研究成果、又以智能船艇挑战赛的形式完成任务目标的落地实现。

    缘何选址日照：环境资源优势和发展海洋智能的决心赢得青睐

    中国智能船艇挑战赛海上段的比赛选址经过多轮论证后，最终确定了山东省日照市。除了当地政府的大力支持和竞赛保障外，专家选址时更看重日照当地得天独厚的环境资源和发展海洋智能产业的决心。

    本次竞赛场地位于山东省日照市的上海交通大学海洋装备智能演进中心。该中心位于日照市万平口风景区，分为岸基指控中心、内湖竞赛区与海上竞赛区。其中，内湖及近岸水域面积约160万平方米，平均水深5米以下，拥有桥梁、堤坝、船闸、防波堤、岛礁、沙滩、亲水平台等实景障碍物，可满足长度20米以下、宽度15米以下、吃水2米以下的智能无人船艇竞赛需求。上海交通大学海洋智能装备与系统教育部重点实验室在日照当地开展海洋智能装备产业规划和技术成果孵化，发挥在海洋智能装备科技和产业的资源优势，以海洋装备智能演进中心的建设带动人才、科技和产业集聚及产业园区的发展。

    山东省日照市东港区人民政府副区长王利侠介绍，去年3月份以来，上海交大海洋智能装备演进中心落户东港以来取得了一系列成果和实效，目前已规划600公顷海域供演进中心长期使用。日照将用最大的热诚、最优的服务和最高的效率接力保障中国智能船艇挑战赛及相关活动顺利举办，相信能够依托竞赛及相关活动为打造“智能日照”品牌营造良好氛围。

    通过一年多的快速建设，海洋装备智能演进中心现已成为能够满足在各种复杂测试科目设定下无人艇的功能调试、性能测试、智能演进需求的多用途试验基地，接下来将为中国智能船艇挑战赛承担最有力的支撑。

    竞赛官方网站现已开通，即日起至8月31日，参赛单位可以通过中国造船工程学会网站或海上争锋竞赛官网进行报名登记。登记后，竞赛组委会将委派专家组到参赛单位进行实船调研，帮助参赛单位更好地完成竞赛任务。（暖暖）
