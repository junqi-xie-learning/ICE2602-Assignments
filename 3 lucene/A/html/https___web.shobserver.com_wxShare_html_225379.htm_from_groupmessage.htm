中组部宣布：杨振斌任上海交大党委书记，姜斯宪不再担任

教育在线 2020-03-17 11:04
来源：上观新闻 作者：徐瑞哲
杨振斌曾任清华大学党委副书记、教育部思想政治工作司司长。

解放日报·上观新闻记者获悉：3月17日，上海交通大学举行教师干部大会，中共中央组织部宣布了中共中央的任免决定，杨振斌同志任上海交通大学党委书记（副部长级）。

因年龄原因，姜斯宪同志不再担任上海交通大学党委书记职务。

【新闻链接】杨振斌简历

杨振斌，男，1963年10月出生，中共党员，1986年8月参加工作，清华大学系统工程专业在职硕士研究生毕业，研究员。

曾任清华大学党委副书记、教育部思想政治工作司司长，2012年4月至2014年10月任厦门大学党委书记（副部长级），2014年10月任吉林大学党委书记（副部长级）。
