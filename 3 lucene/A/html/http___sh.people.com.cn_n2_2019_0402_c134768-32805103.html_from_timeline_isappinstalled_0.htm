青春为祖国歌唱 华东师大网络拉歌接招

【查看原图】

繁花春色洋溢着蓬勃朝气，校园中传来优美歌声，一首首熟悉的旋律，唱出华东师大学子的青春宣言，绽放幸福之花。近日，一场#青春为祖国歌唱#的全国高校自发拉歌活动火爆网络。受上海交通大学邀约，华东师大学子用歌声抒发青春激情，追逐青春理想，表达对祖国的深情。同时，华东师大邀约湖南大学，接力武汉大学、天津大学、厦门大学，通过网络互动的形式携手歌唱祖国。

丽娃河边、樱桃河畔、群贤堂前、尚义桥上、大夏书店中……同学们深情演绎《我在未来等你》《华东师范大学校歌》《共筑中国梦》三首歌曲，歌声悠扬、情绪昂扬。华东师大中山北路校区及闵行校区航拍、延时摄影中呈现的四时风景、建筑风物，浓缩的是筚路蓝缕、融汇中西的校史，更展现了华东师大学子的青春活力及精神气质。

此次华东师大的视频创作拍摄积极调动了学生的主动性，取景地及歌曲选择完全由学生做主。在选取传统地标的基础上，同学们建议加入大夏书店、华东师大出版社24小时阅读空间、闵行校区樱花园这些新兴网红地，体现校园书香文脉的创新传承，更书写了华东师大学子的身份认同。在演唱曲目的选择上，同学们选择了贴近自身心路历程，又能弘扬爱国主义情怀的歌曲。歌曲拍摄中，路过的师生被吸引，驻足围观。政治学系政治学与行政学2016级本科生陶佳宁表示，“去年我和我们师大的140名小伙伴们一起参与了首届中国国际进口博览会的志愿服务活动，深深感受到了青年应该将自己的青春梦融入到中国梦当中，在与祖国同奋斗的过程中，实现自己的青春理想，书写青春担当。”

现场视频通过微信、微博、抖音等新媒体矩阵平台同步发布，发布后三小时微博播放量已达3.8万次。“少年强则国强，为我们的青春打CALL”“合唱part太容易让人感动了”“青春的力量，你们感受到了吗？”网友们互动热烈，纷纷在评论区留言。
