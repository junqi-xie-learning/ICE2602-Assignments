上海交大首届创新设计国际青年学者论坛举行

 

12月27日至28日，上海交通大学2018年第一届创新设计国际青年学者论坛举行。本次会议由上海交通大学设计学院为主承办方， 吸引了来自斯坦福大学、慕尼黑工业大学、哥本哈根大学、米兰理工大学、新加坡国立大学、香港大学、清华大学等海内外知名学府与研究机构的47名青年才俊齐聚交大，分享他们的最新研究成果，深入探讨交流创新设计热点前沿问题。

上海交通大学副校长王伟明在开幕式上表示，设计学院的成立，对上海交大人文艺术与科学技术的进一步融合具有重大意义。而今恰逢设计学院成立一周年，面对着国家创新发展战略需求和上海市“设计之都”建设的需要，希望有更多的设计界青年学者能够加盟交大，为交大创建世界一流大学贡献力量。

论坛特别邀请了上海交通大学访问讲席教授、中国工程院院士徐志磊，中央美术学院建筑学院课程教授、《建筑学报》执行主编黄居正，东南大学建筑学院教授、景观规划设计研究所所长成玉宁作主报告，为嘉宾带来创新设计学科未来发展的关键前沿理论。

作为新中国杰出的机械工程设计专家，他在题为“创新设计的理念·价值与意义”的报告中指出，中国设计如今局限便在于，不同设计领域仍存在不同的设计理念。而创新设计的核心，即是要汇聚不同理念，实现元设计回归。徐志磊通过详细阐述科学、艺术、创新和设计的内在联系，引出创新设计真正的功能理念、价值及意义。他表示，国家的竞争力集中体现在社会经济发展的繁荣程度，体现在技术创新的商业转化，而这一切都与国家的设计能力和竞争力息息相关。徐志磊希望交大设计学院未来能够坚持以此理念发展，进一步融合数字技术、人工智能、商业模式等现代技术，共同创造更多价值，从而更好的服务于人类社会。

据悉，本次论坛共设置五个分论坛，论坛主题分别围绕“建筑设计及其理论”“建筑历史及其理论”“产品与交互设计”“文化与服务设计”及“可持续风景园林”展开，涉及包含建筑设计中的空间布局、人文关怀、绿色节能、建筑设计中的数字化方法应用、建筑遗产的保护方法、价值挖掘与再利用、生态修复等多学科交叉融合下的创新设计前沿热点研究。

记者了解到，上海交大设计类学科着力于人文艺术与科学技术交融，注重理论与实践相结合，于人才培养、学术研究、社会服务、国际交流等领域均取得突出成就，在近年QS世界大学学科排名中，“建筑与人居环境学科”在国内外高校中排名前列，“艺术与设计学科”于2015、2016 年连续进入全球50强。

为充分激发设计类学科发展潜力，2017年12月，学校整合全校设计类学科资源，将船舶海洋与建筑工程学院建筑学系、原媒体与设计学院设计系以及农业与生物学院风景园林系重组成立了设计学院，统筹建设“创新设计”一流学科群，涵盖建筑学、设计学与风景园林学三个一级学科，并将其列入学校“双一流”重点建设的17个学科群之一。
