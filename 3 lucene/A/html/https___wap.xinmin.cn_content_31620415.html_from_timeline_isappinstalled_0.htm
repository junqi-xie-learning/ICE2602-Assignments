新增全国重点文保单位“上海交通大学早期建筑”揭牌

新增全国重点文保单位“上海交通大学早期建筑”揭牌飞入寻常百姓家
来源：新民晚报   作者：易蓉   2019-12-08 15:48:00

图说：上海交通大学早期建筑成功入选第八批全国重点文物保护单位 新民晚报记者 孙中钦 摄（下同）

新民晚报讯（记者 易蓉）今天下午，第八批全国重点文物保护单位——“上海交通大学早期建筑”在上海交通大学徐汇校区红太阳广场举行揭牌仪式。

今年10月，国务院正式印发《关于核定并公布第八批全国重点文物保护单位的通知》，其中，上海市共有12处文物入选，上海交通大学早期建筑成功入选第八批全国重点文物保护单位。

上海交通大学历史悠久，是近代最早由中国人创办的两所大学之一，也是目前国内惟一一所跨越三个世纪、保留发源地与办学地不变的高校。徐汇校区具有独特的、中西结合的校园布局与建筑，集中展示了邬达克等近现代建筑大师的作品，很多建筑都采用了当时最先进的建造材料和技术手段，代表了当时先进的建筑思想及技术上的勇于尝试。不同建筑风格各异的建筑并存，一定程度展现了上海地区近现代建筑的发展变化历程，见证了中国高等教育的诸多重大历史事件和重要变革，是近代中国社会发展、校园发展历程的印证。

上海交大早期建筑的一大独特性来自交大校友对母校的热情和帮助，有一些建筑是由校友捐助建成的，如图书馆、校门、工程馆等，也有些是由交大校友主持修建的，如体育馆和文治堂。这些建筑承载着国家记忆，传承红色基因，已成为上海市革命先烈缅怀基地。

图说：行走校园，阅读建筑

2014年，上海交通大学历史建筑群被上海市人民政府公布为上海市文物保护单位。学校长期以来重视文物保护与管理工作。上海交大建筑遗产保护中心开展了调查研究、制订规划、保护修缮、价值传播、申报文保单位等方面的工作，尤其是价值传播方面，为了推动建筑可阅读，出版了《老房子新建筑》《上海交通大学人文建筑之旅》《南洋筑韵》等图书；制作了《交大有些老房子》《中院》《因为建筑，历史得以长久铭记——百年交大历史建筑群》等视频节目。

这些建筑包括老图书馆、中院、新中院（董浩云航运博物馆）、新上院、工程馆、科学馆、北四楼、执信西斋、文治堂、总办公厅、新建楼、体育馆、盛宅、华山路校门、五卅纪念柱和烈士墓。上海交大建筑遗产保护中心在每幢历史建筑设二维码，参观者扫描即可看到详细介绍内容。
