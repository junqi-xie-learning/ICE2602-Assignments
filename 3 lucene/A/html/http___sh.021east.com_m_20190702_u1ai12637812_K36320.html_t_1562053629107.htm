“00后”双胞胎姐妹花：除了尽享第二杯半价，还要一起上名校

东方网 傅文婧
2019年7月2日 15:32:36

　　蔡亦卓（右）、蔡也越姐妹

　　东方网记者傅文婧7月2日报道：又到了一年一度收录取通知书的时节。在浙江镇海有一对18岁的孪生姐妹蔡亦卓、蔡也越，她们不仅青春靓丽“颜值高”，而且聪明好学“考分高”，可谓“才貌双修”。今年高考过后，她们双双以高分被上海交通大学工科试验班录取，而且两人考分仅相差1分。

　　作为双胞胎，生活中就无法“独一无二”，生命自诞生起就有“世界上另一个我”形影不离。姐姐名叫蔡亦卓，妹妹名叫蔡也越，从她们的“卓越之名”上也能看出父母对她们的殷切期望，而这对姐妹也没有辜负父母的培养。

　童年的两姐妹

　　都说双胞胎之间有神奇的心灵感应，蔡亦卓和蔡也越也会这样。她们经常会同时哼唱同一首歌，同时做出相同的动作，产生相同的念头，对于“单身狗”艳羡的“第二杯半价”，她们从小到大都能尽情享受。这样的同频率，不仅在生活里处处有，连学习的节奏上也总是齐头并进。

　　凭着几乎完全相同的基因，加上长期共同生活带来的默契，蔡亦卓和蔡也越打小就总能在你划我猜、两人三足等游戏中拔得头筹。正所谓“双剑合璧，天下无敌”，在学校乒乓球双打比赛中，右撇子姐姐和左撇子妹妹的默契合作，所向披靡也不在话下。

　　蔡亦卓、蔡也越同被上海交大录取

　　因为是双胞胎，家人难免将两个孩子不停地从小比到大。“妹妹这次数学比姐姐高2分”“姐姐这次语文考得比妹妹好”……她们是亲密无间的姐妹，也是天生的竞争对手。在好学上进的家庭氛围中，姐妹俩总是自觉的拿起课本，认真求学。虽然有时也会因为琐事拌嘴，但这丝毫不破坏姐妹俩的“革命情谊”。姐姐考砸时，妹妹贴心温暖的安慰与鼓励；妹妹骨折时，姐姐风雨无阻地推着轮椅上下学以及无微不至的照顾。姐姐坦言，她们一直都在学习与生活的世界里互相搀扶，互相成就。

　　蔡亦卓和蔡也越毕业于浙江省镇海中学。高一时，她们偶然聆听了一次”学长学姐情系母校”的宣讲，头一次了解了上海交通大学，对学校的教学环境、科研实力和丰富的文化底蕴产生了浓厚兴趣。上海交大以工科见长，蔡家姐妹刚好对工科领域充满向往。她们觉得工科是拿起知识和技术的工具在名为“实践”的高山里磨练技艺。选择工科，意味着要在学习中实践，在实践中学习，将理论用于实际，在一次又一次动手中寻找解决问题的办法。选择工科，就是选择了学习专业技术的道路，这也将是她们坚定的人生选择。

　　如今，高考成绩早已出炉，两姐妹也凭着高分被上海交大工科试验班录取。面对梦想之中的大学和充满挑战的大学生活，她们也将拿出小时候“打怪”的勇气，并肩作战，在更大的人生舞台追寻各自的理想。
