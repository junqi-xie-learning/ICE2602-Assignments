长三角一体化对上海有什么好处？听听委员怎么说

伴公汀 2019-01-26 17:28
来源：上观新闻 作者：曹飞
在长三角一体化发展战略之下，上海将更好地发挥中心城市的功能，可以更好地借助于长三角城市群的腹地来推动现代服务业的发展。

去年11月5日，习近平总书记在首届中国国际进口博览会开幕式上的主旨演讲中向世界郑重宣布，支持长江三角洲区域一体化发展并上升为国家战略，推进更高起点的深化改革和更高层次的对外开放。

 

那长三角一体化发展战略对于上海来说有什么好处？在今天下午举行的市政协十三届二次会议委员与市民网上交流的活动中，有网友提出了这个问题。

 

市政协委员陆铭回答问题时表示，在长三角一体化发展战略之下，上海将更好地发挥中心城市的功能，可以更好地借助于长三角城市群的腹地来推动现代服务业的发展。

 

上海的优势是金融贸易等现代服务业、高科技、文化产业等，而其他的长三角城市则可以相对更多的发展制造业，尤其是在长三角下游地区，沿江的城市港口条件都非常好，在发展出口导向型的制造业方面相对于上海来说，具有更加便宜的劳动力和土地。因此，不同的城市之间完全可以形成更好的产业分工。“总之，不同城市间的分工将更为明确，相互之间的功能定位将更为互补。对于上海经济总量和质量的提升都将起到极大的促进作用。”陆铭委员指出。

 

针对市民提出的关于长三角一体化发展受限原因、最主要障碍的问题，陆铭委员在回答中称，现在制约长三角发展的最主要因素，就在于行政管理边界对于一体化的制约。由于行政管理边界的制约，每个城市的规划是独立作出的，交通基础设施、公共服务的提供也是在行政管辖边界范围之内做的。而每个地方政府又有自己经济发展总量和税收最大化的目标，这些都会阻碍生产要素的自由流动。

 

不过，陆铭委员指出，全国各地都不同程度的存在行政管辖边界对于区域经济一体化的障碍，而在长三角地区，不同城市经济一体化的需求和行政边界对于一体化的障碍，这一对矛盾最为突出。因此，如果长三角地区可以率先改革推进区域经济一体化的话，就可以为全国其他地区区域经济一体化产生示范作用，在这方面我认为上海应该敢于创新，更多地发挥引领发展和制度创新的表率作用。

 

在珠三角地区，广（州）佛（山）同城化发展比较顺利，行政管辖边界对于两个城市之间的阻碍作用得到了较好的克服，交通基础设施的连通性也比较好，以这个案例为榜样，上海还要更好的推进与周边其他城市一体化进程，尤其是在交通基础设施的连通性上可以率先突破。另外，珠三角地区民营经济的发展也比较好，这点也值得上海学习，陆铭委员建议。

 

据悉，市政协今年将把推动长三角更高质量一体化国家战略放在更加突出的位置。围绕建设长三角一体化发展示范区，组织委员进行深入调研，并召开专题议政性常委会进行深入的协商议政，积极建言献策。另外，还将积极推动筹办常设性的长三角一体化发展论坛，邀请三省一市知名企业家和全国的专家学者，聚焦高质量、聚力一体化，开展广泛深入地研讨，广集民智，广聚共识。
