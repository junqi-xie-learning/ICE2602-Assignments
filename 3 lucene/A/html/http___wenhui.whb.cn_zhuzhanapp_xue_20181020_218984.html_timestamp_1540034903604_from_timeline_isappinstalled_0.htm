这些大学里的年轻志愿者要在进博会上岗了！
2018-10-20 15:56:41 作者：樊丽萍

首届中国国际进口博览会的脚步日益临近，沪上高校吹响了志愿服务的集结号，一批朝气活力的青年大学生正在接受系统专业的志愿服务培训。据悉，在今年进博会已经完成的5438名志愿者中，高校大学生志愿者超过5200人，堪称进博会志愿服务军团的绝对主力。今天下午，部分高校的志愿者代表在上海外国语大学松江校区接受志愿服务培训工作。肯担当、耐吃苦、做实事，进博会的志愿服务岗位，也将是中国青年自信与智慧的一次集中展示。

进博会是展示国家形象的重要窗口，提供优质的志愿服务，少不了专业人才的加盟。上海外国语大学阿拉伯语专业大四学生葛学政，是本次进博会会期服务岗位志愿者之一。目前，他已通过外交部遴选，毕业后即将进入外交部工作。

记者获悉，今年，上外师生集体上阵，共有460人入选进博会志愿者人选，人数居沪上高校之首。所有志愿者都是清一色的外语类专业背景，涉及法语、西班牙语、俄语、阿拉伯语、英语、德语、日语、葡萄牙语、朝鲜语、荷兰语、希腊语、意大利语、土耳其语、波兰语、越南语、印尼语、波斯语、瑞典语、泰语、希伯来语、印地语等20多个语种。其中，阿拉伯语、法语、俄语、西班牙语等语种服务人数超40人。在进博会期间，他们将主要服务于国际高级访问团，国外媒体记者团，制证安检团队，以及国家会展工作组等。

“我教的学生在进博会期间几乎全员上岗。”上外俄罗斯东欧中亚学院杨波教授即将参与进博会期间官方活动的同声传译和交替传译等会议翻译工作。近期，她还将为院系的志愿者开展专业培训。

在沪上复旦、上海交大、同济、华东师大等多所高校，世博会志愿者的校内培训眼下也在陆续启动。

上海交大今年有140余名学子入选世博会志愿者。就在不久前，该校安泰经管学院教师黄超芸在受邀为志愿者做内部培训。她说，进博会是“面向世界的国家行为”，因此，志愿者代表国家形象，需要强烈的志愿精神，“志愿者坚守的每个岗位将是一点火光，星星之火，可以燎原。”

在复旦大学，共有200位学子从400多名报名者中脱颖而出，成为进博会的志愿者。在接到进博会指挥部提出的需要招募媒体志愿者等具体的志愿者专业能力方向的要求后，学校在新闻学院中开展定向招募。

同济大学已组建了一支120人左右的进博会志愿者队伍。值得关注的是，同济电信学院的一个项目团队已为本次进博会准备了一套高精度的室内定位导航系统，将在技术上开创行业先河。进博会期间，同济将有多名志愿者服务于该系统，为进博会各参与方提供便利。

在华东师范大学，校方专门为140名即将服务进博会的志愿者“开小灶”，邀请一些有跨文化实践的专家和教师代表做专题报道，参加过米兰世博会、阿斯塔纳世博会等国际展会的学生志愿者也受邀分享自己志愿服务的经历。

此次进博会上，留学生志愿者也将成为一道风景。从尼泊尔来沪留学的浦瑞妍，是东华大学纺织工程专业大四学生，也是进博会的留学生志愿者代表之一。“尼泊尔的纺织品大多都是手工制作，来到中国学习的都是高技术的纺织工程，希望通过进博会能把好技术带回家乡。”

在东华大学招募的进博会志愿者中，有四位是留学生，分别来自塞内加尔、尼泊尔、布隆迪和巴基斯坦。


五月份开始就投入进口博览会实习的上外研究生金铭可以说是上外第一位进口博览会学生志愿者，她先后参与了宣传、会务以及接待等筹备工作，谈到100多天的志愿者工作，她表示“可能每一个人都在做微不足道的事情，这些事情是看不到的，甚至小到只是指示牌坏了需要有人去提醒，每一个人都在排查可能存在的问题。”正是这些默默付出的志愿工作者，汇成了一个强有力的团体，为进博会顺利开展保驾护航。

葛学政是上海外国语大学阿拉伯语专业大四学生，是本次进口博览会会期服务岗位志愿者。他已通过外交部遴选，毕业后将进入外交部工作。

这名“95后”青年在大二期间入选国家留学基金管理委员会优秀本科生项目，被选派至摩洛哥哈桑二世大学留学一年，学习成绩优异，实践经历丰富。“参加进口博览会志愿服务是我们年轻一代在国际平台上培养中国情怀和国际视野的重要经历。”小葛作为小语种特长志愿者，先期被安排在证件中心工作，负责进口博览会数种证件的校对、制作工作，很快成长为志愿者骨干。为了保证证件按时制作完成，他和其他志愿者经常加班至深夜，并就地过夜，“证件是保障进口博览会安全有序进行的一道屏障，工作中就要不怕吃苦，任劳任怨。”

今天，两款首届进口博览会志愿者服装还同时发布。

一款是新中式西装，灵感源于五星红旗，红色明快鲜亮，代表朝气、热情；白色宁静舒缓，代表和平、友善；灰色沉稳大气，代表时尚、专业。红白灰线面交织，领口呈现五星造型，点缀四叶草形纽扣，凸显海派的都市韵味。

另一款以五角星和线形元素组成主体纹样，寓意“一带一路”链接五大洲，体现进口博览会是“一带一路”倡议的生动实践，描绘出进口博览会将促进全球贸易和世界经济增长，推动人类命运共同体建设的美好愿景。两款服饰充分体现进口博览会志愿者自信、开放、阳光、担当的强国一代、大国青年形象。

“为祖国增光，为城市添彩！向前进，博青春！为进博，一起来！”志愿者代表用青春之声庄严宣誓，表达了践行志愿者精神、全心为进博奉献的信心与决心。

根据进口博览会的整体要求和志愿者岗位安排，目前，已经完成志愿者的招募和配岗工作，总计招募5438人。其中152名是长期管理岗位志愿者，主要是来自高校、机关企事业单位的在职人员，已从6月份起陆续脱产上岗服务，参与到进博会各个块面的筹备工作中；5200多名是高校大学生志愿者，这是进口博览会志愿者队伍的主力军，将参与到场馆内外迎博办博的各项服务保障工作中；同时，为体现长三角一体化，团市委也向苏浙皖三省发出了邀请，长三角30名志愿者也将共同参与到进博会志愿者工作中来。
