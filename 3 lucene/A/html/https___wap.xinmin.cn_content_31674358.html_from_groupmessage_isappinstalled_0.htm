沪上高校纷纷启动“在线教育”

沪上高校纷纷启动“在线教育”飞入寻常百姓家
来源：新民晚报   作者：王蔚   2020-03-01 09:50:00

大学的校园，迎来了最清冷的时刻。新冠肺炎疫情仿佛为校园按下了暂停键，空荡荡的教室、空旷的操场等待着青春生气的回归。可是在云端，大学生的学习没有停止。近日，沪上高校已经纷纷“在线教育”，即便无法在校园相聚，高校师生也正以力所能及之力为抗疫贡献自己的力量。团聚在党的周围；细致入微的送温暖活动，更让每位学子切身感受到众志成城的力量。

上海交大“隔空”学习不松劲 “攀登计划”引领专业课在线攀登

“此刻也许我们还没有能力，还没有准备好迎接挑战。但终有一天，当国家需要我们的时候，我们也能站出来，拿出锤炼的勇气和力量……”今天上海交通大学迎来“在线教学第一课”，交大学子可以通过网络选修两千余门课程。“云学习”能否适应？其实在假期里，上海交大电子信息与电气工程学院的学生早就发起了宅家学习的号召，老师“送课”、学霸拉群，“隔空”学习充实了宅家生活。

这是学院本科生学术志向引导工作“攀登计划”的线上版，由学生组织推动。本科生黄鹏程老家在遵义，原本他打算过完春节就到县图书馆去看看书。疫情打乱了计划，各种消息和越来越严格的封村隔离，也带来了紧张和不安的情绪。正在一筹莫展，学院的一批课程上线了，更重要的是，有名师讲解、学长答疑，还有同学一起学。正学习的《电路理论》，是新学期的内容，他希望早早预习，返校之后可以学得更轻松、深入。“主讲老师是很厉害的教授，15分钟的视频恰到好处。通过这样的学习，甚至还有自豪感、愉悦感，何乐不为呢？”黄鹏程说。

家住青岛的陈柏宇把书桌让给了妹妹，自己用妹妹以前的小课桌架上电脑也开始“线上攀登”，“本来离开学校，许多文献无法查看。现在能够‘不停学’，这才是交大学子该有的样子！”学霸笔记、难题讨论，“课外时间”微信群更热闹。

今天下午，上海交大校党委书记姜斯宪和公共卫生学院院长王慧通过网络为交大学子直播讲授“在线教育第一课”，也正式开启“学在交大”的“云端版”。

是不是每个学生都有上网设备？通讯是否顺畅？会不会有学生无法负担在线学习的流量费用？为了保障线上学习无障碍，学校为教师搭建平台，仔细排摸学生情况，要让每个学生都能实现在线学习。

      此外，上海交大还特别设立新冠肺炎疫情专项特殊困难补助。学生处副处长、学生事务中心主任胡艳丽说，延期开学对学生开展科研、实习就业、出国交流比赛等都带来影响，因此学校出台“托底”政策，希望在学生遇到实际困难的时候给予帮助，让学生安心。

同济新生学院网络党课上线 “人必须宅，精神却要放飞”

“我的父亲正在基层从事抗‘疫’工作，我也要不负党员担当，传播正能量”，“‘硬核’医生张文宏的几句话让我印象深刻，党员要冲在最前线，什么是前线？现在这个疫情就是前线”……近日，同济大学新生学院济勤学堂的网络党课又上线了。先开展线上党组织生活，围绕战“疫”中的党员先进性展开了一番讨论。

同济新生学院济勤学堂共有817名大一新生，疫情突至，这些00后萌新们，还来不及感受第一次大学寒假的喜悦，就陷入了焦灼的情绪中。“宅在家里，精神不能‘宅’”，新生院党委第六党支部辅导员们敏锐地感受到了同学们的不安情绪，于是，凝心聚力，借助网络平台，引导萌新们积极配合疫情防控，以信仰启航教育为切入点，为萌新们解疑释惑，丰富学生的精神世界。

这一天，20个班级群的全体学生，又一次聆听入党启蒙教育。在“寻找战疫中的优秀党员”主题思想汇报活动中，同学们在寻找榜样的过程中深深地被感动、被激励，他们写道，“党员在平时没有特权，却要在危急时刻冲在第一线，这让我对党员肃然起敬”

一名湖北籍的学生说：“家在鄂州的我，在这样一个特殊时期，也开始思考，身为一名党员，我能在这场战‘疫’中做点什么。虽然没有医生专业技能，其实，我们也能做很多很多。协助学校完成各类统计，和湖北的小伙伴们互相鼓励，原来，在家，也能实现一名党员的价值。”

刚满18岁的张峰同学，在网上党课郑重提交了自己的入党申请书，“我之所以要加入中国共产党，是因为我要为中华民族的强大出一份微薄而坚强的力量”。

上课完了，还有同学依依不舍，写道：“病毒无情，人间有爱。党员同志奋勇向前并肩作战，守望相助，爱和希望比病毒蔓延得更快。让我们一起打赢疫情防控阻击战。”

上海工程大辅导员李迎的特殊任务 每天收齐2631个“平安福”

“我们要严格执行疫情防控‘零报告’‘日报告’制度，每天及时准确地排摸学生信息，确保数据一个也不能少。”上海工程技术大学电子电气工程学院辅导员李迎说，这段日子，他不是抱着电脑就是拿着手机，家里人很疑惑，殊不知，他是在收集学生的“平安福”。

15个线上联系群，每天数百次的语音、文字交流，终于梳理出第一轮2627名学生的基本信息、寒假行程轨迹及健康状况。为避免出现遗漏，细心的李迎通过学校教学管理系统下载在籍在校学生名单，逐一核对至凌晨两点，终于梳理出在国内外交流、参军复学的另外多位同学信息，并一一电话联系，确保平安。此后，每天收到学院2631名学生的平安福，便是李迎最为心安的事。

防“疫”战打响时，工程大纺织服装学院教师孙琴第一时间关注到班里的三位湖北籍学生。其中一位王姓男生已经返回了北恩施州家中，那几天，小王身体有些不适，爸爸也刚刚从武汉回来。家住乡镇，医疗资源欠缺，家中也没储备口罩等防护物资，小王一下子慌了神。孙琴赶紧通过电话进行心理疏导，并提醒其尽早去大医院检查就医。想到家中还有一些“存货”，她立刻叫了快递员，将口罩寄送至湖北。“我们自己的口罩也不多，寄给学生我们怎么办？”面对父母的疑虑，孙老师解释说：“他比我们更需要口罩，要把最紧缺的物资送到最需要的地方去”。

上大一群教师正在湖北度寒假 每天都在等学生留言”一切都好”

安奕男老师在学生工作办公室负责承担学籍管理工作，上大在湖北省有400多本科生。大年初三，学工办党支部成立了“湖北学生疫情防控工作”临时联合党小组，负责落实湖北学生的疫情防控具体工作。“我的一日三餐都是在卧室解决，我妈端到床头柜，我想起来了扒两口，然后继续工作。每天一醒来第一件事就是看小组的消息，筛选出我的工作内容，然后打开枕边的笔记本电脑，开始一天的奋战，”安老师说。

“冰哥你也多保重。”被称为冰哥的，是上大外语学院2018级学生辅导员崔一冰，寒假里又多了个头衔——学院疫情防控领导小组信息联络员。身在武汉，冰哥可忙了，他说：“学生在，我也该在。起初，我也有过抱怨，认为信息汇总工作繁重、琐碎，给本就有些揪心的‘隔离’生活徒增了压力。但是，令我特别感动的是，前两天收到了一个包裹，是班上一位学生寄来的，是一盒医用口罩。”如今，他组建的抗击疫情群，还被同学们称为“冰哥全国粉丝后援会”。

此外，经济困难学生的家庭生活情况，尤其是湖北籍同学的经济状况、身心状况牵挂着学校老师的心。上海大学学工办、研工部与各院系联动，逐一了解、密切关注学生及其家庭情况，并联动相关部门设立专项资助资金发放给105名湖北学生。“这几天，我们排摸出全校共有52名正在湖北的研究生存在着家庭经济困难问题，目前已经向他们发放了临时补助款，帮助他们度过困难。”上大党委研究生工作部匡可风老师说。

上师大志愿者展开特殊服务 为抗疫医生子女办“爱心学校”

近日，上海师范大学外国语学院青鸟志愿服务团队开办了“抗击疫情医护人员子女网络爱心学校”，为上海中医药大学附属岳阳中西医结合医院参与抗击疫情的15名医护人员子女提供一对一线上英语口语练习、学业辅导。

青鸟志愿服务团队坚持规范化、网络化、专业化的服务原则，主动与岳阳医院对接，前期通过问卷调查了解医务人员需求，排摸网络爱心学校开展的可行度,通过梳理网络课业辅导的具体流程、课程安排、注意事项等，形成了“一家一策”的网络爱心学校开展机制。上岗之前，学院团委专门组织岗前培训，从保护学员隐私、谨慎理性上网、坚守志愿初心等方面做基本工作要求。上岗之前，学院团委专门组织岗前培训，从保护学员隐私、谨慎理性上网、坚守志愿初心等方面做基本工作要求。所有教员线上“云”承诺——让青春在这场疫情防控阻击战中绽放。

17级英语（师范）专业的志愿者黄芷萱说:“网络爱心学校传递的不仅是知识，更是高质量的心灵陪伴，很荣幸能为发挥自己师范生的特长，为医护人员筑起坚强的后盾。”

新民晚报记者 易蓉 张炯强 郜阳 王蔚
