上海高校师生热议改革开放40周年大会

2018-12-18 17:07:01　来源: 科技日报 作者: 王春1

中国科技网·科技日报记者 王春

12月18日上午，庆祝改革开放40周年大会在京举行，习近平总书记出席大会并发表重要讲话。习近平表示，建成社会主义现代化强国，实现中华民族伟大复兴，是一场接力跑，我们要一棒接着一棒跑下去，每一代人都要为下一代人跑出一个好成绩。他号召全党全国各族人民要更加紧密地团结在党中央周围，高举中国特色社会主义伟大旗帜，不忘初心，牢记使命，将改革开放进行到底，不断实现人民对美好生活的向往，在新时代创造中华民族新的更大奇迹，创造让世界刮目相看的新的更大奇迹。上海高校师生通过电视直播观看大会，习近平总书记的讲话，在师生中引发热烈反响。

中国工程院院士、华东理工大学副校长钱锋说，“是改革开放铸就了今天的我。1978年初，我作为高考恢复后的第一批大学生，幸运地登上了改革开放这辆高速‘知识列车’。作为一个普通家庭出身的子弟，能够成为中国改革开放的见证者、亲历者，能够成为社会主义现代化建设事业取得历史性成就的践行者、参与者和受益者，我无比自豪、无比荣幸！深化改革创实绩，不忘初心再出发，站在新的历史起点上，我将带领团队深度融合传统制造业与人工智能，以‘智’为引，努力将具有自主知识产权的核心技术应用于工业装置，助推工业制造向智能化方向提升，这是我不懈耕耘的‘远望’。”

 上海交大船舶海洋与建筑工程学院教授刘西拉表示，听完习近平总书记的讲话，心中无比兴奋激动。“我出生在1940年战乱中的重庆，儿时最清楚的记忆就是在防空洞里看到烛光后面那一个个同胞惨淡的面孔。非常庆幸我们这一代能亲眼看到中国的巨变；更庆幸的是，我们这一代有机会为祖国的巨变奋斗一生。”刘西拉表示，改革开放后，自己有幸被祖国派到国外学习西方先进经验，他用三年零七个月取得了硕士、博士学位，并得到了国际同行的认可。回国后，刘西拉服务于祖国教育科研事业一线，并被世界工程组织联合会授予“卓越工程教育奖章”。“如果没有改革开放，就不会这样充实的今天。我从内心感谢我们的祖国和人民，我深感我们身上的责任，定将不忘初心，继续努力奋斗！”

全球高被引科学家、华东理工大学唐漾教授表示，“回首中国共产党领导下的中国改革开放的40年历程，上海一直勇立潮头，开拓进取，海纳百川。1983年出生于上海的我，经历了上海浦东开发开放的历史，深感无比骄傲和自豪。作为一个普通的海外归国人员和青年教师，能够成为中国改革开放的见证者、亲历者和受益者，我无比荣幸。站在新的历史起点上，我将不忘初心、勇于担当、坚忍不拔、立德树人，在党和政府的领导下，以高质量的人才培养和科学研究为目标，为实现祖国的繁荣昌盛、中华民族伟大复兴贡献自己的一份力量。”

上海交大媒体与传播学院2015级本科生舒天楚表示，“庆祝改革开放40周年大会上，‘1979年，那是一个春天…’当《春天的故事》熟悉的旋律在人民大会堂响起，我不禁热泪盈眶，心潮澎湃。一位位我们熟知并敬重的改革开放先锋名字一遍遍在我心中回响，他们照耀过往峥嵘岁月，引领和改变这个时代，更为我们指向一个更加美好的未来。我坚信，我们国家在党的领导下，必将在千帆竞发、百舸争流的时代越战越强，在新时代创造中华民族新的更大的奇迹！改革开放正青春，强国一代有我在！作为一名青年大学生，一名刚刚加入中国共产党的年轻党员，我会不断在学习工作中不忘初心，牢记使命，为下一个40年新征程贡献自己的青春力量！”

（科技日报上海12月18日电）
