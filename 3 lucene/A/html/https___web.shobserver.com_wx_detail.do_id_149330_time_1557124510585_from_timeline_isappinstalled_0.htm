这本科生近一年4次赴美已直博耶鲁，他们的物理是诺奖得主教的，难怪叫拔尖计划2.0

教育在线 2019-05-06 14:34
来源：上观新闻 作者：徐瑞哲
你在网上读过一篇科普文“养乐多里真的有100亿个乳酸菌吗”，那是致远学院生命科学方向学生戈劼妤在微信公众号上原创的。“妈妈从小给我吃得好，我也对食物带着热情和好奇……”

金哲霆，上海交大致远学院大四学生，毕业季这一年来已先后4次赴美，从出席美国物理学年会作大会报告、走进强磁场实验室发表海报论文，到参加耶鲁大学校园开放日活动，所有参访基本都由院方资助。6日，金哲霆告诉解放日报·上观新闻记者，他最终选择直升耶鲁博士生。

这是交大人才特区——致远学院依托教育部“基础学科拔尖学生培养计划”培养的应届生之一。渊源于2009年29名学生组成的“理科班”，如今全院在读学生规模已近1100人，成为一流高校中的一流人才。

随着教育部日前发布“六卓越一拔尖”计划2.0，全国建设260个基础学科拔尖学生培养一流基地，培养一批杰出的自然、人文、医学科学家。在教育部“基础学科拔尖学生培养计划”实施十年评估总结中获得全优的上海交大致远学院，也酝酿续推拔尖计划2.0版。

 

 

【大一起陆续加入课题组，逾六成学子攻读博士】

 

对于金哲霆而言，在致远学院的大一暑假，是他人生第一次出国。那一年，他去了牛津大学，加入数学和物理学暑期课程。也是在大一，他还加入了人生第一个课题组，跟着导师与师兄师姐从事铁基超导体科研。离开中学并不久的他，从线性代数、凝聚态物理到量子力学，都得加紧自学。“我看导师也是边学习边研究的，所以就拼命锻炼自学能力。”到大四，金哲霆已累计先后参与11个课题组科研。

 

笨鸟要先飞，智鸟更先飞。在实施“项目主任负责制”的致远学院，从事7年物理方向项目主任的郑杭教授，告诉解放日报·上观新闻记者，他主要做两件事：设计课程和邀约名师，一切都是赶早。比如，在其他相关学院，纳米科技课程通常是高年级后半学期开设，而致远学院在大一到大二之间就请来加州理工学院大教授开出16学时专业课；又如，入学第一学期，他们就设计出“专业物理讨论”课程，将21世纪以来的18项诺贝尔物理学家设置为议题，让学生早早掌握世界前沿在哪里，顶尖科学家在干什么……

统计数据证明，致远学子学术志向坚定、学术追求未艾。截至去年，618名毕业生中91%继续攻读研究生，62%选择攻读博士学位，19%的毕业生直接进入“世界大学学术排名”前20的高校深造。迄今，4位毕业生分获北京大学、芝加哥大学、佐治亚理工学院等高校教职，13人继续从事博士后科研工作。基本实现毕业生达到国际一流大学TOP5%的尖子生水平。

 

 

【诺奖得主教物理，图灵奖得主教计算机】

 

学生要拔尖，老师先拔尖。问起致远学生，你们的物理和计算机是谁教的？原来，诺贝尔奖得主、著名理论物理学家Anthony Legget，连续6年为致远学院学生讲授超导体物理，他严谨的治学态度使得学生在耳濡目染中激发自己的学术兴趣和创新潜力；图灵奖得主、世界知名计算机科学家John Hopcroft更是连续9年在学院任教授课，还牵头为学生在康奈尔大学等找到合适导师。作为我国外籍院士，Hopcroft教授说，“坚持让本科生少上一点专业课，让他们把更多的精力用于吸取更广泛的知识，并给他们足够的时间进行自学。”

作为讲授《激光等离子物理》一课的教授，致远学院名誉院长、中科院院士张杰也认为，未来世界的创新人才需要掌握物质科学、生命科学、数据科学、人文科学四大知识结构群的基础知识，致远学院将充分发挥上海交大涵盖理、工、医、经、管、文、法等九大学科门类的办学优势，借鉴世界一流学府“整合科学”的课程经验，将物理、化学、生物、计算机的学科知识进一步整合贯通，为学生从事交叉科学问题研究打下更坚实基础。

 

在致远学院常务副院长章俊良看来，教育部启动“六卓越一拔尖”2.0计划是全面振兴中国本科教育的战略行动，而交大致远正是以包括10名两院院士在内的海内外名师队伍来打造“金课”体系，兑现“不出国门的世界一流本科教育”目标。

 

 

【好奇心驱动自由探索，12个卫星实验室开放】

 

你在网上读过一篇科普文“养乐多里真的有100亿个乳酸菌吗”，那是致远学院生命科学方向学生戈劼妤在微信公众号上原创的。“妈妈从小给我吃得好，我也对食物带着热情和好奇……”这个身材高挑的女生告诉解放日报·上观新闻记者，除了用实验证明乳酸菌的真实含量，她还以科学方法阐释“吃坚果能否代替红肉”“喝什么样的牛奶更健康”等话题。

事实上，戈劼妤最初入院学习的成绩并不理想，甚至担心被学院老师认为“不务正业”，“但他们不仅没有干涉我的兴趣，还鼓励我这样做下去。”在毕业设计时，戈劼妤研究了补充维生素D对中国人群的健康影响，发现按西方人群的剂量摄入维D，是无法满足国人需求的。“科学是没有国界的，但营养学研究却因国家因民族而异，因为不同的人种之间有很大的差异。”目前，她在中国科学院大学上海营养与健康研究院攻读博士学位。“我们还缺乏国外营养学跟踪十几年、几十年之久的队列数据，我的科研路要从中国大地上起步。”

 

为打造学生可以自由探索的创新平台，学院筹建起“致远创新研究中心”，以量子通信与计算、能源材料、生物分子探测、数据科学与技术为核心，汇集12个卫星实验室。今年9月，大一新生将在中心实验室体验首次开放的精选前沿实验：利用光镊来感受光子的力量，操控单光子和量子光源体悟量子纠缠的魅力，操纵原子力显微镜观测亲手搭建的DNA分子机器……
