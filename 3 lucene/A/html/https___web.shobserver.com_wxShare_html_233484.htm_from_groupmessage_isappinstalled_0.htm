上海第三批援鄂医疗队领队陈尔真：“使命完成了，就地免职……”

见识录 2020-04-05 17:11
来源：上观新闻 作者：张驰 司占伟
2020年4月5日，160名上海援鄂医疗队队员结束医学隔离观察，可以回家了。在分批送队员回家的现场，上海第三批援鄂医疗队领队陈尔真十分激动，他对前来接人的同行、朋友说：“人我是一个不少都带回来了……我的使命完成了，就地免职。”

在现场，陈尔真的激动和兴奋溢于言表。

“照片拍完，可以上车走啦！”

“交大附属的同志们，可以上车啦！”

“上车咯！”

“再会”。

“一、二、三……“，交大附属31人，别数错了。

”人都到齐了伐？“

使命完成，陈尔真的心也算落了地。

坐在车头的陈尔真，笑着凯旋。
