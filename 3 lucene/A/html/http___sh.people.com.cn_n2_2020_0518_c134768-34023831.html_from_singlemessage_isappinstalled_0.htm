上海各级领导化身“带货主播” 为这座城市满血复活“冲刺”

葛俊俊 韩庆

2020年05月18日08:00  来源：人民网-上海频道
 

5月17日，五五购物节系列活动之上海消费云峰汇上，上海市委常委、副市长吴清上阵最高级别“直播带货”，为全网推介上海制造硬核科技。实际上，后疫情时代，为拉动经济促增长，上海各区领导纷纷化身“带货主播”，直播逛商圈、探店、推销非遗美食、特色农产品。直播各有千秋，有的当起了“段子手”，有的辣手砍价，有的化身吃播，有的自掏腰包买买买。

据商务委数据统计，5月1日至5月10日，“五五购物节”线上销售快速增长，网络零售额超过400亿元。作为上海首创的大规模消费节庆活动，上海多个区不约而同选择了线上直播引流叠加线下实体消费的模式。其中，“直播带货”逐渐成为一股新潮流。

有网友笑侃：“这都是被领导工作耽误了的主播啊。”没错！为刺激消费提振信心，他们化身“带货达人”，为产品“站台吆喝”，为购物节奋力打CALL，在直播界“一举蹿红”。尽管他们在镜头前还略显拘束，但激增的数据显示他们的直播带货业绩可观。

疫情之后，从“幕后”到“台前”，各级领导“直播带货”，能否激发消费者购物欲，是否也表达了对零售新业态的认可，对在线新经济发展而言，意义几何？就这座城市而言，上海独特的城市精神变得更加直观！

各显神通 区领导直播带货成趋势

5月10日中午12点，青浦区委书记赵惠琴来到设在绿地全球商品贸易港的直播间，像一位女主人，准备好一桌饭菜，向广大消费者推荐青浦优质农产品和贸易港进博同款商品。

“可以先闻一下这饭香，是我们青浦独有的蛙稻米，不撒一滴农药，放养青蛙捉虫子。”“这茭白炒肉丝，一定要尝一尝，这是我们练塘今年的头茬茭白，非常非常嫩......”

青浦区委书记化身带货主播助力五五购物节 来源：绿色青浦

赵惠琴自卖自夸间，还解锁了练塘茭白新吃法，“要想吃到练塘茭白真正的脆嫩，其实最好的吃法是冰镇茭白。将茭白切成片，放在冰块上冰镇，像吃生鱼片一样，蘸少许鲜酱油，味道非常棒！”这招最“懂经”吃法亮相，立马刷新了百万网友对青浦茭白的认知，直播屏幕瞬间被无数“买买买”和“666”刷爆。

宝山区委书记汪泓（左二）直播带货 摄影/胡伟

5月5日晚，宝山区高境镇三邻桥体育文化园的“E购市集”开市，宝山区委书记汪泓走进各品牌的直播间，开启自己的直播带货首秀。与他区领导带货时在镜头前侃侃而谈不同，汪泓每到一个直播间，就拿出手机扫一扫直接“买下”，“今天这个化妆品太划算了，来一盒”“豆制品、虾各来一盒，赚到了”“宣传不如下单”。 而副区长陈尧水的风格则是到现场拿起虾直接品尝，让镜头前的网友垂涎欲滴。

闵行区副区长吴斌直播带货

4月30日，闵行区副区长吴斌率先尝试直播带货。“大家好，我是吴斌，大家可以叫我斌叔”，吴斌开场就和网友拉近距离，当主持人表示：大家都等你好久了。吴斌调侃回应：“我今天也是大姑娘上轿头一回”。他随即喊出带货口号“我有我骄傲，你来你实惠！”“年轻人向往、老年人向往、妇女儿童都向往”……更拿自己的妻子举例，“我家里老是看到我老婆贴个面膜，现在50来岁，看上去年轻好多！”现场金句不断，但更重要的是，吴斌首次试水1分钟卖出1000多单进口面膜。

区领导“直播带货”，为何如此吸睛，消费者会“买账”吗？

“官员本身是一个热点。由政府官员出面推荐产品，是一件新鲜事，无疑更容易获得消费者认可。”上海交通大学国际与公共事务学院教授、中国城市治理研究院研究员、中国公益发展研究院院长徐家良认为原因有两个方面。

一方面，平时很有距离感的领导干部出现在镜头前，所造成的视觉冲击远高于日常进行直播的各类直播，能够吸引有猎奇心理的观众，尝试其推荐商品；另一方面，官员推荐商品，比一般主持推荐可信度可能要高一些，比较可靠，有一种政府为企业背书的意味，无形中显著增强商品的公信力，自然也提升市场认可度，引起观众购买欲。

也有业内人士认为，社会公众尤其是年轻人有了“共克时艰”的内在觉悟，面对领导所推荐的本地特产等商品容易产生心理认同，将自己的消费行为与支持家乡、支持国家相联系，因此商品销量能保持在一个较为理想的水平。

确实如此，那让数据说话。青浦区委书记赵惠琴“尝一尝”鲜嫩的茭白，“品一品”嚼劲十足的蛙稻米， 短短15分钟，直播间就卖出了10万斤大米，1千斤茭白，销售额150万元；虹口区副区长郑宏为上海老品牌“飞跃”站台，直播卖鞋，10分钟，4万销售额，他穿在脚上的那款迅速脱销；金山区副区长张权权直播卖瓜，10分钟，直播间观看人数从2000多增至2.8万余人，共卖出312箱“小皇冠”西瓜；普陀区副书记、区长姜冬冬首次直播带货卖的是汽车，5月1日-5日，5天时间共成交526台，成交金额突破1.8个亿。

直播为产业带来新机遇 互动模式打造有温度城市

区领导“直播带货”的模式，并非上海首创。就全国来看，地级市领导直播带货已经是普遍现象，甚至不乏省级领导出镜推销产品。

“上海区领导身先士卒，打开了销售思路的新方式，也改变了人们对直播购物和电视购物的固有印象。”上海金融与法律研究院院长傅蔚冈认为，与网红IP带货方式的差异在于，区领导直播不仅仅是领导本人简单的出现在直播间，而是他将产品、本地特色等一系列内容结合起来，从而打动消费者。

与全国相比，上海的领导干部除了代言本地土特产农产品之外，还推荐了汽车、化妆品等较为高端的制造业商品，其中不乏进口商品。可谓“五花八门”，各区纷纷重点推出其“拳头产品”。

如本次五五购物节，上海聚焦汽车消费，出台包括沪牌增加投放、发放新能源车充电卡等一系列政策。正是看到这一点，普陀区为区域车企代言，旨在最大限度发挥汽车产业链消费链的带动优势，最大限度促进消费回补和潜能释放；作为进博会溢出效应主平台之一，虹桥进口商品展示交易中心的虹桥品汇是闵行区市委市政府重点打造的进口商品集散高地。闵行区副区长吴斌不停“种草”“安利”，仅仅十分钟带货时间，虹桥品汇2200组面膜库存被抢到只剩下400组；“寰球渔市”是宝山区在拼多多平台海鲜销售量最大的企业，这次“五五购物节”被定为直播带货的首选企业之一，吸引了宝山区党政干部组团“支援”。

“领导带货的首选是当地领先产业，通过直播表明其产业集群在全国具备优势。”傅蔚冈认为，通过产品将产业集群推广出去，将为区域内企业带来新的发展机遇。“领导直播带货是履行当前‘六保’职责要求，具有示范效应，能为区域品牌提供背书。”同济大学经济与管理学院副教授张鑫指出，有利于激活国内消费市场，有利于促进经济内循环。

这也正是上海各区广泛动员与积极参与的动力所在：与时俱进，最大限度促进消费回补和潜力释放，让企业受益得发展，消费者获利得实惠，区级经济撬动得增长，加快推进建设上海国际消费城市重要功能承载区。

“这是一种新型的政社互动形式，甚至可以说是政、商、社三圈互动的新形态。”徐家良指出，政府官员同企业一道，面对面直接向公众推销商品，并通过直播平台进行互动，打破了以往不同圈层的固有界限。这既是一种互动模式的进步，也可能代表着未来的政社互动趋势，即在先进技术的支撑下，不同圈层之间去边界化的交流。对于上海来说，这样的模式将更有利于打造出一座有温度的城市。

“三赢局面”打开，上海经济在全面回暖

如今，人气、商气、官气三气合一，久违的上海经济在全面回暖。

在5月17日举行的五五购物节系列活动之上海消费云峰汇上，上海市委常委、副市长吴清上阵“直播带货”，为全网推介上海制造硬核科技。至此，上海迎来了最高级别的“带货直播”。

吴清介绍联影天眼CT。直播截屏图

10多分钟的时间里，吴清共推荐了三款产品：荣威智能网联汽车、达阀消毒机器人和联影天眼CT。他说：“我的第一个寄语就是买它买它买它，再一个就是我们现在都处在一个新的时代，我们的数字新基建和在线新经济、智能新制造一定会带来我们的智慧新生活。”

在上海各级领导冲锋陷阵，带头示范中，各镇党委书记、镇长也豁了出去。值得一提的是，五五购物节期间，“在局长带你游上海”直播间，上海各区14位文旅局局长陆续走进直播间，展示上海的多样魅力。

“从目前来看，领导干部带货更多是在特殊时期以及电商直播行业发展当前阶段的产物。”徐家良说到。张鑫对此表示认同，他指出重要的是如何把短期的商业模式创新与运用探索形成促进消费增长的长效机制。

“无论哪种促销形式，产品品质依然是核心问题，唯有真正打好品质牌，才是留住消费力的长久之策。”张鑫指出，对上海来说，拉动消费要兼顾短刺激和远谋划，可以探索和建立共享直播带货销售平台，以及线上线下结合的新型销售模式或销售生态新体系。也就是说，上海各级领导带货为小概率事件，接下来能不能刷到直播，全凭缘分，且看且珍惜。

开放，包容，海纳百川是上海的特质。2400万常住人口的广度和深度，使得上海可以接纳更多的新生事物。“傅蔚冈建议，吸引更多的购物达人如更多的“李佳琦们”，在上海创业，为上海产品“代言”。

政府搭台，企业唱戏，吸引全民参与，无疑为上海经济的整体复苏添了一把火、加了一把劲。上海市商务委主任华源表示，从下阶段的重点活动安排来看，在将上海打造成具有国际影响力、国内领先的在线新经济发展高地的展望下，加速推动信息消费相关举措引发关注。

正如上海市经济信息化委员会主任吴金城所说，政府将不断优化营商环境，努力当好金牌“店小二”，细化落实专项支持政策，持续发力、久久为功，为上海产业高质量发展打造新亮点、创造新标杆，形成经济发展新增量。 

(责编：龚莎、韩庆)
