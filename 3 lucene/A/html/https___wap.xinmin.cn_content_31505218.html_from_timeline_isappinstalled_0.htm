这些美图可以“吃”吧？沪上高校推出特色青团，满满的春天味道！飞入寻常百姓家
来源：新民晚报   作者：易蓉 郜阳   2019-03-22 14:57:15

这几日，上海各大高校食堂陆续推出各式各样的青团，柔韧绵软、清香爽口的青团从色彩到口感都散发出春天的气息。除了传统的豆沙、爆款的蛋黄肉松外，不少学校还上线了特色品种：麻香花生碎、黑芝麻核桃、火龙果肉松……

上海交通大学

萃取汁液、揉制面团、配制内馅儿、手工包制、细心烹制……每只青团都蕴含了交大食堂大厨们的心血——让它成为交大学子春日记忆中最好的滋味。

交大食堂今年贴心准备了特色咸蛋黄肉松、秘制麻香花生碎、马兰头香干、荠菜肉丝、养颜红枣葡萄干核桃碎、本帮红豆沙、经典黑洋酥、7种口味的青团。甜品爱好者可以试一试最经典的本帮红豆沙，或者健康养颜的红枣葡萄干核桃仁，人气最高的是特色咸蛋黄肉松，另外还有秘制麻香花生碎、马兰头香干、经典黑洋酥、荠菜肉丝多种口味，可以满足大家“一周青团口味不重复”的愿望。

上海交通大学供图

华东师范大学

一席碧绿，一抔细粉，徐徐注到一起，由一双干净有力的大手和匀压实，麦青汁与糯米粉重新揉合，在桌子上铺张地展示新绿。眨眼间，一块亮眼的绿，已经被捏成一个个小团儿。旁边小小的秤，展示着它精确的“体重”：三两。这个春天，华东师大两个校区推出了红豆沙、绿豆沙、荠菜肉、紫薯、南瓜、蛋黄肉松、黑芝麻核桃、蟹黄鲜肉等多种口味的青团。一口青团入胃，慢尝春光好滋味，若是待到清明过，只得等来年再会。学校后勤保障部说，趁着好时节，捧颗香香软软的青团，华东师大就让你吃遍春天！

华东师范大学供图

上海外国语大学

上外的青团都来自食堂叔叔阿姨们的用心制作。每道工序都认真落实，让学校师生放心食用。今年上外在原有豆沙、蛋黄肉松、萝卜鲜肉、荠菜鲜肉馅的基础上，新推出腌笃鲜、芝麻核桃、紫薯、花生和板栗5种口味，青团的价格也都很亲民。上外食堂对师生们发出邀请：欢迎品尝舌尖上的春天！

上海外国语大学供图

华东政法大学

今年华政青团多达9种口味！青团子分别有：清甜红豆搭配清香艾叶的豆沙馅、自带网红体制、咸香怡人的咸蛋黄肉松馅、把春天的大地包进口中的荠菜鲜肉馅、满嘴鲜烫荡漾舌尖的鲜肉馅。此外还有“五彩斑斓”款，比如南瓜皮豆沙款、紫薯五仁款、火龙果肉松款、胡萝卜伍仁款，简直是“彩虹”团。

华东政法大学供图

上海海洋大学

有人说，“青团的味道就像春天，不甜不腻，却有着清淡又悠长的香气。”上海海洋大学推出了咸蛋黄肉松青团、蛋黄豆沙青团、枣泥青团、玫瑰豆沙青团、马兰头香干青团、紫薯味青团六种口味，前两种单价4元，后四种单价2.5元。有同学点评玫瑰豆沙青团说，软糯、香甜的豆沙馅，外表散发着巧克力色的光泽，嚼上一口，豆沙甜蜜的味道与沙绵的口感，不会过分甜腻，反而有种独特的清新感。

上海海洋大学供图

上海海事大学

上海海事大学在学校多个食堂为师生们准备了多种口味的青团。喜欢甜的，可以花上2.5元来一个豆沙或黑洋酥馅儿的；喜欢咸的，则可以品尝5元一只的马兰头香干或咸蛋黄肉松馅儿。

上海海事大学供图

上海理工大学

“小青”以传统的艾草入皮，带有清淡却悠长的青草香气，配合网红咸蛋黄肉松内馅，油绿如玉，咬起来幸福感满满。红光满面的是胡萝卜，配合爽口开胃的香干马兰头，健康又好吃！紫气东来的是紫薯皮，紫薯可是减肥利器，所以吃再多青团都不会胖的！配合补血养颜的枣泥馅，甜而不腻，越吃越想吃！金光闪闪的是南瓜，甜甜的味道让人欲罢不能，内馅是可以生发的黑芝麻，做题做到秃头的还在犹豫什么？

上海理工大学供图

上海工程技术大学

万物生长的春天，总给人一种清新扑鼻的感觉。若是飘来一股艾草香，是否能将你舌尖上的春天也唤醒？伴着糯米的软糯、蛋黄的咸香、豆沙的绵密、肉松的甜厚、黑芝麻的香浓，今年的工程大青团如期而至，3月20日起，该校推出蛋黄、肉松、豆沙、黑芝麻四种口味的青团。学校还贴心地提醒师生，制作青团的主要材料是糯米，糯米性温滞气，含植物纤维既多又长，吃多了会加重肠胃负担。尤其是胃病和肠道疾病患者不可多吃。

上海工程技术大学供图

上海立信会计金融学院

上海立信金融会计学院推出了七大“网红”口味青团，包括豆沙、黑芝麻、咸蛋黄肉松、红枣、板栗以及香芋口味。学校今年的青团十分考究，颇具特色。例如今年的咸蛋黄肉松改用炒制制作，提香同时增添了松软口感；今年的青团还分为艾草版本和麦青汁版本，前者香味绵长，后者软糯爽口；今年的青团一个约为80-90g，初期试售以传统圆团形状为主，后期还可能加入新的口味和造型。

上海立信会计金融学院供图

上海健康医学院

除了青团，春天的时令“青”之美食还有哪些？上海健康医学院官微特别普及了青团的制作方式，科普了清明节日、春耕节气的科普知识，同时还分享了各地清明时令美食。 比如，广东潮汕地区的鼠曲青粿，用一种鼠曲草和糯米粉制作，包裹花生、冬瓜干、芝麻等馅料，十分美味；江西地区的清明果，形状如饺子或者元宝，外皮用鼠曲草或艾草制作，内以腊肠丁、笋丁、香菇丁等咸味馅料填充，别有一番风味。学校也推出了豆沙、芝麻、紫薯和咸蛋肉松口味的青团，在3个食堂向师生们推出。

上海健康医学院供图

上海电机学院

这个春天，上海电机学院推出了三种口味的青团。荠菜竹笋肉青团单价4元、蛋黄肉松青团单价4元，而红豆沙青团单价2.5元。校方介绍，每个青团剂子都在70克至75克之间，而细腻的揉搓，是做青团最关键的环节。

上海电机学院供图

上海建桥学院

自3月14日起，上海建桥学院食堂早中晚三餐供应莲蓉、豆沙、荠菜鲜肉和蛋黄肉松四种口味的青团。价格也十分亲民，前两种零售价3元/只，后两种4元/只。该校食堂师傅介绍，首先要将艾草汁拌入糯米粉，反复揉搓直至成团，做成剂子；随后搓圆剂子，在面团中间按坑，填入馅料，封口；再沾少许食用油后放蒸锅蒸15分钟左右即可。

上海建桥学院供图

上海海关学院

在上海海关学院，青团不限于“青”色，创意无限，师生也很有口福。今年，海关学院在沿用传统制作技艺的基础上融入创新元素，除了推出传统红豆沙青团，还精心制作、推出了火龙果流沙奶黄、南瓜荠菜鲜肉、紫薯栗子奶油口味的青团，不仅色彩鲜艳，口感也相当丰富，还未推出就受到了师生们的热烈期待和追捧。

创意使“青团”色彩缤纷，例如火龙果流沙奶黄是根据粤式点心流沙包演变而来，将火龙果榨汁，掺入糯米粉中，制成胚皮，再用咸蛋黄、黄油等食材做成流沙奶黄馅，最终二者结合，水果的芳香及细腻的口感在口腔中流淌。

海关学院以青团载情、寄思，预热了节日的氛围，带来了春天的味道，传递浓浓情谊，也让师生们在品尝民族特色食品的同时，感悟和传承中国传统文化。
