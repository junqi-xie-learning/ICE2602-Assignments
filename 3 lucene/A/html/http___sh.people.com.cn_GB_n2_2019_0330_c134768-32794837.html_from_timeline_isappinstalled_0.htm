产学跨界，上海交大首个文创产教基金签约

2019年03月30日23:44  来源：人民网-上海频道
 

人民网上海3月30日电 （记者 姜泓冰） 上海交大探索文创教育新模式，首个面向文创人才设立的产教链通“创基金”今天签约启动。

上海市政协副主席、上海交通大学副校长黄震，上海市教委综改办文教办副秘书长林炊利，上海德必文创产业发展集团股份有限公司创始人、董事长贾波，上海文化广播影视集团有限公司监事长、上海市文联副主席滕俊杰等产业导师代表，上海交大文创学院党总支书记兼院长张伟民及学院部分学术导师、外籍导师等出席了今天下午在外滩举办的签约活动。

产教链通“创基金”由上海交大文创学院首批产业导师贾波发起，旨在以创新解决实践中的真实问题为导向，通过“文创+X”的跨学科合作，“学分+基金”的产教融合激励，以及“校区+园区”的协同工作模式，打通文创产业和文创教育的链接，激发文创领域的创新思维、创意能力和创业精神。

据文创学院院长张伟民介绍，今后，上海德必及其产业园区内企业将根据实际需求，每年发出研究或研发项目5-8个，由文创学生自主组建“文创+X”的跨学科、跨产学团队进行项目申报，在规定时间内完成后，由项目评审小组确定基金奖励。文创学院则将项目纳入《文创产业调研》课程体系，以“学分+资金”的方式鼓励学生参与。此外，在文创学院教研大楼和德必园区都分别设立办公空间，供入选项目同学使用，形成“校区+园区”联动的方式。

上海交大文创学院由上海交通大学与美国南加州大学共同组建，2015年4月成立，以“国际化、学科交叉、产业导向”为培养特色，已聘请50余位产业界创始人或高管担任产业导师,在上海国际电影节、上海设计周、长三角文博会等参与主办官方论坛，每年还邀请著名文创领域教授面向上海文创行业人才进行专业集训，每年覆盖近2000人次。

(责编：邬迪、韩庆)
