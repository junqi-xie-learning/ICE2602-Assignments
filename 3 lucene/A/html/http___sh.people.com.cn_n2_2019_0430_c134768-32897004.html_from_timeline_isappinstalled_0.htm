一扇“睦邻门” 打开社会治理新格局

2019年04月30日13:44  来源：人民网-上海频道
 

党的十九大报告中对新时代社会治理提出新的要求，明确“打造共建共治共享的社会治理格局”，实现“社会充满活力又和谐有序”的目标愿景。“共建共治共享” 已然成为了社会治理创新的重要指导原则和改革方向。

社会各界对于“共建共治共享”的丰富内涵进行了广泛其而深入的讨论，包括价值层面与实践层面、社会层面与个体层面等等。这里的一个关键问题是，“共建共治共享”社会格局的构建前提必须解决“共”的问题，即打破社会隔阂、推动社会融合、实现广泛参与。

上海市杨浦区的一扇“门”的开启，深刻诠释了“共”的内涵与实现过程。这同时也是一个关于不同阶层打破隔阂、和谐共处的故事，故事中涉及到了两个小区，分别是五角场街道创智坊小区和国定路第一社区。创智坊小区是2006年建成的高档商品房社区，小区环境整洁优美，居民中年轻白领居多，基础设施齐全，毗邻五角场繁华商圈，区位优势明显。国定路第一社区是80年底建成的老旧小区，小区基础服务设施陈旧老化，大量外来人口聚集，居民组成较为复杂。

一墙之隔的两个小区

两个小区各为封闭型小区，同时由于社会阶层差异明显，小区之间虽一墙之隔，但是基本不相往来。隔绝状态也给两个小区居民分别带来带来了不便。

由于基础设施完善以及区位优势明显，创智坊小区成为杨浦区“睦邻中心”的建设小区，创智农园、创智天地园区等娱乐休闲场所不断建设完成，特别是集学习、娱乐、休闲于一体“睦邻中心”服务项目不断推出，但是这一切对于国定路第一社区居民而言是“可望而不可及”。另一方面对于创智坊小区居民而言，两个小区的隔绝导致了其买菜、出行、接送小孩等被迫绕远。所以打破两个小区的隔绝状态也是民众的共同愿望。

2017年以来，街道联合社会组织与2个居民区党组织共同组成社区议事会，讨论提出未来打开围墙的可能性，并组织社区小朋友一起在阻隔2个小区的围墙上手绘了缤纷魔法门，表达两个小区打破隔绝状态的愿望，这也进一步激发了两边居民“开门”的意愿。

小朋友手绘的缤纷魔法门

经反复沟通协商、双方居民区党组织围绕打通围墙议题，牵头召开由社区规划师、业委会、物业公司、社会组织、居民群众等社区多元主体共同参与的“百脑汇”议事堂。最终达成了由国定一社区实施破墙开门的决议，确定了围墙破“门”的开门地点、开放时间、物业管理、周边保洁、安防设施安装、党员志愿者巡查安排等细节。这一打破隔阂的“门”被命名为“睦邻门”。2019年3月28日，“睦邻门”正式开门通行。

“睦邻门”的开启

“睦邻门”的打开非但没有给居民带来预想的困扰，还大大方便了居民的生活，提高了生活品质，居民获得感大大提升。目前“睦邻门”的开放时间处于试运行阶段，等各项服务和保障措施，开放时间有望对标地铁运行时间，切实方便民众生活。

“睦邻门”开启不仅仅是空间“围墙”的打破，更重要的是“心墙”的打破，目前两个小区的“共建共治共享”格局构建正在不断升级。

（作者：王奎明 上海交通大学中国城市治理研究院助理研究员）

(责编：严远、轩召强)
