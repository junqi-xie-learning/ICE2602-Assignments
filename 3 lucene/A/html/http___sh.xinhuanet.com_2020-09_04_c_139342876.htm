上海交大“智慧迎新”：线上报到+直播教学

2020年09月04日 19:31:53 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

    新华网上海9月4日电（吴一航）9月4日，在上海交通大学闵行校区，前来报到的新生没有像往常那样排着长队等候登记信息，而是纷纷拿出手机通过学校开发的“交我办”App完成个人信息“线上核验”，动作快的同学完成整套操作甚至只需十几秒。

    新生“线上核验”是上海交通大学疫情背景下全方位推进智慧校园建设，创新运用“人脸识别”与“活体验证”技术进行“智慧迎新”的新举措。这套系统除能核验学生身份信息，学校教务、财务、宿管等多部门多项新生业务均能实现线上办理，新生在入校前即可完成信息查询、缴费、保险等事项，无需线下排队办理。

    充分考虑因疫情防控暂时无法返校学生学习需求，上海交大采用“教室教学+在线直播”模式，系统不仅能支持校内校外4000人同时在线观看，而且还能直播回放，是在原有线上教学基础上进行的新拓展。

    今年3月份，上海交大联合清华大学、浙江大学等国内6所高校发起了“春霖助学行动”，组织在校大学生利用在家的空余时间，给国家贫困地区专项招生计划实施区域的高三学生进行“在线教学”，帮助高三学子梳理学科知识点、错题讲解等。高考放榜后，“春霖助学行动”中有来自15个省份的51名同学成功收到上海交通大学录取通知书。

    开学之际，上海交通大学还积极搭建了多个智慧校园应用场景。校内三大公共交通动态信息，已实现“指尖查询”；各食堂就餐人数和分布情况，通过“就餐指数”实时反映；无人车派送、手机扫码秒取快递等无接触方式也为广大师生带来了安全、放心的快递服务。
