耶鲁在本土之外首个实体研究院在沪开建，中美名校联合攻关癌症免疫治疗等领域|附视频

教育在线 2019-03-23 13:50
来源：上观新闻 作者：徐瑞哲
免疫学被耶鲁大学科学战略委员会规划为耶鲁下一个十年需要优先发展的五大方向之一，并推荐建立研究院（Institute）。

中美科研，在沪牵手。作为耶鲁大学在美国本土以外建立的第一个实体研究院，上海交通大学医学院—耶鲁大学免疫代谢研究院23日在中国上海启动建设。新研究院的主攻方向之一，正是2018年诺贝尔生理学或医学奖授予的癌症免疫治疗领域。

 

 

免疫系统是人体最重要的防御系统，其调控与肿瘤、感染、过敏等多种疾病密切相关。基于免疫学的疫苗和抗体，消灭了天花，控制了脊髓灰质炎，治愈了白血病和淋巴瘤……仅2017年，全球免疫相关类药物的销售总额达845亿美元，在全球医药市场占据绝对优势地位。当今，免疫学和代谢学的交叉产生了免疫代谢这一新兴前沿领域，为有效控制危害人类健康的重大慢病，如糖尿病、癌症、心血管疾病、衰老等，开辟了一条新的预防和治疗途径。

 

 

早在1810年成立的耶鲁大学医学院已走过200多年历史，而“耶鲁大学与中国的交往历史长达180多年，”耶鲁校长苏必德（Peter Salovey）至今记得，“我对5年前在上海交通大学被授予荣誉学位的经历依然记忆犹新。”此访期间，从耶鲁大学免疫学系归国并担任我国首个免疫学研究机构——上海市免疫学研究所所长的苏冰博士，向苏必德校长提议在两校之间建立另一座合作的桥梁，那就是开展免疫学的联合研究项目。“今天我更要感谢上海市政府的支持促成了此次合作。”苏必德通过视频对中国同行说，此次负责全球战略事务的副校长伯里克利·刘易斯（Pericles Lewis）教授来沪启动研究院建设。

 

 

中国科学院院士、上海交通大学医学院院长陈国强向解放日报·上观新闻记者透露，苏必德在此前向他发出的电子邮件中提及，拓展与加深与中国上海交大医学院的合作是自己在任期间最优先级的议程之一。陈国强则表示，新的免疫代谢研究院完全看齐耶鲁大学人才标准，在全球范围吸收最优秀研究者来到上海科创中心，在人才培养上也是最好导师带教最好学生。

 

 

在耶鲁大学医学院，各类教职员工达3007人，占耶鲁大学教职工总数约2/3。其中美国国家科学院院士63人，美国国家医学院院士47人。据统计，耶鲁医学院活跃经费总额超过6亿美元，活跃专利有898项，共建立了77家生物技术公司。而此次强强联合的主体之一，是耶鲁医学院的旗舰学科——免疫生物学系，全系拥有5名美国国家科学院/医学院院士，5名霍华德•休斯学者，多项学术指标排名世界或美国第一。2016年，免疫学被耶鲁大学科学战略委员会规划为耶鲁下一个十年需要优先发展的五大方向之一，并推荐建立研究院（Institute）。

 

上海交大医学院党委书记范先群、耶鲁副校长Pericles Lewis、耶鲁免疫系系主任David Schatz、耶鲁终身教授Richard Flavell、上海交大基础医学院院长程金科、上海市免疫学研究所所长苏冰等共同启动研究院建设。

 

作为免疫生物学创始系主任，美国国家科学院院士及医学院院士、英国皇家学院院士理查德·弗拉维尔（Richard Flavell），在对解放日报·上观新闻记者的表述中，使用最频繁的词语就是“Best”。他也说，通过3到5年的努力将中美名校共建的研究院建成具有世界影响力的科研机构，成为上海融入全球创新网络的重要纽带，“让世界上最好的科研人才，来到世界上最好的科研环境，成就世界上最好的免疫代谢科研机构。”目前，他本人和苏冰教授担任研究院的共同院长。

 

 

据悉，上海交通大学医学院—耶鲁大学免疫代谢研究院一期设在上海交大医学院老校区，有望在浦东新校区进行后续建设，将对接张江实验室的战略部署，构建“基础-临床-产业”全链条创新研究集群，为上海市生物医药产业集群发展提供推力。陈国强院士表示，新研究院还将根据近日“科改25条”《关于进一步深化科技体制机制改革 增强科技创新中心策源能力的意见》，进一步扩大高校院所在科研活动中的自主权。
