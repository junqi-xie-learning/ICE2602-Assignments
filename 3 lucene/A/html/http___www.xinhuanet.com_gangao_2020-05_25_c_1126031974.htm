董建华：涉港国安立法只针对极少数犯下严重罪行者
2020-05-25 23:43:47 来源： 新华网
关注新华网
微信

评论

　　新华社香港5月25日电  全国政协副主席董建华25日通过网上直播表示，全国人大进行涉港国安立法是为了保护国家安全、香港社会安宁和市民生活安宁。香港市民可以对涉港国安立法放心，相关法律针对的只是极少数犯下严重罪行的人。

　　董建华指出，有些人散播谣言，声称一旦完成涉港国安立法，香港就会“失去民主自由”“失去国际金融中心地位”，甚至声称“‘一国两制’已死”。这些都是毫无事实根据的恶言恶语，目的就是企图制造恐慌。事实上，中央已经指明，建立健全香港特别行政区维护国家安全的法律制度和执行机制工作的重要原则包括：坚持和完善“一国两制”制度体系、切实保障香港居民合法权益等。

　　董建华强调，回归20多年来，香港原有的资本主义制度、生活方式一直保持不变，香港人享有的自由权利比历史上任何时候都要多。不同民调都显示，“一国两制”受到香港社会不同阶层、不同政见人士的广泛支持。涉港国安立法是中央经过深思熟虑的慎重决定，绝不会令香港市民目前享有且受到香港基本法充分保障的言论、新闻等自由受影响。

　　董建华指出，涉港国安立法是为了保护国家安全。自去年“修例风波”开始，有些人公然鼓吹“港独”“自决”等主张，甚至勾结外部势力，乞求外国干预香港事务。更令人忧虑的是，一些西方反华势力不惜混淆是非黑白，公然为反中乱港分子撑腰，他们真正的企图只有一个，就是要遏制中国和平发展。大家不能再容许境外势力与香港反中乱港势力串联，危害国家主权、挑战中央权力和香港基本法权威。香港特别行政区有宪制责任维护国家安全。

　　董建华表示，涉港国安立法是为了保护香港社会安宁。香港回归20多年来，香港基本法第23条的立法工作一直未能完成，使香港成为国家安全的风险口，为敌对势力提供了可乘之机，扰乱香港公共秩序，危害社会经济利益！尤其是无休止的示威抗议行动，导致社会秩序混乱，市道变得萧条，加上新冠肺炎疫情影响，香港经济元气大伤。在这一背景之下，反对派政客仍然继续纵容甚至鼓吹违法行为，更威胁特区政府，若是不答应他们的诉求，便会否决特区政府提出的所有议案和拨款申请。大家是否愿意见到议会不能正常运作、政府无法有效管治、社会变得乱七八糟，任由多代香港人努力不懈建设得来的成就到今天付诸流水呢？

　　董建华指出，涉港国安立法是为了保护市民生活安宁。过去一年，“修例风波”中出现一连串暴力违法行为，令香港市民持续生活在惶恐、惧怕、沮丧之中。例如，每逢周末，外出的市民都会特别留意交通情况，要知道哪里有堵路、哪里被纵火。就连在公开场合说话也要小心翼翼。相信这些日子香港人都受够了！当前，香港社会需要的是同心同德，而不是散播恐惧。涉港国安立法能够有效防范、制止、惩治各种扰乱香港的活动，有助恢复香港社会秩序。只有国泰民安、社会和谐稳定，大家才能安居乐业！

　　董建华指出，维护国家安全，不单是特区政府必须履行的宪制责任，亦是整个社会需要思考如何承担责任，如何配合支持，如何推动宣传的工作。他建议，必须发声支持全国人大涉港国安立法的建议；必须积极推广国家安全教育；必须对“一国两制”充满信心。他强调，国家重视香港，珍惜香港。现在是香港人展示责任担当，坚守“一国两制”，维护国家主权，为香港稳定繁荣作出贡献的关键时刻。相信所有热爱香港的市民一定会义不容辞，不负所托！
