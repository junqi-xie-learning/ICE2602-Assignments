乡村振兴的一张“名片”
——金山区漕泾镇试行“埭长制”建“美丽宅基”
2020年03月03日  作者:汤妙兴   编辑:chunchun   审核:刘纯  版面:A2

　　金山区漕泾镇从去年下半年起，在全区率先开展以乡村振兴为目标的“美丽宅基”创建工作，全镇5300多户村民纷纷行动起来，至今已有2386户农户通过镇级“美丽宅基”验收，挂上了“美丽宅基”牌子，三年创建目标有望提前完成。在近日漕泾镇召开的“美丽宅基”创建工作表彰会上，护塘村沈瑞忠等一批优秀“埭长”、优秀创建户和优秀创建工作者得到表彰。

　　漕泾镇在总结“五违四必”河道整治推行“河长制”取得成功经验的基础上，探索试行“埭长制”，至今已成为助推“美丽宅基”创建活动的有效抓手。“埭长”由村干部、村民小组长和骨干党员或有名望的村民代表组成，一般少则十几户，多则二三十户为一个埭。“埭长”按照“五清四美”的创建目标，结合实际情况将创建任务分解到户，并加以指导和监督。海涯村一号“埭长”方明康干一行爱一行，这次村里推荐他为“埭长”，负责全埭34户村民家庭创建的宣传动员和指导督促工作。接到任务后，方明康不顾自己腿脚残疾带来的不便，首先对自己家宅前宅后进行整理。然后他拄着拐杖，一户一户上门进行宣传动员。村民们看到埭长已先行一步，又苦口婆心地做工作，都很快行动起来。有2户人家缺劳力行动落在后头，方明康动员宅基头上村民开展相互帮助，村环卫人员及时到宅基收取整理出来的垃圾。不到2个月，全埭34户村民户户挂上了“美丽宅基”牌子，成为全村创建的“领头雁”。

　　由于“埭长”都由土生土长骨干党员或村民担任，他们对每个户每个宅基的环境情况都了如指掌，甚至对宅基头人员的生活习惯也清清楚楚，因此实行“埭长制”有效提升了创建质效水平。创建半年来，“埭长”们最常用的推进措施，就是“抓两头促中间”，挑选自己熟悉的上进性强的村民先行一步，树立创建样子，然后对创建困难户进行重点帮扶。这样，中间一部分就带起来了。

　　由于“埭长”们对环境整治力气用在刀口上，半年来拆除库外违建5万多平方米，拆除旱厕、地间窝棚等1800处，清理宅前屋后河道200条，新添绿化面积6万平方米，清运垃圾2万余吨，有效推进“五清”“四美”“治三乱”目标的实现，为建设美丽新农村打下了扎实基础。
