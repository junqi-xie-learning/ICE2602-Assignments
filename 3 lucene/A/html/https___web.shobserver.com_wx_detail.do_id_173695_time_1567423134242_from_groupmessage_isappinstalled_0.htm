22万余人登上中国城市之巅，加持千里眼、顺风耳，国内首个全5G高空VR展开拍续集

教育在线 2019-09-02 19:17
来源：上观新闻 作者：徐瑞哲
结果发现，63%的参展者选择“5G带来的未来生活”是最感兴趣的展出内容，但也有几乎相同比例的公众认为自己对5G和XR相关知识了解程度一般。

仅一个月，就有超过22万名市民游客登上中国城市之巅、全国第一高楼——上海中心119层观光厅，加持千里眼、顺风耳等VR、AR、5G装备，体验国内首个全5G环境下的高空科创展。

在上海中心大厦观光层率先完成5G部署试点后，加载的最大应用就是“城市之巅 5G+XR创意科技展”。而2日，这个产学研合作展会又开拍“续集”，启动“心中上海 虚拟现实（XR）创意大赛”。不论来自大学生还是行业人士的获奖成果，都同样有机会在上海中心后续展会上落地。

来自上海交大，上海中心和VR科技企业的代表一同为创意大赛揭幕。

超高速电梯直达600多米“海拔高度”，一出电梯只见智能机器人迎面而来，用歌声迎宾观展。作为互联网原住民，小朋友们在VR座舱里环顾四周进入恐龙世界，戴上增强现实的AR眼镜挥动手柄通关游戏……

种种应用场景，都是超高速5G与智能可穿戴设备带来的虚拟现实场景。通过三大运营商联合部署，连接4G和5G信号的两部手机在现场接受测速，实时数据流量分别为：4G的下载速度22.2Mb、上传速度24.0Mb；而5G的下载速度达740Mb、上传速度达115.85Mb。作为全国最大的VR内容创作社区，内容提供商借此“高速路”带来了覆盖科幻、国漫、艺术、舞台表演等多个类别的20多个VR体验项目。

那么，市民游客对高新技术“点映”的最大感受是什么？在20多万观展人中，上海交大、上海中心、上海市多媒体行业协会等主办方抽样约1500份问卷展开调研，跟踪公众对文化科技融合创新的体验反馈。结果发现，63%的参展者选择“5G带来的未来生活”是最感兴趣的展出内容，但也有几乎相同比例的公众认为自己对5G和XR相关知识了解程度一般。

嘉宾在创意科技展的现场体验VR科技。

解放日报·上观新闻记者了解到，调研显示，公众最感兴趣的XR内容依次是科幻、教育和艺术。文旅场景方面，公众还希望能够体验风光类的作品。值得注意的是，当问及参展者不喜欢XR体验的原因是什么，“内容不够丰富”的选择也高达60%以上。

对此，作为总策展人的上海交大文创学院全球文创合作与教育发展中心主任王昊青表示，一方面，公众渴望有质量、有价值的虚拟现实丰富内容；另一方面，装备创新决定了公众对内容的接受方式和程度，“装备和内容几乎同等重要，甚至设备或装备的舒适性和酷感更能决定其接受度。”

为此，身为上海市教委2019上海文化创意产教融合引领项目“虚拟现实影视创制与装备产业国际人才培养”总负责人，文创学院院长张伟民透露，学院和联合主办单位一起，继续推出以“心中上海”为主题的虚拟现实（XR）创意大赛，面向全球大学生和虚拟现实内容创作者，鼓励参赛者思考5G环境下的城市创新需求，在包括但不限于文化、商务、教育、艺术、娱乐、传播、公益等各个领域，展开创意与创想竞赛。

在赛制上，大学生与产业界不分组别，都通过比赛选拔、作品孵化、商业路演、展览展示等环节，登上这个从内容创意到商业应用的一体化实践平台。上海中心方面也期待为有志于投入5G时代虚拟现实内容创制与装备创新的选手提供机会和舞台，在大赛期间与创作者或机构一起转化创意，使其成为5G+XR 文旅地标的全新体验项目。
