临港将再现一片人才高地！上海交大中英国际低碳学院今入驻，开启“双学位、双导师”联合培养
2018-09-15 13:55:57 作者：樊丽萍

全球气候变暖的大背景下，发展低碳能源，建设低碳城市，减少温室气体排放，使得低碳产业越来越受到关注。

今天，国内首家低碳学院——上海交通大学中英国际低碳学院正式入驻临港。基于上海交大在能源、环境、金融、管理等领域先进的研究基础，结合临港地区“全球低碳示范地标”的建设愿景和产业优势，同时借鉴英国著名学府在低碳技术和人才培养方面的成熟经验，可以预期，这所学院的入驻，将为临港地区的城市建设、产业发展和人才集聚提供新的助力。

中英国际低碳学院于2017年5月在上海交大成立，英国爱丁堡大学、伦敦大学学院等英国高校参与合作，是在上海市大力支持下创办的低碳技术和碳资源管理领域国际化办学试验区。学院致力于培养低碳领域的复合型、国际化顶尖创新人才，组织前瞻性、系统性的低碳产业高端培训，开展前沿性、创新性的科学研究，推动低碳技术的成果孵化，实现产学研一体化协同发展，打造低碳领域国家级双创示范基地，为推进上海乃至全球低碳产业发展提供强有力支撑。目前，学院已开设环境工程（低碳环境方向）与动力工程（低碳能源方向）两个专业，并将陆续开设碳金融与碳管理，低碳制造与低碳城市等专业方向。

经过一年多的紧密筹备，而今，正式入驻临港后的中英国际低碳学院，坐落于临港新城主城区的银莲路3号。学院占地29000多平米，紧邻上海海洋大学与上海电机学院。经改造，学院已拥有设施完善、功能齐备的主楼和实验楼各一栋，总建筑面积达20000多平米。值得一提的是，改造过程中，除保持原有的欧式建筑风貌，学院还在室内设计方面尽可能地融入了建筑利旧、可变空间、绿色材料等设计理念。在建设过程中，学院通过大量保留原有建筑墙体分割、设计上进行空间复合化利用，以减少建筑垃圾的产生和提高空间利用率。在材料应用方面，生态石材、环保地毯、环保地胶的应用也符合一所“低碳”学院的环保期许。

相关负责人表示，学院还在与临港相关主管部门研究制定全新“低碳示范楼”的建设规划。“低碳示范楼”将集中展现和使用目前最先进低碳技术和低碳理念，服务于研究中心的成果转换、低碳技术的产业应用、低碳文化的传播展示。

除提供一流的硬件设施，未来，上海交大也将创新人才培养模式应用于中英国际低碳学院。据了解，通过与爱丁堡大学、伦敦大学学院等英国知名院校的合作，该学院正在打造“双学位、双导师”的创新联合培养模式。学院硕士学制两年半，学生第一学年在上海交大学习，第二年通过选拔后可赴爱丁堡大学学习，最后半年在上海交大完成论文撰写。学生在规定期限内完成中英双方研究生课程学分，完成硕士学位论文并通过答辩，将分别获得上海交大和爱丁堡大学的硕士研究生毕业证书，同时分别授予硕士学位。同时，学院还推行“学术导师+产业导师”的双导师制。产业导师通过对学生提供职业指导、课题参与、企业参访、讲座论坛等方式，帮助学生积累职业经验、熟悉产业结构、提高就业竞争力。另外，上海交大学生创新中心将设立临港分中心，为学院学生在创新创业方面提供咨询与指导。
