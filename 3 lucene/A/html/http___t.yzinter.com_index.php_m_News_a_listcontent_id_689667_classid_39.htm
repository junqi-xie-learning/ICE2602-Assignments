原创|上海交大首个文创产教基金签约，产学跨界共育未来文创人

2019-03-31 18:55:36

扬子晚报网3月31日讯（记者 罗晓娜 文/摄）在承载时代变迁和城市更新的百年外滩，3月30日下午，上海交大首个面向文创人才设立的产教链通“创基金”正式签约启动。这是上海交大首个面向文创领域设立的基金项目，是上海文创教育的又一创新探索。

产教链通“创基金”由上海交大文创学院首批产业导师贾波发起，旨在以创新解决实践中的真实问题为导向，通过“文创+X”的跨学科合作，“学分+基金”的产教融合激励，以及“校区+园区”的协同工作模式，打通文创产业和文创教育的链接，激发文创领域的创新思维、创意能力和创业精神。

据文创学院院长张伟民介绍，上海德必及其产业园区内的企业将根据实际需求，每年发出研究或研发项目5-8个，由文创学生自主组建“文创+X”的跨学科、跨产学团队进行项目申报，在规定时间内完成后，由项目评审小组确定基金奖励。同时，文创学院将项目纳入《文创产业调研》课程体系，以“学分+资金”的方式鼓励学生参与。此外，在文创学院教研大楼和德必园区都分别设立办公空间，供入选项目同学使用，形成“校区+园区”联动的方式。

上海交大文创学院由上海交通大学与美国南加州大学共同组建，自2015年4月成立以来，根据产业价值链和市场需求整合全球优质资源，以“国际化、学科交叉、产业导向”为培养特色，以培育具有“创意、创新和创业”精神的文化创意产业领袖人才为使命，为有志青年学子实现梦想提供全球一流的教育资源。编辑：李燕
