圆叶葡萄跨洋引种成功，不属“真葡萄”但独具抗癌物，还酿成美酒开我国南方葡萄酒先河

知道吗，你能在国内市面上吃到的葡萄，几乎都是“真葡萄”，但葡萄世界实际上还有一大家族。当然，并不是“假葡萄”，而是圆叶葡萄。

14日上午，解放日报·上观新闻记者走进上海交通大学闵行校区农业与生物学院，在大楼对面的篱笆后，有一片20多亩的“神秘园”——上海交大葡萄遗传与资源育种圃。在西区，保存着葡萄核心种质资源200多份，东区则是新品种示范园，其中包括今年进入盛果期的圆叶葡萄。

记者发现，圆叶葡萄与常见葡萄相比，葡萄结串分散但分布多，单串不大而单个大。更重要的区别，正由30位葡萄酒业品鉴专家以及普通消费者进行品鉴试验……


【极易有机种植，单粒均重10克】

“神秘园”里，葡萄架下，绿色的“卡洛斯”，棕色的“格威尔”，红色的“诺贝尔”，紫色的“阿拉紫”，还有像乒乓球一样大的“南洋1号”，十多个品种都是圆叶葡萄……

原来，葡萄分为真葡萄属和圆叶葡萄属两大类。国家葡萄产业技术体系岗位科学家、上海交通大学农生学院葡萄与葡萄酒研究中心主任卢江教授介绍，真葡萄包括欧亚葡萄种、北美葡萄种群和原产我国的东亚葡萄种群；圆叶葡萄则起源于美国东南部，在美国南方得到商业化种植推广，甚至可以自主长到马路边，成为引起葡萄业界瞩目的新品类。

“这里引种的各种葡萄都属于圆叶葡萄，见到成熟的，可以直接采食，绝对没打过药。”卢江颇有信心地告诉解放日报·上观新闻记者，圆叶葡萄是世界公认抗病性最强的葡萄品种，极易进行绿色种植，生产有机葡萄。同时，露地栽培传统鲜食葡萄，1个人只能管几亩地，而种圆叶葡萄1个人可以管40到50亩，也易于田间管理。

在曾被认为是“葡萄禁区”的我国南方湿热地区，在基本上不用农药的情况下，它们还能丰产丰收。事实上，圆叶葡萄在我国南方各省区示范推广面积已达5000余亩，在上海崇明、金山、奉贤、闵行等地已建生产基地。

目前，卢江50多人师生团队拥有试验挂果的优良圆叶葡萄新种20余个，包括鲜食品种、酿酒（制汁）专用品种和酿酒鲜食兼用型品种，其中5个已通过品种审定。其中鲜食品种单粒均重10克以上，最大达到20克，亩产可达2吨。

【富含鞣花酸，研发5种酒】

来自广西农科院葡萄与葡萄酒研究所的副研究员黄羽，也是圆叶葡萄引种合作者。他向记者展示了自己两三岁女儿抓圆叶葡萄的照片：她的小手中，有两个尚未定名、代号M1102的大果型葡萄，似乎抓都抓不住。黄羽说，这种葡萄单颗的最大果重可达30克。

品尝中，圆叶葡萄也各具口感，特色是皮厚实、肉韧性、味甘醇。卢江表示，圆叶葡萄营养价值高，含有丰富的多酚类化合物，白藜芦醇等物质含量高。“而且是唯一富含鞣花酸的葡萄，这种在蓝莓等莓果中存在的物质，具有抗氧化、疏通血管、抗癌防癌、延缓衰老之功效。”

除了吃，还有喝，圆叶葡萄还酿成美酒填补了我国南方无葡萄酒业的空白。包括上海在内的我国南方地区，尚没有葡萄酒产业。黄羽也说，在广西引种地，只有2家酒厂，还不会做葡萄酒。即使在我国北方的葡萄酒产区，国产葡萄酒也一直以干红为主。为此，卢江团队探索圆叶葡萄酒的酿造工艺，研发出5个种类的“思源”圆叶葡萄酒：静止白葡萄酒、起泡白葡萄酒、桃红白葡萄酒、甜红葡萄酒、圆叶葡萄白兰地。

其中，半甜及甜型葡萄酒针对我国占很大比例的葡萄酒入门者，气泡及桃红酒如同目前在我国占很大市场份额的意大利进口莫斯卡托甜起泡酒，加强型圆叶葡萄酒与圆叶葡萄白兰地则符合烈酒文化需求，而比白酒更为健康。专家认为，利用圆叶葡萄同时具有鲜食性和酿酒价值的特点，将葡萄种植、葡萄酒加工、葡萄采摘等融合起来，将形成圆叶葡萄全产业链的发展模式，也带动与乡村振兴相结合的葡萄与葡萄酒文化旅游产业。
