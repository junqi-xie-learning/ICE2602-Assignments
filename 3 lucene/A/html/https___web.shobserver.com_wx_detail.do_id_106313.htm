青年AI科学家联盟在沪成立，最令他们惊讶的人工智能成果是什么？

前沿风 2018-09-19 17:53
来源：上观新闻 作者：舒抒
“科学和产业就像一条河的上游和下游，人工智能是科学驱动的产业，只有科学上的不断突破才能持续带来产业的繁荣。”

9月18日，在徐汇滨江航汇大厦举行的“新世代•新疆界•新引擎：青年AI科学家畅谈”活动上，由20位人工智能（AI）企业的青年创始人和自身科学家组成的青年AI科学家联盟（梧桐汇）正式启动，通过产学研对接、金融服务和人才培养，推动中国本土AI产业与人才不断聚合创新能量。

 

近年来，计算机视觉、语音识别、NLP（自然语言处理） 等研究领域借助深度学习技术取得的突破加速了人工智能在金融、医疗、自动驾驶、传统制造等各个行业的落地应用。论坛上，联盟的青年AI科学家代表分享了他们最关心的人工智能议题，以及过去十年中最令他们惊讶的人工智能发展成果。

 

商汤科技联合创始人兼CEO徐立就表示，如何让AI在工业应用中取得突破，并用中国原创技术创造新的产业价值，是青年AI科学家们肩负的重要使命。曾获《麻省理工科技评论》“35 岁以下科技创新 35 人”（MIT TR 35）殊荣的上海交通大学教授卢策吾表示，取得基础研究突破和应用大规模落地，是人工智能面临的两大挑战。令人欣喜的是，近年来学术界与工业应用领域的交流愈发深入，卢策吾认为，下一步AI迁移学习和通用AI系统应该得到更多关注。

 

提到最令青年科学家们感到惊奇的AI发展成果，同样曾获“MIT TR 35”殊荣的氪信科技创始人兼CEO朱明杰说，自己十多年前曾与计算机“深蓝”的主要研发负责人交流，没想到从1997年“深蓝”打败国际象棋大师卡斯帕罗夫，到阿尔法围棋（AlphaGo）横空出世打败李世石、柯洁等一众围棋大师，期间只过去20年，说明深度学习技术的进展已经对整个行业带来革命性改变。

 

人工智能算法从静态学习到动态学习的发展速度也超出了科学家们的预期。上海交通大学上海高级金融学院教授顾彬表示，以前机器智能通过历史数据进行学习，只是一维的静态发展，而当生成式对抗网络（Generatic adversarial networks，GAN）产生后，机器学会从对抗和博弈状态中进行学习，从互动中增强智能水平。

 

他表示，亚马逊推出的Amazon Go这一新零售模式就是通过自动感知技术营造出无人超市。当人们走进店内，大量不同位置和用途的摄像头可以识别和跟踪顾客的步态和全身运动，人脸识别技术则进一步矫正和提高机器的识别精度，动态识别技术可以检测人们取放商品的首部动作和商品数量的变化。最终，在人们看不到的“智能世界”里，一系列AI技术为人们实现“拿就走”的无人零售体验。

 

对于青年科学家们最期待的AI发展方向，微软亚洲研究院资深研究员韦福如认为，深度学习算法和模型已经广泛应用在自然语言理解多个领域，他看好未来十年，具备一定常识和推理能力的自然语言理解研究可以取得更大的技术突破。

 

“科学和产业就像一条河的上游和下游，人工智能是科学驱动的产业，只有科学上的不断突破才能持续带来产业的繁荣。” 在卢策吾看来，年轻科学家们应该沉下心来思考人工智能的未来，尤其要关注基础研究。

 

作为青年科学家中的企业家代表，朱明杰认为年轻的AI科学家们应该具备“科创家精神”，承担时代的使命。“第一，我们要造就一把坚实、高超的‘AI之锤’，其次找对‘钉子’，也就是海量的数据、真实的应用痛点和长期的研究价值，接着才能成规模地将技术向商业落地迁移，真正推动产业和社会的创新。”
