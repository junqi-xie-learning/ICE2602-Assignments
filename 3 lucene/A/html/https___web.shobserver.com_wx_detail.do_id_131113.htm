为过中国年不回国度假了！申城大学校园有自己的“春晚”，各国留学生也成“年味”主角

教育在线 2019-02-03 18:33
来源：上观新闻 作者：徐瑞哲
他们在寒假期间就提前来了交大，就是为在春节的中国气息中感受中国文化，作为来华留学体验的重要部分。

南湘走到舞台中央，旦角装扮、水袖轻抖，和着悠扬的节奏唱起了京剧《贵妃醉酒》的四平调：“海岛冰轮初转腾……”在华东理工大学为寒假留校学生举办的新春联欢晚会上，唱这出京剧的南湘是一名老挝留学生。

 

作为华理社会学院博士研究生，南湘来华留学已有5年多，不但爱吃小笼、水饺等中华传统美食，更是迷恋上中国国粹——京剧。为在晚会上演好《贵妃醉酒》，她还加入学校京剧协会，每周末跟着师生票友一板一眼学唱，难怪收获了中外师生一片喝彩。

华东理工大学“春晚”集锦。

 

猪年新春之际，在申城大学校园自己办的“春晚”上，各国来华留学生也成了“年味”主角。在数十所高校的数万留学生中，不少学子虽已完成学业，仍不急着回国度假，只为亲身体验真正的中国年。

 

 

结业留校体验中国最热闹的传统佳节

 

华东师大塔吉克斯坦留学生德明（中）、乌克兰留学生玛丽（右）等表演合唱节目。

 

从吉他弹唱《平凡之路》、独唱《rolling in the deep》到小组唱《说爱你》，华东师范大学留学生同学们出的节目，唱的基本都是中文。

其中《说爱你》演唱组合的一员，是来自塔吉克斯坦的留学生德明。“选择这首歌是因为热爱，我们想对华东师大说爱你，也想对中国说爱你。”今年9月第一次来到中国的德明，实际上已经结束了一个学期的交流学习，本可以直接回国度假的，但他选择了留在中国——因为很想体验中国最热闹的传统佳节：春节。

 

德明激动地告诉记者，“最近看到中国人准备过年，千家万户热热闹闹，就越发期待春节的到来。”在校园里，和来自世界各地的同学们一起包饺子、玩游戏、送祝福、抢“红包”，是德明在华东师大2019迎新春师生同乐会上最有获得感的项目。

同样即将结束来华学习的乌克兰留学生玛丽，也感慨：“时间过得太快了！来到师大半年，我已经习惯并爱上了这里的生活。虽然这次只能在中国学习一个学期，但我很热爱中国文化，以后一定要再来。”此时，全场师生在校长钱旭红的带领下齐声喊道：“爱在华师大！”钱旭红说，“每一个师大人都要懂得感恩，”在新的一年要一起为社会创造“幸福解决方案”。

 

 

寒假提前来华学习传统文化“预科”

 

上海交大海内外学子学写福字贴。

 

在上海交通大学，同样有很多留学生加入“年夜饭”活动。在“春晚”节目表演环节中，校留学生会还特别带来了歌曲表演《夜空中最亮的星》，赢得格外热烈的阵阵掌声。

 

其中有这样一群留学生，他们来自美国普渡大学，参加普渡与上海交大机械与动力工程学院联合举办的春季学期交流项目，计划将于春季学期在交大开始一学期的专业课学习。然而，他们在寒假期间就提前来了交大，就是为在春节的中国气息中感受中国文化，作为来华留学体验的重要部分。

为了这项中华传统文化的“预科学习”，上海交大专门开设了“文化学堂”，为他们讲授中国春节习俗，以及艺术、厨艺、茶道、中医等文化知识。于是，初来乍到的美国学子也登上校园春晚舞台，一起合唱《茉莉花》，还进行了功夫表演，与中国学生一起欢度中国年。

解放日报·上观新闻记者了解到，早在2003年，上海交大就与普渡大学签署合作协议，中美两校机械工程相关院系已联合拓展出多种模式的学生联合培养项目。在上海，据市教委最新统计，去年共有来自185个国家和地区的60870名外国留学生，在沪上招收外国留学生的42所高校（科研机构）就读，其中长期生达45830名，学历生达22130名。

 

 

中外学生结伴“打卡”最宜知华友华

 

上海中医大中外学生组队寻找海上年味之旅。

 

校园“春晚”还有户外版，传统与现代的“年俗”在校内外联动。从南京路、人民广场到外滩、陆家嘴，从新天地、田子坊到豫园、城隍庙，上海中医药大学过年时就以快闪定向越野的方式，打卡上海地标建筑，共同寻找中国年味。

中外学生结伴，最宜知华友华。今年留校学生大部分为国际学生，他们的打卡地选择了那些“上海最有年味的地方”。布满近千只彩灯的豫园灯会上，国际学生们在九曲桥广场和黄金广场看生肖主题灯盏，参与各种传统民俗项目：挂幸运树送祝福、猜灯谜赢奖品、品尝各色美食等，在人流攒动中感受浓浓年味。

九曲桥底，鲤鱼红火。中医大康复医学院留校学生杨国辉，与来自泰国和土库曼斯坦的留学生成为一组，共同分享年味之旅。泰国留校生陈金色说：“我们穿梭在上世纪三十年代的老上海与摩登都市之间，闲逛文化风情小街，品味地道民间美食，可以感受不同地域文化的新年习俗。”

突尼斯留学生刘敏那一组去的是新天地和田子坊，参观了上海弄堂，找到的“年味”也有许多：红灯笼、“福”字、春联……她说，“我们在弄堂里感受了上海的年味，美味的小吃，热闹的气氛，热情的上海人，让我们对中国文化有了更美好的回忆。”他们说：“年味，自己吃是一个味道，和大家吃又是一个味道。”
