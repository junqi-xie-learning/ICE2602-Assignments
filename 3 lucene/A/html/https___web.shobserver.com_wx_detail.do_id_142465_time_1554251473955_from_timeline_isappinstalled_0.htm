从2023年开始，中国的老龄化会突然加速，养老产业的“风口”在哪儿？

思想汇 2019-04-03 06:31
来源：上观新闻 作者：王珍
从2023年开始，中国的老龄化会突然加速，预计一直要到2051年才会有所减缓。考虑到平均寿命的延长，这个时间还有可能延后。

“养老行业是关切夕阳的朝阳产业。”3月29日，在“智享变革 洞见新机遇”安泰智库媒体沙龙上，上海交通大学安泰经济与管理学院教授、行业研究院养老行业研究团队首席研究员罗守贵就中国养老问题的挑战与养老行业发展的机遇发言时作出上述表示。

罗守贵指出，从世界经验来看，发达国家的人口结构往往呈宝塔型，发展中国家则呈金字塔型。尽管中国依然是发展中国家，但人口结构已从金字塔型变为宝塔型，这给养老问题带来巨大挑战。相比金字塔型，宝塔型意味着老年人口越来越多，年轻人越来越少。全面二孩政策实施以来，产生了脉冲效应，但是由于人口增长呈现出代际性特征和延迟效应，因此挑战依然严峻。1963年中国爆发了婴儿潮，如果以60岁退休来计算，那么到2023年这一代人将步入老年。从2023年开始，中国的老龄化会突然加速，预计一直要到2051年才会有所减缓。考虑到平均寿命的延长，这个时间还有可能延后。

 

就老龄人口而言，养老最严峻的问题主要集中在高龄独居老人、失能老人和农村老人。有一个预测，2020年中国失能老人可能达到4200万，80岁以上老人达到2900万，空巢、独居老人1.18亿。这部分老年群体是社会重点关注对象，同时也是解决养老问题的关键所在。此外，超过1亿的农村老人面临的问题更大，不但子女多在外打工，身边无人照顾，而且这些老人自身的经济能力也难以支撑其养老。因此可以说，未来30年中国面临所有的挑战中，人口老龄化是最大的一个。

 

为什么说养老产业是朝阳产业？罗守贵同时给出了另一组数据：有统计显示，养老产业占GDP比重，中国只有7%，美国达到22%，欧洲有29%。如果这个数据是可信的，那么中国养老产业发展的空间非常大。从现在开始至少30年，甚至50年，将成为中国养老产业发展的一段黄金时期，也让养老产业成为朝阳产业。之所以这么说，一方面是上面所说的老年人口数量很多，而且随着人的寿命延长，老年这个年龄段会被拉长。另一方面，我们再算一笔账，2020年中国GDP有望突破100万亿元，按8%的占比计算，养老产业的市场规模将达8万亿元。再延伸10年，即便我们达不到欧洲29%、美国22%那样高的比例，但是比现在翻一倍达到15%是完全有可能的，那就意味着，到2030年预测中国养老产业可以到25万亿元的市场规模。“大家可以想想，有什么产业可以达到25万亿？有人说房地产，将来随着人口老龄化的到来，房地产这个行业不会像过去那么发展了，而且房地产里面很多也是和养老产业密切相关的。”

 

养老产业在上海发展前景如何？罗守贵认为，上海的老龄化程度在全国是走在前面的，因此面临的挑战也更为紧迫。经济学讲供给和需求，供给方面有一个很重要的趋势是什么？那就是过去的老人没有钱，但是现在老人有钱了。以储蓄作为保障、以养老金作为流量来支付养老费用，从供给端来说，资源还是比较丰富的。上海新一代老人养老支付能力大大增强，为中国高端养老提供了巨大市场。作为养老支付最有保障的养老金，上海目前企业退休金平均值已达到3700/月，超过5000元甚至10000元的也不在少数。

 

罗守贵认为，我国养老产业的市场潜力很大，正在进入一个爆发式增长的前期。目前，在相应硬件和软件提供上，还有很大提升空间。比如，上海目前只有12.6万张养老床位，占60岁以上老人的比例只有2.9%，缺口巨大。此外，目前的养老服务设施接纳失能、半失能的老人能力还比较弱。因此，在做好政府公共服务的同时，也要更好挖掘包括民营资本在内的社会力量，充分抓住养老产业的“风口”。
