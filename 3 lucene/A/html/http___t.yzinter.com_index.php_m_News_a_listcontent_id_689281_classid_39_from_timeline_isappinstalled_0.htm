原创|养老行业是关切夕阳的朝阳产业

2019-03-29 21:20:09

扬子晚报网3月29日讯（记者 罗晓娜）上海交大安泰经济与管理学院教授、博士生导师上、海交大行业研究院养老行业研究团队首席研究员、应用经济系主任罗守贵教授在“智享变革 洞见新机遇”安泰智库沙龙中表示：我们判断养老行业是关切夕阳的朝阳产业，至少30年，甚至到50年，这个是中国养老产业发展非常非常长的一段黄金时期。

为什么是朝阳产业？

罗守贵教授说：老人的数量很多，中国是大国。大家可以算一下，从退休到离开有多少时间？男的60岁退休，将来到80岁甚至更长。女的有的50岁退休了，寿命更长，可以到80、90岁，养老可以到3、40年的时间。有人算小孩子的教育，那个是比较短，从出生到大学毕业就是10几年，老人后面会持续几十年，这个时间非常长。

他说：我们再算一笔账，2020年中国GDP突破100万亿，刚才说中国现在只有7%，按照8%来算，就意味着到2020年中国养老产业市场8万亿。

将来中国人的老年人比重很高，我们算15%，到2030年预测中国GDP的15%算养老产业可以到25万亿。大家可以想到什么产业可以达到25万亿？有人说房地产，将来随着人口老龄化的到来，房地产这个行业不会像过去那么发展了，房地产里面很多也是和养老产业密切相关。

失能失智老人就有80万

现在有一些敏锐的企业开始进入这个行业，现在养老产业在中国发展时间比较短，总体也不是很好，我们调研了一下，包括民营的养老机构很多在亏损，三分之一盈利，三分之一平衡，三分之一亏损，这个并不意味着这个行业不好，我们可能在摸索、在寻找盈利的模式，以及各种企业的运作，非常找到好的途径，但是机会非常大。

以上海为例，上海的人口老龄化非常严重，还有一点大家注意，经济学讲供给和需求，从这个角度来看，需求非常大，供给呢？供给方面有一个很重要的趋势是什么？过去的老人没有钱，但是现在老人有钱了。其他一些国家，老人有钱，到国外旅游你看都是老人，年轻人没有钱没有时间旅游，中国老人不是普遍有钱，但是中国的人口基数大，有一部分的人有钱。像上海，我周一下午到民政局去，他们给我一些数字，他们告诉我上海企业的平均养老金去年是3700块左右。

大家知道事业单位和公务员退休高很多，上海老人的退休金一个月五六千，一万块的也有。我们把退休金作为流量来做养老，老人通常有一些储蓄，他们这个储蓄可以作为保障，每个月的养老金作为流量，支付养老费用，养老金是多少？意味着养老方面可以支付多少。从这个角度来算，光是上海就有很大的群体要养老。旺盛的需求和很大的供给，注定这个市场有很大的空间。实际的情况如何？我们做了一些教研，总体上包括养老的床位，政府过去是做低端的保障性的，中高端是比较少。接纳失能，半失能的老人能力比较弱。中国光是失能失智老人就有80万，我们接纳他们的床位都不足，还有半失能的。

上海有一些中高端的养老机构，收费从五六千到一两万的都有。甚至有人在筹建年收费50万的养老机构，一个人50万，老两口要100万养老，这个是比较贵，这个是金字塔顶端，但还是有的。

我们养老产业的市场潜力很大，也在进入一个爆发式增长的前期，以资本市场来讲，投资者也好，这个时候要布局养老领域了。
