上海交大成功举办“海洋水下工程技术发展论坛”
中国网 2018-11-22 16:32

中国网11月22日讯 由上海交大海洋水下工程科学研究院主办的“上海交大海洋水下工程技术发展论坛”在上海交通大学徐汇校区成功举行。上海交通大学党委书记姜斯宪和中国军事科学院原副院长糜振玉、北海舰队原司令员王继英等领导出席。

论坛召开当日，正值上海交大海洋水下工程科学研究院建院四十周年。几十年来，该院在我国海洋水下工程科学研究领域，始终瞄准国际先进水平，一次次填补国内空白，一次次取得技术突破，302m 氦氧饱和潜水实验创造了亚洲饱和潜水深度记录。在无人遥控潜水器（ROV）的研制方面，“海马”号ROV的研制成功标志着我国全面掌握了大深度无人遥控潜水器的各项关键技术，并在关键技术国产化方面取得实质性的进展，是我国深海高技术领域继“蛟龙号”之后又一标志性成果，实现了我国在大深度无人遥控潜水器自主研发领域“零的突破”，先后获得国家科技进步奖5项，省部级奖40余项，主编国家及行业标准18项，获得专利授权及软件著作权40余项。目前上海交大海洋水下工程科学研究院已成为国内唯一集水下工程技术研发、工程施工、专业设备开发销售、潜水培训和潜水生理医学为一体的综合性水下工程技术研发单位。

当前，国家正在大力推进军民融合，实现民用高技术服务军队建设，这既是推动发展、改革、奋进的一个重要契机，也是迈向新征程、开创新辉煌难得的历史新机遇。此次论坛的召开，旨在为政府、业界、行业以及企业搭建一个讨论交流、思想碰撞、资源整合、相互协作的平台，共同把脉海洋水下工程技术领域的现状并探讨研究今后的发展方向。论坛嘉宾一致认为，今后要以更高的历史站位、更广泛的国际视野、更深锐的战略眼光，加强合作，共谋发展，抓住国家海洋开发的宝贵机遇，为实现国家海洋战略做出更大的贡献。

交通运输部救助打捞局系统、科技部中国21世纪议程管理中心、中国潜水打捞行业协会、上海市隧道工程轨道交通设计研究院、国内从事海洋及水下工程、潜水医学、水利工程的高校和企业的同行、专家共400余位嘉宾参加了此次论坛。
