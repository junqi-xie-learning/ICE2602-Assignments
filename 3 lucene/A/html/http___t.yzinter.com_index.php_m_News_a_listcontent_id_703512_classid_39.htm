【视频】“国企领导上讲台”上海交大专场报告会举行

2019-05-07 17:52:07

扬子晚报网5月7日讯（记者 罗晓娜 文/摄）5月7日，“国企领导上讲台”上海交通大学专场报告会举行。中国远洋海运集团董事长、党组书记许立荣作主题报告。

此次上海交通大学与中国远洋海运集团一对一“结对子”，邀请国企领导进校园、上讲台，就是要原汁原味地把中国特色现代国有企业制度最鲜活的实践案例带进课堂，拉近青年学生与国有企业的距离，增进青年学生对国有企业的信任。学校和企业共同联手，把思想和行动、智慧和力量凝聚到党中央的要求上来，不断凝聚育人合力，培养合格的中国特色社会主义建设者和接班人。

 

许立荣“海运即国运”主题报告，围绕习近平总书记在上海考察时关于“经济强国必定是海洋强国、航运强国”的重要讲话精神，放眼世界航运史和新中国海运发展历程，结合中远海运70年成长发展和全球化实践，深刻阐述了海运业的战略地位、战略担当及航海精神特质。

据悉，“国企领导上讲台”活动首批选择50家央企和50所高校结对，要求国企领导每年进高校为学生讲授2场公开课，国企遴选2名业务骨干担任高校校外辅导员。编辑：李燕
