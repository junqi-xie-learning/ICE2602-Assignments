超级机场 航天对接 治水专项 游泳冠军……“别人家”能的，中国人更能！

教育在线 2019-12-18 19:18
来源：上观新闻 作者：徐瑞哲
贾广社和“小巷总理”朱国萍、“大国工匠”王曙群、“洱海守护者”孔海南、“游泳世界冠军”陆滢等用自身经历讲励志故事

“从2018年5月2日赴京，到2019年9月30日返沪，我们用517个日日夜夜，助力北京新机场这只‘金凤凰’展翅高飞。”北京大兴国际机场重大工程总进度管控、同济大学教授贾广社在上海交通大学文治堂告诉全场学子，今年国庆节前的那一天，首都大兴机场建成通航，被外媒誉为“世界新七大奇迹之首”。

在18日下午启动的上海市“奋斗的我 最美的国”新时代先进人物进校园首场报告会上，贾广社和“小巷总理”朱国萍、“大国工匠”王曙群、“洱海守护者”孔海南、“游泳世界冠军”陆滢等用自身经历讲励志故事，为沪上各大高校700余名师生代表上了一堂“思政大课”，共同传递出一种态势、一份自信：“别人家”能的，中国人更能。

【文本“独孤求败”一个字也改不了】

据透露，大兴机场本期建设按2025年旅客吞吐量7200万人次、货邮吞吐量200万吨、飞机起降量62万架次的目标设计，用地面积27平方公里，建设“三纵一横”4条跑道、70万平方米航站楼等设施。

超级工程需要超级“管家”，贾广社团队编制完成《北京新机场建设与运营筹备总进度综合管控计划》，共梳理出重点问题41个，识别计划节点366个，被民航部门评价为“为我国创建出大型工程治理体系和治理能力现代化的样板”。

贾广社表示，团队中，有37位90后、95后同学，其中最长的在工地待了13个月。他们忍耐着市郊的枯燥、工地的喧嚣，白天在指挥部做计划管控，晚上在宿舍复习功课，有时周末还要赶回上海上课……

同济驻场团队的工作质量标准是：“一个标点符号也不许错！”月报及汇报文件精益求精，反复修改，有的报告竟然改过20次之多才定稿。为此，同济各类文本被领导戏称为“独孤求败”：一个字也改不了。贾广社常常问他们：“第二个百年目标的工程是什么？你们做好准备了吗？”当天，他也问起在座的各位同学，“为了实现第二个百年奋斗目标，你们准备好了吗？”

【哄睡陪玩带娃仍要参选东京奥运】

中国泳坛“最美蝶后”陆滢，从幼儿园大班开始学游泳，如今已过去20多年了。“如果那时候有人问我，你能坚持吗？那个小小的我都还不知道这两个字的分量有多重。”

陆滢直言，参加国际比赛，到国外集训，能感受到一些人对中国运动员还是存在偏见。他们觉得：一个运动项目从你们那么小就开始变成生活的全部，甚至还牺牲了很多事，这样好像过于执着了。其实，这种刻板成见在国内也有，“你把青春都给了运动不后悔吗？”——“这几乎是我们常常被问到的高频问题。”

在现场，陆滢不回避曾经的低谷和惯常的伤病，把乐观和确信传递给每个人，“大家知道，我现在有了新的身份和新的使命，我成为了一位母亲，我也得哄睡、陪玩、带娃……”她透露，但重回泳池继续训练是目前最重要的事，“还有半年就要选拔了，我也不知道能不能选上参加明年的东京奥运会。”

她表示，“但我不会停下来，我想挑战自我，外国选手可以做到的，我们中国选手为什么做不到？”身着一袭运动装的陆滢说，“我仍然想去尝试，看看我们的运动寿命在哪儿。”

【加完班完全想不起到哪里接儿子】

王曙群是上海航天设备制造总厂有限公司班组长，主要负责神舟飞船、空间站对接机构的总装调试以及探月工程月面巡视器的结构机构总装等。他坦言，1989年中专毕业进入航天，1995年机缘巧合进入神舟飞船对接机构总装组。“那时，我国航天事业正处于发展的低谷，以我所在的航天总厂来说，我们总共才发射了3发长征火箭，型号任务屈指可数。”有人觉得他们傻，到外资企业随随便便就可以找一份几千元的工作，而他们却死守着一二百元的死工资，做着一个永远见不到底的任务。

但王曙群30年追求卓越无极限，并将见证中国人在不久的将来拥有环绕地球的唯一空间站。在整个对接机构研制过程中，最困难的那一阵子，家里人和同事们都说他像中了“邪”一样，走路时想、睡觉时想，有时甚至在饭桌上也会不自觉地用手比划。“有一次，我爱人打电话让我加班结束后去接一下补课的儿子。”王曙群自嘲地说，“结果我加完班，却完全想不起该到哪里去接儿子。”

就是这样一个技术工人，自主研发航天对接期间牵头研发了50多台套专用装备，还完成论文15篇，获得5项国家发明专利。

【积蓄200万元设保护人才教育基金】

“考察船到了湖心，看到了洱海的沉水植物群落从湖底一直长到了湖面，呈现了十几平方公里‘水下森林’景象，十分震撼……”二三十年前，参与中国太湖、滇池、大理洱海与日本霞浦湖、北海道洞爷湖之间比较研究课题的孔海南教授忆起，“我到过世界上许许多多的著名湖泊，看过各种各样的生态美景，洱海的那幅景象让我终身难忘……”

曾经，为了不拖延研究进程，他和团队就用自己的研究经费甚至个人收入预先垫付，不仅团队的“国际合作经费”“横向科研经费”悉数被垫进去，最后动用他本人的存款和工资。“2007年6 月，本人被迫公开宣布‘破产’了，”说到此时，师生笑了，“后来通过向大理当地大学借用住房及科研用房，出差改乘火车等方式节约资金，又有国家水专项技术总师请洱海项目参加单位紧急支援了团队20万元，才最终艰难渡过难关。”

“老人与海”的故事续写到今夏，洱海项目“十二五”期间最后一个课题通过国家正式验收。这位上海交大河湖环境技术开发中心主任今年70岁，就要正式退休。经与家人商量，他拿出自己积蓄200万元，曾参与洱海治理的中国水环境集团也出资200万元，共同发起设立“洱海保护人才教育基金”，并有来自各方的捐赠持续注入，“只有一代一代年轻人前赴后继，才能真正守得洱海水清月明。”

【身退心不退培养更年轻“小巷总理”】

长宁区虹桥街道虹储小区原居委会主任、党总支书记朱国萍也坦陈，“去年我退休了。可身子退了，心不能退。”

前阵子，朱国萍很荣幸被授予“庆祝中华人民共和国成立70周年纪念章”，还受邀现场观摩了在天安门举行的国庆阅兵。上个月，习总书记还来到虹桥街道全国基层立法联系点，鼓励他们“要坚持好、巩固好、发展好，畅通民意反映渠道，丰富民主形式”。朱国萍现场聆听，“更是倍感振奋、深受鼓舞。”

她表示，退休后正用心做好“萍聚工作室”，从服务社区到服务社会，培养更多社区能人和“小巷总理”，回报组织，感恩社会。“我给工作室取名‘萍聚’，并不是用我个人的名义，是希望汇聚各方力量，传承给年轻人，交好接力棒。”今年，工作室跟常熟蒋巷村这个乡村振兴和生态发展的典范城乡牵手，建立“蒋巷萍聚苑”，合作迈出服务长三角第一步，把国家战略的“一盘棋”落实到基层群众的“一家亲”。

解放日报·上观新闻记者了解到，此次进校园活动是上海高校响应教育部等七部委号召，举办的首场市级层面宣讲活动。上海市教卫工作党委、市委宣传部、市教委、市国资委、市总工会、团市委、市妇联、市科协已联合建立“新时代先进人物进校园”同城资源库，根据“同城就近”“双向选择”的原则与高校结对共建，进课堂、进实验室、进宿舍、进食堂，深度参与学生主题党团日、主题班会、实习实训、社会实践等活动，并利用网上网下多种途径与学生交流，打造常态化思想政治教育品牌，构建新时代先进人物进校园长效性机制。
