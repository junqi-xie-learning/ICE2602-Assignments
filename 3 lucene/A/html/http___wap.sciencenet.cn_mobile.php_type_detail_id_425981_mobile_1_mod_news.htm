“国企领导上讲台”上海交通大学专场报告会举行
2019-05-07 21:42
来源：科学网 www.sciencenet.cn

 

5月7日，“国企领导上讲台”上海交通大学专场报告会在徐汇校区文治堂举行。中国远洋海运集团董事长、党组书记许立荣作主题报告，上海交通大学党委书记姜斯宪主持报告会。中远海运集团董事、党组副书记孙家康，上海交通大学党委常委、党委宣传部部长胡昊，中远海运集团相关部门负责人，上海交通大学师生代表出席报告会。

此次上海交通大学与中国远洋海运集团一对一“结对子”，邀请国企领导进校园、上讲台，就是要原汁原味地把中国特色现代国有企业制度最鲜活的实践案例带进课堂，拉近青年学生与国有企业的距离，增进青年学生对国有企业的信任。学校和企业共同联手，把思想和行动、智慧和力量凝聚到党中央的要求上来，不断凝聚育人合力，培养合格的中国特色社会主义建设者和接班人。

许立荣作“海运即国运”主题报告。他结合中远海运70年成长发展和全球化实践，深刻阐述了海运业的战略地位、战略担当及航海精神特质。报告分享了“海运”与“国运”关系的五个方面的思考，一是海运是“强国”难以复制的动力要素。列举了欧美、日韩等国家崛起的“海运”因素，回顾了从唐代以来中国海运业发展与国力强弱同频共振的沧海沉浮，揭示了海运事关民族安危、国家命脉的重要战略地位；二是海运是“强国”不可或缺的经济要素。阐述了海运业作为预判国际市场的“晴雨表”、市场充分竞争的标杆、国内经济融入国际市场的先导，在拉动经济、促进贸易等方面的不可或缺作用；三是深化改革重组是做强海运的最佳路径。介绍了中远海运集团以新发展理念为引领，以服务国家战略为使命，持续深化改革重组，大力推进企业高质量发展，实现了规模与效益、改革与发展同步增长，成为全球最具竞争力的综合航运企业的改革历程和重大成就；四是践行“一带一路”是做强海运的最大机遇。阐述了海运业作为推动“一带一路”落地最成熟的产业力量，在“一带一路”建设中的骨干作用，介绍了中远海运在航运、码头支点网络、全程供应链服务等立体式一带一路业务格局，同时分享了“义利并举、共享共赢”的“一带一路”实践体会；五是海运文化是滋养强国梦的精神财富。结合航海文化和航海精神，勉励在座同学，在远大的人生航程中，“把好舵”，砥砺家国天下的爱国精神；“定好锚”，坚守脚踏实地的实干精神；“开好桨”，激扬战风斗浪的进取精神；“系好缆”，保持同舟共济的合作精神，做青春的奋斗者，做时代的追梦人。

据悉，开展“国企领导上讲台”活动是切实营造全员育人浓厚氛围的重要举措，活动首批选择50家央企和50所高校结对，要求国企领导每年进高校为学生讲授2场公开课，国企遴选2名业务骨干担任高校校外辅导员。后续上海交通大学还将与校外辅导员加强联系，发挥校外辅导员的专业特长，助力学生成长成才，逐步推动活动制度化、常态化。
