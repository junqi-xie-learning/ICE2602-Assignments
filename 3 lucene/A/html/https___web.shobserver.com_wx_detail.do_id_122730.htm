洋学生眼里的中国什么样？看看他们拍的短视频吧！上海交大联合30多高校开启比赛

互联网观察 2018-12-19 20:11
来源：上观新闻 作者：刘璐
在表达留学生的真情实感和有趣发现的同时，向世界展示中国真实的国情、文化和社会生活。

“今天，短视频是世界各地年轻人表达自己的重要形式，感谢你们为留学生提供了一个有意思的平台，让我们表现自己眼中的中国。”

 

说这话的Arina Yakupova，是上海交通大学的一名研究生。她本科就读于莫斯科国立大学，此次是作为留学生代表，出席2019“我眼中的中国——在华留学生短视频大赛”启动仪式。

 

Arina所在的上海交通大学媒体与传播学院，是本次大赛的发起方和组织方。依托交大的理工科优势，学院在发展文理交叉学科上自成特色。建院16年来，已成为国内新闻传播院系中留学生比例最高的院校之一。目前在读留学生300余人，来自世界上58个国家，包括普林斯顿大学、芝加哥大学、加州大学洛杉矶分校、莫斯科国立大学等世界名校的毕业生。据该院李本乾院长介绍，大赛旨在为国际学生提供了解中国国情、中国文化和真实中国生活的机会，尤其是针对新媒体全英文硕士和博士项目，提高学生创新、创业、创意的能力，让学生以第一人称的独特视角，通过现场观察，用他们的聪明才智和数字设备，向世界反映一个真实的中国。

上海交大宣传部新闻中心负责人李厚锐介绍了学校近年来在国际化办学方面的积极探索，众多留学生通过学校开展的丰富活动，走访中国城市、企业，近距离感受和了解现在的中国。“现在短视频已进入传播的风口期，即将迎来快速发展期。目前中国的网络视频用户已经超过6亿人。而随着5G时代的到来、移动设备的进一步发展，制作和传播短视频会变得越来越便捷。”他表示，尤其对于90后、00后这些主要的在校大学生群体，短视频成为他们接收信息、自我表达、认知世界的重要途径，本次短视频大赛正好迎合了这一重要的媒介趋势。

 

解放日报·上观新闻记者了解到，为了更广泛地提升中外文化交流水平、搭建中外青年学生交流的桥梁，大赛并不局限于一校一院，而是联合了北京大学、清华大学、复旦大学、中国人民大学、中国传媒大学、武汉大学、华中科技大学、南京大学、浙江大学、暨南大学等三十余所大学的新闻重点院校，主要面向具有外国国籍、并正在中国就读的留学生，鼓励他们用手机或其他设备拍摄短视频，讲述他们眼里的中国模样。

 

12月19日在上海交通大学闵行校区举行的发布会上，上海报业集团新闻办副主任唐颖豪表示，大赛可以看作是上海报业集团与上海交大媒体与传播学院部校共建项目的首个重要活动，也象征着双方长期战略合作的良好开端。

 

据悉，大赛将于2019年1月1日至3月15日接受投稿，参赛团体或个人需要围绕“我眼中的中国”这一主题，用自己的母语或英语录制一段1-5分钟的短视频，上传到组委会指定的网站。经过专家评审，组委会将在2019年4月中旬公布最终结果，获奖作品有机会在国内外知名媒体播放，在表达留学生的真情实感和有趣发现的同时，向世界展示中国真实的国情、文化和社会生活。
