诺贝尔化学奖得主做客交大表示：

《道德经》道出点击化学哲学真谛

青年报记者 刘昕璐

青年报记者 刘昕璐

本报讯 点击化学是一种实用的哲学概念在合成化学上的映射，已被广泛应用于有机化学、聚合物材料合成、生物分子标记、抗体修饰、药物开发等一系列重要的研究和生产领域。昨天，不对称催化和点击化学领域的奠基人，诺贝尔化学奖卡尔巴里夏普莱斯教授做客上海交通大学大师讲坛，作《SuFEx点击化学在生物方面的应用》报告，向交大师生分享其近期在研究具有抗癌活性的小分子方面的工作。

据悉，夏普莱斯喜欢中国文化，特别是可以团结很多人的家族观念。他也非常欣赏老子《道德经》里的一句话，“故有之以为利，无之以为用”，认为这句话道出了“点击化学”中的哲学真谛：平时不会反应的碳-杂原子的结合，看起来似乎没什么用处，可当带上了特殊的官能团后，就会具有很高的选择性。比如夏普莱斯教授正在研究的第二代“点击化学”反应中，带上硫氟基团的小分子，可以精准识别出能与之结合的蛋白质，并形成相当稳固的共价键。

夏普莱斯不对称双羟基化反应与夏普莱斯环氧化反应，是现代有机合成中最重要的反应之一，并因这一贡献获得2001年诺贝尔化学奖。最近，他在点击化学的核心理念上又发展出来新一代的点击化学——基于“硫氟交换（Sulfur(VI) Fluoride Exchange，SuFEx）”的高价硫氟类化学。

上海交通大学大师讲坛创办于2012年4月，是面向全校师生的高层次学术活动。大师讲坛定期邀请国内外学术大师、著名科学家、学者等走进讲坛，讲述科学知识、科学道德、科研精神和研究方法，与交大师生面对面交流，以成功者的典范指引学生的学术航向，激励学生投身学术道路，为祖国科技事业的兴旺发展贡献力量。
