沪上各高校团委干部表示：做青年学生的知心人

在时代脉动中奉献青春

青年报记者 刘昕璐 周胜洁

五四精神穿越了时空，璀璨着整个世纪。习近平总书记在纪念五四运动100周年大会上发表的重要讲话，连日来在申城高校中引发持久反响，特别是对新时代青年提出的“树立远大理想、热爱伟大祖国、担当时代责任、勇于砥砺奋斗、练就过硬本领、锤炼品德修为”等要求，正不断激励青年群体，也让团学工作者进一步思考和行动。   

青年报记者 刘昕璐 周胜洁

做青年学生的知心人、引路人

复旦大学团委书记赵强表示，总书记的讲话充分肯定了五四运动的历史作用，充分肯定了青年在救亡图存、国家建设、民族复兴伟大历程中的作用，高扬着爱国主义的伟大旗帜，充满对青年的厚爱和希冀，激励着每一位青年永久奋斗，是对马克思主义青年观的又一次深刻阐释。

“青年工作，抓住的是当下，传承的是根脉，关系到中华民族永续发展。作为先进青年的政治组织，共青团要始终坚持党的领导，党旗所指就是团旗所向，要始终把为党和国家的事业培养建设者和接班人作为根本任务，要始终谨记巩固和扩大党的青年群众基础的政治责任，要始终围绕中心服务大局，团结带领广大团员青年，投身伟大时代，激扬奋斗梦想，为中华民族伟大复兴贡献磅礴的青春力量。”赵强说道。

上海交通大学团委副书记钱文韬表示，纪念五四运动100周年大会的胜利召开，为新时代青年提气明志，在历史回顾中传承五四精神，在时代脉动中唤起情怀梦想，在接续奋进中号召责任担当。值此纪念五四运动百年之际，作为实现两个百年奋斗目标的见证者、参与者、建设者和推动者，我们定要深入学习、把握五四精神的核心及其新时代下的内涵，让五四运动凝聚的爱国、进步、民主、科学的精神旗帜高扬、薪火相传；我们定要增强“四个意识”、 坚定“四个自信”、做到“两个维护”，紧密团结在以习近平同志为核心的党中央周围，以最昂扬的精神面貌积极拥抱新时代、奋进新时代，在奉献中绽放青春光彩。

同济大学团委书记陈城表示，作为一名团干部，将继续用青春和热血践行五四精神，着力做青年朋友的知心人、青年工作的热心人、青年群众的引路人，进一步强化基层建设、强化政治引领、强化成长服务，进一步做强做优传统青年成长平台，引导青年扎根基层、扎根大地，为国家发展、民族复兴贡献力量。

上海理工大学团委书记曲德强认为，高校的共青团组织，党旗所指，团旗所向，党有号召，团有行动。我们要认真学习贯彻落实习总书记的讲话精神，牢牢把握新时代中国青年运动的方向，引领广大团员青年坚定“永远跟党走”的理想信念，号召广大团员青年把自身的前途命运同国家和民族的前途命运紧紧联系在一起，努力为共同的理想和目标贡献青春力量。

引领青年热爱伟大祖国

上海财经大学团委副书记袁海萍说，现在接触的学生都是“95后”“00后”的青年学生，在他们的身上，看到了爱国、勤奋、明辨、求实、创新等宝贵的精神特质。

“习总书记的讲话，为我们高校共青团下一步工作指明了方向，当代青年生逢其时，重任在肩，今后高校共青团要进一步创新工作载体和方法，切实加强青年思想政治引领，引领青年坚定理想信念，巩固和扩大党执政的青年群众基础；引领青年热爱伟大祖国，厚植家国情怀，知行合一，在最美的年纪，选择最有意义的事情，将青春之花绽放在祖国最需要的地方；要贴近青年，做好服务，关心厚爱和严格要求相统一，充分信任青年，真情关心青年，严格要求青年，成为青年朋友的知心人，青年工作的热心人，青年群众的引路人；引领青年练就过硬本领，明德修身，珍惜新时代的机遇与机缘，做德才兼备、与祖国发展同心同向、积极向上的社会主义建设者和接班人。”袁海萍说道。

华东理工大学团委书记侯宇认为，作为一名基层团干部，要牢记总书记的嘱托，做青年的引路人，引领广大团员青年进一步赓续传承五四精神，厚植家国情怀与价值引领，成长为信仰坚定、本领过硬的青年马克思主义者，激发新时代的使命与担当。

在集体观看纪念五四运动100周年大会，听取并学习习近平总书记发表的重要讲话的前一天，上海第二工业大学的学生们还积极参与了上海青少年纪念五四运动100周年主题歌会，用歌声和舞蹈缅怀五四先驱崇高的爱国情怀和革命精神，用飘扬的旗帜指引青年走向未来奋斗前进的正确方向。团委书记董凌俐感到，作为高校共青团的工作者，一是要加强思想政治引领建设，教育引导青年正确认识世界、了解国情，既要热情鼓励，也要教育引导；二是要创新活动思路内容，办青年喜闻乐见的活动，让青年在参与中有所收获，有所成长；三是要关心服务学生成长。

我们，生逢其时，重任在肩，赓续精神，永久奋斗！上海对外经贸大学团委书记缪韵笛认为，总书记的重要讲话既是动员令，也是路线图。学校共青团组织更要将共青团的根本任务、政治责任和工作主线落到实处，带领全校团员青年高举团旗跟党走。
