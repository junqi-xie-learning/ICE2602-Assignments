人生“彩排”：这里的学生刚上大一就开始“玩”毕业……

东方网 傅文婧、汪伟秋
2019年7月6日 10:33:01

　　东方网记者傅文婧、汪伟秋7月6日报道：每年七月，当高考生收到录取通知书的时候，也是毕业生收拾行囊离开校园的时候。带着大学四年甚至更久，青春年华最美好的回忆，走向新的方向，踏上新的旅程。

　　7月3日，上海交通大学办了一场名叫“Re:Set”的毕业晚会，台前幕后的全部工作都由交大学联艺术中心的学生们一手包揽，而他们都是大一到大三的学生。交大学生出了名的敢玩，敢”自黑“，”排名全国前三的十所高校之一“成为了今年高校招生广告里最红的搞笑梗。为了给即将毕业的学长学姐们搭好最后的舞台，这群”会玩“的孩子忙活了两个月。

　　导演组、场务组、主持、导播、摄影、化妆、舞美、灯光，乃至转场短片的拍摄制作，每一个环节都看起来充满专业感，而背后却是一群热血又热心的大学生们在默默“为爱发电”。节目单的安排丰富多彩，有热辣街舞、有流行风的民乐、有唱出青春感悟的摇滚，有爆笑脱口秀，也有宿管阿姨的T台秀和退休老师的“最后一课”。

　　一时令人笑痛肚皮，一时又令人眼眶发热，这台晚会在调动观众情感的节奏把握上，专业感十足，又处处充满大学生单纯的真情流露，有笑有泪的参与感一气呵成，两个小时的晚会不知不觉就走到了尾声。这台晚会的总导演刘煜伟同学，只是一名大二学生――真是江山代有人才出。

　　刘煜伟说，自己大一的时候就参与了学校毕业晚会，当时自己只能算是个“小朋友”，过程中得到了很多前辈的指点。现在轮到自己来“统领全局”了，在策划落实过程中，接触了许多学长和他们的家长，了解到了很多学长的故事，对自己很有触动，“整个人发生很大变化，对自己也是个激励。现在晚会顺利结束了，希望可以学长学姐可以喜欢这份毕业礼物。”

　　刘煜伟从新人成长为了主心骨，工业工程专业的年梓帆同学则从当年的幕后一员坐上了今年的观众席――这次，轮到他毕业了。年梓帆也曾经和刘煜伟一样，为学长学姐的毕业晚会忙里忙外，今年他要离开母校，舞台上的许多节目都令他感慨不已。在舞台上演出的，有他认识的同学，在幕后默默付出的，同样也有他认识的队友，“就感觉自己真的毕业了，在学校最后的日子里勾起了很多回忆，四年的时光都涌现了出来，真的很感触。”

　　正在读大三的黄炎康同学今年是第三次参与毕业晚会。他说三年前的自己还“啥都不懂，只是觉得新奇”，去年他忽然意识到有一群他认识的同学要毕业了，开始感受到毕业的味道。今年，当初带他工作的学长学姐要毕业了，这是他最熟悉的一群人，也是他感受离别情最浓烈的时刻，“这一次轮到我来为他们做毕业晚会。”对黄炎康来说，大家都同在交大成长，团队的工作一代又一代接力下去，这份传承的情谊是永远不会变的。“我想帮助毕业生记住四年的大学时光，这也是我一直想做的事。”

　　今年毕业的王昱雯是一位颇有“创造101”风范的“漂亮小姐姐”。虽然念的是行政管理专业，但王昱雯的最爱还是跳舞，毕业晚会也是她在舞团的告别演出。四年来，她在“525社团”里和队友一起练舞、一起演出，而毕业季这场“大秀”是她对自己的告别，也是四年来社团生活的总结。在这次的排练过程中，她遇到了许多学弟学妹，他们帮忙拍照、制作宣传视频，一次次协助他们走台排练，令她非常感动。“希望学弟学妹们好好珍惜接下来的日子，也祝福他们能有越来越好的发展。”

　　晚会落幕时分，走出场外竟然还有一场惊喜。大学四年级的孙逸豪同学给自己的毕业季准备了一份“终极彩蛋”――就在会场门外，他向自己的女朋友求婚了。毕业时告别青涩，又迅速为自己开启下一段精彩旅程。这份坚定选择人生下一站的勇气和年轻人的大胆浪漫，实在令人感叹。

　　每年都会遇见毕业季。拥抱、不舍、离别，喜悦伴着泪水留在校园各个角落。每年也都会有懵懂的新鲜面孔，带着好奇与憧憬开启大学生活。新的友谊、新的目标、新鲜的热情不断在校园里传递。这是大学的魅力，这是青春的魅力，而下一个毕业季，又从这一刻开始倒计时。
