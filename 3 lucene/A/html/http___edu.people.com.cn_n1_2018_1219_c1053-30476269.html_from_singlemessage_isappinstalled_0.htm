庆祝改革开放40周年大会引发热议 32位大学书记校长这样说

2018年12月19日10:27  来源：人民网-教育频道
 

人民网北京12月19日电（李依环）庆祝改革开放40周年大会18日上午在北京人民大会堂隆重举行。中共中央总书记、国家主席、中央军委主席习近平在大会上发表重要讲话，引发高校教育工作者的热烈反响。

高校书记、校长一致表示，将以习近平新时代中国特色社会主义思想为指导，不忘改革开放初心，继续高举改革开放旗帜，坚定改革开放再出发信心和决心，为推进改革开放伟大事业、加快推进“双一流”大学建设、实现中华民族伟大复兴的中国梦而努力奋斗。

坚定改革开放再出发的信心和决心

习近平指出，改革开放是党和人民大踏步赶上时代的重要法宝，是坚持和发展中国特色社会主义的必由之路，是决定当代中国命运的关键一招，也是决定实现“两个一百年”奋斗目标、实现中华民族伟大复兴的关键一招。

中国人民大学党委书记靳诺表示，习近平总书记的重要讲话，深刻总结了改革开放40年来党和国家事业取得的伟大成就和宝贵经验，高度赞扬了中国人民为改革开放事业作出的杰出贡献，郑重宣示了改革开放只有进行时没有完成时、改革开放永远在路上的信心和决心，为新时代以新担当新作为推动改革开放再出发指明了前进方向、提供了根本遵循。

浙江大学党委书记邹晓东指出，总书记的重要讲话让我们更加充分地感受到党团结带领全国人民锐意进取、永不懈怠、勇立潮头的坚定决心和信心。浙江大学将在新一轮改革开放中，坚持用改革的思维、创新的举措、开放的视野，矢志立德树人，扎根中国大地加快建设中国特色世界一流大学，为新时代改革开放提供更加有力的人才、科技和思想文化支撑，为民族复兴和人类进步作出更加卓越的贡献！

“习近平总书记的讲话高屋建瓴、求真务实、直抵人心，是科学把握当今中国马克思主义的重要文献。”上海交通大学党委书记姜斯宪表示，学校将结合新时代建设中国特色世界一流大学的要求和上海交大的改革发展实际，增强深化改革开放的思想自觉、行动自觉，牢牢抓住培养社会主义建设者和接班人这个根本，为国家和民族广育栋梁之才。

“多年来，中国的经济与科技发展发生了翻天地覆的变化。回想起来，能够参与祖国改革开放的伟大历史进程，并为之做出自己的努力，非常有意义。”中国科学院院士、武汉大学校长窦贤康表示，未来，武汉大学将持续建设中国特色世界一流大学，致力于办成一所更加崇尚学术、更加追求卓越、更加关爱学生、更加担当有为的大学。

中国工程院院士、天津大学校长钟登华表示，改革开放40年来，中国教育呈现历史性变化，特别是高等教育不仅规模成为世界最大，而且在人才培养、科学研究、社会服务等方面也做出了巨大贡献。进入新时代，天津大学将落实立德树人根本任务，系统推进以“新工科”为牵引的人才培养模式改革，培养德智体美劳全面发展的社会主义建设者和接班人，为实现中华民族伟大复兴的中国梦贡献天大力量！

中国科学院院士、兰州大学校长严纯华表示，习近平总书记的讲话，全面论述了改革开放的重大理论和实践问题，是新时代指导推进我国改革开放的重要文献，为未来全方位推进改革开放、实现中华民族伟大复兴，提供了强有力的思想武器和行动指南。作为改革开放的先行者和受益者，教育工作者未来要做新旧动能转换的助力者和奋斗者，为国家培养大批优秀人才，打造大批改革事业的中流砥柱。

“作为改革开放的亲历者、受益者，我们见证了党在40年矢志改革中，团结带领全国各族人民披荆斩棘、砥砺奋进的光辉历程。”中国工程院院士、华中科技大学校长李元元指出，我国高等教育进入了以提高质量为核心的内涵式发展的新时期，要坚持社会主义办学方向，高举新时代改革开放旗帜，办好让党和国家满意、让人民群众满意、让广大师生员工满意的大学。

“整场大会既催人奋进，又令人心生感动。”南开大学党委书记杨庆山认为，高校是改革开放40年的重要参与者和生力军，要用改革开放铸就的伟大改革开放精神教育引导广大师生，坚定信仰、培塑信念、铸牢信心，厚植爱国主义情怀，实现爱党爱国爱社会主义相统一，激励师生在把新时代改革开放继续推向前进的伟大历程中，写下与国家民族同频共振的爱国奋斗新篇章。

“作为乘着改革开放东风成长起来的知识分子和教育工作者，对于改革开放带来的辉煌意义，感受更深、体会更切！”中国工程院院士、太原理工大学校长黄庆学表示，学校将高举中国特色社会主义伟大旗帜，将改革开放进行到底，在新时代创造新的一流业绩，为把学校早日建设成为高水平国际化创新型大学不懈奋斗。

中国科学院院士、西北工业大学常务副校长黄维表示，总书记的重要讲话，站在新时代、新格局、新定位、新征程的战略制高点，兼具理论深度、战略高度、历史广度、民族气度、全球维度、实践厚度，是继党的十九大报告之后，又一篇党的建设史上光辉的历史性文献，将会进一步指导我们党带领全国人民迈向“两个一百年”、实现中华民族伟大复兴的伟大实践。

伟大梦想是拼出来的、干出来的

“伟大梦想不是等得来、喊得来的，而是拼出来、干出来的。”习近平强调，改革开放已走过千山万水，但仍需跋山涉水，摆在全党全国各族人民面前的使命更光荣、任务更艰巨、挑战更严峻、工作更伟大。

“四十年的伟大实践坚实有力，四十年的伟大改革举世瞩目，四十年的伟大成就鼓舞人心。”西安交通大学校长王树国指出，站在新的起点，西安交大人将继续弘扬西迁精神，肩负新的历史使命，时不我待、敢为人先，以建设中国西部科技创新港为抓手，促进大学与社会的融合发展，探索中国特色世界一流大学新形态。

西北农林科技大学校长吴普特表示，面向新时代，西农要切实承担起党和国家赋予的服务旱区农业现代化的使命和担当，要敢于攻坚克难，以更大的政治勇气和智慧，不失时机地推进以质量为导向的内涵发展方式转变，为实施乡村振兴战略、建设美丽中国贡献西农智慧和力量。

中国科学院院士、西安电子科技大学党委书记郑晓静表示，40年来，我们党始终坚持对教育工作的领导，坚定不移地把教育放在优先发展战略地位，走出了一条中国特色社会主义教育发展道路，中国教育发生了历史性变革、取得了历史性成就。西安电子科技大学将积极稳妥推进各项改革任务，为建设世界一流大学给出西电方案、作出西电特色，为实现中华民族伟大复兴的中国梦不懈奋斗！

“40年来，北京邮电大学已经从传统的通信学院发展成为具有突出互联网特色的现代大学。”北京邮电大学校长乔建永表示，学校将进一步深化综合改革，牢记新时代中国特色社会主义大学的神圣使命，在抢抓信息化发展机遇，加快建设网络强国的时代征程中，弘扬改革开放铸就的伟大改革开放精神，不断贡献北邮智慧和北邮力量。

中国政法大学校长黄进表示，40年来，特别是党的十八大以来，我们党运用法治思维和法治方式深化改革，发挥法治对改革的引领和推动作用，确保重大改革于法有据。在全面深化改革总体框架内全面推进依法治国各项工作，坚定不移推进法治领域各项改革，以改革思维和改革方式推进法治中国建设，取得了巨大成就。中国政法大学将不忘初心、牢记使命，继续深化综合改革，扩大教育对外开放，为建设世界一流法科强校而努力奋斗。

“改革开放深深影响着全世界和每一个人。40年来，国家各项事业飞速发展、人民生活翻天覆地，从‘站起来’到‘富起来’到‘强起来’，也是我们教育工作者当仁不让的使命担当。”湖南大学党委书记邓卫表示，站在新的起点，湖南大学将始终坚持正确方向、坚持立德树人、坚持服务国家、坚持改革创新，为国家发展、人民幸福、人类文明进步作出湖大应有的新贡献！

“步入新时代，学校将不忘初心、牢记使命，持续推进改革开放。”北京科技大学校长杨仁树指出，作为新中国钢铁工业的最高学府，40年来，北京科技大学在助力“钢铁强国、科技兴邦”进程中取得了一定成就，未来也将积极探索特色发展道路，全面激发学校发展活力，争取早日建成国内一流、国际知名的高水平研究型大学。

“作为新中国第一所石油高校，学校坚持扎根中国大地，服务国家重大战略需求，与国家能源发展相依同行，成就了学校办学地位和特色优势。”中国石油大学（北京）党委书记山红红表示，今天受到隆重表彰的石大杰出校友、改革先锋“新铁人”王启民身上，正是体现了一代代石大人薪火相传的“实事求是、艰苦奋斗、爱国奉献、开拓创新”精神。

北京中医药大学党委书记谷晓红表示，作为新中国最早成立的高等中医药院校之一，40年来，学校在中医药教育、医疗健康服务、科学研究与技术应用、文化传播与国际化进程等方面勇于改革，不断开拓与提高，为中医药教育和卫生健康事业做出了积极贡献。今后也将全面深化教育教学改革，为实现高等教育内涵式发展，建设教育强国和健康中国作出新的更大贡献。

“总书记的讲话回顾了改革开放40年的光辉历程，动员全党全国各族人民在新时代继续把改革开放推向前进。”西北政法大学党委书记宋觉指出，全校各级党组织和各部门要积极组织师生深入学习贯彻，切实从思想和行动上坚决拥护改革、积极支持改革、自觉投身改革，不忘初心、砥砺前行，在大有可为的时代大有作为，交出无愧于历史和人民的满意答卷。

“改革开放以来的40年，是西南财经大学复校的40年，也是学校发展历史上最重要的40年。”西南财经大学党委书记赵德武表示，学校始终与时代脉搏同频共振、与国家发展同向同行，未来也将坚持改革创新、内涵发展，凝心聚力、攻坚克难，全面推进中国特色世界一流学科建设，全面开启国际知名财经特色鲜明高水平研究型大学建设新征程。

“改革开放给我们最大的财富就是要坚定改革发展信念，推动思想再解放、改革再深入、工作再抓实。”陕西师范大学校长游旭群指出，学校目前已进入加速建成以教师教育为主要特色的综合性研究型大学、开启特色鲜明世界一流大学新征程的重要阶段，未来将拼搏赶超，砥砺奋进，为实现学校的发展目标、实现中华民族伟大复兴的中国梦而奋斗。

实现中华民族伟大复兴要一棒接着一棒跑下去

习近平指出，建成社会主义现代化强国，实现中华民族伟大复兴，是一场接力跑，我们要一棒接着一棒跑下去，每一代人都要为下一代人跑出一个好成绩。

中央民族大学党委书记张京泽表示，40年来，党和国家在奋进中发展、在变革中新生，用40年时间实现了别人数百年的发展历程。站在新的起点，改革开放仍是中央民族大学未来发展进步的关键一招。各族师生只有抓改革的思路、制度、措施、效果，通过改革砥砺前行、顽强拼搏、接力奋斗，才能实现民大的“双一流”建设事业，才能在民族复兴的伟大征程中建功立业。

“有幸现场聆听了习近平总书记的重要讲话，深感能够成为改革开放事业的亲历者、受益者和建设者，我何其幸运，何其自豪！”中国工程院院士、北京交通大学校长宁滨表示，今后将带着这份自豪感，在人生路上继续奋斗，在实现中华民族伟大复兴的‘接力跑’中，不仅跑好自己这一棒，还要教育、培养更多优秀的接棒者，为创造中华民族新的更大奇迹做出贡献。

北京林业大学党委书记王洪元今年也参加了大会。“习近平总书记的讲话振奋人心。我国改革开放40年伟大实践雄辩证明，马克思主义具有强大生命力，中国共产党具有强大生命力，中国特色社会主义具有强大生命力，中华民族具有强大生命力。”会后接受采访时，王洪元激动地说。

“中央财经大学40年不断改革进取和开放发展的历程，也是中国高等教育改革发展史的缩影。”中央财经大学校长王瑶琪表示，站在改革开放再出发的重大历史时点，学校将抓紧机遇、奋发有为，扎根中国、融通中外，立足时代、面向未来，通过双一流建设工作，作出中央财经大学应有的贡献。

长安大学校长陈峰表示，中国高等教育已经站在新的历史起点上，开启了加快推进教育现代化、建设教育强国、办好人民满意的教育新征程。长安大学将继续深化教育改革开放，努力写好教育奋进之笔，培养德智体美劳全面发展的社会主义建设者和接班人，为实现中华民族伟大复兴的中国梦作出新的更大贡献。

“总书记是当之无愧的新时代领路人，将继续带领中国人民靠‘重要法宝’和‘关键一招’赢得明天。”今年也适逢中国矿业大学在徐州建校40年，中国矿业大学党委书记刘波表示，中国矿大始终与改革开放同向同行。面向未来，学校将弘扬敢闯敢试、敢为人先的改革精神，一代接着一代跑好立德树人的接力赛，为把这所拥有109年历史的高校，建成能源资源特色的世界一流大学而不懈努力。

“我校因贸易强国而生，因对外开放而兴，因全面深化改革而盛，因新时代对外开放事业而新。”对外经济贸易大学党委书记蒋庆哲表示，进入新时代，学校将持续服务国家开放大局，努力培养能够担当民族复兴大任的国际化领军人才，朝着建设以开放型经济学科群为引领的特色鲜明世界一流大学踏步迈进。

哈尔滨工程大学校长姚郁指出，学校将立足自身办学定位、特色和服务领域，紧密围绕一流大学建设的时代命题，围绕服务支撑“三海一核”领域的国家战略与助力龙江发展的战略任务，在推进学校综合改革中“大胆的试、勇敢的改”，找准切入点、结合点、服务点和贡献点，汇聚起推进制造强国、网络强国和海洋强国建设的磅礴力量。

“中国的改革开放深刻地改变了中国，也深刻地影响了世界。”中国民航大学党委书记景一宏表示，全校师生要紧跟党的领导，坚定“四个自信”，学习习近平总书记在本次大会上提出的新目标和新要求，结合学校的实际工作落实贯彻，为实现民航强国目标、实现中华民族伟大复兴的中国梦贡献力量。

“40年来，学校已从一所学科专业相对单一的行业高校，发展成为地学类人才培养规模居全球高校首位，地球科学一流、多学科协调发展的高水平大学。”中国地质大学（武汉）校长王焰新指出，学校将以更加开阔的视野、更加开放的心态、更加强劲的举措，全面落实立德树人根本任务，全面推进地球科学领域国际知名研究型大学建设。

“百十五载，一代代工大人不忘立德树人的使命初心，为国家经济社会发展、区域经济协同发展做出了突出贡献。”今年也是河北工业大学建校115周年，校长韩旭表示，站在继往开来的崭新起点，学校将哿ν心，锐意进取，在建设世界一流学科、实现中华民族伟大复兴的中国梦的征程中继续书写工大人更加昂扬激越的时代华章。
