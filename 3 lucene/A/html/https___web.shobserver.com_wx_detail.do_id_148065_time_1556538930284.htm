24小时开放，全年无休，上海交大新建的学生创新中心很拼！

见识录 2019-04-29 17:13
来源：上观新闻 作者：赖鑫琳
交大闵行校区的学生创新中心目前已建成机器人、无人机、IT和智能制造4个开放实验室……

这一科创活动“航母级”平台的工作模式，被大学生戏称为“007”，也就是从当日零点到次日零点，并且一周七天无休。

4月29日，记者走进新装亮相的上海交通大学学生创新中心，这里的三大幢建筑物7×24小时向学生开放，提供专业实验场地、全面工程服务、多样科技竞赛和前沿企业课程，成为“科技领军人才的练武堂”、“工业界和大学的握手区”。

位于闵行校区的学生创新中心目前已建成机器人、无人机、IT和智能制造4个开放实验室，并下设模型、3D打印、切割和设备借用等分中心，鼓励学生自主设计模型并加工，还购置了大量工程正版软件，一期已上线软件工具17款。

通过与英特尔、IBM、微软、腾讯、霍尼韦尔等70多家企业合作，中心聘任了120多名企业导师，开设企业课程近30门。
