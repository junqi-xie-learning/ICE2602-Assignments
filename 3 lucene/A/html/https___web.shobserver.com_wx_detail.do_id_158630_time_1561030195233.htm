来！认识一下这位上海的全国十大“最美高校辅导员” ，他是《钱学森》背后的人

教育在线 2019-06-20 19:29
来源：上观新闻 作者：徐瑞哲
辅导员、思政课教师、学生话剧社团指导老师、校辩论队总教练、校艺术教育和高水平艺术团负责老师……这些，都是汪雨申的多重身份标签。

今年新中国成立70周年，也是“五四”运动百年。教育部首次从历届在岗的“高校辅导员年度人物”和在读的“大学生年度人物”中，再次评选出10名“最美高校辅导员”和10名“最美大学生”。优中选优，来自上海交通大学的汪雨申，日前荣膺全国这十大“最美高校辅导员”的殊荣。

上个月，汪雨申正在西安参加中国高教学会美育专业委员会的学术会议，白天会议结束后，他不忘和上海交大学生话剧社团的学生远程连线，指导他们准备大型原创话剧《钱学森》的新生季演出。原来，汪雨申是《钱学森》一剧背后的人。他组织排演的《钱学森》自2012年公演以来，在校内以及全国巡演13轮39场，观众人数累计超过6万余人，曾获中国校园戏剧节最高荣誉。

【“一部好话剧，就是一堂课”】

汪雨申与学生在一起

“我的祖国给了我生命和智慧，如果我的爱能对祖国有一点点贡献，我将百死不辞，毕生有幸！”原创话剧《钱学森》全国巡演期间，剧中这句台词一度在大学学子中成为流行语。这部由上海交大师生根据新中国“两弹一星”元勋、交大1934届校友钱学森生平事迹创作而成的“热剧”，带给年轻人心灵的触动和震撼，而它的幕后制作人就是当时上海交大校园里一名普通的辅导员汪雨申。

自2001年留校任教，汪雨申老师在这条文化育人的路上一走就是18年。“与其理性地说教，不如感性地示范。”汪雨申说，“什么是最有效的思政教育方法？想必我已经找到答案，一部好话剧，就是一堂德育课！”去年，他又指导创排了医学生题材的原创话剧《白衣赤诚》，把握时代脉搏，聚焦青年成长，随风潜入夜，润物细无声。

汪雨申与学生在一起

辅导员、思政课教师、学生话剧社团指导老师、校辩论队总教练、校艺术教育和高水平艺术团负责老师……这些，都是汪雨申的多重身份标签。从教以来，他“身兼数职”，忙并快乐着。这个文艺青年带教更多充满热血的“文青”，由他指导的“阳光剧社”、“思辩学社”、“吉他协会”等学生社团，连续多年获评上海交通大学五星社团。

【学生的事就是“汪sir”的事】

汪雨申给学生上思政课

“汪sir（先生）”是学生们对汪雨申老师的亲切称呼。在学子心中，他不仅为学生传道授业解惑，更愿意随时与学生分享喜怒哀乐。“周六保证不休息，周日休息不保证”，汪雨申的手机24小时为学生开机——学生半夜生病，他第一时间赶到医院；学生成绩下滑，他促膝谈心、对症下药；学生感情受挫，他细心开导、化解心结……谆谆如父语，殷殷似友亲，学生的事就是“汪sir”的事。人文学院一位毕业校友这样评价：“‘汪sir’像我们的兄长、老师、引路人，我们从他那里学到了很多。”

经师易求，人师难遇。除了在生活中对学生关爱有加，汪雨申还十分注重全方位育人。他担任交大青年马克思主义党校主讲教师为学生党员开党课；为港澳台学生开设关于上海历史变迁的专题报告；为国际与公共事务学院学生开设演讲与辩论专题课程；还担任《思想道德修养与法律基础》、《形势与政策》、《社会实践》、《艺术理论与实践》等课程的主讲教师。

汪雨申与学生在一起

给学生一杯水，自己就要有一桶水，其实汪雨申还是《平易近人——习近平的语言力量》一书作者之一。他独立负责“俗文俚语”篇的撰写，在自我学习总书记讲话精神的同时，不忘饮水思源，从语言文化角度为大学生提供了读得进、记得住、用得上的读本。

【为留学生讲好中国故事】

汪雨申指导学生排练话剧


汪雨申的辅导员工作对象，除了中国学生，还包含2000多名留学生。他指导创建的“仁·闻”网络文化工作室，在工作室微信公众平台上以“语伴”为主要版块，面向全体交大学生发布语伴信息，既解决留学生的语言学习问题，也为中国同学提供了很好的中外交流机会。

果不其然，在汪雨申支持和帮助下，2018届挪威留学生欧阳海在全校评选中获得“上海交通大学校长奖”，并立志做一名在挪威和中国之间架桥的友谊使者；2017级意大利留学生马克也是如此，从一名五星级酒店的大厨转变成一名研习汉语的大学生，不仅在央视“汉语桥”节目上有着精彩表现，去年还获上海交大留学生之星和“感动校园”人物称号，立志要为意中世代友好做当代马可·波罗。

汪雨申与学生在一起

“我们组织了各种各样的中国文化体验活动，让留学生了解中国、喜爱中国，并逐渐成为知华、友华的国际使者，” 汪雨申告诉记者，“只要他们用自己的亲身经历感受着中国，既会讲好中国故事，也会向世界传递出中国的声音。”
