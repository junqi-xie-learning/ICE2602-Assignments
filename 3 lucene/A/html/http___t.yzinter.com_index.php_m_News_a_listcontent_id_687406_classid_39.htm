原创|交大校友企业已深入到AI+各领域

2019-03-25 20:41:56

交大校友发言

扬子晚报网3月25日讯（记者 罗晓娜）对接国家发展战略和上海未来产业发展布局，依托上海交大的优势学科，加快行业性校友组织的建设，上海交大校友分会先后成立了人工智能、集成电路、医疗器械等重点行业的分会，促进校友之间的信息互通、资源共享、产业合作，助力校友和校友企业的加速成长，也支撑上海相关重点产业的发展。

今年初成立的人工智能行业校友会把众多交大人工智能领域的校友凝聚起来，促进相关领域的校友的深度合作，促成更大的事业发展，也为上海的人工智能产业建设成为世界级产业群做出贡献，去年在美国纽交所和纳斯达克上市的两家上海企业都是AI领域，也都是交大校友企业。

一大批交大校友，特别是很多青年校友抓住了时代机遇，勇于创新创业，在人工智能领域彰显了交大影响。流利说、商汤、依图、触宝科技、云天励飞、第四范式等等，众多人工智能领域的头部企业都是交大校友创办；事实上，交大校友企业已经深入到AI+的各个领域，在教育领域的流利说、乂学教育，在人脸识别领域的商汤科技、依图科技、云天励飞，医学影像大数据挖掘的健培科技和让机器读懂病历森亿医疗科技等助力医疗健康产业发展，在通用平台和数据发掘领域的第四范式、极熵数据，在生活服务方面的助理来也，文本智能处理方面的达观数据等等，众多的“交大系”人工智能公司已经下沉到各个AI+的各个应用场景。编辑：李燕
