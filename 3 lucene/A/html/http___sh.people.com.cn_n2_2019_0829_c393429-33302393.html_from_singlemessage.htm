中国城市治理研究院副院长陈高宏讲授“城市精细化管理的实践与思考”

2019年08月29日16:05  来源：人民网-上海频道
 

人民网上海8月29日 （董志雯、陆嘉呈）8月26日到30日，由中央组织部与民政部联合举办、上海市委组织部承办的全国城市基层党建引领基层治理培训示范班在上海浦东新区区委党校举行。

随着社会的快速发展与进步，城市管理在提升城市综合竞争实力方面发挥着越来越突出的影响。但城市管理水平和运行效率如何与时俱进，如何通过精细化管理提高城市管理效率成为一个重要课题。8月29日上午，上海交通大学中国城市治理研究员副院长陈高宏以“城市精细化管理的实践与思考”为主题，给学员带来一场精彩的讲座。

2017年3月5日，习近平总书记在参加全国人大上海代表团审议时指出，走出一条符合超大城市特点和规律的社会治理新路子，是关系上海发展的大问题。城市管理应该像绣花一样精细。城市精细化管理，必须适应城市发展。要持续用力、不断深化，提升社会治理能力，增强社会发展活力。

陈高宏用浅显易懂的语言从理解精细化管理的重要意义、明确精细化管理的基本要求、提高精细化管理的综合素质三方面进行分析和解读，引起了学员的极大兴趣。

对于城市精细化管理的意义，陈高宏指出，城市精细化是城市化的前进方向，是新时代的现实命题，是一流城市的卓越标志。从城市发展的历史来看，城市的发展也经历了一个从粗放到精细的过程，从最初仅仅只是建城，到之后强调建管并举，管理在城市治理中所占的分量越来越重。上海这样的特大城市，城市管理者面对的问题多且广，城市管理必须细化到每一个末端。换言之，对城市末端的管理水平，体现了对城市真正的管理水平。

陈高宏认为，城市精细化管理的基本要求分为六点：一是以人为本的导向；二是多元协同的主体；三是量化专业的标准；四是法治规范的依据；五是长效常态的机制；六是科学智能的手段。

城市精细化管理水平很大程度反映着政府的治理水平，城市精细化管理质量很大程度影响着居民的生活质量，城市精细化管理能力很大程度关系着区域的竞争能力。如何提高精细化管理的综合素质？陈高宏总结为四个能力：耐压脱敏能力、互动博弈能力、预警纠偏能力、平衡整合能力。

陈高宏指出，现代社会，是一个信息化的社会，也是一个多元化的社会。面对复杂多变的群体利益，如何平衡各方面的要求，如何协调整合各方面的利益，如何应对突发的风险。种种问题都对新时代的管理者提出了更高的要求。陈高宏认为，新时代的城市管理者首先要有担当精神，要能顶得住压力，听得进闲言碎语，经得起冲击，俯得下身子。在多元利益格局下，有所担当才能有所作为。同时，要懂得科学的协调，处理，平衡各方的利益。要引导各方把各自的诉求合理表达出来，再科学的予以协调最终达成一致。陈高宏还指出，复杂的现实必定会带来许多的不确定性，现代的城市管理者一定要有如临深渊，如履薄冰的意识。要增加底线思维，随时准备打好化险为夷，转危为安的遭遇战。

陈教授的授课既有理论分析，又有对具体工作的实际指导，让学员们听的十分解渴。学员们纷纷表示，陈院长的讲座思路清晰，内容丰富，深入浅出，案例生动，对于今后工作开展具有非常实用的指导作用。 讲座结束后，陈教授与学员们进行现场互动，就学员们关心的一些热点问题进行了探讨交流。（摄影：梁晶）

(责编：严远、韩庆)
