AI创业界有个“交大帮”：他们是些什么人，要些什么人？

教育在线 2019-03-26 18:20
来源：上观新闻 作者：徐瑞哲
去年，他们获得国家发改委“人工智能创新发展重大工程”专项，成为科大讯飞之外另一个智能语音技术国家队；今年，他们携手中芯国际注资成立半导体企业，在张江走上了软硬兼修的AI造芯之路。

人工智能企业创始人，有什么共同点？从流利说、商汤、依图、触宝科技、云天励飞到第四范式等，其中包括去年在美国纽交所和纳斯达克上市的两家上海企业，它们都是人工智能（AI）领域的头部企业，也都是上海交大校友参与创办的企业。

 

曾几何时，AI创业界出现了一种“交大帮”现象。这些圈内领军人才是什么样的人，喜欢什么样的人，相互之间又有何种学科渊源？解放日报·上观记者日前探寻了一番究竟。

 

 

 

“同学同学”，你我皆同学

 

去年9月27日，“流利说”在纽交所敲响上市钟声；无巧不巧，就在次日，“触宝科技”也紧接着在美上市。有意思的是，这两家AI企业可谓“同学企业”，创始人都有同校甚至同院学习的“交集”。

 

 

作为“流利说”的3位创始人之一，身兼首席技术官的胡哲人在2002年考入上海交大，他与触宝的首席执行官王佳梁都是电子信息与电气工程学院的同学。当年校园里，一个在读本科，一个在读研究生。2005年到2006年，两人一前一后从交大毕业。2008年，触宝成立；2013年，流利说上线。

 

“我当年读过ACM班，”胡哲人说的这个国际化试点班，是“ACM国际大学生程序设计大赛”的校内特班。以同班学子为班底，交大ACM队曾在世界上6度斩获大赛金牌、3次将全球总冠军带回中国。如今创业圈里，胡哲人在学校里就认识的老同学还真不少。

 

胡哲人认为，当年人工智能的应用领域和前途前景还不十分清晰，但学校已然开始采用“实验班”、“创新班”、“联合班”这样的培育模式，前瞻性地培养前沿人才。“十多年后，终于纷纷生根、开花、结果。”王佳梁告诉解放日报·上观新闻记者，在他眼中，交大同学的风格都“很务实”，不玩概念，这同样也是他们这批人成功创业的“画风”。

 

 

“同学同学”，学长招学友

 

眼下，春招进行时。与往年一样，交大创业者也招学弟学妹。“英雄莫问出处，当年学校还没有开设什么创业教育模块。”作为学长，王佳梁学电子专业出身，创业则从触宝输入法起步走向全球化的手机内容分发平台。他对创业人才的专业背景倒不在意，而更看重新一代学友是否拥有扎实的数理基础以及思辨能力，“没有经验也无所谓，更重要的是创新力和使命感。”

 

 

在面试中，如果应聘者涉猎甚广，除了编程，还爱文学，又会乐器，那绝对是“加分项”。王佳梁认为，这种跨学科的综合素养，正是潜在的创新点。另外，如果有人喜欢搞或曾搞过学生社团，也可以说明他或她希望创造额外的价值，而不仅仅是为了一份工作，“这是创业企业最需要的价值观。”

 

在互联网这个世界里，总能“回首又见他”。“饿了么”创始人张旭豪也是上海交大校友，而且属于“同龄创业人”。在去年阿里巴巴对其并购之后，饿了么的人才形成外溢效应。胡哲人向解放日报·上观新闻记者透露，“流利说”也引进了其中一部分人才，“他们是企业高速成长期间涌现的人才，非常适用当下之需。”胡哲人认为，这表明人才本身是企业成长的加速器，也使得上海作为创新创业中心的人才密度快速提高，而且在同行企业之间、上下游产业链上，出现了可观可喜的人才联动效应。

 

 

 

“同学同学”，学校聚校友

 

事实上，众多的“交大系”人工智能公司已经下沉到AI+的各个应用场景。商汤科技的联合创始人兼CEO徐立、第四范式的创始人兼CEO戴文渊、依图科技的联合创始人林晨曦、思岚科技的创始人兼CEO陈士凯、云天励飞联合创始人兼CEO陈宁等，都是一批AI创业校友及教师。教育领域，有流利说、乂学教育；健康领域，有医学影像大数据挖掘的健培科技、让机器读懂病历的森亿医疗科技；还有通用性平台的极熵数据，生活服务类的助理来也，文本智能化的达观数据，等等。

 

 

“正如业内一句话：有多少人工，才有多少智能。”上海交大人工智能研究院常务副院长杨小康告诉解放日报·上观新闻记者，上海交大校友会人工智能分会今年宣告成立，进一步助推人工智能行业校友之间、校友企业之间、校友与母校之间的互动与合作。AI“交大帮”有了归属感，也成了AI教育与实践的嫡系导师，共同构筑人工智能学科发展新高地。

 

 

吴梦玥就有产教结合的双重身份，她既是思必驰上海交大智能人机交互联合实验室助理教授，也是上海交大计算机系助理教授。去年，他们企业获得国家发改委“人工智能创新发展重大工程”专项，成为科大讯飞之外另一个智能语音技术国家队；今年，他们携手中芯国际注资成立半导体企业，在张江走上了“软硬兼修”的AI造芯之路。她说，思必驰的智能语音交互技术脱胎于上海交大，出自企业创始人兼首席科学家俞凯教授，“AI创业人才应当工程能力与科学思维得兼，在校培养，送往企业。”
