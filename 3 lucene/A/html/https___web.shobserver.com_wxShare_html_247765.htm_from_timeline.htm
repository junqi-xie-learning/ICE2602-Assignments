上海这所著名高校多了无人值守“书店”，如需购买自助扫码

文化观澜 2020-05-14 17:01
来源：上观新闻 作者：施晨露
“初夏花开，专辟一隅，展售精品，传播文化，博雅君子，扫码购读”

5月14日，新生态阅读空间“菊生书苑”在上海交通大学正式开放。“菊生书苑”由上海交大图书馆与交大出版社共同建设，在交大图书馆内专门辟出开放性空间，展陈交大出版社的精品图书，不定期开展阅读推广活动，并以“诚信购书”理念，开通无人值守扫码购书功能。

“初夏花开，专辟一隅，展售精品，传播文化，博雅君子，扫码购读”，在典雅的“菊生书苑”褐色铭牌上，不仅揭示了这一空间的功能，也解释了命名由来。我国近代著名出版家张元济字筱斋，号菊生，曾任上海交大出版社前身——南洋公学译书院首任院长。上海交大出版社社长李芳介绍，“菊生书苑”寓意秉承先贤的精神和传统，在知识分享中实现价值引领，在信息传播中坚定文化自信。

位于交大闵行校区图书馆主楼一楼的“菊生书苑”面朝树荫，绿意从大片落地玻璃门窗沁入，与靠墙成片落地书架的书香相得益彰。这里是精品图书的展示空间，展售图书定期更换，首批包括《査医生援鄂日记》《新冠肺炎防治精要》《人类未来》《天生我材》《工业人工智能》《东京审判——为了世界和平》《全球视野下的中国文学系列》《信息经济学》《古典诗词鉴赏与创作》《漫画脑》等数百种。交大出版社专门制作了好书推荐牌，知名专家或资深编辑的推荐语将为读者选书指点迷津。

“菊生书苑”所有图书均供师生、读者免费取阅，如需购买可自助扫码。换句话说，这是一个依赖诚信售书的空间。借助出版社和高校资源，在这里不定期举办的读者见面会、小型讲座，将让读者与好书背后的人充分交流、互动。作为走进图书馆、走进校园的新尝试，交大出版社还有意探索可复制推广的模式，让“菊生书苑”走进其他大学和图书馆。

揭牌仪式后举办的“书之道·交大读书会特别活动”《科技成就中国》作者见面会上，该书作者、上海交大马克思主义学院副教授黄庆桥与现场数十位读者分享了中国科技发展的感人故事，回顾中国科技发展历程、理解中国科技崛起之路。

5月下旬，“菊生书苑”将邀请《点赞青年英雄》作者、上海交大马克思主义学院教授施索华为读者上一堂“微党课”，解读习近平总书记在纪念“五四运动”100周年大会上的重要讲话中列举的马克思、恩格斯等青年英雄以及嫦娥、神舟、北斗等青年英雄团队的精神品格。
