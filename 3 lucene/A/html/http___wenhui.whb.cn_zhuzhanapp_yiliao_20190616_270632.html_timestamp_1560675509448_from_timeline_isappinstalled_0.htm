“全球健康学院”成立！交大医学院牵手中国疾控，首任院长高福：快乐与长寿值得研究
2019-06-16 15:01:18 作者：唐闻佳

▲ 国家卫健委副主任曾益新、中国疾病预防控制中心主任高福、上海交通大学党委书记姜斯宪、上海交通大学医学院党委书记范先群、中国疾病预防控制中心寄生虫病预防控制所党委书记陈晓红、中国科学院北京生命科学研究院院长康乐共同为“全球健康学院”揭牌

在参与全球健康治理领域，上海迈出崭新一步，一座全新的高水平“健康学院”在沪成立！

今天，由上海交通大学医学院、中国疾病预防控制中心寄生虫病预防控制所（国家热带病研究中心）共同主办的“全球健康论坛”上传来消息，上海交通大学医学院和国家热带病研究中心携手成立“全球健康学院”，首任院长为中国疾病预防控制中心主任、中国科学院院士高福，中科院院士陈竺担任学术委员会主任。

这是上海首个以学院命名的全球健康机构，合作双方约定通过学科建设、科学研究、人才培养等领域的强强联手，积极参与全球健康治理。

伴随全球化的快速发展与大势所趋，全球性挑战日益增多，加强卫生与健康领域的全球治理是大势所趋，催生了“全球健康” （global health）这一概念。

作为新兴国家和发展中国家的代表，中国已成为世界卫生领域的重要行为体，上海交通大学医学院-国家热带病研究中心“全球健康学院”应运而生，瞄准了积极推动我国全球健康学科发展、加快全球健康人才培养的建设目标和发展重任。

▲ 在国家卫健委科教司副司长吴沛新、中国疾病预防控制中心副主任冯子健、上海市卫健委副主任衣承东、上海市教育科学研究院院长桑标、上海交通大学医学院党委书记范先群、中国疾病预防控制中心寄生虫病预防控制所党委书记陈晓红等的共同见证下，中国疾病预防控制中心寄生虫病预防控制所所长周晓农和上海交通大学医学院院长、中科院院士陈国强签署合作协议

“全球健康学院”将依托上海交通大学医学院医学科学领域的雄厚实力和上海交通大学多学科的综合优势，整合中国疾病预防控制中心在热带病防控领域的优势力量和全球分布的国际合作网络，以国际一流全球健康学科为发展目标，培养具备扎实学科基础，创新思维、全球视野的全球健康治理的领军人才；共同在全球热带病防控、生命全周期健康、全球卫生政策与治理等方面，推动全球健康学科的发展，促进中国方案、中国产品更好地为全球人类健康服务，为我国参与全球健康治理提供有力支撑。

陈国强院士称，“全球健康学院”将率先重点聚焦热带病与突发传染病、全球卫生政策与标准、全球妇幼卫生、全球营养与食品安全、全球精神卫生这五大发展方向。

根据合作框架，在学科建设与科学研究方面，双方共同推动“国家热带病研究中心”“全球健康学院”的建设，使我国更多科研成果推广应用；在人才建设方面，共同开展热带病和全球健康研究领域人才引育工作，筹建全球健康研究生学位教育，并构建全球健康继续教育平台；在国际交流方面，依托现有海内外基地（网络）等各类合作关系，在海内外合理布局协同创新网络。

合作双方表示，将认真研究学院的组织形式和运行机制，出台一系列有利于学院发展的政策方案，给予学院发展空间、经费投入、人才引育等方面的政策支持，探索打造“学术特区、人才特区”，保证这座全球健康学院的可持续发展；并将积极拓宽国内外合作，从关注“全球人类健康”向“人-动物-农业-生态系统”的“大健康”概念发展，积极拓展与世界卫生组织、联合国粮食及农业组织、世界动物卫生组织的合作，在推动全球健康、全球卫生治理上形成多方合力。

“全球健康不仅是地域的概念，因为疾病包括传染病和慢性非传染性疾病从来不遵守国界，它还是一个生物圈的概念。”首任院长高福谈到对全球健康学院的愿景时称，希望目光不止步于当下，而看到下一代乃至下下一代，希望这座学院的研究能着眼未来20年、50年。

他举例谈到精神健康问题，希望不要等许多人都被抑郁所困扰，科研人员才大量投身其中。

“比如，快乐与长寿是十分值得研究的课题。到底如何保持快乐，如何延缓衰老，遗传因素、微生物组、饮食、慢性感染等因素到底再其中扮演什么角色，研究领域尚有许多空白等待我们去填空，找答案。”

中国科学院院士、全球健康学院学术委员会主任陈竺特意发来祝贺视频。

陈竺谈到，近些年来，在世界卫生组织推动下，在联合国千年发展目标和联合国2030可持续发展议程的推动下，全球卫生合作风起云涌、势不可挡，“全球健康学院”的成立是合作双方为人类命运共同体，为“健康中国”建设、为“健康丝绸之路”作出重要贡献的一项意义深远举措，体现了双方领导和专家们的远见卓识，希望中国模式、中国智慧在全球化进程中更好地给全人类带来福祉。


链接：热带医学、热带病研究、国家热带病研究中心

热带医学是研究热带地区的各种感染性疾病（热带病）的诊断、治疗和预防以及如何控制和消灭这些疾病的科学。

热带病狭义上主要是指发生在热带或亚热带地区的常见多发的感染性疾病，多指传染病、寄生虫病和虫媒病等。

自十九世纪英国籍Sir Patrick Manson开创热带医学学科至今，热带医学一直与国际健康后称全球健康息息相关，是其最为重要的构成部分。

为使我国热带病防治研究成果及相应技术服务于长三角、全国的疾病控制工作，并将中国经验推向全球，时任卫生部陈竺部长和上海市韩正市长于2011年7月共同签署《卫生部 上海市人民政府共建共管在沪卫生部管理单位补充协议》，明确在“十二五”期间，卫生部和上海市共同筹建“国家热带病研究中心”，即以中国疾病预防控制中心寄生虫病预防控制所为核心单位，联合在沪高校及相关资源，形成集科学研究、教育培训、国际交流、疾病临床治疗与防控于一体的国家级热带病研究中心。

经过长期的筹备与申请工作，中编办于2017年正式批复同意中国疾病预防控制中心寄生虫病预防控制所加挂“国家热带病研究中心”。

中国疾病预防控制中心寄生虫病预防控制所（简称“寄生虫病所”）是全国寄生虫病预防控制工作业务指导中心。此次寄生虫病所与上海交通大学医学院合作共建“国家热带病研究中心”，使之成为与英国伦敦热带医学与卫生学院、瑞士热带病与公共卫生研究所等国际一流单位相提并论的热带病研究中心。
