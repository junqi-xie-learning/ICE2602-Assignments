上海交大师生热议习近平总书记在纪念五四运动100周年大会上的讲话

2019年04月30日21:21  来源：人民网-上海频道
 

4月30日上午，纪念五四运动100周年大会在人民大会堂举行。中共中央总书记、国家主席、中央军委主席习近平出席大会并发表重要讲话。上海交通大学师生通过电视、广播、网络等收听、收看，热切关注大会内容，结合自身学习、工作和生活实际，交流感想。

上海交通大学环境科学与工程学院教授孔海南表示：习近平总书记在讲话中提到“五四运动以来的100年，是中国青年一代又一代接续奋斗、凯歌前行的100年，是中国青年用青春之我创造青春之中国、青春之民族的100年”。五四运动倡导的爱国、进步、民主、科学精神，至今仍是激励我们的宝贵精神财富。青年时期的我选择赴日深造，刻苦钻研最先进的水体污染治理技术。2000年，我选择回国，用自己的知识和积累来治理洱海、报效祖国。十几年间，许许多多的青年人加入了我们的队伍，与我们一起扎根洱海，为守护家园的理想情怀而奋斗，将个人的理想融入国家发展和民族复兴的伟大理想。

上海交通大学马克思主义学院教授胡涵锦表示：总书记的重要讲话，寓意深刻，语重心长，娓娓道来，好似与青年朋友促膝谈心，有很强的思想性、理论性和亲和力、针对性。作为思政课教师，我们深受启发。历史和现实一再表明，青年象征着未来，赢得了青年，就赢得了未来。我们要扎实推进习近平新时代中国特色社会主义思想的“三进”工作，引导广大青年学生坚持党的领导，听党的话，坚决跟党走，在实现中华民族伟大复兴中国梦的征途上，谱写壮丽青春篇章。

上海交通大学马克思主义学院青年教师刘伟表示：作为新时代青年，我们要树立远大理想，用中国梦激扬青春梦，勇挑时代重任不畏艰难，为祖国和人民建功立业。作为新时代青年，我们要练就过硬本领，不辜负伟大的时代，把握人生的黄金时期，在增长知识和实践作为书写“中国梦”，以真学实干描绘青春最靓丽的底色。作为新时代青年，我们要勇于砥砺奋进，在爱国奉献中实现人生价值，用青春之我创造青春之中国，让青春在不懈奋斗中闪光。

上海交通大学机械与动力工程学院团委书记王国锋表示：如今，历史的接力棒传到新时代的青年人手中，新时代中国青年处在中华民族发展的最好时期，我们应该珍惜这个时代的际遇和机缘。作为院系团委书记，应该将联系青年作为主要工作，主动走进青年，在与青年的交流沟通中了解青年诉求，发现问题所在，在青年成长路上注重过程引导，做好青年的知心人、热心人和引路人。

上海交通大学材料科学与工程学院团委书记朱彦彦表示：不忘本来才能开辟未来，善于继承才能更好地创新。新时代我们应继续传承五四精神，结合当代青年学生奋斗主题和责任担当，为中华民族伟大复兴而不懈努力。材料科技的进步是国民经济发展的基础，创新材料研发是材料学子所肩负的使命。作为高校基层团干部，今后我将发挥好党联系青年的桥梁纽带作用，锤炼真本领，将论文写在祖国大地上，将科技报国印入当代材料学子心中。

上海交大-南加州大学文化创意产业学院思政教师热米拉・阿扎提表示：作为学习、工作和生活在内地十余年的维吾尔族青年，与各民族青年一起奋斗的场景如今俨然成为常态。各族青年在这片充满生机的沃土上，在祖国最需要的地方，谱写各自青春的绚丽篇章，为了共同的中国梦而奋斗。我始终认为，作为一名生活在美好新时代的青年，既要有对家乡饮水思源的情感与责任，更要有翱翔海阔天空，以促进各民族交往交流交融，筑牢中华民族共同体为己任的使命感。责任与担当，二者永远是辩证统一的，没有轻重，没有取舍的先后。这是不忘初心，更是牢记使命。

上海交通大学中美物流研究院2018级硕士研究生、上海交通大学学生联合会副主席池松恒表示：面对新时代，我们当代青年，必将按照总书记的谆谆教诲，用马列主义的崇高信仰、对社会主义的坚实信念、对伟大复兴的坚定信心树立人生的远大理想；用一辈子的时间，在对祖国和人民的奉献中表达对党和祖国的热爱；用刚健勇毅的担当精神迎难而上挺身而出，勇往直前不惧艰险，担当起属于我们的时代责任。我们将从小事做起，勇于砥砺奋斗，珍惜青春韶华，孜孜不倦地学习知识，锤炼品格，增长才干，练就本领。将永怀饮水思源的感恩之心，明辨笃实外修内省，开拓眼界突破局限，情系家国心怀四海，增强“四个意识”，坚定“四个自信”，牢记“两个维护”，接过先辈的接力棒，跑出更好成绩，实现我们青年一代作为社会主义建设者和接班人的人生价值。

上海交通大学媒体与传播学院2015级本科生舒天楚表示：在历史大浪淘沙的轨迹中，青春奋斗正是中国精神、中国力量的生命力所在，我们的成长期、奋斗期与“两个一百年”目标高度重合，“强国一代”的我们，更要以青春之我，不怕艰险、排除万难、勇于担当，投身改革开放大潮。“爱国是本分，是职责，是心之所向，情之所系”，这句话让我尤为触动。作为高校青年学生，要在更多生动的社会实践中深入观察社会、形成正确的价值观念和理性全面的社会观察，让爱国成为引领人生的航向标。“党旗所指就是团旗所向”，作为一名团学工作的学生骨干，我们首先要自己学深学透，深刻体会和领悟讲话精神，引领青年大学生坚定信念跟党走，努力成长为第二个百年目标的中流砥柱。

上海交通大学农业与生物学院 2017级本科生杨浩表示：作为“00后”的新生一代，在这样契机多样的时代背景下，我们“00后”更加多元与包容，每个人都对这个世界有着多样化的特别理解。但是，趋于多样的我们也有着相同的共同追求。不仅要努力奋斗实现自我的人生价值，还肩负着国家与时代赋予的使命与责任。作为新时代青年的我们，要用自己的行动将五四精神传承下去，用自己的行为弘扬爱国主义传统，响应时代的号召。明确自己身处的新时代历史方位，努力扩展自己的通识知识背景与深化自己的专业素养，丰富个人的阅历，为将来投身社会主义现代化强国建设作准备。

2019年是五四运动100周年，依托此重要契机，共青团上海交通大学委员会在全校班团集体和团员青年范围内，组织开展丰富的主题纪念活动。在全校共青团范围内深入组织开展“青春心向党・建功新时代”主题团日活动，共有589个班团支部成功立项，活动依托校内外红色教育资源，组织全体共青团员在庄严肃穆的仪式中接受“沉浸式”精神洗礼。开展主题宣传教育，在校园内发起“青春，为祖国歌唱”网络拉歌活动，“新青年耀青春”纪念“五四”百年系列活动之――时光宝盒万里巡展，号召全国青年学子献歌新中国七十华诞，重温五四历史，展现新时代青年风采。组织行业引导与行业教育相关活动，五四青年节前后，结合行业引导与行业教育主题，校内组织开展多场讲座，邀请多个重点领域的专家和学者与学生进行深入交流。通过长跑活动、大学生辩论赛、“我的青春”主题人物纪录片等相关文体活动助力营造氛围，同时各基层院系团委也组织开展了丰富的主题活动。

(责编：唐小丽、轩召强)
