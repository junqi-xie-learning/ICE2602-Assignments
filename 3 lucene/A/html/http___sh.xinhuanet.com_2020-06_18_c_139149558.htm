“区校企”三方深化合作 “大零号湾”或将成为闵行区闪亮的“创新名片”

2020年06月18日 21:02:06 来源： 新华网
  • 评论0
  • 打印
  • 字大
  • 字小

    新华网上海6月18日电（吴一航） 18日，由上海交通大学、上海市闵行区人民政府、上海地产集团合作的创新创业培育平台——“零号湾”迎来了5周岁生日。

    5年的发展，让这片区域成为全球创新创业者青睐的沃土；5年后，三方再次携手，将在“零号湾”原有基础上，合作共建“大零号湾全球创新创业集聚区”和“大海洋科研创新平台及产业化基地”，并以区域功能、产业、形态作为心开发点，打造医疗机器人、人工智能、海洋装备、新材料等高科技产业集群，促进各功能区交互融合、协同发展。

活动现场，嘉宾们参观高科技海洋装备成果。新华网吴一航 摄

    作为“大海洋科研创新平台及产业化基地”的主要建设单位，上海交通大学制定了“大海洋”专项战略，并计划依托“区校企”三方优势资源，加速建设海洋装备战略新兴产业创新基地和上海海洋装备研究中心，进一步服务海洋强国战略，助力上海科创中心建设。

    “此次落户闵行的‘大海洋科研创新平台及产业化基地’是三方同心配合的一个缩影。我们将以此为契机，对标国家实验室建设要求，全力支持国家海洋强国发展战略，为海洋装备领域自主创新提供源头供给。”上海交通大学校长林忠钦表示，学校将发挥一流高校与一流学科优势，将此科研创新平台打造成海洋装备新高地，与“大零号湾”一起迈向新的征程。

    为进一步推动高校、科研院所的创新资源溢出，闵行区正以多形式、多渠道促进科技成果与区域主导产业衔接，形成以市场为导向、产学研用深度融合的协同创新机制。依托区内优势资源，聚力推进零号湾全球创新创业集聚区建设。

    闵行区委书记倪耀明表示，“大零号湾”启动建设和“大海洋科研创新平台及产业化基地”落户开发区，标志着闵行在建设上海南部科创中心核心区、国家科技成果转型转化区的道路上又迈进了坚定的一步。

    “我们的目标是将‘大零号湾’打造成国际科技成果的转化地、‘硬科技’创业的首选地、高品质生活的示范地。”在倪耀明看来，创新决定未来，本次“区校企”三方共建“大零号湾”，将在一定程度上促进区域科创资源转化为发展动力。“大零号湾”或将成为闵行区一张闪亮的“创新名片”。
