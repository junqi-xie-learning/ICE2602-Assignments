“城市治理理论与实践丛书”亮相2019上海书展

2019年08月14日16:00  来源：人民网-上海频道
 

人民网上海8月14日电 （记者 曹玲娟）8月14日，2019上海书展开幕当天，由上海交通大学出版社举办的“城市治理的上海细节经验分享会”在书展现场举行。

中国城市治理研究院王奎明以及上海市徐汇、金山、杨浦等区的相关管理人员，围绕“城市治理理论与实践丛书”，从各自多年地区工作的实践经验出发，阐述了对上海城市精细化治理的理解，并根据各区的区位特点分别分享了中心城区、远郊区、老城区的精细化治理经验。

精细化管理是现代科学管理思想的集中体现，也是实现国家治理体系和治理能力现代化的内在要求。习近平总书记在参加2017年全国“两会”上海代表团审议时强调：“城市管理应该像绣花一样精细”。垃圾分类、便民措施、人性化养老、城市微更新……这些看似不起眼的细节无不与城市居民的获得感息息相关。

为全面梳理和总结中国特色城市治理的理论和实践，提出具有中国特色的本土性、原创性和指导性的理论体系，深度总结、积极推广上海和其他地区城市治理的先进经验，为全球城市治理贡献中国范本，上海交通大学出版社和中国城市治理研究院联合推出“城市治理理论与实践丛书”。

据上海交通大学出版社策划编辑徐唯介绍，该套丛书中，实践系列已出版《像绣花一样精细：城市治理的徐汇实践》《像绣花一样精细：城市治理的金山实践》，后续还将推出《像绣花一样精细：城市治理的杨浦实践》《像绣花一样精细：城市治理的浦东实践》等。

分享会现场，徐汇区地区工作办公室副主任张滋介绍，徐汇区是上海市七个中心城区之一，中心城区不像近郊区或远郊区有大面积可延展的空间，不能实行大拆大建的粗放式改造，所以徐汇区从本区域历史文化风貌和生态环境特点出发，形成了一系列精细化的城市微更新、微改造、微治理特色实践。

金山区委办公室副主任朱建华表示，金山区是上海的农业大区，既有高速发展的滨海城市地区，也有恬静美丽的广大农村区域。金山区在党建引领下，在美丽乡村、特色小镇的建设上，形成了一系列颇具特色的精细化治理实践经验。

杨浦区地区工作办公室副主任薛敏谈到，杨浦区作为老城区，从制度精细化、分工精细化、资源对接精细化出发，在睦邻家园建设上进行了从点到面的全方位尝试。最后，嘉宾们就上海垃圾分类中精细化治理方式与理念的体现分享了各区的经验。

(责编：严远、轩召强)
