0000001号上海高校首份录用协议！女博士圆国防梦，与同学签下前十份Offer

教育在线 2019-11-06 12:04
来源：上观新闻 作者：徐瑞哲
2020届沪高校首份就业协议书，出自上海交大毕业生，她去了哪儿？

解放日报·上观新闻记者今日获悉，上海交通大学机械与动力工程学院2020届博士毕业生高娇近日被中国工程物理研究院核物理与化学研究所录取，圆了她的国防兴国梦。

高娇不仅成为今年第一位成功签约的交大毕业生，也成为编号0000001的上海高校首份录用协议签约人。继第1号协议签约后，编号为0000002至0000010号协议书的签约毕业生均被国家重点单位录取，投身祖国最需要的地方建功立业。

【编号0000001】

高娇博士从小到大在不同城市间求学，交大读博期间，目前已累计发表高质量SCI论文9篇，申请国家发明专利4项，也获得了“挑战杯”、“节能减排”大赛等多项国内顶级创新创业竞赛的特等奖。

毕业之际，高娇参加了中物院的招聘宣讲会。看到宣传册上 “铸国防基石，做民族脊梁”几个大字十分醒目，她记得在这里曾经工作过并且获得共和国勋章、“两弹一星”功勋的于敏先生说过：“国家需要我，我一定全力以赴。”无论工作性质还是工作地点，中物院都是她心中最佳的选择。

由于不是核专业的学生，在投递简历时她还有过一丝担忧，但是最终在学校和学院的鼓励与帮助下，她如愿可以为祖国国防事业的参天大树增添一份新绿。

【编号0000002】

无独有偶，编号0000002的录用协议出自上海交大材料科学与工程学院的硕士毕业生王志恒，他签下的是中国船舶集团公司第七一九研究所。硕士学习期间，他继续选择材料专业，这源于自己对材料成形科学的执著认同。在学习之余，他还热心于公益与志愿者活动，免费帮助同学维修家电、手机等设备。

当看到习近平授予93岁的黄旭华老先生 “共和国勋章”的新闻时，他深受触动。隐“功”埋名三十载，终生报国不言悔，这位交大校友、中国核潜艇之父的一生深深地感动了他……

怀着一腔热血，王志恒放弃了来之不易的互联网公司高薪工作机会，毅然决然地选择了武汉，选择了黄老先生所在的七一九研究所，“懂材料成形理论又能够运用理论进行工程实践的人才，是建设工业强国的不可或缺的力量。”

【编号0000003……】

编号0000003的协议签字人，是上海交大电子信息与电气工程学院的硕士毕业生张仕强，他被中国电子科技集团公司第十三研究所录取。从本科到硕士，他在科研上对射频电路表现出极大兴趣，在上海交大读研时选择了微波无源器件作为研究方向。期间，他参与导师与中电十三所的项目，不仅学以致用，更是活学活用，由此对中电十三所有了更深入了解。中电十三所肩负着“振兴民族半导体，实现国家信息化”的历史使命，“投身于微波无源器件领域是幸运的，能参与到和中电十三所的项目中是更加幸运的。”

协议书编号0000004的李晓旺作为航空航天学院的一名博士毕业生，目前已签约中国航天科工二院206所。以国家战略需求为牵引，在毕业生就业引导下，其后编号0000005到0000010的协议也都出自上海交大学子：

机械与动力工程学院的博士毕业生齐文元签约上海航天技术研究院（航天八院）；电子信息与电气工程学院的硕士毕业生彭至鹤签约中国船舶集团公司第701研究所；机械与动力工程学院硕士毕业生袁麟签约航空工业第一飞机设计研究院；船舶海洋与建筑工程学院博士研究生周文俊签约中国船舶集团公司第七〇八研究所；船舶海洋与建筑工程学院博士毕业生毛立夫签约江南造船（集团）有限责任公司；电子信息与电气工程学院电子系博士毕业生蒋冬冬签约中国电子科技集团公司第十四研究所。
