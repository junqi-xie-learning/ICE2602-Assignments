经略海洋｜院士专家眼中的日照机遇
2018-10-21 12:37:41　 　日照日报社
A+ A-

拥抱智能 深耕蓝海

——参加海洋智能装备与未来航运系统创新论坛院士专家眼中的日照机遇

　　核心提示：海洋是日照最鲜明的特征,也是发展的最大优势和潜力所在。必须以时不我待的姿态加快向海经济发展，深耕海洋这片沃土。

　　10月20日，我市与上海交通大学主办的海洋智能装备与未来航运系统创新论坛吸引了200余位“人工智能 海洋科技”相关领域的院士、专家的参加。对此，本报特派记者专访了部分嘉宾，以期在经略海洋中获得更多真知灼见。

探索深海装备智能化发展，日照迈出踏实一步

专家名片：曾恒一  海洋石油工程专家、中国工程院院士

　　第一次来日照，就见证了日照与上海交大合作建设的海洋智能装备重点实验室的揭牌，这一举措，为走向更大范围更大水深装备的智能化，迈出了踏踏实实的第一步，我感到非常高兴。

　　实验是走向产业化、工程化必经环节，而智能化是工业、制造业转型升级的必由之路，我们海洋石油也在推动海上智能油田的开发，所以日照与上海交大的合作很有意义。

　　日照有良好的地理环境、海上实验条件，这为重点实验室今后研发工作打下了很好的基础。我们中海油在开发新的油田，开发水下侦探系统，开发深海空间站，走向深水的时候，可以跟这个实验室有机结合起来，充分发挥重点实验室多功能的军民融合优势。未来，我们将把更多的浅水中试试验放在这里，为建设海洋强国贡献自己的力量。

　　他还表达了对上海交大学子们的殷切期盼。他说，一直以来，上海交大有着非常优良的治学传统，希望学子们一定继承下去，为海洋强国做出贡献，不辜负国家期望。

以智慧海洋为起点，示范带动日照其他工业走向智能化

专家名片：朱英富 水面舰船工程专家，船舶设计大师，中国工程院院士

　　这是我第一次来日照，虽然是小城市，但是有大城市的气魄和抱负。今天参加这个论坛让我对日照有了很多的了解，这里环境优美，经济发展快速，特别是跟上海交大合作以后，新的项目在这里开展，对日照各方面的进步将有很大影响。

　　智能化是中国包括世界各国发展的潮流，不仅在海上，多个方面都在呈现智能化发展趋势。日照与上海交大合作的实验室就是很好起点，利用智慧海洋技术作为发展契机，把更多单位聚集于此，把更多学者、科研院所、企业精英以及力量，吸引过来，拓展日照发展网络，同时也将给日照工业其他方面的发展起到示范带动效应。

　　我上午听了你们海洋渔业局局长作的报告，他描绘了利用日照本地资源，新的思维将海洋经济、养殖业、旅游业有机地结合起来，描绘了很好的发展蓝图，我相信再过几年到日照，会有面目一新的感觉。

　　这个月28号，日照到武汉的航班就要开通了，随着高铁、航空交通体系的完善，以后会有更多的人来到这里。我很早就知道，你们是第一个见到太阳的城市，我相信以后随着你们声誉的扩大，未来发展不可限量。

期待像管理社区一样管理海洋，像生活在社区一样享受海洋

专家名片：易宏  教授，博士生导师 上海交通大学海上装备与系统研究所所长

　　因为工作的原因多次来到这里，也算是半个日照人了，对这里产生了很深的感情。日照水美，激发了“好想法”，这么一块好地方，一定能干出好的事情。

　　根据自己的特点和国家需要，日照确定自己的发展思路，并非“人家干啥我也干啥”。举办本次论坛就是很好的特色，不仅撬动了智能神经，也让大家认识智能，反思未来。

　　智能，是一个概念，更是发展方向。如果我们把智能做好了，以后大家想到智能会想到日照 ，想到日照会想到智能，日照这个宜居城市加上智能的概念，“居”的含金量会更高。

　　他说，日照正在大力推进中央活力区建设，活力靠智能，智能便是撬动活力的因素，这也是牵引他们来到日照的力量。基于这样的基础和共鸣，未来，他们将从小到无人船技术，大到四大技术(船体技术、环境感知识别、任务智能规划、智能技术节能减排)的研究，与日照海洋驿站、海洋牧场的发展进行融合，打造“牧、游、控”海上休闲中心，要像管理社区一样管理海洋，像生活在社区一样享受海洋。

　　在这一方向牵引下，日照会有很多发展机遇，比如说深海鱼礁，是直接投放还是将海底做成美丽的公园，这是完全不同的概念。真正的瓶颈在于概念，不在于建设，未来的海洋，在统一规划下，在概念介入下，相信会发展的很好。

积极发挥院校科研和人才培养优势，助力日照海洋经济腾飞

专家名片：吴卫国  武汉理工大学 首席科学家、教授

　　这是我第三次来日照了，每次来都有全新的感受。基于从事海洋工程教学科研四十年的经验，我认为，日照有非常好的海岸线和海洋资源，尽管早期开发稍慢，但近年来发展势头迅猛，后发优势存在。

　　尤其是举办本次论坛，吸引了院士、将军、全国知名高校专家学者以及业内领军人才，智能企业等等，搭建了一个很好的沟通交流平台。武汉，科技高校特别是涉海的院校很多，是培育海洋装备人才最多的城市。随着本月28号机场航线的开通，相信两地交流合作的机会更宽广。

　　国家提出海洋强国战略，山东省也大力支持海洋开发利用，在这样的背景下，我们将积极发挥院校在专业科学研究及人才培养等方面优势，日照的海洋经济在未来定会腾飞。

　　近期，半个月内，我们会专门带领船舶海洋研发团队再次来到日照，从海上休闲观光装备、海洋牧场开发、近海消浪等方面做好服务，提供产品和设计理念，在这里落地生根，配合好日照的海洋开发和利用。

　　今后，我们还打算与上海交通大学、华中科技大学等几所国内船舶海洋领域影响较大的高校，在日照市建立海洋装备研发中心，研发符合日照海洋开发需求的海洋装备，拉动本地海洋装备制造业发展，进而推动日照海洋经济的发展。

（统筹策划/刘新英 采写/日照报业全媒体记者 史树梅  李敏）
