青少年走进上海交大医学院实验室：科学原来这么有趣！
2019年05月25日 15:59   来源：中新网上海  





　　中新网上海新闻5月25日电 (杨静 陈静)“实验室除了小白鼠 还有小灰鼠！”“薄薄的一层皮肤其实结构很复杂”“脂肪也分好坏，让人发胖的是白色脂肪！”……25日起，2019科学训练营全国“实验室开放日”――生命科学专题活动迎来了对科学充满好奇心的青少年，他们。由上海交通大学公共卫生学院、上海交通大学基础医学院联合上海市免疫研究所、中国细胞生物学会、上海市细胞生物学会推出

　　据悉，实验室开放日活动是为响应中央对于科普活动力度的号召，拓展公共卫生健康教育服务范围，提高青少年科技创新意识和科学实践能力发起的，已成为交医品牌科普活动之一。上海交大医学院方面透露，今年的活动，该校共有13个课题组向公众开放了实验室，包括学科带头人、专业院系主任聚集在校园里，以科普讲座、实验室参观、实验设备介绍和实验动手操作相结合的方式，向青少年学生通俗易懂地解释他们所从事的研究内容，帮助青少年走进科学，走近科研。吸引了300余名青少年学生和家长参与。

　　 《天“肠”地久，从“肠”计议》《显微镜下的微观世界》《细胞奥秘和抗氧化复杂性》《抗体――神奇的子弹》等专题讲座和实验，从主题到内容展示都做了精心设计。科研人员从细胞、基因和免疫因子等微观层面向青少年解读肿瘤诊疗领域的最新科研进展，同时还从肠道免疫、皮肤免疫等方向梳理人体的自我防御系统，科普内容中还包括了多能干细胞的再生医学研究、抗体靶向治疗等最新技术。

　　活动中，青少年在指导下直接操作移液器、离心机、荧光显微镜等仪器，并近距离观察和接触实验小鼠、斑马鱼等实验动物和各种细胞。通过现场解答、互动体验等形式，大家充分感受到了科学的魅力，整个过程学生们都非常兴奋，提出了许多奇思妙想的问题，纷纷表示这是一次难得的体验经历，开阔了眼界、增长了知识，激发了对科学知识探索的兴趣。

　　据了解，与往年的实验室开放相比，本次还重点关注到生活中常见的肥胖、病原真菌和饮水健康等公共卫生问题。公共卫生学院院长王慧课题组让青少年直接通过显微镜观看生活中常见的病原菌，帮助青少年建立起科学客观的防菌卫生意识；环境与健康系田英教授课题组会通过漂白粉中有效氯含量测定、饮用水余氯测定等小实验，帮助青少年直观了解家庭中常见的饮用水安全问题。从“身边事”说“科研理”，这些科普内容更加深入日常生活，调动了青少年的兴趣和积极性，也能切实促进全民健康知识的普及。

　　据悉，为使更多人了解了科研工作者的工作内容，激发青少年对生命科学的兴趣，除了此次“实验室开放日”活动外，主办方还推出多种线上、线下科普讲座， 切实推进科技改革发展成果的全民共享。
