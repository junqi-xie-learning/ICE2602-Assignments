中国国旗在人类商船航线的最高纬度、最险区段升起！他们穿越北极航道，开辟冰上丝路

教育在线 2019-09-15 20:21
来源：上观新闻 作者：徐瑞哲
一艘载着远洋船员与研究人员的中国航船，成功驶过全球所有商船航线能够抵达的最北点，也是北极航线上最艰险的区段。

北极圈传回最新消息！解放日报·上观新闻记者15日获悉，一艘载着远洋船员与研究人员的中国航船，成功驶过全球所有商船航线能够抵达的最北点，也是北极航线上最艰险的区段。

从8月底开始，历经半个多月航程，中远海运特种运输股份有限公司“天佑轮”不仅进入北极圈，而且行驶到北极航行任务的纬度最高点：北纬77°57′、东经93°24′。

而此时此刻，在天佑轮驾驶舱的顶层甲板上，一场形式简单却庄严神圣的升旗仪式同步举行。极地凛冽寒风中，船员们在船长和政委的带领下，与来自上海交通大学的博士教师一起，20多人一边高唱国歌，一边目送国旗，飘扬在此次航行任务的最高纬度，壮怀激烈、溢于言表。

北极航线是中国许多沿海港口通往欧洲的最短路线，且北极航线将是“冰上丝绸之路”经济带建设的重要载体，在“一带一路”倡议中扮演着重要角色。

2019年8月26日，作为执行2019年度北极航行任务的中远海运特运公司旗下特种运输船舶，去年下水的36000吨级“天佑轮”，满载着大型风电设备，驶离上港集团张华浜码头，一路向北经东海，过朝鲜海峡。北京时间8月30日晚，天佑轮穿行通过日本海宗谷海峡。9月6日晨穿越白令海峡，进入楚科奇海，走北极东北航道；9月7日，全船正式进入北极圈。至9月12日下午3点左右，“天佑轮”又驶抵人类商船航行纬度的最高点——长约104公里、全年覆盖浮冰的维利基茨基海峡。按计划，此番航程一个月左右，在运抵德国港口后，于9月下旬返航上海港。

那么，目的并非科考，高校专家又怎么会出现在远洋商船上，与普通船员同舟共济，饱经冰海风浪考验呢？

追溯到今年5月7日，中远海运集团董事长、党组书记许立荣走上上海交通大学讲台，为师生作“海运即国运”主题报告，聚焦总书记在上海考察时关于“经济强国必定是海洋强国、航运强国”的重要讲话精神，阐发了海运业战略地位、战略担当及航海精神特质。由此，这对名校与央企开启新一轮产学研合作，上海交大船舶海洋与建筑工程学院交通运输工程系党员教师戴磊博士和姚帅寓博士，成为交大首批派出的随船专家，搭乘中远海运特运“天佑轮”参加极地航运，在一线与前沿开展科研活动。这一切，对于减少我国对常规航线的依赖、保障我国海上能源运输安全、降低远洋运输成本、降低碳硫排放等方面有重要价值与意义。

有缘万里，同船渡水，北极共建。在途经每一个标志性航点的过程中，天佑轮党支部书记黄爱军政委等，与上海交大交通航运党支部这两位青年老师，都一起牵头开展“临时联合支部”的主题党日系列活动，将“不忘初心 牢记使命”主题教育活动的一部分，带到了船上、带到了北冰洋上、带到了北极圈内。

顶着冰区严寒，扛着风大浪急，双方登上驾驶台甲板重温入党誓词，又在舱内会议室集体学习习近平有关“开展北极航道开发和利用，打造‘冰上丝绸之路’”的重要论述，并作校企之间交流发言。此外，博士教师还带来一场船舶航行风险管理的学术讲座，加深船员实操的理论认识。船员们还要求交大专业老师在后续航程中再做几场讲座，进一步拓宽知识面。作为此行北极航运研究小组的两名成员，他们融入船员生活，现场观察航海实务，详细收集北极航行的相关数据。
