上海交大师生暖心快闪献礼新中国70周年华诞

2019-01-02 21:11 光明日报客户端 曹继军 魏娜

“我和我的祖国,一刻也不能分割……” 随着饱含深情的旋律响起,正在钱学森图书馆参观的观众们停下脚步,跟随人群,加入合唱队伍。近日,上海交大师生以“我和我的祖国”为主题,在钱学森图书馆开展了一场暖心快闪教育活动,献礼新中国70周年华诞。

现场,来自上海交大教授、学子,钱学森图书馆的志愿者和工作人员,老人、小孩,还有许许多多普通市民加入合唱队伍,歌声洪亮,每个人脸上都洋溢着激动的神情,每个人心里都充满着对祖国的祝福。

在图书馆序厅,一位女士正伴着钢琴、弹奏跳动的音符,从容优雅:一位长者正拉着小提琴,精神矍铄。他们就是上海交通大学教授刘西拉与陈陈。刘西拉是上海市教学名师、上海交大“最受欢迎教师”、首届教书育人一等奖获得者;2017年6月,刘西拉作为团长与一群平均年龄72.3岁的老年合唱团含着热泪,在央视《出彩中国人》的舞台上唱响了《我爱你中国》,得到广泛传播,无数网友为之动容、落泪。

对于这场活动,上海交通大学教授刘西拉、陈陈表示:“在祖国庆祝改革开放40周年和新中国70周年华诞之际,能参加这次活动感到十分幸运。我们出生在日本侵略的战乱时代,经历了多次逃难和敌机轰炸;年轻时代,我们有机会参加了西南‘大三线’的建设,并为教育事业拼搏。今天亲眼看到祖国走到了世界的中央,真的非常感动。我们不仅看到了祖国的变化,而且为此付出了青春,深感幸运和充实。”

“当所有的观众跟随同一个旋律高唱同一首歌的时候,当所有人的爱国热情在那一刻驱散了冬日的寒冷的时候,我们心中对于国家和民族的归属感、认同感和自豪感油然而生。”上海交通大学中美物流研究院2018级硕士研究生、上海交大学生联合会副主席池松恒表示,这是一堂生动的爱国主义课程,而我们也在这样的氛围中倍感肩头责任重大,未来,必将把自己的青春像先辈一样融入到国家振兴和民族复兴的伟大事业中。

参与此次快闪活动,来自上海交通大学机械与动力工程学院钱学森班2016级本科生孙欣钰说:“歌曲唤起了我们共同的精神情感——爱国。在百废待兴的年代,钱学森等一辈科学家突破重重阻碍,在戈壁滩上建起了我们的国防工业。时代在变,爱国情不变,我们青年学子更应该拓展国际视野,完成好自己的使命,将个人发展和国家社会相联系,让中国声音更加嘹亮。”(光明融媒记者 曹继军 通讯员 魏娜)
