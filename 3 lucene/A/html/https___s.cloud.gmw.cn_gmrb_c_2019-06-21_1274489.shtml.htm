“最美高校辅导员”汪雨申：以文化人 以美育德

2019-06-21 19:58 曹继军

辅导员、思政课教师、学生话剧社团指导老师、校辩论队的总教练、校艺术教育和高水平艺术团负责老师……这些都是上海交通大学辅导员汪雨申的多重身份标签。18年来，汪雨申一直走在文化育人的道路上。近日，记者获悉在中宣部、教育部首次从历届在岗的“高校辅导员年度人物”和在读的“大学生年度人物”评选中，汪雨申获评“最美高校辅导员”。

“我喜欢学生叫我汪sir。”汪雨申告诉记者，这个称呼让他和学生的距离更近了。学生半夜生病，他第一时间赶到医院；学生成绩下滑，他促膝谈心；学生感情受挫，他细心开导……在学生的眼中，他不仅是一位为学生传道授业解惑的老师，更是一位愿意随时与学生分享喜怒哀乐的兄长。

在具体工作中，汪雨申围绕着“育人”这个学校的中心任务，从理想信念、创新创业、社会实践、志愿服务、审美素养等方面挖掘学生潜质，将美育传统引入思政实践。

2012年上海交大原创话剧《钱学森》公演，在校内以及全国巡演13轮39场，观众人数累计超过6万余人，曾在2013年获得中国校园戏剧节最高荣誉。这部由上海交大师生根据钱学森生平事迹创作而成的“热剧”，带给无数青年学子心灵的触动和震撼，而它的幕后制作人正是当时上海交大校园里一名普通的辅导员汪雨申。

与商业剧有固定演员不同，校园剧最大的难度就是演员会不停的更换。对此，汪雨申却有自己的理解，他说，“一部话剧，就是一堂精彩的德育课！与其理性的说教，不如感性的示范。”正是在话剧的排演过程中，汪雨申润物无声地将学生思想道德建设融入教育实践中。

18年来，汪雨申坚持在学生工作的第一线，他的工作对象除了中国学生之外，还包含了2000多名留学生。2017级来自意大利的留学生马克，在汪雨申的支持和帮助下，不但汉语水平有了突飞猛进的提高，从一名五星级酒店的大厨，转变成一名研习汉语的大学生，他立志要做当代的马可·波罗，为意大利和中国的世代友好交流付出自己的努力。

以美育人、以文化人。在上海交大，还有众多和汪雨申一样的辅导员，他们驻守在学校思政工作一线，关注学生发展与成长，成为交大一道“最美”的风景线。

记者了解到，为了培养和发展辅导员队伍，上海交大将原有的学生工作相关部门进行了整合，成立了上海交通大学学生工作指导委员会，将沿用多年的“政治指导员”改称为“思政教师”，让辅导员兼具管理干部与教师双重身份，并建立相应的辅导员职称系列以打造“最美”思政队伍。

（光明日报全媒体记者 曹继军 通讯员 魏娜）
