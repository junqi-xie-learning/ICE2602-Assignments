重观赏更重科研
——2019上海（国际）花展开幕
2019年03月29日  作者:王毅俊   编辑:ydm   审核:杨冬梅  摄影:杨浦东  版面:A1

昨天起至5月5日，2019上海（国际）花展在上海植物园举办，一场集缤纷花卉、精致景点、时尚展览和新颖活动于一体的“园艺盛宴”将上海的春天点亮。

本次花展最大的亮点莫过于首次采用朱顶红和花毛茛为“双主题花”，来延长花展的观赏期。花毛茛花朵形似牡丹、叶片犹如芹菜，又称“洋牡丹”，朱顶红则花朵硕大、花色繁多，又名百支莲、喇叭花，是球根花卉之首，两种花卉都很适合家庭栽植、观赏，很契合“精致园艺、美丽家园”的花展主题。

上海植物园副园长毕庆泗表示，花毛茛和朱顶红品种繁多、色彩艳丽、花型优美，具有“寓意美好、颜值过人、性格亲民、品种繁多”等优点。

作为沪上最大规模的综合性花展，此次花展进一步加大与国内外园艺机构、花卉企业的合作力度，30余家国内知名园艺设计、园林建设、花卉生产单位及10余家来自瑞士、泰国、丹麦、法国、比利时等国的专业机构共同参展。

花展设“主题庭院、新优植物、特色园艺、体验互动”四大版块。在中外庭院展区，一座座具有国际元素的庭院“样板间”令人感受到园艺的巨大魅力。由瑞士园林专家设计的典型瑞士居家花园中，除了花卉绿植、瑞士木屋外，还有精致的“昆虫旅馆”，用来留宿花园中的益虫。法国设计师设计的浪漫花园，则将可食用的鲜蔬香草与艳丽的观赏性花卉混搭，使景观更为生动活泼。

另外，主办方还将通过展示荷兰鸢尾、铁线莲、海棠等新优植物，展现上海植物园在新优观赏植物收集与种质创新上的最新科研成果，助推上海科创中心建设。

毕庆泗介绍，要从花展走入百姓家庭，不少花卉还有一大段路要走，其中关键一步是克服原生地与上海气候环境的迥异问题，否则无法适应存活。以往有市民反映，花展主推的花很漂亮，但在家里养就是养不活。如今，上海（国际）花展也正转型成为产业平台。毕庆泗表示，为迎接上海气候环境的挑战，上海植物园和相关园艺单位组成了专门的课题组，将对10多个品种的花毛茛进行“淘汰赛”，最终留下适应性良好的品种，进而重点培育，最终向上海城市绿化输送。
