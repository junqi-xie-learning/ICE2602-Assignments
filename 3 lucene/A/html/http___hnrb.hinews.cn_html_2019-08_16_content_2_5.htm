上海交通大学三亚崖州湾深海科技研究院成立
　　本报三亚8月15日电 （记者徐慧玲）8月15日，上海交通大学三亚崖州湾深海科技研究院成立。该研究院将围绕深海科技领域，开展科研工作，对接三亚深海科技产业发展需求，推进落实科研合作项目，打造深海科技公共平台。

　　根据合作协议，上海交通大学将充分发挥自身优势，全力支持三亚崖州湾深海科技研究院发展，建设好深海科技公共平台。同时，三亚将为研究院的运行提供支持和保障，并以此为契机，在教育、科研、产业等领域全面深化合作，共同推动崖州湾科技城和三亚高新技术产业发展，为海南自贸区、自贸港建设作出积极贡献。

　　上海交通大学三亚崖州湾深海科技研究院成立后，双方将共同努力推进研究院和科技公共平台建设，深化深海科技合作，通过科技成果转化，力促高新技术产业发展。

　　省委常委、三亚市委书记童道驰，上海交通大学党委书记姜斯宪出席成立大会。
