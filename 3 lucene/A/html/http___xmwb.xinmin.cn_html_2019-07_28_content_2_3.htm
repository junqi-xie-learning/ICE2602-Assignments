让北斗更好地导航百姓生活
——记上海北斗导航创新研究院院长郁文贤
江跃中
　　前天，从北京传来好消息：在全国退役军人工作会议上，上海北斗导航创新研究院院长、上海交通大学讲席教授郁文贤，荣获“全国模范退役军人”的称号。

　　郁文贤长期从事军民融合高新技术研究和产业化，率领研究团队在雷达目标识别、遥感信息处理、融合导航定位技术等方面，取得一系列实践创新研究成果，为我国国防关键技术发展、装备研制和产业化，为北斗更好地导航百姓生活，作出了突出贡献。他先后获得国家科技进步二等奖5项，部委级科技进步一等奖6项等。

　　1981年9月，毕业于松江二中的郁文贤，考入国防科学技术大学无线电测控与数传专业，1985年7月毕业。此后，他又先后攻读通信与电子系统专业硕士学位和通信与信息系统专业博士学位。

　　博士毕业后，郁文贤在国防科学技术大学正式服役和任教，他32岁就被破格晋升为正教授。“十五”期间，郁文贤带领的团队，在我国3个舰队的4个观通站开展科研攻关，累计在海岛上的工作时间有近两年，完成了上万次的实际数据采集与识别实验，取得了大量的研究成果和研究经验。在国防科学技术大学的27年里，郁文贤和他的团队，研发了我国首台套雷达目标自动识别系统，研制了我国第一套空间目标监视战略相控阵雷达目标识别装备等，多次立功受奖。

　　2008年，已经是正师职领导的郁文贤，从国防科技大学转业，他放弃组织行政移交安排，选择到科技创新创业一线工作，并参加了全球院长竞聘，被上海交通大学录用，担任电子信息与电气工程学院院长。

　　郁文贤瞄准北斗卫星导航系统的应用，开展重点研发。他创建了“北斗导航与位置服务”部市共建重点实验室，致力于解决北斗导航与位置服务中的核心技术问题，技术发展成效显著。针对城市环境存在信号遮挡、多径反射等因素导致卫星导航定位精度严重恶化的问题，他主持开发了结合运动情景感知的多源信号融合定位与真实场景库测试验证方法，推动企业产品提升复杂环境下的定位可靠性，典型场景下定位精度和稳定性与国际同期最高水平相当，同时，智能化低功耗多模融合定位终端率先在司法社区矫正、老人儿童关爱等领域推广应用。相关成果获得了2016年度上海市科技进步特等奖。　　　　

　　科研攻关屡屡成功，郁文贤靠的是执着、专注与毅力，他在负责一个项目的算法软件技术定型和材料总结时，曾3天3夜没有合眼。技术研发过程中连走路、吃饭和睡觉做梦都和项目有关，有个项目研发的方法，就是他走在路上想起，立刻返回实验室修改调试告捷的。

　　2011年，上海交通大学开始规划论证建设北斗西虹桥基地，郁文贤率领团队埋头苦干。2013年底一期开园，郁文贤带动和集聚了多个退役军人创业团队入驻园区，打造出主题鲜明的、高学历高素质退役军人创业模式的典范，现在有的创业团队吸纳的退役军人就业者高达70%。

　　郁文贤把北斗西虹桥基地建成了国内北斗产业第一园，形成了产业链集群创新与集聚发展的良好生态环境，累计集聚近200家导航产业相关企业。 本报记者 江跃中
