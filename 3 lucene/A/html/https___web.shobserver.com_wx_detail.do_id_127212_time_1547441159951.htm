中国主导、国际最佳！南极“冰立方”加入，700米地下的2万吨中微子探测器后年建成

教育在线 2019-01-14 12:45
来源：上观新闻 作者：徐瑞哲
在世界几个最大规模的中微子实验项目中，包括中国JUNO、美国DUNE、日本HyperK等，江门中微子实验有望首次测定中微子质量顺序，对三类中微子振荡参数的完备性测量也将达到前所未有的1%精度。

“每一秒钟，你的身体就放射出大约3亿个中微子！”14日，南极“冰立方”合作组研究者徐东莲告诉解放日报·上观新闻记者，神秘中微子就在你身边，如果吃了一个香蕉，那么香蕉中的钾元素经衰变后也会产生中微子。

 

当天，位于上海交大闵行校区的李政道研究所集聚国际上各路中微子专家，透露出中国主导的江门中微子实验最新进展：这项由17个国家和地区、77所高校与科研院所、600多位科研人员加入的超级实验，目前正从五年来的筹备和基建工作，转段进入探测器安装和整合。

 

江门中微子实验站地理位置示意图。

 

家族“三丁”，占已知基本粒子四分之一

 

尽管公众未必熟知，但中微子无处不在，每秒钟就有上万亿个中微子自由穿透我们的身体；商用核电站在发电的同时，每秒释放出大约万亿亿个中微子……

 

江门中微子实验项目发言人、中科院高能所所长王贻芳院士进行了科普：中微子是一种不带电、质量极其微小的基本粒子。它们家族“三丁”，分3种类型，即电子中微子、缪（μ）中微子和陶（τ）中微子，占构成物质世界的已知12种基本粒子的四分之一。在微观粒子物理和宏观宇宙起源及演化中，中微子扮演着极为重要的角色；然而，由于中微子仅参与非常微弱的弱相互作用，难以被察觉到，除非通过超大体积和超级灵敏的粒子探测器。

 

中国在中微子探索领域处于全球第一方阵。曾经多少年来，中微子被认为是和光子一样没有质量的粒子；然而20年前，中微子被发现可以在飞行中从一种类型转变成另一种类型——这种“中微子振荡”揭示出中微子一定具有微小的质量，相关突破性成果获得了2015年诺贝尔物理学奖。我国科学家主导的大亚湾中微子实验于2012年发现第三种全新的中微子振荡模式，完善了对三类中微子相互转变的理解，也因此获得2015国际基础科学突破奖（Breakthrough Prize），以及2016年国家自然科学一等奖。

 

江门中微子实验探测器示意图。

 

从南极到江门，探索中微子天文学

 

谜团，还有众多未解。3种中微子的质量谁最重、谁最轻？到底有多轻？又如，宇宙中剧烈的天体活动，如超新星爆发等也产生大量的中微子。而中微子可轻松穿透宇宙中致密的天体环境，包括地球在内，因此让人类找到了揭秘超高能宇宙射线起源的理想探针。近年来，中微子天文学也成为一门前沿交叉学科。

 

去年，位于南极的“冰立方”中微子实验(IceCube)就首次发现来自于猎户座40亿光年以外的“耀变体”中微子，证明了其中心有超级黑洞的活动星系核，确实可以加速宇宙射线到几万万亿电子伏特，比目前人类最强大的加速器高几千倍。这一发现，被《科学》杂志遴选为2018年国际最重大科学突破之一。

南极冰立方（IceCube）中微子望远镜冰面实验室。

 

“位于南极极点的‘冰立方’，探测器深入冰层之下2.5公里，已接近岩层。”在刘江来教授团队加入大亚湾中微子实验并成为实验成果重要贡献者的基础上，上海交大现已引进南极“冰立方”合作组成员徐东莲，她长期致力于寻找高能天体“陶中微子”和高能中微子的瞬变源，在加入江门中微子实验合作组后，计划开展中微子天文学方面研究，进一步拓宽这项实验的科学潜力。同时，刘江来也作为江门中微子实验合作组执行委员会成员之一，承担着研制中心探测器刻度系统的课题。刘江来和徐东莲，也是今年第13届江门中微子实验国际合作大会共同主席。

 

冰立方望远镜探测元件，共5000多个掩埋在1.5到2.5公里的南极冰川冰层中，形成长宽高各1公里的探测阵列，是世界最大的中微子探测器。

 

有望首测中微子质量顺序，率先达1%精度

 

在南极点之外，我国南部沿海地区正成为中微子探测的核心地带。在大亚湾中微子实验站点以南、珠江口以西，中科院高能物理研究所牵头的江门中微子实验（JUNO），正在广东江门市开平附近建设埋深700米的地下实验室。该实验的核心是具有超高纯净度和国际最佳能量精度的液体闪烁体中微子探测器，直径35米、重2万吨。通过测量来自临近广东阳江和台山核电站的中微子，可以测定中微子质量顺序、精确测量中微子振荡，同时开展对超新星中微子、大气中微子、太阳中微子、地球中微子、惰性中微子、核子衰变、暗物质间接探测等前沿方向的研究。

 

今天，首次来沪举行的江门中微子实验国际合作大会透露，作为目前中国主导的大型国际科学合作项目，江门中微子实验预计2021年全面完成建设，运行周期达20年以上，合作组将使液体闪烁体这一核心技术推进至前所未有的成熟度。王贻芳院士表示，实验运行后，不仅能对理解微观的粒子物理规律做出重大贡献，也将对宇宙学、天体物理乃至地球物理做出重大贡献，其项目经验对未来中国发起国际大科学计划和工程有重要参考价值。

 

项目副发言人之一、来自于意大利国家核科学研究所的资深研究员吉尔切诺·拉努奇表示，在世界几个最大规模的中微子实验项目中，包括中国JUNO、美国DUNE、日本HyperK等，江门中微子实验有望首次测定中微子质量顺序，对三类中微子振荡参数的完备性测量也将达到前所未有的1%精度。在基础研究基础上，带动国内新型建筑、化工工业、特种材料、特种分析测试仪器、光敏探测、微电子等相关高科技企业的核心技术创新。
