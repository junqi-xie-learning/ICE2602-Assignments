【附视频】上海最美教师：有实力的偶像派

教育在线 2018-09-28 21:30
来源：上观新闻 作者：徐瑞哲
她收到了数不清的小纸条，上面最常用的话就是“我爱你，孙老师”。

“最美教师”长啥样？她们到底美在哪？上海“最美教师”典型人物、典型案例选树的20人，带来了最佳答案。

 

今天（28日），在上海市教卫工作党委、市教委指导下，由市教育发展基金会、上海师范大学和解放日报共同主办的“第十四届新解放教育讲坛”上，无论小学还是大学的 “最美教师”代表都是“有实力的偶像派”，与师生观众们分享了何为师者中的“最美”。

 

爱+专业+才艺

 

有爱就够吗？上课就够吗？当好“最美教师”实属不易。

 

高安路第一小学教师孙怡青。李立基 摄

 

来自高安路第一小学的数学教师孙怡青，今年正好执教十年。入职时，她的最初标准就是爱，“爱每一个学生”，弯下腰、蹲下地，常常笑着说，“我很高兴认识你，我很喜欢你。”为此，她收到了数不清的小纸条，上面最常用的话就是“我爱你，孙老师”。

 

但爱与被爱之间，孙老师渐渐反思自省出一条公式：“爱+专业+才艺=最美教师”。她不仅研究国内外十几套数学教材和教辅，还自学信息技术软件来制作Flash，使孩子对数学概念有更直观理解，她甚至自费学习心理学，考出国家二级心理咨询师证书。

 

对低年级用平板电脑讲述卡通故事，对中年级以漫画形式分组完成游戏，对高年级在真实情境模拟算术任务，孙怡青在不同年级有不同教学策略。而为了彻底“征服”学生，美美哒孙老师还常常在校园中展示不一样的自我，从跳舞到画画，她告诉孩子们，“我学数学，还学其他，你们也一样哦。”

 

从“一桶水”到“一杯水”

 

从“多才多艺”的小教，到了教研并重的高教，才是真正“上得厅堂，下得厨房”。上海交通大学医学院的细胞生物学教授易静，已有太多荣誉的光环：市劳动模范、市先进工作者、市三八红旗手……但这位最美教师为自己设定的目标，则是“做更好的老师”，“研究型大学的教师必须既能从事原创性科研，又能通过课堂教育培育下一代创新人才。”

 

上海交通大学医学院细胞生物学教授易静。李立基 摄

 

在易静看来，生长在互联网时代的90后和自媒体时代的00后，具有特殊的认知模式和知识获取方式；面对这些变化，固守传统的讲课方式往往难以获得良好课堂效果，甚至会给教师带来挫败感。

 

她的方式，是从“一桶水”才能倒出“一杯水”。易静20多年来长期负责国家自然科学基金项目等，在医学细胞生物学领域形成研究特色，在国际学术期刊发表数十篇论文，才能在课堂上给出一系列新的研究案例，有时为把近期重要科研突破引入课件而备课到深夜。

 

就是这样的医学家，十多年来每年平均给医学生和研究生讲课的课时数竟超过100个。她负责跨越多个学科的新型整合课程，主导建设并主讲四门课，同时主编两部出版教材，还连续多年担任大学生科研活动导师和班导师。解放日报·上观新闻记者观察到她的“课程表”，多年累积至今的选课听课人数超过了7000人。

 

嘉宾与现场观众互动交流。李立基 摄

 

心有大道 行有大爱 学有大志

 

“我希望自己孩子遇到怎样的好老师，我就应该成为那样的老师。”易静坦言，自己既是老师，也是家长，每每看到学生积极参与学习并发出”这才是大学的学习”的赞叹，她和团队成员感到所有投入的精力都是值得的。

 

这些上海“最美教师”的代表，让中国教育伦理学会会长、上海师德研究与评价中心主任、上海师范大学教授王正平，想起了苏霍姆林斯基的教育名言：“为了在学生眼前点燃一个知识的火花，教师本身就要吸取一个光的海洋，一刻也不能脱离那永远发光的知识和人类智慧的太阳。”根据王正平倡议的《新师德宣言——我们的师德信条》，他表示：新时代的新师德，应更加重视弘扬教师仁爱与宽容精神，促进学生健全发展；同时更加依靠教师道德主体积极性和进取心，促进教师职业精神的养成。

 

中国教育伦理学会会长、上海师德研究与评价中心主任、上海师范大学教授王正平。李立基 摄

 

“心有大道，行有大爱，学有大志。”上海市教卫党委副书记、市教委副主任李昕表示，一个人遇到好老师是人生的幸运，一所学校拥有好老师是学校的光荣，一座城市源源不断涌现出一批又一批好老师则是城市的希望。正如这届新解放教育讲坛的主题“致敬最美教师，弘扬师德正能量”，上海市教育发展基金会和上海师范大学联合组织开展了上海市“最美教师”典型人物典型事迹宣传活动，依托解放日报社等主流媒体，分类别、分阶段推出了一批“最美教师”典型报道。

 

上海市教育发展基金会理事长王荣华。 李立基 摄
