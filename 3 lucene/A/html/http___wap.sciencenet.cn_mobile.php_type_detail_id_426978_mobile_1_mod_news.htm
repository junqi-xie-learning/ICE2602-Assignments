首届科技人文国际学术研讨会将在上海交通大学举行
2019-05-31 16:55
来源：科学网 www.sciencenet.cn

 

科技与人文是理性与感性的化身，更是命运共同体。在“机器人类化、人类机器化”时代来临之际，“智能人”和“自然人”将如何“和平与和谐”相处？科学家和人文社会学者将携手承担怎样的历史使命和责任担当？科技人文未来的发展之路在哪里？

5月31日，上海交通大学宣布：今年10月10日至13日，以“对话与融通：首届科技人文国际学术研讨会”将在上海交大召开，会议由上海交通大学协同联合国教科文组织哲学与人文国际理事会（UNESCO）、亚洲新人文网联（Asia New Humanities Network）、中国外国文学学会比较文学与跨文化研究会、浙江越秀外国语学院共同举办。

据悉，来自国内外院士、人工智能专家、国内外重要刊物主编、世界大学评估专家、著名人文学者将参加此次跨学科研究学术盛会。

会议将围绕科技人文，数字人文，医学人文，科技、文学和比较文学，科技与文化多样性和人文质量科学标准与世界大学评价等主题进行深入探讨。届时将推出包括国际期刊、科技人文丛书和相关学术组织在内的多项举措，为中国构建科技人文学术话语体系贡献交大智慧。

会议期间，上海交通大学还将举办第一届科技人文“融通杯”英语辩论赛，并在研讨会中展示。此次辩论赛主题为“人工智能与人类社会的关系”，拟邀请九校联盟（C9 League）成员参加，宏辞论道，思辩青春。

上海交通大学是一所理工科见长、文科发展迅速的研究型、综合型和国际化的“双一流”建设大学。改革开放以来，学校以“敢为天下先”的精神，锐意推进改革，在科技人文研究方面取得令人瞩目的成就。2004年，学校就展开了具有前瞻性的文理交叉研究，逐步呈现出以学者自由探索为基础、以项目培育为牵引的学科交叉，利用专项布局培育面向现实问题的跨学科团队，形成了举校兴建的学科交叉创新科研平台和科学合理的体制机制。自2010年文理交叉基金项目启动后，至今已持续资助项目200项左右，资助金额超过3500万元，在87项已结项的文理交叉项目中，有85个项目先后分别承担了国家社科基金、教育部人文社科和上海市哲社规划等纵向项目，其中包括21项国家社科重大项目和3项教育部重大攻关项目立项。

近年来，上海交大发挥自身的学科优势，成立了多个学术平台。如，人工智能研究院、心理与行为科学研究院、语言智能中心、健康长三角研究院和城市治理研究院等。立足于理工科的学科优势和多学科背景，上海交通大学主动瞄准世界学术前沿和国家战略需要，让科技人文再多学科交叉融合中逐步释放活力、激发动力、提升能力，加快建成世界一流大学、创建世界一流学科的步伐。

专家表示，由上海交通大学举办的第一届科技人文国际学术研讨会是学校继续探索文理交叉跨学科研究的新举措。

上海交通大学负责人表示，本次科技人文大会是国内首次以科技人文为主题的国际学术研讨会。其价值三。其一,有利于科技人文工作者抓住第四次科技浪潮的机遇，构建以中国学者为主导的科技人文话语体系；其二，有利于汇聚世界顶级学者的智慧，打破科技与人文的界限，改变传统科技与人文相互割裂的局面，促进自然科学与人文社会科学的交叉融合，生产和传播新知识，提出新思想和新方法；其三，有利于促进人文学科建设，新型人文评价体系的创立将“反哺”人文学科的建设，为学科建设提供新思路，新视野。
